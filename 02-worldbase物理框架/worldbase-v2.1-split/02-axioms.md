# 第二部分：状态空间与十公理体系

## 2.1 状态空间的最小选择

设状态空间为：

$$\mathcal{X} = \{0,1\}^N$$

这是一个 $N$ 维二进制超立方体，共有 $2^N$ 个状态。对任意状态 $x = (x_1, \dots, x_N)$，定义：

**汉明距离**：
$$d_H(x, y) = |\{i : x_i \neq y_i\}|$$

**汉明重量**：
$$w(x) = \sum_{i=1}^N x_i$$

其中 $\mathbf{0} = (0, \dots, 0)$ 是唯一的全零态。

**本体论解释**：每一个比特不表示物理量的数值，仅表示在这一维度上是否存在差异。因此 $\mathbf{0}$
是唯一的无差异参照态，其他所有状态都是相对于 $\mathbf{0}$ 的差异模式。

$\mathcal{X} = \{0,1\}^N$
的最小性体现在两个方面。第一，任何更小的状态空间（如单比特或固定重量层）都无法同时承载后续十条公理所要求的层级结构、横向自由度与循环闭合性。第二，$N$
的具体取值不是先验固定的，而是由各相互作用的推导过程中公理约束联合确定（见第五部分 $k=3$ 锁定、第六部分 WLEM 中 $N$
的物理固定方式）。

## 2.2 十条公理（统一表述）

| 公理           | 约束内容               |
|--------------|--------------------|
| **A1 原初差异**  | 存在唯一无差异态，差异沿层级单调累积 |
| **A1′ 横向涌现** | 层级正交方向存在连续对称涌现     |
| **A2 二元具象**  | 差异的基本单元是二值的        |
| **A3 有限离散**  | 状态空间有限             |
| **A4 最小变易**  | 每次演化仅改变一个维度        |
| **A5 差异守恒**  | 差异总量守恒             |
| **A6 不可逆性**  | 演化图为有向无环图（DAG）     |
| **A7 循环闭合**  | 稳定态必须参与有向循环        |
| **A8 对称偏好**  | 系统偏好对称差异分布         |
| **A9 内生完备**  | 不引入公理之外的自由度        |

各公理在数学结构中的精确含义如下。

**A1**：存在唯一元素 $\mathbf{0} \in \mathcal{X}$ 使得对所有 $x \in \mathcal{X}$，$d_H(\mathbf{0}, x) = w(x)$
，且差异沿汉明重量方向单调累积，即演化路径上 $w$ 不减。

**A1′**：在与 A1 层级方向正交的子空间中，存在连续旋转对称性，其最小实现为二维（见 §3.3 的维度推导）。

**A2**：$\mathcal{X} = \{0,1\}^N$，每个坐标 $x_i \in \{0,1\}$，差异单元不可再细分。

**A3**：$N < \infty$，$|\mathcal{X}| = 2^N < \infty$。连续极限 $N \to \infty$ 是方法论工具，不改变基本本体论。

**A4**：每次允许的演化步骤改变恰好一个坐标，即相邻状态之间 $d_H = 1$。在演化图意义下，每条有向边连接汉明距离恰好为 1 的两个状态。

**A5**：存在守恒量 $Q(x)$，在所有允许演化下保持不变。在引力推导中 $Q$ 对应差异流总通量，在电磁推导中对应电荷，在弱力推导中对应约束度变化的平衡条件。

**A6**：演化图 $G = (\mathcal{X}, E)$ 是 DAG，即不存在有向环路。这保证了演化的时间偏序，是弱力手征结构的代数来源（见第六部分）。

**A7**：稳定态集合 $S \subset \mathcal{X}$ 中的每个元素都参与至少一个有向闭合循环。A6 与 A7 的表面矛盾由尺度分离调解：局部演化图是
DAG（A6），全局稳定结构参与闭合循环（A7），两者在不同尺度上同时成立。

**A8**：系统对汉明重量 $w(x) = N/2$ 的状态赋予最高偏好权重，偏好权重随 $|w(x) - N/2|$
单调递减。精确形式为 $\rho(w) = \binom{N}{w}/\binom{N}{N/2}$（见第六部分 §6.13）。

**A9**：系统的自由度集合恰好等于公理 A1–A8 所明示要求的自由度集合，不多不少。这是一个封口公理，其作用是将所有其他公理所确定的结构锁定为最小充分实现。

## 2.3 公理张力与必然调解

十条公理并非彼此独立地施加约束，它们形成一个**最小张力系统**，其中若干对公理之间存在表面矛盾，而这些矛盾的必然调解本身就产生新的物理结构。

**A1 × A5**：局部差异单调增加（A1）× 全局差异守恒（A5）。若差异只增不减，守恒无从实现；唯一的调解方式是存在循环结构，使差异在某些方向增加的同时在另一些方向减少。这一调解必然要求
A7 的循环闭合。

**A6 × A7**：局部不可逆（A6，DAG）× 全局闭合（A7，稳定态参与循环）。两者的调解产生"局部 DAG + 全局环路"
的结构，这正是量子相干演化的离散原型：局部演化不可逆，但稳定模式通过全局闭合维持自身。

**A1′ × A9**：至少二维横向自由度（A1′）× 禁止额外自由度（A9）。A1′ 要求横向维度至少为 2 以承载连续旋转对称性，A9
要求不超过公理明示要求的最小值。两者联合将横向维度精确锁定为 2（见 §3.3）。

**A4 × A6**：单步最小变易（A4）× 演化不可逆（A6）。单步有向跃迁算符在 DAG 约束下必然是非厄米的，这是弱力 $\mathfrak{su}(2)$
手征结构的代数起点（见第六部分）。

## 2.4 有效维度的结论

由 A1 与 A1′ 的正交性，有效空间维度由两个方向的贡献叠加：

$$D_{\text{eff}} = D_{A1} + D_{A1'} = 1 + 2 = 3$$

A1 提供一个层级深度方向（$D_{A1} = 1$），A1′ 提供两个横向方向（$D_{A1'} = 2$，由 A9 精确锁定，详见 §3.3）。

> **命题（强命题）**：在十公理体系下，有效空间维度被强约束为三维。

严格的维度定理仍需补充两个形式化步骤：A1 的"层级深度"与几何维数的严格等价，以及 A9
作为维度上界的精确自由度计数语言。这两个缺口的存在是将本命题标注为强命题而非定理的原因。该结论构成后续全部推导的稳定前提。
