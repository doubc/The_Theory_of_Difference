## 第一部分：元素周期表的公理重构

主流化学用量子力学（薛定谔方程）解释周期表。WorldBase 认为量子力学本身也是公理涌现的。因此，周期表是**公理的直接投影**。

### 1. 空间维度与轨道形状（为什么是 s, p, d, f？）
*   **主流解释**：3 维空间中的球谐函数解。
*   **WorldBase 公理推导**：
    *   **公理来源**：**A1 (层级) + A1' (涌现对称) + A9 (完备性)**。
    *   **推导链**：
        1.  V1.2 已严格证明：这三条公理锁定有效空间维度 **$D_{eff} = 3$**。
        2.  3 维空间中的旋转对称群是 **$SO(3)$**。
        3.  A1' 要求涌现空间具有连续旋转对称性。
        4.  $SO(3)$ 群的不可约表示由角动量量子数 $\ell$ 标记，对应轨道类型：
            *   $\ell=0 \rightarrow s$ 轨道 (球形对称)
            *   $\ell=1 \rightarrow p$ 轨道 (哑铃形)
            *   $\ell=2 \rightarrow d$ 轨道 (花瓣形)
            *   $\ell=3 \rightarrow f$ 轨道 (复杂形)
    *   **结论**：周期表的区块结构（s 区、p 区、d 区、f 区）是 **3 维空间几何性质的必然结果**。如果公理导致 $D=4$，周期表结构将完全改变（出现 g 轨道等）。

### 2. 量子化与泡利不相容原理（为什么每轨道 2 个电子？）
*   **主流解释**：波函数反对称性，自旋 1/2。
*   **WorldBase 公理推导**：
    *   **公理来源**：**A2 (二元具象) + A3 (有限离散)**。
    *   **推导链**：

        2.  **泡利原理 (A2)**：状态空间是二值的 ${0,1}^N$。每个分量 $x_i$ 只能是 0 或 1。
            *   在物理投影中，一个“轨道态”对应状态空间的一个分量。
            *   因为 $x_i \in \{0,1\}$，所以一个轨道态只能被“激活”一次（占据数为 0 或 1）。
        3.  **自旋 (A2+A1')**：A2 的二值性结合 A1' 的横向对称性，自然给出两种自旋态（向上/向下，或相位 0/$\pi$）。
        4.  **容量**：每个空间轨道 ($\ell, m_l$) 有 2 个自旋态 $\rightarrow$ 容量为 2。
    *   **结论**：壳层容量公式 $2(2\ell+1)$ 来自 **$SO(3)$ 对称性 (A1')** 乘以 **二元自旋 (A2)**。

### 3. 填充顺序与构造原理（为什么按 1s, 2s, 2p... 填充？）

### 4. 周期性与稀有气体（为什么每周期结尾稳定？）

## 第二部分：化学键本质的公理重构

主流化学用轨道重叠和能量降低解释化学键。WorldBase 将其重构为**差异流的平衡**。

### 1. 化学键的统一定义
*   **定义**：化学键是两个或多个原子（A4 路径端点）之间，为了满足 **A8 (对称偏好)** 而进行的 **A5 (差异守恒)** 交换通道。
*   **物理本质**：势场梯度的耦合。

### 2. 离子键 (Ionic Bond)
*   **公理机制**：**A8 (对称偏好) + A5 (差异转移)**。
*   **过程**：
    1.  原子 A (如 Na, $\delta=-1$) 偏离对称态，有多余差异。
    2.  原子 B (如 Cl, $\delta=+1$) 偏离对称态，缺少差异。
    3.  通过 A4 路径，A 将差异完全转移给 B。
    4.  结果：两者都达到稀有气体构型 (对称态，$\delta=0$)。
*   **特征**：差异完全转移，无方向性 (球对称势场)，对应 A1 层级主导。

### 3. 共价键 (Covalent Bond)
*   **公理机制**：**A4 (最小变易) + A5 (差异共享)**。
*   **过程**：
    1.  两个原子 (如 H-H) 都无法通过单方面转移达到对称态。
    2.  根据 A4，它们建立局部连接（共享分量）。
    3.  差异在两个原子之间流动（离域），模拟出一种“平均对称态”。
*   **特征**：差异共享，有方向性 (轨道重叠对应 A1' 横向对称性的特定模式)，对应 A1' 主导。
*   **V1.2 对应**：对应电磁力部分提到的“相位差异守恒”。共价键是相位差异的相干叠加。

### 4. 金属键 (Metallic Bond)
*   **公理机制**：**A8 (对称偏好) + A3 (有限离散的大数极限)**。
*   **过程**：
    1.  大量原子聚集。
    2.  差异在整个集合中完全离域，形成“差异海”。
    3.  系统在全局尺度上达到统计对称性。
*   **特征**：高导电性 (差异流动阻力小)，延展性。

### 5. 键能与键长
*   **公理来源**：**A4 (最小变易距离) + A5 (势场梯度)**。
*   **解释**：
    *   **键长**：对应两个原子状态在状态空间中的**最佳汉明距离**。太近 (距离<1) 违反 A4 (不能重叠)，太远势场耦合弱。
    *   **键能**：对应成键前后**势场方差的变化量** ($\Delta H_\phi$)。成键使系统总势场更均匀 (方差减小)，释放能量。
