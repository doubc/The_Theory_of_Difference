## 第一章：生成序列的一般定义

### 一、形式语言的借用说明

本章使用的形式语言（划分、约束、转换、不动点）借用了操作系统和软件工程中已被工程实践检验过的数学结构。这些结构在 $\{0,1\}^N$ 上有精确的定义。

但本章不声称操作系统是WorldBase公理体系的实例。操作系统的转换规则来自工程决策，不是从A1-A9推导的。我们借用的是形式语言，不是具体机制。

---

### 二、生成序列

设状态空间为 $\{0,1\}^N$（A2+A3），两个差异态 $x, y \in \{0,1\}^N$ 之间形成界面时，发生以下生成序列：

$$\text{公理约束} \xrightarrow{\text{驱动}} \text{最优划分} \xrightarrow{\text{决定}} \text{转换规则}$$

**不是**预先给定划分再施加约束，而是约束驱动出最优划分，划分决定转换规则。顺序不可颠倒。

### 2.1 约束层（公理输入）

界面处的转换受以下公理约束：

- **A5守恒**：界面处总差异守恒，无损耗
- **A4最小变易**：转换以最小步数完成
- **A8对称偏好**：同等代价时选择最对称的分配

这些约束不预设任何划分。它们是对"两个差异态如何相互作用"的全局约束。

### 2.2 划分层（约束的输出）

每个差异态的偏移被分解为内部和界面两部分：

$$\delta_i = \delta_i^{(int)} + \delta_i^{(face)}$$

这个分解不是预先给定的。它是以下优化问题的解：

$$\min_{t, \, s \geq 0} \; |\delta_1 + t - s| + |\delta_2 - t - s|$$

其中：
- $t$：使元素1偏移值变化的量（$t > 0$ 表示元素1的内部偏移沿趋零方向增加）
- $s$：双方各贡献到共同界面的共享量（$s \geq 0$）

A4驱动系统选择使内部偏移绝对值之和最小的 $(t,s)$。A8在多个最优解中选择 $|t|$ 最小的（最对称的界面）。

**划分是约束的输出，不是输入。**

### 2.3 转换层（划分的决定）

划分 $(t^*, s^*)$ 确定后，界面处的转换规则被自动确定：

| 划分结果 | 转换类型 |
|---------|---------|
| $s^* = 0, t^* \neq 0$ | 单向转移 |
| $t^* = 0, s^* > 0$ | 双向共享 |
| $t^* \neq 0, s^* > 0$ | 混合 |
| 无解 | 无转换 |

**转换规则是划分的输出，不是独立规定的。**

## 第二章：各层实例

### 三、化学层实例

#### 3.1 化学偏移

化学层的差异态以族偏移标记：碱金属 $\delta = -1$，卤素 $\delta = +1$，稀有气体 $\delta = 0$，以此类推。

偏移的公理来源：A8的对称态（$\delta = 0$），其他元素偏离对称态，偏移量标记偏离程度。

#### 3.2 生成序列的运行

**实例1：NaCl**

约束输入：$\delta_1 = -1$（Na），$\delta_2 = +1$（Cl）

优化问题：$\min_{t,s \geq 0} |-1+t-s| + |+1-t-s|$

| $(t,s)$ | $\|\delta_1^{(int)}\| + \|\delta_2^{(int)}\|$ |
|---------|---------------------------------------------|
| $(1, 0)$ | $\|0\| + \|0\| = 0$ |
| $(0, 1)$ | $\|-2\| + \|0\| = 2$ |
| $(0.5, 0.5)$ | $\|-1\| + \|0\| = 1$ |

最优解：$t^* = 1, s^* = 0$。划分：$\delta_{Na}^{(int)} = 0, \delta_{Cl}^{(int)} = 0$。转换：单向转移。

**实例2：H₂**

约束输入：$\delta_1 = +1, \delta_2 = +1$

优化问题：$\min_{t,s \geq 0} |+1+t-s| + |+1-t-s|$

| $(t,s)$ | $\|\delta_1^{(int)}\| + \|\delta_2^{(int)}\|$ |
|---------|---------------------------------------------|
| $(0, 1)$ | $\|0\| + \|0\| = 0$ |
| $(1, 0)$ | $\|2\| + \|0\| = 2$ |
| $(0, 0)$ | $\|1\| + \|1\| = 2$ |

最优解：$t^* = 0, s^* = 1$。划分：$\delta_H^{(int)} = 0, \delta_H^{(int)} = 0$。转换：双向共享。

**实例3：He + 任意元素**

约束输入：$\delta_1 = 0, \delta_2 = \text{任意} \neq 0$

结论：不存在 $(t, s \geq 0)$ 使总偏移严格减小，即 He 不与任何元素形成界面。

**严格程度**：结论正确，证明待补全。

当前尝试的证明（三角不等式直接应用）是错误的——三角不等式给出 $|s-t| + |\delta_2 - t - s| \geq |\delta_2 - 2t|$，而非 $|\delta_2|$。当 $t = 0$ 且 $0 < s < \delta_2$ 时，$|\delta_2 - s| < |\delta_2|$，不等式不成立。

正确的证明路径需要分区讨论 $t$ 和 $s$ 的取值范围，证明对所有 $t \geq 0, s \geq 0$：

$$|s - t| + |\delta_2 - t - s| \geq |\delta_2|$$

且等号当且仅当 $t = s = 0$ 时成立。这是V0.2的核心引理之一（引理L1），待补全。

#### 3.3 键型分类的涌现

键型不是人为标签，而是生成序列在不同输入下的自然输出：

```
输入偏移 (δ₁, δ₂)
    │
    │ A4+A8驱动优化
    │
    ▼
最优划分 (t*, s*)
    │
    │ s*=0 → 离子键
    │ t*=0 → 共价键
    │ 混合 → 极性共价键
    │ 无解 → 不成键
    │
    ▼
转换规则
```

同一个生成序列，不同输入，不同输出。分类是涌现的，不是预设的。

### 四、多界面耦合

#### 4.1 问题

当一个元素同时与多个配体形成界面时（如H₂O中O与两个H），每个界面独立满足生成序列，但多个界面在中心元素（对称偏好）——稀有气体是化学层处发生耦合|\delta_l|$ 在各步中不变。

#### 4.4 分子式唯一性

给定 $\delta_c$ 和 $\delta_l$，配体数目 $n$ 由以下条件确定：

$$n^* = \arg\min_n |\delta_c^{(n)}| \quad \text{subject to:} \quad |\delta_c^{(i)}| < |\delta_c^{(i-1)}| \; \forall \, i = 1, \ldots, n$$

即：找到使中心元素残余偏移最小的配体数目，且每增加一个配体都必须使残余偏移严格减小（否则没有动力继续添加配体）。

| 分子 | $\delta_c$ | $\delta_l$ | $n^*$ | $\delta_c^{(n^*)}$ |
|------|-----------|-----------|-------|-------------------|
| H₂O | $+2$ | $+1$ | 2 | 0 |
| NH₃ | $+3$ | $+1$ | 3 | 0 |
| CH₄ | $+4$ | $+1$ | 4 | 0 |
| CO₂ | $+4$ | $+2$ | 2 | 0 |
| CaCl₂ | $-2$ | $+1$ | 2 | 0 |
| AlCl₃ | $+5$ | $+1$ | 3 | $+2$（残余） |

分子式是生成序列在多界面耦合下的唯一输出。

#### 4.5 残余偏移

AlCl₃中 $\delta_{Al}^{(3)} = +2 \neq 0$。这个残余偏移不是模型错误，而是生成序列的自然输出——它对应AlCl₃的Lewis酸性（可进一步形成AlCl₄⁻）。

定义残余偏移：

$$R = |\delta_c^{(n^*)}|$$

$R > 0$ 的分子具有进一步界面化的潜力——它是"未耗尽的界面容量"。

**公理来源**：A4最小变易不要求每个元素内部偏移归零，只要求总代价最小。残余偏移是总代价最小解中允许的非零内部偏移。

### 五、极性（待扩展）

#### 5.1 问题的重新提法

在当前生成序列中，H₂和HF都给出 $t^* = 0, s^* = 1$（纯共价键）。模型无法区分非极性和极性。

用生成序列的语言重新提问题：

> **给定两个元素的偏移 $(\delta_1, \delta_2)$ 和壳层参数 $(n_1, l_1, n_2, l_2)$，A4+A8驱动的最优划分是否必然是不对称的？在什么条件下？**

#### 5.2 极性的涌现条件

当两个元素的壳层参数相同（$n_1 = n_2, l_1 = l_2$）时，A8的对称偏好要求等分——$t^* = 0$，无极性。

当壳层参数不同时，A8的对称偏好被壳层深度的不对称打破。更深的壳层（更大的 $n$）意味着更多的累积约束（A1），对界面处差异的拉力更强。

优化问题扩展为：

$$\min_{t, s \geq 0} |\delta_1 + t - s| + |\delta_2 - t - s| + \lambda \cdot \Phi(n_1, l_1, n_2, l_2, t, s)$$

其中 $\Phi$ 是壳层不对称引入的额外代价项。当 $n_1 = n_2$ 时 $\Phi = 0$，恢复原始问题。当 $n_1 \neq n_2$ 时 $\Phi \neq 0$，$t^*$ 偏离零，极性涌现。

#### 5.3 待完成

$\Phi$ 的具体形式需要推导。候选来源：A1（层级累积）在界面语境下的表现——更深壳层的元素每接收一份界面差异，约束度增加更少（因为已有更多累积约束作为缓冲），因此更倾向于接收界面差异。

形式化：

$$\Phi(n_1, l_1, n_2, l_2, t, s) = \left|\frac{s}{n_1} - \frac{s}{n_2}\right| \cdot g(t)$$

当 $n_1 = n_2$ 时 $\Phi = 0$。当 $n_1 > n_2$ 时，元素1（更深壳层）接收更多共享差异的代价更低，$t^*$ 偏向使元素1接收更多差异的方向。

**严格程度**：$\Phi$ 的函数形式是候选，尚未被严格推导。这是V0.2的核心工作。

### 六、VSEPR（待扩展）

#### 6.1 问题

当中心元素与 $n$ 个等价配体形成界面时，这些界面的空间分布取什么几何构型？

#### 6.2 预期推导路径

A8（对称偏好）要求多个界面在同一中心点的空间分布取对称性最高的构型。在三维空间中（维度定理 $D_\text{eff} = 3$），不同 $n$ 值的最对称构型是：

| $n$ | 构型 | 对称群 |
|-----|------|--------|
| 2 | 直线型（180°） | $D_{\infty h}$ |
| 3 | 平面三角形（120°） | $D_{3h}$ |
| 4 | 正四面体（109.5°） | $T_d$ |
| 6 | 正八面体（90°） | $O_h$ |

这些恰好是VSEPR理论预测的几何构型。

**严格程度**：构型列表是已知的，A8→构型的推导路径是候选，待严格化。需要证明"在给定约束下，对称性最高的空间分布"确实给出上述构型。这需要三维空间中点群对称性的严格处理。

### 七、与核结合能的关系

#### 7.1 SEMF作为核子层实例

SEMF的五项对应核子层界面形成过程中**五种不同的约束来源**，每种约束对应生成序列约束层的一个分量：

| SEMF项 | 生成序列约束层的分量 |
|--------|---------------------|
| 体积能 | A1（层级累积） |
| 表面能 | A1（边界修正） |
| 库仑能 | A2（二元具象，有方向的差异量） |
| 对称能 | A8（对称偏好） |
| 壳层修正 | A1'（涌现结构） |

SEMF是核子层生成序列的能量输出。五项对应约束层的五个分量，不是生成序列三个步骤（约束→划分→转换）的直接对应。

化学层的界面形成有三个主要约束（A4+A5+A8），核子层有五个（A1+A1'+A2+A5+A8）。约束层的分量数目随层级变化，但生成序列的三步结构不变。

### 八、与象界的关系

生成序列是象界第二章（边界→界面）在 $\{0,1\}^N$ 上的精确数学表达。象界描述的是差异如何形成界面，生成序列给出的是这个过程在离散状态空间中的具体运算规则。

后续象界机制（自维持、记忆、复制、筛选、功能）的数学表达将遵循同样的模式：先建立生成序列，再找化学层和核子层的实例。
