## 踩过的坑
1. git clone 走不通 → 改用 GitHub API（`/contents/` 端点）
2. URL 中文字符编码报错 `ordinal not in range(128)` → `urllib.parse.quote` 编码路径
3. Sina API 无 Referer 返回 403 → 加 `Referer: https://finance.sina.com.cn` header
4. Sina 批量请求被限流 → 分批（每批≤8）+ sleep 0.3s
5. gen 函数返回值数量不一致 → 统一为 6 元组 (product, chains, entities, polarity, pricing_models, relations)
