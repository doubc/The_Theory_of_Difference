# Glossary (术语表)

| English | 中文 | Definition |
|---------|------|------------|
| Axiom | 公理 | Formal constraint on the discrete state space |
| Constraint Derivation | 约束推导 | Deriving physics by eliminating incompatible structures |
| WorldBase | WorldBase | The physics framework derived from the ten axioms |
| Hypercube | 超立方体 | The discrete state space {0,1}^N |
| Hamming distance | 汉明距离 | Number of differing bits between two states |
| LNCIM | LNCIM | Local Nonlinear Conservative Irreversible Multiscale class |
| Enstrophy | 涡拟能 | Integral of squared vorticity: Ω = ∫ω²d²x |
| Enstrophy cascade | 涡拟能级串 | Forward transfer of enstrophy to small scales |
| Inverse energy cascade | 能量反级串 | Backward transfer of energy to large scales |
| Inertial range | 惯性区间 | Scale range between injection and dissipation |
| DAG | DAG | Directed Acyclic Graph (A6 structure) |
| VEV | VEV | Vacuum Expectation Value |
| CFT | CFT | Conformal Field Theory |
| Noether charge | Noether 荷 | Conserved quantity from continuous symmetry |
