## §1 问题定位

论文二（定理 B）已证明：对所有 $n \geq 2$，$J(2n,n)$ 的每条边至少包含在一个三角中。由此可得 $\text{im}(\partial_2)$
覆盖所有边，$H_1 = 0$。

但该证明依赖一个隐含步骤：从"每条边在三角中"到"$\text{rank}(\partial_2) = f_1 - f_0 + 1$"的推断在论文中标注为 Conjecture
1（支持 $n=2,3,4$ 的数值验证，但未给出一般证明）。

TASK-01 的目标：通过离散 Morse 函数给出一个独立的代数证明，绕过 Conjecture 1。

## §2 定理陈述

**定理 CLEM-MORSE**（$b_1 = 0$ 的代数证明）

设 $\mathcal{K}_2 = \mathrm{Cl}(J(4,2))$ 为 $J(4,2)$ 的团复形（八面体边界），$f$-向量为 $(6, 12, 8)$。

定义离散 Morse 函数 $f: \mathcal{K}_2 \to \mathbb{R}$：

- 顶点：$f(v_i) = i$，$i = 1, \ldots, 6$（字典序，来自 A6 DAG 拓扑排序）
- 边：$f(e_{ij}) = j + 0.5$（$i < j$）
- 三角：$f(t_{ijk}) = k + 1.5$（$i < j < k$）

配对 $\mathcal{M}$（共 12 对）使得临界单纯形恰为：

$$\text{0-临界点：} v_1, \qquad \text{2-临界点：} t_8 = \{v_4, v_5, v_6\}$$

配对导出的有向图 $\Gamma$ 是以 $v_1$ 为汇点的 DAG，无环。

由弱 Morse 不等式 $b_k \leq c_k$ 得 $b_1 \leq c_1 = 0$，故 $b_1 = 0$。

结合 $\chi(\mathcal{K}_2) = 2$、$b_0 \geq 1$、$b_0 \leq c_0 = 1$，得

$$\boxed{b_0 = 1, \quad b_1 = 0, \quad b_2 = 1}$$

从而 $H_*(\mathcal{K}_2; \mathbb{Z}) \cong H_*(S^2; \mathbb{Z})$，$\mathcal{K}_2 \cong S^2$。

该证明不依赖任何数值矩阵秩计算，完全基于离散组合配对和 Euler 示性数约束。$\square$

详细配对构造、无环验证和临界点计算见 `calculations/morse_function.md`。

## §3 一般 $n$ 的 $H_1 = 0$

### §3.1 定理 B 的陈述

**定理 B**（论文二，§4）：对所有 $n \geq 2$，$J(2n,n)$
的每条边都包含在至少一个三角形中。从而 $H_1(\mathcal{K}_n; \mathbb{Z}) = 0$。

**证明要点**：给定边 $\{A, B\}$，令 $S = A \cap B$（$|S| = n-1$），取任意 $c \in [2n] \setminus (S \cup \{a,b\})$
，令 $C = S \cup \{c\}$，则 $\{A, B, C\}$ 构成三角形。这样的 $c$
存在当且仅当 $|[2n] \setminus (S \cup \{a,b\})| = n-1 \geq 1$，即 $n \geq 2$。$\square$

### §3.2 与 Conjecture 1 的关系

论文二 Conjecture 1 主张：对所有 $n \geq 2$，$\operatorname{rank}(\partial_2) = f_1 - f_0 + 1$，从而 $b_1 = 0$。

定理 B 证明了"每条边在三角中"，这意味着 $\operatorname{im}(\partial_2)$ 覆盖所有边方向，但严格的秩等式仍需要证明边界矩阵没有意外的线性相关。

CLEM-TASK-01 的 Morse 方法对 $n=2$ 给出了比 Conjecture 1 更强的结果——不仅 $b_1 = 0$，而且给出了明确的配对结构。对一般 $n$
，Morse 方法的推广方向是：在 $J(2n,n)$ 上构造类似的高度函数（利用顶点的字典序），证明配对后无 1-临界点。这是 CLEM-OPEN-05
的候选任务。

## §4 公理连接

| 公理             | 角色        | 在证明中的体现                                                                       |
|:---------------|:----------|:------------------------------------------------------------------------------|
| **A1'**（横向涌现）  | 边的选择规则    | 精确距离 $d_H = 2$ 定义了 $J(4,2)$ 的边集，决定了复形的 1-骨架                                   |
| **A4**（最小变易）   | 每步只改变一个比特 | 保证 Morse 配对的局部性——每条边只连接 Hamming 距离 2 的顶点，配对不会跨越多个比特翻转                         |
| **A6**（DAG 有向） | 高度函数来源    | DAG 的拓扑排序给出顶点的严格全序 $v_1 < v_2 < \cdots < v_6$，这就是 Morse 函数 $f(v_i) = i$ 的公理基础 |
| **A7**（循环闭合）   | 三角面的存在性   | A7 要求演化中存在闭合路径，这在单纯复形语言中就是三角面的填充，保证了 $\partial^2 = 0$ 和链复形的合法性                |
| **A8**（对称偏好）   | 中截面选择     | 权重 $n$ 层（中截面）统计权重最大，保证 $J(2n,n)$ 的中截面是主导结构，$n=2$ 时给出最干净的 $S^2$                |
