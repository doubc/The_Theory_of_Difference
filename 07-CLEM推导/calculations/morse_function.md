## §1 顶点标记与三部结构

$J(4,2)$ 的顶点集是 $[4] = \{1,2,3,4\}$ 的所有 2-元子集，共 6 个。按字典序编号：

|  编号   |    子集     |    特征向量     |
|:-----:|:---------:|:-----------:|
| $v_1$ | $\{1,2\}$ | $(1,1,0,0)$ |
| $v_2$ | $\{1,3\}$ | $(1,0,1,0)$ |
| $v_3$ | $\{1,4\}$ | $(1,0,0,1)$ |
| $v_4$ | $\{2,3\}$ | $(0,1,1,0)$ |
| $v_5$ | $\{2,4\}$ | $(0,1,0,1)$ |
| $v_6$ | $\{3,4\}$ | $(0,0,1,1)$ |

**三部结构**（互补对，Hamming 距离 4，不相邻）：

$$P_1 = \{v_1, v_6\}, \quad P_2 = \{v_2, v_5\}, \quad P_3 = \{v_3, v_4\}$$

$K_{2,2,2}$ 的边规则：同一部内不相邻，不同部之间全部相邻。12
条边恰好是三部之间的所有跨部连接。三角形恰好是从三个部各取一个顶点的所有组合，共 $2^3 = 8$ 个。

## §2 离散 Morse 函数回顾

对单纯复形 $K$，Forman 离散 Morse 函数 $f: K \to \mathbb{R}$ 的配对需满足：每个单纯形至多出现在一个配对中，且导出的有向图无环。未配对的单纯形即为临界点。

## §4 Morse 配对的构造

### §4.1 配对策略

目标：保留 $v_1$ 不配对（作为唯一的 0-临界点），其余 5 个顶点各与一条边配对，7 条未配对边各与一个三角配对，剩余 1 个三角为
2-临界点。

### §4.2 顶点-边配对

按边的 $f$ 值从小到大处理，每次将边与未配对的较小端点配对：

| $f$ 值 |            边             | 动作                           |       配对        |
|:-----:|:------------------------:|:-----------------------------|:---------------:|
| $2.5$ | $e_{12} = \{v_1, v_2\}$  | $v_1$ 保留，$v_2$ 未配对 → 配 $v_2$ | $(v_2, e_{12})$ |
| $3.5$ | $e_{13} = \{v_1, v_3\}$  | $v_3$ 未配对 → 配 $v_3$          | $(v_3, e_{13})$ |
| $3.5$ | $e_{23} = \{v_2, v_3\}$  | 两端均已配对 → 搁置                  |        —        |
| $4.5$ | $e_{14} = \{v_1, v_4\}$  | $v_4$ 未配对 → 配 $v_4$          | $(v_4, e_{14})$ |
| $4.5$ | $e_{24} = \{v_2, v_4\}$  | 两端均已配对 → 搁置                  |        —        |
| $5.5$ | $e_{15} = \{v_1, v_5\}$  | $v_5$ 未配对 → 配 $v_5$          | $(v_5, e_{15})$ |
| $5.5$ |     $e_{35}, e_{45}$     | 两端均已配对 → 搁置                  |        —        |
| $6.5$ | $e_{26} = \{v_2, v_6\}$  | $v_6$ 未配对 → 配 $v_6$          | $(v_6, e_{26})$ |
| $6.5$ | $e_{36}, e_{46}, e_{56}$ | $v_6$ 已配对 → 搁置               |        —        |

**结果**：

- 5 个顶点-边配对：$(v_2, e_{12})$，$(v_3, e_{13})$，$(v_4, e_{14})$，$(v_5, e_{15})$，$(v_6, e_{26})$
- 未配对顶点：$v_1$（唯一 0-临界点）

## §6 无环性验证

### §6.1 有向图 $\Gamma$ 的构造

$\Gamma$ 的边来自两类：

**顶点-边配对产生的反向边**（从边指向其另一个端点或传递目标）：

| 配对              | $\Gamma$ 中的边     | 追踪                                                   |
|:----------------|:-----------------|:-----------------------------------------------------|
| $(v_2, e_{12})$ | $e_{12} \to v_1$ | $v_1$ 是临界点，终止                                        |
| $(v_3, e_{13})$ | $e_{13} \to v_1$ | 终止                                                   |
| $(v_4, e_{14})$ | $e_{14} \to v_1$ | 终止                                                   |
| $(v_5, e_{15})$ | $e_{15} \to v_1$ | 终止                                                   |
| $(v_6, e_{26})$ | $e_{26} \to v_2$ | $v_2$ 已配给 $e_{12}$，传递 $e_{26} \to e_{12} \to v_1$，终止 |

**边-三角配对产生的反向边**（从三角指向其其他边）：

| 配对              | $t$ 的其他边                                                                                   | 追踪至临界点    |
|:----------------|:-------------------------------------------------------------------------------------------|:----------|
| $(e_{23}, t_1)$ | $e_{12} \to v_1$，$e_{13} \to v_1$                                                          | 终止于 $v_1$ |
| $(e_{24}, t_2)$ | $e_{12} \to v_1$，$e_{14} \to v_1$                                                          | 终止于 $v_1$ |
| $(e_{35}, t_3)$ | $e_{13} \to v_1$，$e_{15} \to v_1$                                                          | 终止于 $v_1$ |
| $(e_{45}, t_4)$ | $e_{14} \to v_1$，$e_{15} \to v_1$                                                          | 终止于 $v_1$ |
| $(e_{36}, t_5)$ | $e_{23} \to t_1 \to e_{12} \to v_1$，$e_{26} \to e_{12} \to v_1$                            | 终止于 $v_1$ |
| $(e_{46}, t_6)$ | $e_{24} \to t_2 \to e_{12} \to v_1$，$e_{26} \to e_{12} \to v_1$                            | 终止于 $v_1$ |
| $(e_{56}, t_7)$ | $e_{35} \to t_3 \to e_{13} \to v_1$，$e_{36} \to t_5 \to e_{23} \to t_1 \to e_{12} \to v_1$ | 终止于 $v_1$ |

### §6.2 结论

所有路径均终止于 $v_1$（0-临界点）。$\Gamma$ 是以 $v_1$ 为汇点的有向无环图（DAG），无环性得证。$\square$

## §7 临界点与 Betti 数

$$c_0 = 1 \quad (v_1), \qquad c_1 = 0, \qquad c_2 = 1 \quad (t_8)$$

**弱 Morse 不等式**：$b_k \leq c_k$

$$b_0 \leq 1, \quad b_1 \leq 0, \quad b_2 \leq 1$$

由 $b_1 \leq 0$ 和 $b_k \geq 0$ 得 $b_1 = 0$。

**Euler 示性数**：

$$\chi = b_0 - b_1 + b_2 = 6 - 12 + 8 = 2$$

代入 $b_1 = 0$：$b_0 + b_2 = 2$。由 $b_0 \geq 1$（连通）且 $b_0 \leq 1$，得 $b_0 = 1$，$b_2 = 1$。

$$\boxed{b_0 = 1, \quad b_1 = 0, \quad b_2 = 1} \qquad \square$$
