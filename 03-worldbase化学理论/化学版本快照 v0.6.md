# 化学的公理化：指导原则与方法论

**WorldBase化学投影 · v0.6 · 2026年3月23日**

---

## 一、问题

化学是一门成功的科学——几百年来，它帮助人类炼铁、制药、造芯片。但化学缺乏自己的公理基础。化学的所有核心概念——元素、电子、轨道、化学键、电负性——都来自物理学，或者来自经验归纳。

这意味着：化学能用，但不知道为什么能用。

本文档试图建立一种方法论，使得化学的"为什么"可以从公理推导，同时诚实地标注哪些部分必须依赖经验。

---

## 二、核心主张

**化学的结构框架可以从差异公理推导，但化学的具体知识需要经验接口。**

这不是折中，而是化学作为科学的本质特征。物理学追求"从公理推导一切"，化学追求"用公理框架组织经验知识"。两者认识论地位不同，不能用同一个标准衡量。

---

## 三、六条公理

以下六条公理是整个框架的起点。它们不依赖任何物理概念（电子、力、场），只依赖"差异"这一个原始概念。

**A1（层级）：** 差异有累积性。存在从少到多的层级结构。

**A2（二值）：** 差异是二值的。每个独立的差异方向只有两种状态：存在或不存在。

**A3（有限）：** 差异是有限离散的。独立的差异方向的总数是有限的。

**A4（局部性）：** 差异的演化是局部的。每次只能改变一个差异方向的状态。

**A5（守恒）：** 差异总量守恒。已激活的差异方向的总数在演化过程中不变。

**A8（对称偏好）：** 系统偏好对称态。差异分布越均匀的状态越稳定。

这六条公理联合产生一个数学结构：超立方体 $\{0,1\}^N$，其中N是独立差异方向的总数。

---

## 四、从公理到化学：结构性推导

以下是从公理推导出的化学结构。这些推导不依赖经验数据，只依赖公理和数学。

### 4.1 对称态 → 稀有气体

A8说系统偏好对称态。在超立方体中，对称态是差异分布最均匀的状态。

在化学中，对称态对应稀有气体——差异（电子）均匀分布在所有可用的方向（轨道）上，化学性质最不活泼。

**这是推导，不是经验。** 公理说"对称态被偏好"，化学观测到"稀有气体最稳定"。两者对应。

### 4.2 距对称态的距离 → 偏移值δ

A8说系统偏好对称态。那偏离对称态的程度就决定了化学性质。

定义偏移值：

$$\delta = Z_{\text{noble}} - Z$$

其中 $Z_{\text{noble}}$ 是元素所在周期的稀有气体原子序数。

- $\delta < 0$：比稀有气体多电子（金属，倾向失去电子）
- $\delta > 0$：比稀有气体少电子（非金属，倾向获得电子）
- $\delta = 0$：就是稀有气体（化学惰性）

**δ的定义形式从公理推导（距对称态的距离），δ的具体数值依赖经验输入（原子序数Z）。**

### 4.3 差异交换 → 化学键类型

A4（局部性）+ A5（守恒）：元素之间的差异交换必须是局部的，且守恒。

差异交换的方式决定了键的类型：

- **差异完全转移** → 离子键（Na给Cl一个电子，两者都趋向对称态）
- **差异部分共享** → 共价键（C和H共享差异，各减小一部分偏移）
- **差异离域** → 金属键（大量Na原子之间差异自由流动）

**键类型的分类从公理推导，不依赖电子概念。**

### 4.4 差异守恒 → 反应方向

A5（守恒）+ A6（不可逆性）：反应有方向，从高差异态到低差异态。

判断规则：

$$\Delta|\delta| = \sum_{\text{产物}}|\delta_i| - \sum_{\text{反应物}}|\delta_i|$$

- $\Delta|\delta| < 0$：反应自发
- $\Delta|\delta| > 0$：反应不自发

**反应方向的判断规则从公理推导。** 但δ的具体数值依赖经验输入。

## 五、经验值的角色：接口方法论

### 5.1 经验值不是"噪声"

在传统科学观中，经验数据是"原材料"——理论从经验中提炼出来。

在本文档的方法论中，经验数据是**接口**——连接公理框架与现实世界的桥梁。

没有接口，框架是纯粹的逻辑结构——在逻辑世界中自洽，但和现实世界没有连接。有了接口，框架才能"落地"——从逻辑世界映射到物理世界。

### 5.2 四种接口

**锚定接口——确定框架的"零点"**

公理说"存在基态"，但不说"基态对应什么"。原子序数Z=1（氢）是框架在现实中的锚定点。这是经验输入。

**标度接口——确定框架的"单位"**

公理推导出δ的排序关系（同族元素δ相同），但不能推导δ的1个单位对应电负性的多少Pauling。标度是经验输入。

**边界接口——确定框架的"范围"**

公理推导出框架可以描述任意Z的元素。但"已知最重元素是Z=118"是实验事实。覆盖范围是经验输入。

**验证接口——确认框架的"正确性"**

公理推导出"Δ|δ| < 0 → 反应自发"。"Na + Cl → NaCl确实自发"是实验验证。验证是经验确认。

### 5.3 接口不是弱点

任何理论都需要接口来连接逻辑和现实。几何学需要"比例尺"来对应实际地形——比例尺不是从几何推导的，但没有比例尺，地图无法使用。

化学框架需要"原子序数"来对应实际元素——原子序数不是从公理推导的，但没有原子序数，框架无法落地。

**接口的存在恰恰证明了框架的完整性。** 如果框架不需要任何经验输入就能描述所有化学事实，那框架就不是"化学的理论"，而是"化学本身"——这不可能。

### 5.4 化学和物理学的关系

化学和物理学不是"独立"也不是"依赖"，而是**通过接口连接**。

```
公理 → 物理学投影（D=3, 轨道类型, 壳层容量）
     → 化学投影（特征向量, δ, 分类, 反应方向）
     → 接口层（原子序数, 壳层容量, 周期长度）
     → 现实世界（118个元素, 化学反应, 材料）
```

两者有各自的结构（物理学描述单粒子行为，化学描述组合模式），但通过共享的接口（原子序数、壳层容量）保持一致。

**化学的结构框架不使用物理学的概念（电子、轨道、原子核）。但化学的接口连接物理学的测量（原子序数来自质子数，壳层容量来自SO(3)）。**

## 七、诚实的局限

### 7.1 特征向量是设计的，不是推导的

14位特征向量的编码方式（哪些特征、几位、什么编码）是我设计的，不是从公理推导的。公理说"差异存在"和"系统偏好对称态"，但没有说"必须用14位二进制向量编码"。

这是框架中最脆弱的部分。如果特征向量的设计方式不同，分类结果可能不同。

### 7.2 分类规则是拟合的，不是推导的

"d/f区→金属"、"p区用δ'_p判断"——这些规则是我根据化学知识设计的。公理没有说"d区元素必须是金属"。

长江学者的批评（"你在用已知的化学知识设计规则，然后用规则来分类——这是循环论证"）在这个点上是有力的。

### 7.3 化合物中的差异分布没有编码

"化合物中各元素的δ视为0"是一个工作假设，不是推导。它导致5个反应Δ|δ|=0、框架失去判断力。

### 7.4 周期长度不能独立推导

周期长度（2, 8, 8, 18, 18, 32, 32）来自壳层容量（物理学投影），不能从化学投影独立推导。

### 7.5 无量纲常数（α）没有被推导

精细结构常数α = 1/137.036目前没有从公理推导。豆包的两版推导都不自洽。

### 7.6 验证样本太小

反应方向验证只用了20个反应（精心挑选），不是随机抽样。元素分类只覆盖了112个元素（已知的118个中有6个未分类）。

## 九、方法论总结

**原则一：先问"何以不能"，再问"何以可能"。**

不从经验归纳规律，而是从公理推导约束，排除不可能，锁定必然。

**原则二：诚实标注推导和拟合的区别。**

严格推导的结果（D=3, γ=1）和参数拟合的结果（分类规则）不能混为一谈。前者可以发表，后者需要继续验证。

**原则三：接口不是弱点，是必要组成部分。**

化学不能被完全推导。经验值是框架与现实的桥梁，不是框架的缺陷。

**原则四：用预测检验理论，不用解释检验理论。**

能解释已知事实的理论太多了。能预测未知事实的理论才有价值。框架的最终检验是：它能预测什么传统化学不能预测的东西？

**原则五：错误比正确更有信息量。**

每次失败都被正确地诊断：不是公式写错了，而是问错了问题。电负性公式的四次失败最终引出了"什么是元素"的根本问题。p区分类的12个错误指向了框架需要扩展的方向。

## 3. 核心卡点（讨论接口）

### 卡点1：特征向量是设计的，不是推导的

**问题：** 14位特征向量的编码方式是我设计的。换个编码方式，结果可能完全不同。长江学者的批评（"你在用已知的化学知识设计规则，然后用规则来分类——这是循环论证"）在这个点上是有力的。

**当前的特征向量：**
```
位 1-7：  Z的二进制编码（原子序数）
位 8-9：  区块类型（00=s, 01=p, 10=d, 11=f）
位 10-13：δ = Z_noble - Z（距稀有气体的偏移）
位 14：   p区归一化位置的方向位
```

**讨论接口：** 能否从公理直接推导出特征向量的编码方式？公理说"差异存在"和"系统偏好对称态"，但没有说"必须用14位二进制向量编码"。有没有一种方法，不预设编码方式，而是让公理自动选择最优编码？

**可能的方向：**
- 信息论方法：最小描述长度（MDL）原则，让数据自动选择最优特征维度
- 公理约束方法：A9（无额外自由度）可能约束了特征维度的上界

- A7（循环闭合）可能约束了周期结构，使得周期信息从公理涌现

### 卡点3：化合物中的差异分布

**问题：** "化合物中各元素的δ视为0"是一个工作假设，不是推导。它导致5个反应Δ|δ|=0、框架失去判断力。

**讨论接口：** 化合物中各元素的"残余偏移"应该如何定义？NaCl中Na⁺有Ne的电子构型但有11个质子——这个不匹配如何编码？

**可能的方向：**
- 残余偏移 = |Z - Z_noble(离子的电子构型对应的稀有气体)|
  - Na⁺: Z=11, 电子构型对应Ne(Z=10), 残余偏移 = 11-10 = +1
  - Cl⁻: Z=17, 电子构型对应Ar(Z=18), 残余偏移 = 17-18 = -1
  - 这恰好和原始δ相反——需要验证这个定义是否正确
- 残余偏移可能与离子的电荷有关——需要建立电荷和偏移之间的关系

### 卡点4：周期长度的独立推导

**问题：** 周期长度（2, 8, 8, 18, 18, 32, 32）来自壳层容量（物理学投影），不能从化学投影独立推导。

**讨论接口：** A7（循环闭合）说"稳定态必须参与长度≥N的有向循环"。化学周期性是否是A7的直接推论？如果是，周期长度能否从A7推导？

**可能的方向：**
- A7 → 化学性质随Z周期性变化 → 周期长度由"循环的最小长度"决定
- 循环的最小长度可能由A8（对称偏好）和A4（局部性）联合约束
- 如果A7能独立推导周期长度，化学投影就真正自洽了

### 卡点5：无量纲常数α（精细结构常数）

**问题：** α = 1/137.036是物理学中最重要的无量纲常数之一。目前没有从公理推导。豆包的两版推导都不自洽（数值不自洽、有未解释的因子128、修正因子15/14疑似从实验值反推）。

**讨论接口：** α能否从公理推导？如果能，它可能从哪里来？

**可能的方向：**
- α是"框架的某个结构性量的比值"——比如对称态的数量比、路径数的比值
- α可能与A7（循环闭合）有关——循环的"效率"或"阻力"
- α可能需要扩展框架——当前十条公理可能不够

**诚实的警告：** 这个问题极其困难。任何声称"已经推导出α"的说法都需要严格的数值验证。豆包的两次尝试都失败了，不要重复同样的错误。

### 卡点6：验证样本太小

**问题：** 反应方向验证只用了20个反应（精心挑选），不是随机抽样。元素分类只覆盖了112个元素。

**讨论接口：** 如何设计更大规模的验证？

**可能的方向：**
- 从化学数据库（ICSD、NIST）随机抽取1000个反应，用δ规则判断方向
- 从化合物数据库随机抽取10000个化合物，用Hamming距离预测稳定性
- 用盲测方式——隐藏实验结果，先做预测，再对比

### 卡点7：化学和物理学的接口

**问题：** 化学投影不能完全独立于物理学投影。周期长度、壳层容量等需要从物理学导入。

**讨论接口：** 化学和物理学的接口应该如何精确定义？哪些信息从物理学流向化学，哪些从化学流向物理学？

**可能的方向：**
- 物理→化学：壳层容量、周期长度、轨道类型
- 化学→物理：元素分类的验证、反应方向的验证
- 接口的数学形式：降维映射 π: {0,1}^N → {0,1}^M

## 4. 方法论约束

任何建议都需要满足以下约束，否则我们不会采用：

**约束一：不能引入经验参数。** 如果一个"推导"需要从实验数据拟合参数，它就不是推导，是拟合。拟合可以做，但必须诚实标注。

**约束二：必须有可证伪的预测。** 如果一个理论不能被任何可能的观测结果证伪，它就不是科学理论。任何建议都应该包含"如果X不成立，理论被证伪"的条件。

**约束三：必须与十条公理兼容。** 不能引入公理体系之外的假设。如果需要新假设，它要么从现有公理推导，要么需要明确标注为"新公理提案"。

**约束四：必须诚实标注严格程度。** 严格证明、结构性论证、概念层面——三者不能混淆。
