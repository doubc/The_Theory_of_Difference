## 引理L1：稀有气体不成键

**命题**：设 $\delta_1 = 0$，$\delta_2 \neq 0$。则对所有 $t \geq 0$，$s \geq 0$，有：

$$|\delta_1^{(int)}| + |\delta_2^{(int)}| = |{-t+s}| + |\delta_2 - t - s| \geq |\delta_2|$$

且等号当且仅当 $t = s = 0$ 时成立。

**含义**：不存在任何 $(t, s)$ 使总内部偏移绝对值之和严格小于原始值 $|\delta_2|$，因此生成序列无解，稀有气体不与任何元素成键。

---

### **证明**

设 $\delta_2 \neq 0$，$t \geq 0$，$s \geq 0$。需要证明：

$$f(t,s) = |s - t| + |\delta_2 - t - s| \geq |\delta_2|$$

**核心工具**：三角不等式的正确应用形式。

注意到：

$$(s - t) + (\delta_2 - t - s) = \delta_2 - 2t$$

直接三角不等式给出 $|s-t| + |\delta_2 - t - s| \geq |\delta_2 - 2t|$，当 $t > 0$ 时右边可能小于 $|\delta_2|$，所以这条路不够用。

换一个分解方式。将 $f(t,s)$ 写成关于 $s$ 的函数，固定 $t$，分析 $s$ 的最优选择。

**对固定的 $t \geq 0$，求 $\min_{s \geq 0} f(t,s)$：**

$$f(t,s) = |s - t| + |\delta_2 - t - s|$$

令 $u = s$（$u \geq 0$），$a = t$，$b = \delta_2 - t$，则：

$$f = |u - a| + |b - u|$$

这是关于 $u$ 的两个绝对值之和，其最小值由以下情形决定：

- 当 $a \leq b$（即 $t \leq \delta_2 - t$，即 $t \leq \delta_2/2$）时，$\min_u |u-a| + |b-u| = |b-a| = |\delta_2 - 2t|$，在 $u \in [a,b]$ 时取得。但 $u = s \geq 0$，需要检查 $[a,b] \cap [0,\infty)$ 是否非空。
- 当 $a > b$ 时，最小值同样是 $|b-a| = |\delta_2 - 2t|$，在 $u \in [b,a]$ 时取得。

所以对任意固定 $t$，$\min_{s \geq 0} f(t,s) = |\delta_2 - 2t|$，**只要最优的 $s$ 区间与 $[0,\infty)$ 有交集**。

现在分情形讨论：

---

**情形一：$\delta_2 > 0$**

此时 $b = \delta_2 - t$。

**子情形1a：$0 \leq t \leq \delta_2/2$**

$a = t \geq 0$，$b = \delta_2 - t \geq \delta_2/2 > 0$。最优 $s \in [t, \delta_2 - t] \subset [0,\infty)$，可达。

$$\min_{s \geq 0} f(t,s) = \delta_2 - 2t$$

要使 $f \geq \delta_2$，需要 $\delta_2 - 2t \geq \delta_2$，即 $t \leq 0$。由于 $t \geq 0$，等号当且仅当 $t = 0$。

当 $t = 0$：$\min_s f = \delta_2$，最优 $s \in [0, \delta_2]$。等号在 $s \in [0, \delta_2]$ 处成立，但此时 $\delta_1^{(int)} = s - 0 = s$，$\delta_2^{(int)} = \delta_2 - s$，总和 $= s + (\delta_2 - s) = \delta_2$（当 $0 \leq s \leq \delta_2$）。**等于** $|\delta_2|$，不严格小于。

当 $t > 0$：$f \geq \delta_2 - 2t < \delta_2$。**等等——这说明当 $0 < t \leq \delta_2/2$ 时，$f$ 的最小值 $\delta_2 - 2t < \delta_2$，不等式不成立？**

---

### **暂停：发现一个真实的反例**

取 $\delta_2 = +2$，$t = 1$，$s = 1$：

$$\delta_1^{(int)} = s - t = 0, \quad \delta_2^{(int)} = \delta_2 - t - s = 0$$

$$f(1,1) = 0 + 0 = 0 < 2 = |\delta_2|$$

**引理L1在 $\delta_1 = 0$ 的一般情形下是假的。**

---

### **重新审视命题**

这个反例说明：如果稀有气体的偏移是 $\delta_1 = 0$，而另一个元素的偏移是 $\delta_2 = +2$（比如O，氧族元素），那么 $t=1, s=1$ 使两者内部偏移都归零，总代价为0。这意味着O理论上可以与稀有气体"成键"？

但物理现实是稀有气体确实不成键。问题出在哪里？

问题出在**命题的表述**，不是物理直觉。

---

### **重新表述正确的命题**

回到生成序列的定义。成键的条件（命题B1'）是：存在 $(t,s)$ 使**总内部偏移绝对值之和严格小于原始总偏移绝对值之和**：

$$|\delta_1^{(int)}| + |\delta_2^{(int)}| < |\delta_1| + |\delta_2|$$

对于 $\delta_1 = 0$，$\delta_2 \neq 0$：

$$\text{原始总偏移} = 0 + |\delta_2| = |\delta_2|$$

成键条件要求存在 $(t,s)$ 使 $f(t,s) < |\delta_2|$。

上面的反例（$\delta_2 = 2, t=1, s=1$）给出 $f = 0 < 2$，说明**成键条件在 $\delta_1=0, \delta_2=+2$ 时是满足的**。

这意味着：**引理L1的正确版本不是"稀有气体不成键"，而是一个更有限的命题。**

---

### **回到物理约束：$t$ 的符号问题**

重新检查 $t$ 的定义。在V0.1的文档里：

> $t > 0$ 表示元素1的内部偏移沿**趋零方向**增加。

这个定义意味着 $t$ 不是自由变量，它的符号有物理约束：**$t$ 的符号必须使元素1的内部偏移趋向零，而不是远离零**。

对 $\delta_1 = 0$（稀有气体）：其内部偏移已经是零，已经处于对称态。任何 $t \neq 0$ 都会使 $\delta_1^{(int)} = -t + s$ 偏离零（当 $t \neq s$ 时）。

更精确地说，$t$ 的定义域约束是：

$$t \text{ 的符号} = \text{sign}(-\delta_1) = \text{sign}(0 - \delta_1)$$

即 $t$ 使元素1从当前偏移趋向零。当 $\delta_1 = 0$ 时，元素1已在零点，**不存在趋零方向**，因此 $t = 0$ 是唯一允许的值。

这是 $t$ 的定义中隐含的约束，在V0.1的文档里没有被明确写出。

---

### **引理L1的正确表述**

**引理L1（修正版）**：设 $\delta_1 = 0$（稀有气体），$\delta_2 \neq 0$。在生成序列的约束下，$t$ 的允许值为 $t = 0$（因为 $\delta_1$ 已在对称态，无趋零方向）。

在 $t = 0$ 的约束下：

$$f(0, s) = |s| + |\delta_2 - s| = s + |\delta_2 - s| \quad (s \geq 0)$$

**当 $\delta_2 > 0$ 时**：

- $0 \leq s \leq \delta_2$：$f = s + \delta_2 - s = \delta_2$
- $s > \delta_2$：$f = s + s - \delta_2 = 2s - \delta_2 > \delta_2$

所以 $\min_{s \geq 0} f(0,s) = \delta_2 = |\delta_2|$，等号在 $s \in [0, \delta_2]$ 时成立。

**当 $\delta_2 < 0$ 时**：

$$f(0,s) = s + |\delta_2 - s| = s + s - \delta_2 = 2s - \delta_2 \geq -\delta_2 = |\delta_2|$$

等号当 $s = 0$ 时成立。

**结论**：在 $t = 0$ 的约束下，$f(0,s) \geq |\delta_2|$ 对所有 $s \geq 0$ 成立，等号可以取到但不能严格小于。因此不存在使总内部偏移严格减小的 $(t,s)$，稀有气体不成键。$\square$

---

### **证明的关键：$t$ 的方向性约束**

这个证明揭示了一个此前没有被明确写出的重要结构：**$t$ 不是自由实数，它的符号由元素1的偏移方向决定**。

完整的约束是：

$$t \cdot \delta_1 \leq 0 \quad \text{（}t\text{ 使 }\delta_1\text{ 趋向零）}$$

即：
- $\delta_1 > 0$ 时，$t \leq 0$（元素1需要减小偏移，$t$ 为负）
- $\delta_1 < 0$ 时，$t \geq 0$（元素1需要增大偏移，$t$ 为正）
- $\delta_1 = 0$ 时，$t = 0$（元素1已在对称态）

这个约束的公理来源是**A6（涌现方向，不可逆）**：演化路径只能沿趋向对称态的方向进行，不能使元素远离其自然稳定态。

---

### **对V0.1文档的影响**

这个证明要求在V0.1的生成序列定义里补充一条约束：

**$t$ 的方向性约束（来自A6）**：

$$t \cdot \delta_1 \leq 0$$

这条约束需要加入优化问题的约束集：

$$\min_{t, s \geq 0} |\delta_1 + t - s| + |\delta_2 - t - s|$$
$$\text{subject to: } s \geq 0 \text{（A2）}, \quad t \cdot \delta_1 \leq 0 \text{（A6）}, \quad \delta_1^{(int)} + \delta_2^{(int)} + 2s = \delta_1 + \delta_2 \text{（A5）}$$

加入这条约束后，NaCl和H₂的最优解不变（它们本来就满足这条约束），稀有气体不成键的证明也变得严格。
