## 转移引擎

1. 筛选候选通道（开放 + 类型匹配 + 有容量）
2. 按有效成本排序（最小变易原则）
3. 执行转移（容量足则全转，不足则部分转）
4. 剩余压力积累

## 守恒检查

initial_total = transferred + accumulated + transformed + cost

无记录消失即模型错误。

## 破缺机制

积累压力超阈值触发破缺事件：
- 价格跳变
- 基差异常
- 近月逼仓
- 流动性枯竭
- 保证金追缴
- 强制平仓
- 规则调整
- 交割失败
- 主体退出

破缺后差异压力部分释放（50%）。

**破缺是必要的压力阀**：没有破缺，压力单向积累到系统崩溃（exp_007a 验证）。

## 锁定更新

lock_in += amount * 0.001（上限 1.0）

## 最近稳态判定

条件：总压力低于阈值 + 主要差异有承接 + 连续若干轮无破缺 + 通道选择稳定 + 主体退出放缓

**最近稳态≠均衡**：
- 均衡暗示自动回归
- 最近稳态只说明当前条件下暂时可维持
- 稳态越持续，越需要观察其内部差异是否正在重新积累

## 变形链（Phase 2）

差异经通道转移后，按变形规则生成新类型差异。队列式递归处理，最大深度 5。

```
inventory → delivery → basis → price → margin → liquidity
```

变形不消灭差异，只改变形式。守恒贯穿整条链。

---

## 反馈循环（Phase 2）

承接体承压后生成新的反馈差异，注入世界。

三个层次：
1. **保证金反馈**：capacity_ratio > 0.6 → margin 差异
2. **流动性反馈**：liquidity < threshold → liquidity 差异
3. **强平反馈**：capacity <= 0 → position + liquidity 差异

**约束**：反馈仅在 depth 0 生成，防止递归爆炸。

## 干预机制（Phase 2）

交易所是二阶承接位置——当市场承接不足时介入，但介入不是消除差异，而是重组差异结构。

三种干预：
1. **降低差异生成率**：recurrent_rate *= (1 - reduction)
2. **扩大通道容量**：channel.capacity *= (1 + expansion)
3. **释放承接力**：entity.release(used_capacity * release)

每种干预产生副作用差异（expectation/rule/liquidity）。

**实验验证（exp_006）**：干预延长了破缺间隔（平均从4步延长至9步，2.25倍），但系统仍不稳定。副作用差异会增加总压力。交易所是二阶承接，不能替代一阶运动。

## 收敛条件

系统收敛的充要条件：

```
Σ(recurrent_generation) < Σ(channel_drainage)
```

**实验验证（exp_007）**：
- 无破缺（threshold=9999）：压力直线上升到321，系统崩溃
- 快衰减（decay=0.80）：1次破缺后收敛到~52
- 通道翻倍（capacity×2）：2步收敛到6.42

**通道容量是真正的杠杆。**

## 叙事机制（理论，待实现）

叙事是市场组织对未来判断的方式。

**功能**：
- 把分散信息压缩为方向
- 通过投机承接、套保节奏、套利判断进入价格
- 可以自我强化，但最终要面对现实检验

**叙事的命运**：
- 能吸收现实修正的叙事 → 延续并调整
- 无法解释现实反向变化的叙事 → 破裂、退场、被替代

**叙事破裂**：头寸重组 + 波动上升 + 市场暂时失去共同解释框架

## 交割机制（理论，待实现）

交割是期货市场回流现实的机制。

**仓单**：合约与商品之间的中介
- 被交易所认可的商品权利凭证
- 影响市场价格：仓单充足→价格受约束；仓单稀少→交割紧张

**基差收敛**：不是自动发生的，依赖交割制度+套利行动+资金能力+现货条件

**交割约束近月**：远月可以讲故事，近月必须面对仓单

## 极端行情（理论，待实现）

极端行情不是市场的偶然失常，而是市场限度的集中显影。

**来源**：多种差异在短时间内集中到同一个价格界面上

**特征**：
- 价格跳跃（承接结构断裂）
- 流动性消失（流动性本身也是最近稳态）
- 叙事破裂（市场失去共同解释框架）
- 交割压力从后台走到前台

**干预的正当边界**：不在于维持某个价格水平，而在于恢复市场处理差异的最低功能。

## 铁律

1. 不改第一性定义
2. 不改写为价格预测/交易信号系统
3. 不引入真实下单
4. 不把差异清零
5. 不只给结论必给路径
6. 不把最近稳态解释为最优
7. 不把模拟解释为现实预测
8. 反馈差异不是修复
9. 干预不消除差异
10. 变形不消灭差异
