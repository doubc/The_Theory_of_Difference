# 弱公理与研究假设

## 弱公理集（最小集）

以下公理构成整个系统的逻辑基础。它们是**弱公理**——不声称绝对真理，但提供足够的结构使得研究可以开始。

```
A1  价格路径 = 时间有序离散状态序列
    ─ 价格是采样点，不是连续函数
    ─ 每个 Bar 是一次状态观测

A2  结构由点间关系生成，不由单点定义
    ─ 单根K线没有"结构"
    ─ 结构在至少两个极值点之间产生

A3  结构在容差下保持同一性
    ─ 极值点位置微调、幅度微调不改变结构身份
    ─ 同一性由关系不变量判定，不由精确坐标判定

A4  结构围绕关键区组织
    ─ Zone 是结构的"锚点"
    ─ Cycle 以进入/离开 Zone 为边界

A5  结构通过阶段转换生成、确认、破坏、反演
    ─ 结构不是静态的，有生命周期
    ─ 阶段转换是结构演化的基本单位

A6  随机扰动为残差，不参与结构定义
    ─ noise 不是结构的一部分
    ─ 结构定义只关心过滤噪声后的主干
```

## 研究假设列表

以下假设将被系统性验证，假设本身可以被证伪。

### 核心假设

| 编号 | 假设 | 可证伪条件 |
|------|------|-----------|
| H1 | 同类型结构在不同品种中存在可识别的关系不变量 | 跨品种相似度随机 |
| H2 | 结构的阶段（生成/确认/破坏）与后续价格行为有条件关联 | 阶段标签对后续收益无区分力 |
| H3 | 速度比和时间比是结构分类的有效特征 | 这两个特征的信息增益接近零 |
| H4 | 围绕 Zone 组织的结构比无锚点的价格段更有规律性 | Zone-based 结构的后验分布不比随机段集中 |
| H5 | 历史相似结构的后验分布具有信息价值 | 相似检索的后验分布与无条件分布无显著差异 |

### 次级假设

| 编号 | 假设 | 可证伪条件 |
|------|------|-----------|
| H6 | 结构在时间压缩下保留关系不变量（不同时间尺度的结构同态） | 跨尺度相似度不高于随机 |
| H7 | "慢涨急跌"和"急涨慢跌"是不同的结构类型，不是同一类型的噪声 | 两者的后验分布无法区分 |
| H8 | Zone 的强度（被试探次数）与结构的稳定性正相关 | 强度与后续波动率无相关 |
| H9 | 结构的典型度与后续行为的可预测性正相关 | 典型度高的结构后验分布不更集中 |
| H10 | 结构可以在生成阶段被早期识别（阶段预测可行） | 生成阶段的结构与确认阶段的后验无差异 |

## 验证策略

1. **每个假设独立验证** —— 不混在一起
2. **严格的统计检验** —— 不靠主观感觉
3. **跨样本验证** —— 时间切分 + 跨品种
4. **假设可以被证伪** —— 如果数据说"不成立"，就记录并调整理论

## 阶段0 验收

能用一句话回答：
> **研究对象是什么？** 价格路径中的关系不变量——围绕关键区组织的、由速度比/时间比/试探聚集度等度量定义的结构性关系。
>
> **研究对象不是什么？** K线图形、单根K线涨跌、短期价格预测。
