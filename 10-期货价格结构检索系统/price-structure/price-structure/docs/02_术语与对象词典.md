# 术语表与对象词典

## 基础术语

| 术语 | 定义 | 备注 |
|------|------|------|
| **Bar** | 单根K线，包含 O/H/L/C/V/OI 等字段 | 时间有序离散状态的一个采样 |
| **价格路径** | 按时间排列的 Bar 序列 | 研究的原始输入 |
| **极值点 (Pivot)** | 价格路径中的局部最高/最低点 | 由摆动幅度和持续时间阈值过滤 |
| **段 (Segment)** | 两个相邻极值点之间的连线 | 有方向、幅度、持续时间、平均速率 |
| **区 (Zone)** | 价格聚集或反复试探的区间 | 有中心、带宽、来源、强度 |
| **循环 (Cycle)** | 一段进入关键区到离开关键区的过程 | 有进入段、离开段、速度比、时间比 |
| **结构 (Structure)** | 围绕一个区组织的多个 Cycle 的集合 | 有阶段、有不变量 |
| **丛 (Bundle)** | 共享生成约束的多个 Structure 的集合 | 用于描述跨尺度/跨品种的结构关联 |

## 对象属性

### Point（极值点）
- `t`: 时间戳
- `x`: 价格位置

### Segment（段）
- `start / end`: 起止 Point
- `duration`: 持续时间
- `delta`: 价格变化量（end.x - start.x）
- `avg_rate`: 平均速率（delta / duration）
- `direction`: 方向（+1 上涨, -1 下跌, 0 横盘）
- `noise_level`: 噪声水平

### Zone（关键区）
- `price_center`: 价格中心
- `bandwidth`: 带宽
- `source`: 来源类型（high_cluster / low_cluster / pivot / volume）
- `strength`: 强度（被试探的次数 × 距离衰减）

### Cycle（循环）
- `entry`: 进入段（Segment）
- `exit`: 离开段（Segment）
- `zone`: 关联的 Zone
- `speed_ratio`: |v_exit| / |v_entry|
- `time_ratio`: Δt_entry / Δt_exit

### Structure（结构）
- `zone`: 核心 Zone
- `cycles`: Cycle 列表
- `phases`: 阶段列表（生成 → 确认 → 破坏/反演）
- `invariants`: 关系不变量字典

### Bundle（丛）
- `structures`: Structure 列表
- `generator_constraint`: 共同生成约束

## 阶段术语

| 阶段 | 含义 |
|------|------|
| **生成 (Formation)** | 结构开始出现，区形成 |
| **确认 (Confirmation)** | 多个 Cycle 出现，关系趋于稳定 |
| **破坏 (Breakdown)** | 价格突破区的边界，原有结构失效 |
| **反演 (Inversion)** | 结构类型转换（如顶部→底部） |

## 度量术语

| 度量 | 公式 | 意义 |
|------|------|------|
| **速度比** | \|v_exit\| / \|v_entry\| | 进入和离开的不对称性 |
| **时间比** | Δt_entry / Δt_exit | 节奏变化 |
| **试探聚集度** | stddev(high_i) | 高点的分散/集中程度 |
| **结构典型度** | [0, 1] | 与"典型"案例的接近程度 |
