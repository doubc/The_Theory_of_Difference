# 弱公理与研究假设

## 弱公理集（最小集）

以下公理构成整个系统的逻辑基础。它们是**弱公理**--不声称绝对真理，但提供足够的结构使得研究可以开始。

```
A0  价格是影子：价格 = Π(现实差异)，Π 有损、不可逆、非线性、时变
    ─ 代码分析的是 Π(D) 而不是 D 本身
    ─ 系统必须有投影觉知（知道自己在看影子）
    ─ V1.6 D0.1

A1  价格路径 = 时间有序离散状态序列
    ─ 价格是采样点，不是连续函数
    ─ 每个 Bar 是一次状态观测

A2  结构由点间关系生成，不由单点定义
    ─ 单根K线没有"结构"
    ─ 结构在至少两个极值点之间产生

A3  结构在容差下保持同一性
    ─ 极值点位置微调、幅度微调不改变结构身份
    ─ 同一性由关系不变量判定，不由精确坐标判定

A4  结构围绕关键区组织，关键区有共同反差驱动
    ─ Zone 是结构的"锚点"
    ─ Cycle 以进入/离开 Zone 为边界
    ─ 同一价位、不同反差类型的极值点不应归为同一结构 (D2.2)

A5  结构通过阶段转换生成、确认、破坏、反演
    ─ 结构不是静态的，有生命周期
    ─ 阶段转换是结构演化的基本单位

A6  随机扰动为残差，不参与结构定义
    ─ noise 不是结构的一部分
    ─ 结构定义只关心过滤噪声后的主干

A7  量纲守恒与归一化原则
    ─ 绝对价格受宏观货币总量影响发生漂移
    ─ 真正的结构不变量必须是无量纲的比例（如速度比、时间比、相对带宽）
    ─ 跨周期比较必须建立在对数收益率或相对变化的基础上

A8  差异守恒：不能被无代价清零，沿低阻通道转移
    ─ 被压平的差异不会消失 (D4.1)
    ─ 低阻通道: shorter_timeframe > volume > OI > cross_symbol (D4.2)
    ─ 波动率下降时必须检查差异转移到了哪里

A9  最近稳态：系统先滑向最近能稳住的安排，不是最优解
    ─ 变化沿阻力最小的方向 (D3.4)
    ─ 最近稳态最先可用，不最优 (D3.5)
    ─ exit 段后扫描后续极值，寻找最先停驻的位置

A10 系统 = 结构 × 运动
    ─ 没有运动的结构是死骨架 (D9.1)
    ─ 每个 Structure 必须带 MotionState
    ─ 运动从结构自身演化趋势中推导，不是外部输入
```

## 研究假设列表

以下假设将被系统性验证，假设本身可以被证伪。

### 核心假设

| 编号 | 假设 | 可证伪条件 |
|------|------|-----------|
| H1 | 同类型结构在不同品种中存在可识别的关系不变量 | 跨品种相似度随机 |
| H2 | 结构的阶段（生成/确认/破坏）与后续价格行为有条件关联 | 阶段标签对后续收益无区分力 |
| H3 | 速度比和时间比是结构分类的有效特征 | 这两个特征的信息增益接近零 |
| H4 | 围绕 Zone 组织的结构比无锚点的价格段更有规律性 | Zone-based 结构的后验分布不比随机段集中 |
| H5 | 历史相似结构的后验分布具有信息价值 | 相似检索的后验分布与无条件分布无显著差异 |

### 次级假设

| 编号 | 假设 | 可证伪条件 |
|------|------|-----------|
| H6 | 结构在时间压缩下保留关系不变量（不同时间尺度的结构同态） | 跨尺度相似度不高于随机 |
| H7 | "慢涨急跌"和"急涨慢跌"是不同的结构类型，不是同一类型的噪声 | 两者的后验分布无法区分 |
| H8 | Zone 的强度（被试探次数）与结构的稳定性正相关 | 强度与后续波动率无相关 |
| H9 | 结构的典型度与后续行为的可预测性正相关 | 典型度高的结构后验分布不更集中 |
| H10 | 结构可以在生成阶段被早期识别（阶段预测可行） | 生成阶段的结构与确认阶段的后验无差异 |

### V1.6 新增假设

| 编号 | 假设 | 可证伪条件 | V1.6 对应 |
|------|------|-----------|----------|
| H11 | 最近稳态预测优于随机：exit 后最先停驻的位置具有信息价值 | 稳态位置的后验与随机位置无差异 | D3.2 |
| H12 | 同价位、不同反差类型的结构具有不同的后验分布 | 反差类型对后续行为无区分力 | D2.2 |
| H13 | 守恒通量可预测回流：压缩期之后的释放期具有可识别模式 | 通量变化与后续波动无相关 | D4.1 |
| H14 | 投影压缩度高的结构，其相似性检索的置信度更低 | 压缩度与检索准确率无相关 | D0.3 |
| H15 | 运动态的 phase_tendency 能提前于 Phase 枚举识别阶段转换 | tendency 与实际转换无相关 | D9.2 |

## 验证策略

1. **每个假设独立验证** -- 不混在一起
2. **严格的统计检验** -- 不靠主观感觉
3. **跨样本验证** -- 时间切分 + 跨品种
4. **假设可以被证伪** -- 如果数据说"不成立"，就记录并调整理论

## 阶段0 验收

能用一句话回答：
> **研究对象是什么？** 价格路径中的关系不变量--围绕关键区组织的、由速度比/时间比/试探聚集度等度量定义的结构性关系。
>
> **研究对象不是什么？** K线图形、单根K线涨跌、短期价格预测。
