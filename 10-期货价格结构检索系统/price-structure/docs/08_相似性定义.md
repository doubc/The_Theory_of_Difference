# 相似性定义

## 三层相似度模型

```
sim_total = w1·sim_geometric + w2·sim_relational + w3·sim_family
```

默认权重：w1=0.4, w2=0.4, w3=0.2

## 第一层：几何相似 (sim_geometric)

基于归一化不变量向量的 1 - 欧氏距离。

不变量向量维度：
- cycle_count
- avg_speed_ratio
- avg_time_ratio
- high_cluster_stddev

各维度除以经验量级归一化，计算欧氏距离后 clip 到 [0, 1]。

**含义**：两个结构的"形状"有多接近。

## 第二层：关系相似 (sim_relational)

离散关系的一致性评分，4 个二值/连续子项取平均：

| 子项 | 计算方式 |
|------|----------|
| Zone 来源一致 | 同为 high_cluster 或同为 low_cluster → 1.0 |
| Cycle 数相近 | 1 - |n1-n2|/5，clip 到 [0, 1] |
| 速度比方向一致 | 同为 >1 或同为 <1 → 1.0 |
| 时间比方向一致 | 同为 >1 或同为 <1 → 1.0 |

**含义**：两个结构的"拓扑关系"是否一致。

## 第三层：结构族相似 (sim_family)

| 条件 | 分数 |
|------|------|
| 同标签 | 1.0 |
| 镜像对 | 0.6 |
| 不同标签 | 0.0 |
| 均无标签且 zone 来源一致 | 0.5 |
| 均无标签且 zone 来源不同 | 0.2 |

镜像对定义：
- SlowUpFastDown_TopReversal ↔ SlowDownFastUp_BottomReversal
- FastUpSlowDown_TopDistribution ↔ TripleTest_BottomBreakout

**含义**：两个结构是否属于同一"结构族"。

## 后验统计

检索到近邻后，聚合前向演化结果：

| 指标 | 说明 |
|------|------|
| mean_ret_5d/10d/20d | 各窗口平均收益 |
| median_ret_20d | 20 日收益中位数 |
| prob_positive_10d | 10 日正收益概率 |
| mean_max_dd_20d | 平均最大回撤 |
| mean_max_rise_20d | 平均最大涨幅 |

## 验证策略

- 时间顺序切分（禁止随机打乱）
- walk-forward 回测
- 跨品种验证（待多品种数据就绪）
- 报告样本内/外差距

## v2.5 升级：四层相似度模型

v2.0 为三层（几何+关系+族），v2.5 升级为四层：

```
sim_total = w1·sim_geometric + w2·sim_relational + w3·sim_motion + w4·sim_family
```

默认权重：w1=0.35, w2=0.35, w3=0.15, w4=0.15

### 第三层：运动相似 (sim_motion) — v2.5 新增

两个几何相似但运动方向相反的结构，后验行为可能完全不同。

| 子项 | 计算方式 |
|------|----------|
| 阶段趋势一致 | 同为 breakdown 或同为 confirmation → 1.0 |
| 守恒通量方向一致 | 同为正或同为负 → 1.0 |
| 稳态距离相近 | 1 - |d1 - d2|，clip 到 [0, 1] |
| 反差类型一致 | 相同 contrast → 1.0, unknown → 0.5, 不同 → 0.0 |

### DTW 相似度 — v2.5 新增

- Cycle 速度比序列的 Dynamic Time Warping 距离
- 捕捉"形状相似但时间错位"的结构
- Sakoe-Chiba 带宽约束

### 段形状相似度 — v2.5 新增

- Cycle 编码为离散符号 (方向×速度×时间)
- Levenshtein 编辑距离度量序列差异

## v3.0 升级：质量加权相似度

v3.0 引入质量分层后，检索排序从纯相似度变为质量加权：

```
final_score = sim_total × quality_weight
```

其中 `quality_weight` 由质量分层决定：
- A 层: 1.0
- B 层: 0.6
- C 层: 0.25
- D 层: 0.0（不进入检索候选）

后验统计只使用 A+B 层的检索结果，避免低质量结构稀释统计。
