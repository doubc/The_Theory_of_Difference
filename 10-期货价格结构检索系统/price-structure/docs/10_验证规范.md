# 验证规范

本文档定义模型验证的核心原则、数据切分方案和评估指标。

## 核心原则

1. **时间顺序切分** — 禁止随机打乱，禁止未来信息泄漏
2. **walk-forward 回测** — 滚动窗口训练/测试
3. **跨品种验证** — 同一模型在不同品种上评估
4. **报告样本内/外差距** — 过拟合检测

## 数据切分方案

```
|--- 训练集 ---|-- 验证集 --|-- 测试集 --|
  2005-2018      2019-2021    2022-2026
```

Walk-forward：每 N 个月滑动一次窗口。

## 分类任务验证

| 指标 | 说明 |
|------|------|
| Accuracy | 整体准确率 |
| Macro F1 | 各类别的 F1 平均 |
| 混淆矩阵 | 哪些类型容易混淆 |
| 基线对比 | vs RuleClassifier |

## Embedding 验证

| 指标 | 说明 |
|------|------|
| Recall@K | Top-K 检索中同类型的比例 |
| MRR | 平均倒数排名 |
| 类内距离 vs 类间距离 | 同类近、异类远 |

## 条件分布验证

| 指标 | 说明 |
|------|------|
| 校准曲线 | 预测概率 vs 实际频率 |
| Brier Score | 概率预测精度 |
| 样本外收益分布 | vs 无条件分布的区分度 |

## 可证伪条件

| 假设 | 证伪条件 |
|------|----------|
| H1 跨品种不变量存在 | 跨品种相似度随机 |
| H2 阶段与后续行为关联 | 阶段标签无区分力 |
| H3 速度比/时间比有效 | 信息增益接近零 |
| H4 Zone-based 有规律性 | 后验不比随机集中 |
| H5 历史相似有信息价值 | 后验与无条件无差异 |

## 已知局限

- 跨品种验证需要多品种数据支撑
- 当前样本量有限，统计检验力不足
- 部分分类器尚未完成训练
- 条件分布验证尚未实现
