# 相似性定义

本文档定义四层相似度模型（Similarity Model），用于度量两个价格结构之间的相似程度。

## 四层相似度模型

```
sim_total = w1·sim_geometric + w2·sim_relational + w3·sim_motion + w4·sim_family
```

默认权重：w1=0.35, w2=0.35, w3=0.15, w4=0.15

质量加权后，最终排序分数为：

```
final_score = sim_total × quality_weight
```

其中 `quality_weight` 由质量分层决定：A 层 1.0、B 层 0.6、C 层 0.25、D 层 0.0（不进入检索候选）。后验统计只使用 A+B 层的检索结果。

## 第一层：几何相似（Geometric Similarity）

基于归一化不变量向量的 1 - 欧氏距离。

不变量向量维度：
- cycle_count
- avg_speed_ratio
- avg_time_ratio
- high_cluster_stddev

各维度除以经验量级归一化，计算欧氏距离后 clip 到 [0, 1]。

**含义**：两个结构的"形状"有多接近。

## 第二层：关系相似（Relational Similarity）

离散关系的一致性评分，4 个二值/连续子项取平均：

| 子项 | 计算方式 |
|------|----------|
| Zone 来源一致 | 同为 high_cluster 或同为 low_cluster → 1.0 |
| Cycle 数相近 | 1 - |n1-n2|/5，clip 到 [0, 1] |
| 速度比方向一致 | 同为 >1 或同为 <1 → 1.0 |
| 时间比方向一致 | 同为 >1 或同为 <1 → 1.0 |

**含义**：两个结构的"拓扑关系"是否一致。

## 第三层：运动相似（Motion Similarity）

两个几何相似但运动方向相反的结构，后验行为可能完全不同。

| 子项 | 计算方式 |
|------|----------|
| 阶段趋势一致 | 同为 breakout 或同为 confirmation → 1.0 |
| 守恒通量方向一致 | 同为正或同为负 → 1.0 |
| 稳态距离相近 | 1 - |d1 - d2|，clip 到 [0, 1] |
| 反差类型一致 | 相同 contrast → 1.0, unknown → 0.5, 不同 → 0.0 |

**补充度量**：

- **DTW 相似度**：Cycle 速度比序列的 Dynamic Time Warping（动态时间规整）距离，捕捉"形状相似但时间错位"的结构，使用 Sakoe-Chiba 带宽约束
- **段形状相似度**：Cycle 编码为离散符号（方向×速度×时间），用 Levenshtein 编辑距离度量序列差异

**含义**：两个结构的"运动状态"是否一致。

## 第四层：结构族相似（Family Similarity）

| 条件 | 分数 |
|------|------|
| 同标签 | 1.0 |
| 镜像对 | 0.6 |
| 不同标签 | 0.0 |
| 均无标签且 zone 来源一致 | 0.5 |
| 均无标签且 zone 来源不同 | 0.2 |

镜像对定义：
- SlowUpFastDown_TopReversal ↔ SlowDownFastUp_BottomReversal
- FastUpSlowDown_TopDistribution ↔ TripleTest_BottomBreakout

**含义**：两个结构是否属于同一"结构族"。

## 验证策略

- 时间顺序切分（禁止随机打乱）
- walk-forward 回测
- 跨品种验证
- 报告样本内/外差距
