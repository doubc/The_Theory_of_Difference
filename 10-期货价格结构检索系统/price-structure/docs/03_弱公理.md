# 弱公理与研究假设

本文档定义系统的逻辑基础——弱公理集，以及将被系统性验证的研究假设。

## 弱公理集

以下公理构成整个系统的逻辑基础。它们是弱公理——不声称绝对真理，但提供足够的结构使得研究可以开始。

| 编号 | 公理 | 代码映射 |
|------|------|---------|
| A0 | 价格是影子：价格 = Π(现实差异)，Π 有损、不可逆、非线性、时变；系统必须有投影觉知（ProjectionAwareness） | `ProjectionAwareness` |
| A1 | 价格路径是时间有序离散状态序列，每个 Bar 是一次状态观测 | `DataLoader` |
| A2 | 结构由点间关系生成，不由单点定义；至少两个极值点之间才产生结构 | `pivots.py` |
| A3 | 结构在容差下保持同一性，由关系不变量（invariant）判定，不由精确坐标判定 | `models.py` |
| A4 | 结构围绕关键区（Zone）组织，Zone 有共同反差（ContrastType）驱动；同一价位、不同共同反差类型的极值点不归为同一结构 | `zones.py` |
| A5 | 结构通过阶段（Phase）转换生成、确认、破坏、反演；阶段转换是结构演化的基本单位 | `cycles.py` |
| A6 | 随机扰动为残差，不参与结构定义；结构定义只关心过滤噪声后的主干 | `pivots.py` |
| A7 | 量纲守恒：真正的结构不变量必须是无量纲比例（速度比、时间比、相对带宽）；跨周期比较基于对数收益率 | `relations.py` |
| A8 | 差异守恒：差异不能被无代价清零，沿低阻通道转移（shorter_timeframe > volume > OI > cross_symbol） | `check_conservation` |
| A9 | 最近稳态（NearestStableState）：系统先滑向最近能稳住的安排，不是最优解；exit 段后扫描后续极值，寻找最先停驻的位置 | `_find_nearest_stable` |
| A10 | 系统 = 结构 × 运动态（MotionState）；没有运动的结构是死骨架；运动从结构自身演化趋势中推导 | `MotionState` |

## 研究假设

以下假设将被系统性验证，假设本身可以被证伪。

### 核心假设

| 编号 | 假设 | 可证伪条件 |
|------|------|-----------|
| H1 | 同类型结构在不同品种中存在可识别的关系不变量 | 跨品种相似度随机 |
| H2 | 结构的阶段与后续价格行为有条件关联 | 阶段标签对后续收益无区分力 |
| H3 | 速度比和时间比是结构分类的有效特征 | 这两个特征的信息增益接近零 |
| H4 | 围绕 Zone 组织的结构比无锚点的价格段更有规律性 | Zone-based 结构的后验分布不比随机段集中 |
| H5 | 历史相似结构的后验分布具有信息价值 | 相似检索的后验分布与无条件分布无显著差异 |

### 扩展假设

| 编号 | 假设 | 可证伪条件 |
|------|------|-----------|
| H6 | 结构在时间压缩下保留关系不变量（不同时间尺度的结构同态） | 跨尺度相似度不高于随机 |
| H7 | "慢涨急跌"和"急涨慢跌"是不同的结构类型 | 两者的后验分布无法区分 |
| H8 | Zone 的强度与结构的稳定性正相关 | 强度与后续波动率无相关 |
| H9 | 结构的典型度与后续行为的可预测性正相关 | 典型度高的结构后验分布不更集中 |
| H10 | 结构可以在生成阶段被早期识别 | 生成阶段的结构与确认阶段的后验无差异 |
| H11 | 最近稳态预测优于随机：exit 后最先停驻的位置具有信息价值 | 稳态位置的后验与随机位置无差异 |
| H12 | 同价位、不同共同反差类型的结构具有不同的后验分布 | 共同反差类型对后续行为无区分力 |
| H13 | 守恒通量可预测回流：压缩期之后的释放期具有可识别模式 | 通量变化与后续波动无相关 |
| H14 | 投影压缩度高的结构，其相似性检索的置信度更低 | 压缩度与检索准确率无相关 |
| H15 | 运动态的 phase_tendency 能提前于阶段枚举识别阶段转换 | tendency 与实际转换无相关 |

## 验证原则

1. 每个假设独立验证，不混在一起
2. 严格的统计检验，不靠主观感觉
3. 跨样本验证：时间切分 + 跨品种
4. 假设可以被证伪——如果数据说"不成立"，就记录并调整理论
