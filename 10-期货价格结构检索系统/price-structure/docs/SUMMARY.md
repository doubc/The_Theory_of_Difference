# 价格结构形式系统 — 项目摘要

> 从期货价格序列中提取结构不变量，通过四层相似度检索历史先例，辅助交易决策。

## 核心能力

- **结构编译器**：从原始 K 线到完整结构描述的四层流水线（极值提取 → 段生成 → 关键区识别 → 循环组装）
- **四层相似度检索**：几何相似 + 关系相似 + 运动相似 + 结构族相似
- **质量分层**：A/B/C/D 四层质量评估，控制检索候选质量
- **跨品种共振**：按交易所分组检测板块方向一致性
- **多时间维度对比**：5 分钟 vs 日线、1 小时 vs 日线的跨尺度结构一致性检查
- **交易信号**：假突破 5 模式识别、突破确认 5 维评分、回踩确认、结构老化检测

## 技术架构

```
数据层                    编译层                    检索层                    应用层
┌──────────┐          ┌──────────┐          ┌──────────┐          ┌──────────┐
│ CSV      │          │ 极值提取  │          │ 几何相似  │          │ 研究工作台 │
│ MySQL    │ ──────── │ 段生成    │ ──────── │ 关系相似  │ ──────── │ 日报扫描  │
│ Parquet  │          │ 关键区    │          │ 运动相似  │          │ 主动匹配  │
│ 新浪API  │          │ 循环组装  │          │ 结构族    │          │ 合约检索  │
└──────────┘          └──────────┘          └──────────┘          └──────────┘
       │                    │                    │                    │
  C 扩展加速            C 扩展加速            质量加权            信号生成
  (_pivots.c)          (_compiler.c)         A/B/C/D 分层       5维评分+5模式
```
