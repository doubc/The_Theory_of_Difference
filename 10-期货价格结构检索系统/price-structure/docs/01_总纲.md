# 价格结构形式系统 — 项目总纲

本系统是一个从期货价格序列中识别、编译、检索结构关系的研究底座。

## 研究对象

价格路径中的**组织关系**——不是单根K线的涨跌，而是多根K线之间形成的结构性关系。

| 做 | 不做 |
|---|---|
| 结构本体定义 | 自动交易 |
| 结构对象化 | 短期价格预测 |
| 相似性检索 | K线形态库（视觉匹配） |
| 条件分布研究 | 端到端深度学习黑箱 |

## 系统架构

```
行情数据
  → 结构编译器（九机制形式化定义）
    → 结构对象（Point/Segment/Zone/Cycle/Structure/Bundle）
      → 质量分层（A/B/C/D）
        → 规则引擎（DSL）
          → 结构样本库（人工标注）
            → 相似性检索 → 后验统计
              → 共振检测（板块联动）
                → 生命周期追踪（formation→breakdown）
                  → 交易信号
```

## 弱公理（核心）

1. **A0 投影**：价格 = Π(现实差异)，Π 有损、不可逆、非线性、时变；系统必须有投影觉知（ProjectionAwareness）
2. **A1 离散**：价格路径是时间有序离散状态序列，每个 Bar 是一次状态观测
3. **A2 关系**：结构由点间关系生成，不由单点定义；至少两个极值点（Extremum）之间才产生结构
4. **A3 容差**：结构在容差下保持同一性，由关系不变量（invariant）判定，不由精确坐标判定
5. **A4 锚定**：结构围绕关键区（Zone）组织，Zone 有共同反差（ContrastType）驱动
6. **A5 阶段**：结构通过阶段（Phase）转换生成、确认、破坏、反演，有生命周期（Lifecycle）
7. **A6 噪声**：随机扰动为残差，不参与结构定义
8. **A7 量纲**：结构不变量必须是无量纲比例（速度比、时间比、相对带宽），跨周期比较基于对数收益率
9. **A8 守恒**：差异不能被无代价清零，沿低阻通道转移；波动率下降时检查差异转移到了哪里
10. **A9 最近稳态**：系统先滑向最近能稳住的安排（NearestStableState），不是最优解
11. **A10 运动**：系统 = 结构 × 运动态（MotionState）；没有运动的结构是死骨架

完整公理定义与研究假设见 [03_弱公理.md](03_弱公理.md)。

## 哲学前提

1. 价格是高维信息与约束状态的低维压缩投影
2. 信息沿迟滞、分层、反馈的时序过程进入价格
3. 组织关系相似 → 价格路径保留结构同态
4. 传播效率变化 → 结构发生时间压缩与表观变形，但关系不变量可能保留
5. 宏观与制度约束 → 结构演化具阶段性稳定性
6. 研究对象是比例关系的拓扑结构，不是K线图形或绝对价格
