# 价格结构形式系统 — 项目总纲

## 一句话

**第一年的目标不是"做出赚钱策略"，而是"做出一个能稳定描述、识别、比较价格结构的研究底座。"**

## 研究对象

价格路径中的**组织关系**——不是单根K线的涨跌，而是多根K线之间形成的结构性关系。

具体来说：
- 一个价格区间被反复试探了几次？
- 进入区间的速度和离开区间的速度比值是多少？
- 高点的聚集程度如何？
- 这些关系在历史中出现过吗？之后发生了什么？

## 我们做什么

| 做 | 不做 |
|---|---|
| 结构本体定义 | 自动交易 |
| 结构对象化 | 短期价格预测 |
| 相似性检索 | K线形态库（视觉匹配） |
| 条件分布研究 | 端到端深度学习黑箱 |

## 系统架构

```
行情数据 → 结构编译器 → 结构对象
                              ↓
                        规则引擎(DSL)
                              ↓
                        结构样本库 ← 人工标注
                              ↓
                        相似性检索 → 后验统计
                              ↓
                        学习模型(embedding/分类)
                              ↓
                        研究工作台(每日可用)
                              ↓
                        质量分层(A/B/C/D) ← 5维度评分
                              ↓
                        共振检测(板块联动) ← 跨品种信号
                              ↓
                        生命周期追踪(formation→breakdown)
                              ↓
                        交易信号(PythonGO策略)
```

## 哲学

1. 价格是高维信息与约束状态的低维压缩投影
2. 信息沿迟滞、分层、反馈的时序过程进入价格
3. 组织关系相似 → 价格路径保留结构同态
4. 传播效率变化 → 结构发生时间压缩与表观变形，但**关系不变量**（无量纲比率）可能保留
5. 宏观与制度约束 → 结构演化具阶段性稳定性
6. 研究对象是**比例关系的拓扑结构**，不是K线图形或绝对价格
