# 对象模型

本文档定义系统中所有结构对象的类型、字段与关系。

## 对象层级

```
Point → Segment → Zone → Cycle → Structure → Bundle
                                          ↓
                              QualityAssessment / SectorSignal
                              LifecycleRecord / BreakoutReversalSignal
```

## Point（极值点）

| 字段 | 类型 | 说明 |
|------|------|------|
| t | datetime | 时间 |
| x | float | 价格位置 |
| idx | int | 原始 Bar 序列索引 |
| log_x | float | 对数价格（只读属性） |

## Segment（段）

两个相邻极值点之间的连线。

| 字段 | 类型 | 说明 |
|------|------|------|
| start | Point | 起点 |
| end | Point | 终点 |
| duration | float | 持续天数 |
| delta | float | 价格变化量 (end.x - start.x) |
| avg_rate | float | 平均速率 (delta / duration) |
| direction | Direction | UP / DOWN / FLAT |
| noise_level | float | 噪声水平 |

派生属性：`abs_delta`, `abs_rate`, `log_delta`, `log_rate`

## Zone（关键区）

价格聚集或反复试探的区间。

| 字段 | 类型 | 说明 |
|------|------|------|
| price_center | float | 价格中心 |
| bandwidth | float | 带宽（半宽度） |
| source | ZoneSource | high_cluster / low_cluster / pivot / volume |
| strength | float | 强度 = Σ decay^i |
| touches | list[Point] | 触及该区的极值点 |

派生属性：`upper`, `lower`, `relative_bandwidth`

## Cycle（循环）

进入关键区到离开关键区的过程。

| 字段 | 类型 | 说明 |
|------|------|------|
| entry | Segment | 进入段 |
| exit | Segment | 离开段 |
| zone | Zone | 所属关键区 |

派生属性：`speed_ratio` (|v_exit|/|v_entry|), `time_ratio` (Δt_entry/Δt_exit), `amplitude_ratio`

## Structure（结构）

围绕一个区组织的多个循环的集合。

| 字段 | 类型 | 说明 |
|------|------|------|
| zone | Zone | 核心区 |
| cycles | list[Cycle] | 循环列表 |
| phases | list[Phase] | 生命周期阶段 |
| invariants | dict | 关系不变量 |
| typicality | float | 典型度 [0, 1] |
| label | str | 规则引擎标签 |
| symbol | str | 品种代码 |

不变量字典包含：`avg_speed_ratio`, `avg_time_ratio`, `high_cluster_stddev`, `cycle_count`

## Bundle（丛）

共享生成约束的多个结构的集合。按速度比/时间比聚类。

## QualityAssessment（质量评估）

| 字段 | 类型 | 说明 |
|------|------|------|
| tier | QualityTier | A/B/C/D 层级 |
| score | float | 综合质量分 [0, 1] |
| dimension_scores | dict[str, float] | 各维度得分 |
| flags | list[str] | 质量标记（问题/亮点） |
| recommendations | list[str] | 改进建议 |

## SectorSignal（板块共振信号）

| 字段 | 类型 | 说明 |
|------|------|------|
| sector | str | 板块名称 |
| direction | str | bullish/bearish/mixed |
| resonance_score | float | 共振强度 [0, 1] |
| participating | list[dict] | 参与品种列表 |
| quality_density | float | A/B 层结构占比 |
| direction_consistency | float | 方向一致性 |

## LifecycleRecord（生命周期记录）

| 字段 | 类型 | 说明 |
|------|------|------|
| date | str | 日期 |
| symbol | str | 品种代码 |
| zone_center | float | Zone 中心 |
| quality_tier | str | 当日质量层级 |
| quality_score | float | 当日质量分 |
| phase_tendency | str | 当日阶段趋势 |
| lifecycle_id | str | 生命周期唯一 ID |

## BreakoutReversalSignal（假突破反转信号）

| 字段 | 类型 | 说明 |
|------|------|------|
| signal_type | str | bearish_reversal / bullish_reversal |
| zone | Zone | 关联的 Zone |
| breakout_price | float | 突破时的价格极值 |
| reversal_price | float | 反转后的当前价格 |
| breakout_depth | float | 突破幅度（相对 bandwidth） |
| reversal_speed | float | 反转速度 |
| quality_score | float | 信号质量分 |
| quality_tier | str | A/B/C/D |

## SessionStats（日内时段统计）

| 字段 | 类型 | 说明 |
|------|------|------|
| session | str | 时段标识 |
| bar_count | int | K 线数 |
| structure_count | int | 结构数 |
| avg_speed_ratio | float | 平均速度比 |
| avg_quality_score | float | 平均质量分 |
| dominant_direction | str | 主要方向 |

## 关系算子

不存储，纯计算：

| 算子 | 说明 |
|------|------|
| `first_diff(p1, p2)` | 一阶差分 |
| `log_diff(p1, p2)` | 对数差分（品种无关） |
| `second_diff(p1, p2, p3)` | 二阶差分（曲率） |
| `distance_to_zone(p, z)` | 点到区距离 |
| `extrema_dispersion(points)` | 极值点变异系数 |

## 序列化

所有对象支持 `to_dict()` / `from_dict()`，JSON 友好。
