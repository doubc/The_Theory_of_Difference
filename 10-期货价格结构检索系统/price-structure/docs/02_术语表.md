# 术语表

本文档定义系统中所有术语的统一命名与含义。术语首次出现时括注英文，后续统一使用中文术语。

## 1. 基础术语

| 术语 | 英文 | 定义 |
|------|------|------|
| Bar | Bar | 单根K线，包含 O/H/L/C/V/OI 等字段，时间有序离散状态的一个采样 |
| 价格路径 | PricePath | 按时间排列的 Bar 序列，研究的原始输入 |
| 极值点 | Extremum | 价格路径中的局部最高/最低点，由摆动幅度和持续时间阈值过滤 |
| 段 | Segment | 两个相邻极值点之间的连线，有方向、幅度、持续时间、平均速率 |
| 关键区 | Zone | 价格聚集或反复试探的区间，有中心、带宽、来源、强度 |
| 循环 | Cycle | 一段进入关键区到离开关键区的过程，有进入段、离开段、速度比、时间比 |
| 结构 | Structure | 围绕一个关键区组织的多个循环的集合，有阶段、有不变量 |
| 丛 | Bundle | 共享生成约束的多个结构的集合，用于描述跨尺度/跨品种的结构关联 |

## 2. 对象属性

### 极值点属性
- `t`: 时间戳
- `x`: 价格位置
- `log_x`: 对数价格（消除宏观量纲漂移，关注比例关系）

### 段属性
- `start / end`: 起止极值点
- `duration`: 持续时间
- `delta`: 价格变化量（end.x - start.x）
- `log_delta`: 对数差分（跨品种、跨周期可比）
- `avg_rate`: 平均速率（delta / duration）
- `direction`: 方向（+1 上涨, -1 下跌, 0 横盘）
- `noise_level`: 噪声水平

### 关键区属性
- `price_center`: 价格中心
- `bandwidth`: 带宽
- `source`: 来源类型（high_cluster / low_cluster / pivot / volume）
- `strength`: 强度（被试探的次数 × 距离衰减）

### 循环属性
- `entry`: 进入段
- `exit`: 离开段
- `zone`: 关联的关键区
- `speed_ratio`: |v_exit| / |v_entry|（速度比，无量纲）
- `time_ratio`: Δt_entry / Δt_exit（时间比，无量纲）
- `log_speed_ratio`: 基于对数速率的比例

### 结构属性
- `zone`: 核心关键区
- `cycles`: 循环列表
- `phases`: 阶段列表
- `invariants`: 关系不变量字典

### 丛属性
- `structures`: 结构列表
- `generator_constraint`: 共同生成约束

## 3. 阶段术语

| 阶段 | 英文 | 含义 |
|------|------|------|
| 生成 | Formation | 结构开始出现，关键区形成 |
| 确认 | Confirmation | 多个循环出现，关系趋于稳定 |
| 破坏 | Breakdown | 价格突破关键区边界，原有结构失效 |
| 反演 | Inversion | 结构类型转换（如顶部→底部） |

## 4. 度量术语

| 度量 | 公式 | 意义 |
|------|------|------|
| 速度比 | \|v_exit\| / \|v_entry\| | 进入和离开的不对称性 |
| 时间比 | Δt_entry / Δt_exit | 节奏变化 |
| 试探聚集度 | stddev(high_i) | 高点的分散/集中程度 |
| 结构典型度 | [0, 1] | 与"典型"案例的接近程度 |

## 5. 高级术语

### 质量分层（QualityTier）

将编译出的结构按质量分为 A/B/C/D 四层，基于 5 维度加权评分。

| 术语 | 定义 |
|------|------|
| 质量维度 | 完整性(25%)、运动可信(25%)、守恒一致(20%)、时间成熟(15%)、后验可追溯(15%) |
| A 层结构 | 质量分 ≥ 0.75，可直接用于检索和后验统计 |
| B 层结构 | 0.50 ≤ 质量分 < 0.75，检索时降权(×0.6) |
| C/D 层结构 | 质量分 < 0.50，仅参考或丢弃 |
| 检索权重 | A=1.0, B=0.6, C=0.25, D=0.0 |

### 共振（Resonance）

| 术语 | 定义 |
|------|------|
| 板块共振 | 同板块多个品种同时出现 A/B 层结构，跨品种联动信号 |
| 共振强度 | 质量密度 × 方向一致性 × Zone聚集度，取值 [0, 1] |
| 强共振 | 同板块 ≥3 个品种同时触发，最高优先级 |

### 生命周期（Lifecycle）

| 术语 | 定义 |
|------|------|
| 生命周期 | 结构从生成→确认→破坏的完整轨迹，跨日期追踪 |
| 生命周期 ID | 品种_ZoneCenter_首次出现日期，唯一标识 |
| 质量趋势 | improving / declining / stable，质量分在生命周期中的变化 |

### 假突破（FakeBreakout）与突破确认（BreakoutConfirmation）

| 术语 | 定义 |
|------|------|
| 上破回落 | 价格突破 Zone 上方 → 反折回到 Zone 下方 → 看跌信号 |
| 下破反弹 | 价格跌破 Zone 下方 → 反弹回到 Zone 上方 → 看涨信号 |
| 突破深度 | 突破 Zone 的幅度（相对 bandwidth），越深反转越有意义 |
| 反转速度 | 回落幅度 / 突破幅度，越快信号越强 |

### 市场运动类型（Market Movement Type）

价格运动只有三种基本类型（加上反转作为边界事件），全部由稳态（NearestStableState）的跃迁关系定义：

| 类型 | 英文 | 值 | 定义 |
|------|------|-----|------|
| 上涨趋势 | Uptrend | `trend_up` | 时序上后一个稳态比前一个稳态的位置高（`zone_b.price_center > zone_a.price_center`） |
| 下跌趋势 | Downtrend | `trend_down` | 时序上后一个稳态比前一个稳态的位置低（`zone_b.price_center < zone_a.price_center`） |
| 震荡 | Oscillation | `oscillation` | 在同一个稳态内部的价格往复运动，不产生跨 Zone 的稳态跃迁 |
| 反转 | Reversal | `reversal` | 趋势方向的转换（上涨→下跌，或下跌→上涨），是趋势的边界事件 |

**判定逻辑：**

1. 取最近两个有稳态的 Cycle，比较其 `next_stable.zone.price_center`
2. 两个 Zone 中心价差 > 半个带宽 → **跨 Zone**，进入趋势判定
   - 后 > 前 → 上涨趋势
   - 后 < 前 → 下跌趋势
   - 若之前已有趋势记录且方向不同 → **反转**
3. 两个 Zone 中心价差 ≤ 半个带宽 → **同 Zone**，判定为震荡

**与现有对象的关系：**

- **趋势**依赖 `NearestStableState.zone.price_center` 的时序比较，不依赖价格涨跌方向本身
- **震荡**对应 `stable_velocity ≈ 0` 且 Zone 未切换的状态
- **反转**是趋势的边界事件：前一段趋势的方向标志发生变化

**注意**：运动类型（MovementType）和阶段（Phase）是两个正交维度。阶段描述结构的生命周期（生成→确认→破坏→反演），运动类型描述价格在稳态间的运动方式。一个处于"确认"阶段的结构，其运动类型可能是震荡、上涨趋势或下跌趋势。

### 运动态（MotionState）

| 术语 | 定义 |
|------|------|
| 运动态 | 结构的运动态，从结构自身演化趋势中推导，不是外部输入 |
| phase_tendency | 阶段趋势，能提前于阶段枚举识别阶段转换 |
| movement_type | 运动类型：trend_up / trend_down / oscillation / reversal，由稳态跃迁关系判定 |

### 投影觉知（ProjectionAwareness）

| 术语 | 定义 |
|------|------|
| 投影觉知 | 系统对自身认知边界的觉知，知道在看影子而非实体 |
| compression_level | 投影压缩度 [0,1] |
| blind_channels | 差异可能藏在的通道 |

### 稳定性判定（StabilityVerdict）

| 术语 | 定义 |
|------|------|
| 稳定性判定 | 对结构稳定性的判断，包含守恒通量（conservation_flux）评分 |
| 守恒通量 | flux_score ∈ [-1, 1]，正=释放，负=压缩，绝对值>0.3=结构正在重新分配差异 |

### 共同反差（ContrastType）

| 类型 | 条件 |
|------|------|
| PANIC | span < 30天 ∧ |P| ≥ 3，恐慌性密集试探 |
| OVERSUPPLY | span > 180天，供需失衡的长期堆积 |
| POLICY | 30 ≤ span ≤ 180 ∧ gap_CV < 0.5，政策驱动，间隔均匀 |
| UNKNOWN | 不满足以上条件 |

### 多时间维度

| 术语 | 定义 |
|------|------|
| 跨维度一致性 | 同品种在 5min/1h/1d 上编译出的结构匹配度（Zone重叠×方向×速度比） |
| 重采样 | 5分钟线 → 1小时线/日线的降采样（OHLCV 聚合） |

### 数据存储

| 术语 | 定义 |
|------|------|
| LocalStore | Parquet 本地数据仓库，列式压缩存储 |
| 增量更新 | 只下载新数据追加到本地，避免重复下载 |
| 编译缓存 | 编译结果 pickle 缓存，避免重复编译 |
