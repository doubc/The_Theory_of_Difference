# **价格结构形式系统 工程设计书 v2.5**

> 一个语法，九次展开。价格结构是差异即世界在市场上的显影。
>
> v1.0 → v2.0：引入投影框架、运动态、守恒、最近稳态。
> v2.0 → v2.5：统一语法形式化，补全九机制映射。

---

## **一、项目定义**

构建一个**价格结构形式语言 + 结构编译器 + 历史检索引擎 + 结构学习系统**。

系统每日读取本地行情，将价格序列编译为结构对象，识别其类型与阶段与**运动态**，检索历史相似结构，输出后验统计。

**不做**：自动交易、短期价格预测、K线形态库、端到端深度学习黑箱。
**做**：结构本体定义、结构对象化、相似性检索、条件分布研究。

---

## **二、统一语法**

本系统建立在一套统一的生成语法之上——《差异即世界》的九个机制。
不是九个并列概念，是一条链的九次展开：

```
聚簇 → 层级 → 守恒 → 先天完备 → 最小变易 → 破缺 → 循环 → 锁定 → 自指
```

价格结构系统做的事情：**从价格序列中识别差异走到了这条链的哪一步。**

| 机制 | 系统映射 | 核心对象 |
|------|---------|---------|
| 聚簇 | Zone 识别 | Zone |
| 层级 | Point→Segment→Zone→Cycle→Structure | 全部对象 |
| 守恒 | 守恒通量 + 转移通道检测 | check_conservation |
| 先天完备 | 多候选结构并存 | CompileResult.structures |
| 最小变易 | 极值点提取 + 最近稳态 | pivots + NearestStableState |
| 破缺 | 阶段转换（确认/破坏/反演） | Phase |
| 循环 | Cycle 自维持 + 结构复制 | Cycle + similarity |
| 锁定 | 不变量跨 Cycle 保留 | invariants |
| 自指 | 投影觉知 + 反身性追踪 | ProjectionAwareness + ReflexivityTracker |

## **四、对象模型**

```
Point(t, x, idx)
  ↓ first_diff / log_diff
Segment(start, end, duration, delta, avg_rate, direction)
  ↓ 聚簇
Zone(price_center, bandwidth, source, touches, context_contrast)
  ↓ 进出配对
Cycle(entry, exit, zone, next_stable)
  ↓ 组装
Structure(zone, cycles, phases, invariants, motion, projection)
  ↓ 封装
SystemState(structure, motion, projection, stability)

辅助：
  NearestStableState — 最近稳态
  MotionState — 运动态
  ProjectionAwareness — 投影觉知
  StabilityVerdict — 错觉检测
  ContrastType — 共同反差
  Phase — 生命周期阶段
  Bundle — 共享生成约束的 Structure 集合
```

## **五、编译器流水线**

```
bars → extract_pivots → build_segments → merge_micro_segments
     → detect_zones → build_cycles → assemble_structures
     → for each structure:
         infer_narrative_context()
         check_conservation()
         compute_motion()
         compute_projection()
         build_system_state()
     → detect_bundles()
     → StructureGraph.from_structures()
     → CompileResult
```

## **六、相似性检索**

```
sim_total = w1·sim_geometric + w2·sim_relational + w3·sim_motion + w4·sim_family

sim_geometric  : 不变量向量的归一化欧氏距离
sim_relational : Zone 来源 / Cycle 数 / 速度比方向一致性
sim_motion     : 阶段趋势 / 守恒通量 / 稳态距离一致性
sim_family     : 标签一致 / 镜像变体

v2.5 新增（辅助维度）：
  dtw_similarity         : Cycle 速度比序列的 DTW 距离
  segment_shape_similarity: 段形状编辑距离
```
