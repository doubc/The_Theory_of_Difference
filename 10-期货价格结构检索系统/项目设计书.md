# 价格结构形式系统 — 项目设计书

> 一个语法，九次展开。价格结构是差异即世界在市场上的显影。

## 项目定义

构建一个**价格结构形式语言 + 结构编译器 + 历史检索引擎 + 结构学习系统**。

系统每日读取本地行情，将价格序列编译为结构对象，识别其类型与阶段与运动态，检索历史相似结构，输出后验统计。

**做**：结构本体定义、结构对象化、相似性检索、条件分布研究。
**不做**：自动交易、短期价格预测、K线形态库、端到端深度学习黑箱。

## 统一语法

本系统建立在《差异即世界》的九个机制之上——不是九个并列概念，是一条链的九次展开：

```
聚簇 → 层级 → 守恒 → 先天完备 → 最小变易 → 破缺 → 循环 → 锁定 → 自指
```

| 机制 | 一句话语法 | 系统映射 | 核心对象 |
|------|-----------|---------|---------|
| 聚簇 | 临近差异自动归并 | Zone 识别 | Zone |
| 层级 | 对象逐层组装 | Point→Segment→Zone→Cycle→Structure | 全部对象 |
| 守恒 | 差异不可消灭，只能转移 | 守恒通量 + 转移通道检测 | check_conservation |
| 先天完备 | 多候选并存 | CompileResult.structures | CompileResult |
| 最小变易 | 系统先滑向最近稳态 | 极值点提取 + NearestStableState | pivots |
| 破缺 | 阶段转换不可逆 | Phase 生命周期 | Phase |
| 循环 | 结构自维持与复制 | Cycle + similarity | Cycle |
| 锁定 | 不变量跨周期保留 | invariants | invariants |
| 自指 | 系统觉知自身投影 | ProjectionAwareness + ReflexivityTracker | 投影觉知 |

## 模块总览

| 模块 | 名称 | 职责 |
|------|------|------|
| M1 | 对象系统 | Point→Segment→Zone→Cycle→Structure→Bundle 的数据模型 |
| M2 | 数据基础 | 清洗与统一行情（Parquet + MySQL 双源） |
| M3 | 结构编译器 | 价格→结构+运动+投影的四层流水线 |
| M4 | 规则引擎 | YAML 结构规则定义与 DSL 匹配 |
| M5 | 相似性检索 | 四层相似度（几何/关系/运动/族）+ 后验统计 |
| M6 | 样本库 | 结构标注与历史样本沉淀 |
| M7 | 知识图谱 | 结构演化链 + 叙事递归 |
| M8 | 反身性追踪 | 规则有效性衰减检测 |
| M9 | 研究工作台 | 日更扫描 + 交互研究界面（Streamlit） |

## 对象模型

```
Point(t, x, idx)
  ↓ first_diff / log_diff
Segment(start, end, duration, delta, avg_rate, direction)
  ↓ 聚簇
Zone(price_center, bandwidth, source, touches, context_contrast)
  ↓ 进出配对
Cycle(entry, exit, zone, next_stable)
  ↓ 组装
Structure(zone, cycles, phases, invariants, motion, projection)
  ↓ 封装
SystemState(structure, motion, projection, stability)
```

辅助对象：NearestStableState、MotionState、ProjectionAwareness、StabilityVerdict、ContrastType、Phase、Bundle。

详细定义见 [04_对象模型.md](price-structure/docs/04_对象模型.md)。

## 编译器流水线

```
bars → extract_pivots → build_segments → merge_micro_segments
     → detect_zones → build_cycles → assemble_structures
     → infer_narrative_context() + check_conservation()
     → compute_motion() + compute_projection()
     → build_system_state() → detect_bundles()
     → StructureGraph → CompileResult
```

详细定义见 [06_编译器.md](price-structure/docs/06_编译器.md)。

## 相似性检索

```
sim_total = w1·sim_geometric + w2·sim_relational + w3·sim_motion + w4·sim_family

sim_geometric  : 不变量向量的归一化欧氏距离
sim_relational : Zone 来源 / Cycle 数 / 速度比方向一致性
sim_motion     : 阶段趋势 / 守恒通量 / 稳态距离一致性
sim_family     : 标签一致 / 镜像变体
```

辅助维度：DTW 速度比序列距离、段形状编辑距离。

详细定义见 [08_相似性定义.md](price-structure/docs/08_相似性定义.md)。
