广义相对论是20世纪最精确的物理理论，但它的地基是一组假设——等效原理、协变性、最小耦合……爱因斯坦是天才，但他没有解释这些假设*
*为什么**成立。

WorldBase（差异论的物理学分支）做的事是：**把这些假设变成推论**。

从离散比特空间出发，用10条公理，推导出：

* 时空的连续性（不是假设，是极限行为）
* 等效原理（不是假设，是守恒结构的涌现）
* Einstein 场方程（不是假设，是统计力学的必然结果）

同一套公理体系，换一个尺度，就解释了社会学里的权力集中、制度锁定、历史路径依赖。

**这是差异论的核心主张：世界在所有尺度上都在做同一件事——差异在约束下生成结构。**
