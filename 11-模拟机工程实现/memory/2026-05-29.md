## 09:14 心跳 — CounterfactualEngine 激活 Bug 修复

### 问题
Phase 3 实验子实验 D（反事实探索）失败：`counterfactual_active` 在所有 ODI 水平下均为 false。

### 根因
`DivergencePointTracker.is_significant` 使用 AND 逻辑（entropy >= 0.5 AND ratio >= 2.0）。
自动生成的随机候选方向产生均匀概率分布（ratio ~1.0），永远无法通过 ratio 阈值。
→ 从未检测到分歧点 → 从未创建反事实分支 → 引擎永远不激活。

### 修复
1. `is_significant` 改为仅使用熵检查（高熵 = 多个可行方向 = 分歧点）
2. HierarchicalEvolver 回调中传递实际 `odi_result` 而非 `None`
