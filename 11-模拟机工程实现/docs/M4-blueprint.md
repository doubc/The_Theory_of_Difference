# M4 蓝图：从"检测象界"到"生成象界"

## 理论依据

### 象界形式化V0.8 核心洞察
1. **生成链是涌现的，不是外加的**：边界→自维持→记忆→复制→筛选→功能→前主体态，每一步都是前一步在{0,1}^N上的必然延伸
2. **A5是守恒约束**：差异总量在转换中守恒，保证涌现不"无中生有"
3. **A9是封闭性保证**：所有结构从内公理内生，不引入外部参数
4. **三个内部张力**（工程可映射）：
   - 记忆要求G|𝒮不强连通，自维持要求G|𝒮强连通 → 工程上=结构需要多稳态
   - PS-Emergence步骤3缺严格证明 → 工程上=需要验证耦合结构是否确实比独立结构更稳定
   - A1'不可还原性缺精确定义 → 工程上=粗粒化后高层属性不能从低层完全预测

### 差异论V1.7 核心洞察
1. **底图事件 vs 结构事件**：当前引擎只有"演化步"，缺少"底图改写"机制
2. **五方法论螺旋循环**：P→E→M→S→R→P'，每轮改写可能性空间
3. **叙事递归**：高语义层的核心机制，结构开始"解释"自身
4. **差异分层**：D0(原初可区分性)→D1(结构差异)→D2(分布差异)→D3(语义差异)→D4(事件差异)
