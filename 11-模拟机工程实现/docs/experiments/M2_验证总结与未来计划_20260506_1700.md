## 三、未来验证计划

### 阶段 1：让聚簇真正发生（M3.1）

**目标：** 稳定区域从分散碎片聚合成连通结构。

**方法：**
- **增加训练步数**：300 → 1000+，给结构更多时间聚合
- **调整空间多样性损失**：当前 `spatial_diversity` 权重 0.3，可以增加**连通性奖励**——对连通的稳定区域给予正向激励
- **引入吸引子动力学**：在 `DifferenceReactor` 中，让稳定区域对周围活跃区域有微弱吸引，模拟《象界》中"从分隔到界面"的过程

**验证标准：** closure_ratio ≥ 0.5

### 阶段 2：升维实验（M3.2）

**目标：** L0 稳定结构粗粒化为 L1，L1 上独立运行演化。

**方法：**
- 运行 500+ 步，检测稳定结构
- 调用 `layer.coarse_grain(structures)` 得到 L1 层
- 在 L1 上启动新的 `DifferenceReactor`，观察 L1 是否也能产生稳定结构
- 如果 L1 也产生稳定结构，再次粗粒化为 L2

**验证标准：** L1 上能独立运行 + L1 也能检测到稳定结构

**理论对应：** 这是《差异即世界》中"层级"机制的直接验证——**结构获得递归展开的能力**。如果 L0→L1→L2 都能工作，说明九机制生成链中的"聚簇→层级→循环"路径是可行的。

### 阶段 3：象界前主体态链（M4，远期）

**目标：** 验证《象界》八章生成链。

《象界》描述了从"存在"到"前主体态"的八个阶段。当前模拟机只覆盖了前三个：

| 阶段 | 当前状态 | 需要做什么 |
|------|---------|-----------|
| 1. 从存在到底象 | ✓ A1 注入差异，A2 离散编码 | 已完成 |
| 2. 从分隔到界面 | △ 边界存在但未成为交换通道 | 需要实现**界面层**——边界区域的双向交换机制 |
| 3. 从闭合到自维持 | △ 有循环但不够稳定 | 需要验证结构在扰动后能否自我修复 |
| 4. 从痕迹到记忆 | ✗ 未实现 | 需要实现**历史偏置**——过去路径影响未来演化 |
| 5. 从再现到复制 | ✗ 未实现 | 需要实现**模式复制**——稳定结构在新区域再构造 |
| 6. 从并存到筛选 | ✗ 未实现 | 需要多个结构竞争，优胜劣汰 |
| 7. 从耦合到功能 | ✗ 未实现 | 需要结构内部关系沉积出不对称贡献 |
| 8. 前主体态 | ✗ 未实现 | 上述机制汇聚 |

**关键突破点：** 第 4 步"记忆"是整个链条的转折。当前模拟机是无记忆的——每步演化只依赖当前状态。需要引入**路径依赖**：过去 16 步的演化历史不仅用于检测稳定性，还要**直接影响下一步的演化方向**。

## 四、理论验证的层次结构

```
已验证（代码+实验）          待验证（代码已有，实验不足）      未实现
─────────────────────      ────────────────────────      ──────────
```
