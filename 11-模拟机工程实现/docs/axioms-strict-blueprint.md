# 九公理严格化 — 完整构型

## 理论立场

九公理严格化是在验证一个元物理学主张：**物理规律可以从差异约束中必然涌现**。

这不是"调参数"，而是回答希尔伯特第6问——物理学的公理化。

## 当前九公理实现状态

### A1 原初差异
- **当前**：观测类，仅记录差异水平
- **理论要求**：存在唯一无差异态 **0**，差异沿层级方向单调累积
- **差距**：无层级深度概念，state tensor 没有"汉明重量层级"结构
- **严格化方向**：引入汉明重量 w(x) = Σx_i，确保演化路径上 w 不减

### A2 二元具象
- **当前**：约束类，`state * (1-state)` 损失
- **理论要求**：{x_i ∈ {0,1}}，差异单元不可再细分
- **差距**：用 sigmoid 连续近似，不是真正二值
- **严格化方向**：硬投影 + Gumbel-Softmax 直通估计

### A3 有限离散
- **当前**：约束类，CNN 结构保证局域性
- **理论要求**：N < ∞，状态空间有限
- **差距**：CNN 卷积核是连续参数，不保证离散跃迁
- **严格化方向**：汉明距离=1 的邻域约束

### A4 最小变易
- **当前**：约束类，`(next_state - state)^2` 损失
- **理论要求**：每步汉明距离恰好为 1
- **差距**：连续近似，不保证单比特翻转
- **严格化方向**：单比特翻转掩码 + 汉明距离=1 硬约束

### A5 守恒律
- **当前**：约束类，封闭/开放双模式
- **理论要求**：差异总量守恒，Q(x) 在所有允许演化下不变
- **差距**：近似实现，残差计算用 MSE
- **严格化方向**：精确守恒量跟踪 + 通量平衡

### A6 不可逆性（DAG）
- **当前**：观测类，仅记录梯度大小
- **理论要求**：演化图是 DAG，不存在有向环路，时间偏序
- **差距**：完全没有方向性约束！演化可逆
- **严格化方向**：跃迁方向矩阵，禁止逆向跃迁

### A7 循环闭合
- **当前**：约束类，`stability_violation` 检测时间稳定性
- **理论要求**：稳定态必须参与有向闭合循环
- **差距**：只检测时间稳定性，不检测循环闭合
- **严格化方向**：循环检测器 + 闭合性损失

### A8 对称偏好
- **当前**：观测类，仅标记活跃
- **理论要求**：偏好 w=N/2 态，权重随 |w-N/2| 单调递减
- **差距**：完全没有对称偏好机制！
- **严格化方向**：A8 权重函数 ρ(w) = C(N,w)/C(N,N/2)

### A9 封口
- **当前**：触发类，升维判定
- **理论要求**：最小充分实现，不引入额外自由度
- **差距**：有判定但无自由度封口机制
- **严格化方向**：自由度计数 + 封口约束

## 严格化方案

### 核心转变：从连续近似到离散严格

当前所有约束都是连续近似（sigmoid、MSE、CNN）。严格化需要：

1. **汉明几何替代欧氏几何**
   - 状态空间：{0,1}^N 超立方体
   - 距离：汉明距离 d_H(x,y) = |{i: x_i ≠ y_i}|
   - 重量：w(x) = Σx_i
   - 中截面：w = N/2（最大差异层）

2. **离散跃迁替代连续演化**
   - 每步恰好翻转一个比特（A4 严格）
   - 翻转方向由 DAG 约束（A6 严格）
   - 翻转概率由 A8 权重调制

3. **子空间分解**
   - 引力子空间：层级深度方向（A1）
   - 弱力子空间：手征方向（A6）
   - 强力子空间：色荷轨道（A4+A8）
   - 电磁子空间：规范联络（A1'+A5）

### 模块设计

```
acl/axioms_strict.py  ← 严格化九公理（新文件）
  ├── AxiomEngineStrict  ← 严格化公理引擎
  ├── A1_HammingLevel    ← 汉明重量层级
  ├── A2_BinaryStrict    ← 严格二值（Gumbel-Softmax）
  ├── A3_HammingNeighbor ← 汉明距离=1 邻域
  ├── A4_SingleBitFlip   ← 单比特翻转
  ├── A5_ConservationStrict ← 精确守恒
  ├── A6_DAGConstraint   ← DAG 方向约束
  ├── A7_CycleClosure    ← 循环闭合检测
  ├── A8_SymmetryWeight  ← 对称偏好权重
  └── A9_MinimalRealization ← 自由度封口

engine/hamming_engine.py  ← 汉明几何引擎（新文件）
  ├── HammingState        ← 汉明状态空间管理
  ├── HammingTransition   ← 离散跃迁算子
  └── HammingMeasurement  ← 汉明距离/重量测量

layers/hamming_layer.py  ← 汉明格点层（新文件）
  ├── HammingLattice      ← {0,1}^N 格点
  ├── MidSurface          ← w=N/2 中截面
  └── SubspaceView        ← 子空间视图
```

### 与现有代码的关系

不是替换，而是**并行实现**：
- 当前 reactor/trainer 继续工作（连续近似版本）
- 新增 hamming_engine（严格离散版本）
- 两个版本共享相同的公理接口（AxiomBase）
- 实验对比：连续版本 vs 严格版本的涌现结果

## 关键约束
- 不声称模拟现实物理
- 不做完整力/量子
- 不训练大模型
- 不用视觉图案替代验证器
- N 保持小规模（≤24），避免组合爆炸
