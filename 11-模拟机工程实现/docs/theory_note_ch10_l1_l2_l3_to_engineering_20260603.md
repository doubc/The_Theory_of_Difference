## 一、重大发现：理论定义的 L1/L2/L3 与我们工程层的对应关系

### 1.1 理论中的三层定义

| 层级 | 理论名称 | 核心功能 | 判定标准 |
|------|---------|---------|---------|
| **L1** | 命名层 | 使分散经验获得聚类能力 | 能命名，但不一定解释 |
| **L2** | 因果层 | 组织条件、机制、路径和后果 | 能说明"为什么这么生成" |
| **L3** | 框架重组层 | 改写对象域的可能性空间 | 改变问题提出方式和路径判断 |

### 1.2 理论三层与工程层级的映射

**核心洞察**：差异论第十章定义的 L1/L2/L3 是**功能性层级**（理论在对象域中的生成位置），而非我们要实现的**结构性层级**（模拟机的封装层级）。但两者有深刻映射关系。

| 工程层级 | 结构含义 | 理论对应 | 功能含义 |
|---------|---------|---------|---------|
| **L0** | 原始差异场 (MINI) | **L1 命名层** | 差异的初次聚类，形成可识别的"对象"边界 |
| **L1** | 制度记忆 (INSTITUTIONAL) | **L2 因果层** | 对聚簇进行再组织，形成稳定机制和约束条件 |
| **L2** | 文明叙事 (CIVILIZATION) | **L3 框架重组层** | 改写可能性空间，改变底层如何被判断 |

### 1.3 为什么这个映射成立

**L0 → L1 (命名层)**：
- L0 的聚簇 (clustering) 使差异从背景中提出，形成可识别的"对象"
- 封口 (sealing) 相当于命名：把某些比特关系凝聚为稳定边界
- 差异论："命名使经验获得聚类能力" — L0 的聚簇机制做了同样的事

**L1 → L2 (因果层)**：
- L1 的制度记忆对 L0 聚簇进行再组织
- 不是简单复制 L0，而是增加约束条件（哪些聚簇被保留，哪些被抑制）
- 差异论："L2 要说明机制何时启动，何时失效" — L1 的约束场施加了条件

**L2 → L3 (框架重组层)**：
- L2 的文明叙事改变了 L0-L1 的判断方式
- Track B8 发现 L1 是被动投影而非主动 agent — 理论上的"框架重组"正好对应
- 差异论："L3 改变对象域中问题如何被提出、路径如何被判断"
- L2 的叙事自我 (NSI) 正是这种"判断的累积"

## 二、从 B9 到 B10 的理论路径

### 2.1 B9 完成的状态

```
L0 (原始差异场, 聚簇形成边界) 
  ├─ partial seal → 形成 L1 (制度记忆 = 命名层约束)
  └→ ConstraintBiasedCoupling → L2 (文明叙事 = 因果层)
```

B9 验证了：L1 的冻结结构通过偏置场影响 L2，但 L2 有独立聚类能力。

### 2.2 B10 的理论问题

> 差异论指出层级深度有上限吗？可以继续 L2→L3 吗？

第十章的核心发现：**层级不是固定等级**（§10.4）。同一个理论可以浮动。L1/L2/L3 是功能位置，不是结构固定性。

这意味着在理论上：
- 没有理由认为 L2→L3 的级联不能继续
- 但 L3 需要的不是"更多结构"，而是**框架重组能力**
- L3 的关键：改写可能性空间（即改变 L0 的约束条件）

### 2.3 B10 的工程问题

工程实现的核心问题：
1. L2 能否产生"冻结结构"（类似 L0 的 partial seal）？
2. L2 的冻结结构能否通过偏置场影响 L3？
3. L3 的独立性能否创造新的叙事层次？

目前 B9 中 L2 是通过 ConstraintBiasedCoupling 独立聚类的，但 L2 本身没有 partial seal 机制。为了 L2→L3，需要：
- **L2 的 partial seal 机制**（类似 B7 中 L0 的部分封口）
- **L3 独立聚类**（类似 B5 中 L2 的独立聚类）
- **L2→L3 偏置场**（类似 B9 的 ConstraintBiasedCoupling）

## 三、B10 理论设计

### 3.1 核心架构

```
L0 (原始差异场) → partial seal → L1 (命名层约束)
                                    ↓ (ConstraintBiasedCoupling)
                                    L2 (因果层叙事) → partial seal → L3 (框架重组层)
                                                                       ↓ (ConstraintBiasedCoupling v2)
                                                                       L3 独立叙事
```

### 3.2 L2 的 partial seal 机制

L2 要产生"冻结结构"需要：
- **稳定性积累**：L2 的稳定性持续高于某个阈值
- **封口事件**：L2 的部分比特冻结（模拟"命名层约束"的积累）
- **偏置场转化**：L2 冻结比特成为 L3 的偏置场

### 3.3 L3 的独立叙事

L3 相对于 L2 需要：
- 自己的独立聚类空间
- 从 L2 偏置场获得约束，但不被 L2 完全决定
- 产生独立的 NSI（叙事自我指数）
- **关键测量**：L3 的 NSI 是否改写了 L0→L2 的约束条件

### 3.4 核心假设 H54-H57

| 假设 | 描述 | 通过标准 |
|------|------|---------|
| **H54** | L2 能产生冻结结构（模拟 partial seal） | ≥5/8 seeds 有冻结事件 |
| **H55** | L2→L3 偏置效应可测量 | 均值偏置 > 0.10 |
| **H56** | L3 有独立叙事动力学 | L3 NSI 滚动自相关 > 0 |
| **H57** | L2→L3 级联不影响 L1→L2 稳定性 | L1-L2 corr 保持 < 0.85 |

## 五、理论意义总结

1. **第十章提供了 L1/L2/L3 的功能定义**，与我们的工程结构层级高度映射
