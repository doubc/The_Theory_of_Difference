## 0. 一句话定义

**差异论模拟机**是在九公理约束下构造有限离散世界，观察稳定结构能否递归生成高层单元的计算实验系统。
它不是数值模拟或机器学习，而是**公理约束下的离散世界实验机**——从 `{0,1}^N` 上的比特翻转出发，
让九条公理自己决定哪些结构能存活、哪些能涌现、哪些能封装成更高层级的单元。

### Phase 5-9: 相变发现（核心突破之一）

**时间**: 2026-05-30 ~ 2026-06-03
**范围**: N0 扫频实验、L1 形成率测量、绑定规则参数扫描

**发现 1: 一阶相变 — N0* ≈ 30.5 (Phase 5-9)**

从随机初始化的 bit 空间中，当初始活跃 bit 数超过 N0* ≈ 30.5 时，系统从不密封（无序相）跳跃到密封（有序相）。这是一个**不连续（一阶）相变**。

| N0 | L1 形成率 | 系统状态 |
|----|-----------|---------|
| ≤29 | 0% | 纯 L0 混沌 |
| 30 | ~44% | 临界区（双峰分布） |
| ≥31 | 100% | L1 有序相 |

**发现 2: 绑定(Binding)是结构涌现的驱动力**

Bits 之间通过约束密度(neighbors 数量)产生 binding_strength，当平均 binding_strength 达到阈值时，bits 被"逼在一起"形成 L1 结构。这正是差异论"共同反差"(Common Contrast)的操作性表达。

**发现 3: 密封(Sealing)是结构确定性的标志**

当系统达到临界密度时，所有 bits 在同一瞬间被分类为"保持"(active)或"冻结"(sealed)。这是一次性分化决策(cascade event)，不是渐进冻结。

**文档**: `docs/PHASE_SUMMARY_PHASE5to9.md`

### Phase 10-11: 子空间分解与跨层架构

**时间**: 2026-06-03 ~ 2026-06-05
**范围**: 多子空间耦合、A6 DAG 方向性修复、密封步数记录

**发现 4: 子空间是差异分化的自然结果**

两个独立子空间可以通过耦合产生"救援效应"(rescue effect)——一个已密封的子空间可以"拉动"另一个低于临界密度的子空间进入有序相。

**发现 5: 单向性约束的工程必要性**

A6 DAG 方向性确保差异传递是单向的（非对称），防止反向流绕过密封。这是差异论"循环(机制7)"和"锁定(机制8)"的精确实现。

### Phase 12: 聚簇时空动力学（核心突破之二）

**时间**: 2026-06-07
**范围**: P1 (exp_153: 单空间聚簇动力学), P2 (exp_154: N 扫频 24-96)

**发现 6: 密封是一次性相变，是所有 N 的普适性质**

P1 (exp_153, N=48): 49/50 密封, 98% 单次 cascade。密封发生在单次 cascade 事件中，一次性冻结所有 bits。这不是渐进过程，而是一阶相变。

P2 (exp_154, N=24-96, 70 runs): 所有产生密封的 N 值(36-96) 和所有 runs 中，cascade_events_mean = 1.0, single_event_ratio = 1.0。**一阶相变是聚簇时空动力学的普适性质，不是 N=48 的特例。**

**发现 7: 普适标度律 — cascade_size = 0.40 × N**

| N | cascade/N |
|---|-----------|
| 36 | 0.47 |
| 42 | 0.38 |
| 48 | 0.40 |
| 54 | 0.39 |
| 60 | 0.40 |
| 72 | 0.38 |
| 84 | 0.40 |
| 96 | 0.41 |

**加权平均值: 0.40**。不论系统大小，密封 moment 总是恰好淘汰约 40% 的激活空间。
这是差异论世界的第一个普适量纲关系，对应差异论"共同反差的淘汰比例是系统固有性质"的核心命题。

**发现 8: 相变边界修正 — N0* ≈ 34**

Phase 5-9 测得 N0* ≈ 30.5 基于 L1 形成率。Phase 12 P2 使用精确密封指标给出修正值：**N0* ≈ 34**。N=36 是第一个出现密封的 N 值，仅 40% runs 密封（临界涨落区域）。

**文档**: `12-聚簇时空动力学_v2.md`

### Phase 13: 密封后持久性动力学

**时间**: 2026-06-07
**范围**: 密封后稳定性分析

**发现 9: 密封是拓扑不变量（"死秩序"的起源）**

虽然 Phase 12 证明了密封是一次性相变，Phase 13 发现密封的后果是系统进入一种"死秩序"(Dead Order)状态：
- 密封后 bits 不再翻转（flips = 0）
- 密封结构在演化窗口中永恒不变
- 这不是参数问题，而是动力学拓扑不变量

**文档**: `docs/phase13_planning.md`

### Phase 14: 死秩序的拓扑不变性验证（核心突破之三）

**时间**: 2026-06-07 ~ 2026-06-08
**范围**: P1 (时间结构化注入 exp_158), P2 (竞争性密封核 exp_159), P3 (主动约束塑造 exp_162)

**发现 10: 三个方向全部失败，确认死秩序不可打破**

| 子阶段 | 方法 | 结论 |
|--------|------|------|
| P1 | 时间结构化注入（脉冲/斜坡/随机） | ALL 8 modes L2=0.0%, HW var=0.0 |
| P2 | 竞争性密封核（双核耦合） | 边界增强来自几何效应，非耦合本身 |
| P3 | 主动约束塑造（反馈驱动新层） | 0% secondary sealing |

**关键数据**:
- P1: 时间结构化注入比恒定注入产生 **更少的** 活性信号
- P2: 所有耦合强度 × 方向组合 → enhancement_rate ∈ [0%, 25%]（全来自几何效应）
- P3: 所有约束类型 → 0% secondary sealing

**结论**: 死秩序抵抗时间扰动比抵抗空间扰动更强。密封后的拓扑不变量不受任何形式的注入或约束影响。

## 4. 理论意义：差异论能做什么，不能做什么

### ✅ 差异论精确解释了

1. **从混沌到秩序如何可能**: 当差异密度超过临界值（N0*≈34），bits 通过 binding 机制形成稳定的 L1 结构。这是一个一阶相变，不是渐进过程。

2. **秩序是一次性确立的**: 密封是一次性 cascade 事件，所有 bits 在同一瞬间被分化。这正是差异论"差异是同时性的，不是历时性的"的操作性证明。

3. **结构化是有成本的**: 每次密封淘汰约 40% 的差异空间。淘汰比例是系统普适常数 (0.40×N)，不依赖于系统大小。

4. **死秩序是拓扑必然**: 一旦密封，系统进入拓扑不变量状态。这不是缺陷，而是系统能够保持稳定性(identity)和抵抗扰动(robustness)的机制。

### ❌ 差异论当前形式无法解释

1. **多层级涌现（生命、意识）**: 4 个理论扩展方向、21 个假设、~260 个条件全部失败。差异论只能产生单层秩序。

2. **结构传播**: L1 总是收敛到自己的动力学吸引子，不继承 L0 的结构信息。跨层映射无论采用 cluster/random/feedback 策略，结果完全一致。

3. **"活秩序"（动态持续演化）**: 密封后系统永恒不动。环境耦合、能量流、动态阈值——全部无法使系统在密封后持续演化。解封后也无法重新密封。

### 🔮 对理论扩展的建议

要解释多层级涌现，需要**更根本的理论扩展**，目前的四个方向（开放系统、非局部交互、可变阈值、多层并行）已被实验证明不够。

可能的根本性扩展方向：
1. **非平衡态热力学**: 引入真正的随机性+能量流+熵变，而不是在确定性引擎上叠加环境脉冲
2. **随机微分方程框架**: 将密封机制从确定性阈值改为概率性相变
3. **信息-能量耦合**: 让能量和信息同时影响密封决策，而不是分别施加
4. **连续空间**: 从离散 `{0,1}^N` 扩展到连续状态空间，允许分数级密封

关键洞察：**"死秩序"不是一个需要解决的问题。它是差异论必须接受的边界。**
知道一个理论**不能**做什么，和知道它**能**做什么，同样重要。
