## 0. Phase 13 回顾：死秩序与 L2 涌现的三种可能路径

Phase 13 通过三个实验（exp_155/156/157）建立了核心结论：

### 核心发现：密封产生「死秩序」

> **死秩序（Dead Order）**：系统有序但无生成潜力。L1 密封后，HW 稳定、方差压扁、相关长度不增长、无记忆信号。

### 被排除的三种路径

| 路径 | 实验 | 结果 | 含义 |
|------|------|------|------|
| 被动等待 | exp_156 | ❌ L2 不涌现 | 密封后继续运行 L0 无果 |
| 增加差异密度 | exp_156 H2 | ❌ L2 不涌现 | 48→72 扩张也无效 |
| 空间模板增长 | exp_157 | ❌ 仅 HW+5.6%，无结构化效应 | 向密封结构附近注入差异无效 |

### 未来方向（Phase 13 推论）

Phase 14 需要探索新的 L2 涌现条件：

1. **时间结构化的注入** — 脉冲式差异注入而非恒定注入
2. **竞争性密封核** — 多个密封结构互相竞争而非单一密封
3. **主动约束塑造** — 密封结构主动参与约束动力学而非静态偏向

## 1. Phase 14 核心问题

> **如果密封产生死秩序，什么条件可以打破死秩序、产生新的层级结构？**

核心假设链：

1. **时间结构**可能比空间结构重要 — 差异论的「事件」是时间性的，不是空间性的
2. **竞争**可能比合作重要 — 单个密封核是吸引子，多个核创造张力
3. **反馈**可能比注入重要 — 结构需要「知道」自己在约束景观中的位置并修改它

## 4. P1: 时间结构化注入（时间结构的必要性）

### 4.1 理论动机

Phase 13 的核心发现：**恒定注入无法驱动 L2**。

差异论对「事件」的描述 — «变化以事件的方式断裂推进» — 暗示时间结构可能比空间结构更重要。自然界中，几乎所有层级涌现都涉及时间模式：心跳、节律、脉冲、周期。

恒定注入是「无时间结构」的极限情况 — 每步 +1，均匀、连续、不可区分。如果 L2 本质上是时间性的涌现现象，那么恒定注入恰好排除了 L2 产生的最关键条件。

### 4.2 核心假设

**H14-1（脉冲假设）**：脉冲式差异注入（短时 burst + 长时静默）比恒定注入更有利于 L2 涌现，因为脉冲创造了「差异密度梯度」，使系统可以「感受」到密度的变化并作出响应。

**H14-2（频率假设）**：L2 涌现存在最理想的脉冲频率 — 太快（无静默期）退化为恒定注入；太慢（静默期过长）系统退化回 L0。

**H14-3（幅度假设）**：脉冲幅度（注入的差异量）需要超过某个阈值才能触发 L2 — 小幅度脉冲被密封结构吸收，大幅度脉冲突破密封。

### 4.3 实验设计（exp_158）

**框架**：单子空间，密封后继续运行，修改差异注入模式。

**参数**：
- N = 48（Phase 12 标准配置）
- 恒定期（Phase 12 标准注入）直到密封
- 密封后注入模式切换：

| 模式 | 描述 | 参数扫描 |
|------|------|---------|
| **恒定** | baseline：每步 +1（对照） | — |
| **脉冲** | burst 步数 + 静默步数交替 | burst ∈ [5, 10, 20, 50]，静默 ∈ [10, 20, 50, 100] |
| **斜坡** | 注入强度从 0 线性增至 max 再骤降 | 周期 ∈ [50, 100, 200] |
| **随机** | 注入步数随机（Poisson 过程） | 平均间隔 ∈ [3, 10, 30] |

**运行时参数**：
- 密封后运行步数：1000
- 每次参数组合运行 5 次（不同种子）

**测量指标**：
- L2 形成率（按模式/参数分组）
- HW 重新激活窗口（sealed bits 中是否有重新激活的）
- HW 方差时间序列（方差增长 → 活跃度信号）
- 相关性长度 ξ 的时间演化

### 4.4 预期结果

- **最可能**：某些脉冲模式比恒定注入产生更高的 HW 方差（方差→活跃度信号）
- **中等可能**：存在最优脉冲频率窗口（太快=恒定，太慢=退化）
- **较不可能**：真正产生 L2 层级结构（因为单核密封的单子空间可能本身就不足以支撑 L2）

### 4.5 理论意义

即使 P1 未产生 L2，结果仍然重要：
- 确认「时间结构 ≠ 层级涌现」— 进一步强化死秩序的拓扑性
- 量化脉冲注入 vs 恒定注入的差异（活性差异）
- 为 P2（竞争核）或 P3（反馈）提供 baseline

## 5. P2: 竞争性密封核（结构的张力）

### 5.1 理论动机

自然界中的层级涌现几乎都涉及**多重竞争结构**：
- 细胞分化：多个竞争性的信号中心
- 神经网络：多个竞争性的振荡群
- 经济系统：多个竞争性的市场

差异论的「多重聚簇」概念也暗示：聚簇相遇处的边界可能是新层涌现的「热点」。

Phase 11 的子空间机制本身就是为竞争核设计的。Phase 11 P3 实验中耦合因工程 bug 失效（exp_150/151），修复后（exp_152）尚未在「竞争核」场景下验证。

### 5.2 核心假设

**H14-4（边界假设）**：两个独立密封的子空间通过耦合相互作用时，边界区域（bit 分配的交界附近）会产生比单核系统更高的差异活跃度，可能构成 L2 涌现的基底。

**H14-5（不对称假设）**：大小不对称的子空间耦合（大核 vs 小核）比对称耦合更有利于产生新层 — 小核在大核的「压力」下产生更大的应变形变。

**H14-6（多核假设）**：K>2 个子空间耦合产生的「多体效应」无法还原为两两耦合的叠加 — 多核系统中的层级结构是涌现性的。

### 5.3 实验设计

#### exp_159：双核耦合边界

**参数**：
- k=2 子空间，各自 N=48（均能独立密封）
- 分配模式：block（各自独占一段连续的 bits）
- 耦合强度扫描：[0.0, 0.3, 1.0, 3.0]
- 耦合方向：bidirectional 和 unidirectional 各一组

**测量**：
- 每个子空间密封后，边界 bits（子空间交界 ±5 bits）的翻转活性
- 边界区域的 HW 方差 vs 内部区域
- 边界区域的 L2 有无
- 密封时间的相关性（同步密封 → 竞争弱；异步密封 → 竞争强）

#### exp_160：不对称竞争核

**参数**：
- k=2，大核 N=48，小核 N∈[12, 20, 30, 40]
- 单向耦合，大核→小核
- 耦合强度：[0.0, 0.5, 1.0, 2.0]

**测量**：
- 小核密封率 vs 耦合强度 + 大小组合
- 小核的方向场是否被大核「牵引」
- 小核密封后，边界区域是否有分层信号
- 小核的密封时间分布

#### exp_161：多核系统（可选 — K>2）

**参数**：
- K=3, N 各=32（均接近 N0*~34，依赖耦合实现密封）
- 全连接耦合（complete graph），强度均一
- vs 链式耦合（line，相邻连接）

**测量**：
- 密封顺序（谁先密封，谁后密封）
- 密封率 vs 耦合拓扑
- 是否存在「领导者」子空间（率先密封后驱动他人）
- 全耦合 vs 链式的相变差异

### 5.4 预期结果

- **最可能**：边界区域确实有更高的活性（HW 方差增大），但不足以形成 L2
- **可能**：不对称耦合中，小核被「拖入」大核的结构 — 不是 L2 涌现而是 L1 合并
- **较可能**：K=3 系统中，链式耦合产生梯度结构（两端的子空间先密封，中间的滞后）
- **较不可能**：真正观测到 L2 层级结构出现

### 5.5 理论意义

即使未产生 L2，P2 将为「差异论的多重聚簇」提供第一个定量数据：
- 密封边界的行为（边界是生成的还是消耗的？）
- 不对称耦合的「牵引」效率
- 多核系统的密封顺序统计

## 6. P3: 主动约束塑造（结构与环境的反馈循环）

### 6.1 理论动机

这是三个方向中最「激进」的假设。

Phase 13 发现：密封结构是**被动的** — 它被密封了，但不参与后续动力学。密封不是「地基」（支持上层建筑），而是「障碍」（系统被它锁死）。

但如果密封结构本身能主动修改约束景观呢？不是「密封后消失」，而是「密封后成为了约束制定者」。

差异论（Ch7）讨论制度时指出：制度（已经固化的差异）不是被动的，而是主动地塑造后续差异产生的条件。这个「制度反馈」可能是层级涌现的必要条件。

### 6.2 核心假设

**H14-7（反馈假设）**：如果密封结构能修改 `coupling_bias` 或 `binding_strength`（即密封后的 bits 主动影响约束景观），则系统更可能产生 L2 涌现。

**H14-8（密度依赖假设）**：密封结构对约束的塑造强度与密封规模（被密封的 bits 比例 ~0.40×N）成正比 — 密封规模越大，约束越强。

### 6.3 实验设计（exp_162）

**框架**：单子空间或双核系统 **+** 密封后主动反馈回路。

**反馈机制**：

```python
# 伪代码：密封结构主动修改约束
def active_feedback_callback(step, state, constraints, sealed_mask):
    # sealed_mask: bool tensor, True for bits that are sealed
    
    # 方案 A: 密封 bits 发射「约束信号」
    # 非密封 bits 接收到与密封结构距离成正比的 bias
    active_bias = torch.zeros(N)
    for i in range(N):
        if not sealed_mask[i]:
            # 到最近 sealed bit 的距离
            dist_to_sealed = min(|i - j| for j where sealed_mask[j])
            active_bias[i] = feedback_strength * exp(-dist_to_sealed / decay_length)
    
    constraints.coupling_bias += active_bias
```

**参数扫描**：
- feedback_strength ∈ [0.0, 0.1, 0.3, 1.0, 3.0]
- decay_length ∈ [1, 3, 10, N]（覆盖局部到全局）
- 密封结构：单核（N=48）vs 双核（k=2, 各 N=48）

**测量**：
- 反馈开启后，非密封 bits 的翻转活性变化
- 是否有新的密封事件（二次密封）
- 方向场的空间模式（是否存在梯度）
- 相关性长度的时间演化

### 6.4 预期结果

- **最可能**：强反馈 + 短程作用产生边界聚集 — 非密封 bits 向密封结构靠拢
- **中等可能**：产生二次密封事件 — 但不是 L2 层级，而是 L1 的扩张
- **较可能**：反馈 + 双核可能产生「拉伸」效应 — 两核之间的 bits 受到双向约束，形成有结构的中间区域

### 6.5 理论意义

这是三个方向中最具理论深度的：
- 如果反馈确实能产生新结构，则验证了差异论「制度塑造差异」的核心命题
- 如果反馈也失败，则死秩序可能是一种**更深层的拓扑不变性**，需要更根本的突破

## 8. 理论预测汇总

| 实验 | 最可能结果 | 中等可能结果 | 较不可能结果 |
|------|-----------|-------------|-------------|
| P1 脉冲 | 某些脉冲模式 > 恒定（方差↑），但无 L2 | 存在最优脉冲频率 | 产生真正 L2 |
| P2 双核 | 边界活性↑（方差↑），但无 L2 | 不对称耦合的牵引效应 | 边界产生 L2 |
| P2 多核 | 链式耦合产生梯度密封顺序 | 全耦合加快密封同步性 | 多体效应涌现 L2 |
| P3 反馈 | 短程强反馈产生边界聚集 | 产生二次密封事件 | 反馈+双核产生中间结构 |

**整体预期**：Phase 14 的三种路径中，任何一种单独可能都不足以产生 L2。真正的 L2 涌现可能需要 **组合条件** — 例如时间结构化注入 + 竞争核 + 反馈的联合效应。

如果 Phase 14 全部三条路径都失败，则需要回到理论层面：
- 死秩序可能是差异论世界的**内在拓扑约束**，而非参数问题
- L2 可能需要**三代差异**（L0→L1→L2）才能涌现，而非两代
- 或者需要引入新机制（非线性 binding、场论效应）
