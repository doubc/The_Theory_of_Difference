## 0. 为什么 Phase 11 现在开始

Phase 10 回答了"模拟机告诉差异论什么"，Phase 11 要回答：

> **一个统一的离散差异空间可以涌现层级结构。如果我们将这个空间分解为多个耦合的子空间——每个子空间由不同的约束集支配——会出现什么？**

这是模拟机工程从"统一空间验证"走向"物理多样性"的第一步。

---

## 1. 核心问题

**在一个统一的离散空间中，所有比特受到相同的九公理约束。如果将空间分裂为多个耦合的子空间，每个子空间受到同一组九公理但不同参数化（不同绑定强度、不同方向性偏好、不同守恒残差阈值）——会涌现出什么样的子空间特异性？**

这直接对应差异论的理论问题：
- 不同的"物理力"（引力/电磁/强/弱）是差异空间中的不同约束参数化吗？
- 子空间之间的耦合接口——差异论术语中的"跨差异流"——如何定义？

### 2.2 P1: 子空间基础设施（Subspace Infrastructure）

设计子空间分解的核心数据结构：

```
SubspaceField(bit_indices, rules, coupling_ports)
  ├── bit_indices: 属于该子空间的比特索引集（子集 [0, N0-1]）
  ├── rules: 该子空间特有的参数化（binding_multiplier, direction_bias, conservation_tightness）
  └── coupling_ports: 与该子空间耦合的其它子空间的接口（通信信道）
```

**关键设计决策**：

1. **子空间分配策略**：
   - 静态分区：N0 个比特预先划分为 k 个等尺寸子空间（如 N0=30, k=3, 每子空间 10 比特）
   - 交错分配：比特交替分配给不同子空间（如 bit_id % k 决定子空间归属）
   - 随机分区：每个比特随机分配到 k 个子空间之一
   - 优先级：先做静态分区（最简单，最容易解释结果），再做随机分区（验证稳健性）

2. **子空间特异性**：
   - 绑定强度乘子（binding_multiplier ∈ [0.2, 5.0]）
   - 方向偏好（direction_bias ∈ [0, 1]，控制源→汇方向性的强弱）
   - 守恒紧度（conservation_tightness ∈ [0.01, 0.1]，控制 A5 残差容忍度）
   - 升维阈值乘子（seal_threshold_multiplier ∈ [0.5, 2.0]）

3. **跨子空间耦合**：
   - 耦合强度端口（coupling_strength ∈ [0.0, 1.0]）
   - 通信方向（单向 / 双向 / 全连通）
   - 通信内容（绑定强度传递 / 偏置场传递 / 封口状态通知）

## 3. 实验计划

### 3.1 P2: 子空间独立性验证（Isolation Verification）

**目的**：验证子空间在物理隔离但共享空间中能否独立演化。

**实验 exp_148**：静态三分区，各子空间零耦合

| 参数 | 值 |
|------|-----|
| N0 | 30（适中，接近已知相变点） |
| k（子空间数） | 3 |
| 耦合强度 | 0.0（完全隔离） |
| 分配策略 | 静态分区（等尺寸） |
| 公理 | 九公理全开，各子空间相同参数 |
| 种子 | 16 |
| 步数 | 2000 |

**预期**：
- 各子空间独立涌现 L0-L3，如同规模为 N0=10 的独立世界
- 相变临界点在子空间层面重新校准（预期 N0_sub ≈ 10 → 无 L1 形成，验证相变不可缩放）
- 子空间之间的相关性 = 0（零耦合→零传递）

**成功标准**：✅ 零耦合下子空间独立演化，无串扰

**exp_149**：随机分区 + 零耦合，验证 robust

| 参数 | 值 |
|------|-----|
| N0 | 60 |
| k | 4 |
| 耦合强度 | 0.0 |
| 分配策略 | 随机分区 |
| 种子 | 16 |
| 步数 | 2000 |

### 3.2 P3: 子空间耦合探索（Coupling Exploration）

**目的**：引入子空间之间不同程度的耦合，观察跨子空间信息传递效应。

**exp_150**：静态三分区，对称耦合扫描

| 参数 | 值 |
|------|-----|
| N0 | 60 |
| k | 3 |
| 分配策略 | 静态分区 |
| 耦合强度 | 0.0, 0.1, 0.3, 0.5, 0.7, 1.0（6 个水平） |
| 种子/水平 | 8 |
| 步数 | 3000 |

**测量指标**：
- 子空间内：H1-H8 通过率、L1 形成率、绑定强度、封口时序
- 跨子空间：相关性（Pearson r）、互信息（MI）、因果流方向（Granger causality proxy）
- 涌现指标：跨子空间组织形成（binding_matrix 跨块强度）

**预期**：
- 低耦合（<0.3）：子空间近乎独立，跨块绑定强度 ≈ 0
- 中耦合（0.3-0.7）：出现跨子空间结构传递，但各子空间保留特征
- 高耦合（>0.7）：子空间边界模糊，系统趋近于 N0=60 的统一空间

**成功标准**：✅ 确定耦合强度的三个相区（弱/中/强），描述各相区的涌现特征

**exp_151**：非对称耦合（单向主-从结构）

| 参数 | 值 |
|------|-----|
| N0 | 60 |
| k | 2（主子空间 40, 从子空间 20） |
| 耦合强度 | 主→从: 0.5, 从→主: 0.0 |
| 种子 | 16 |
| 步数 | 3000 |

**目的**：模拟"高层级对低层级的单向约束"——这是 Phase 8 发现的 L1→L0 被动约束的子空间类比。

### 3.3 P4: 子空间特异性与物理映射（Parameter Specialization）

**目的**：为不同子空间赋予不同的参数化，观察子空间特异性涌现。

**exp_152**：绑定强度分化

| 子空间 | 绑定强度乘子 | 预期物理类比 |
|--------|-------------|------------|
| S0（引力） | 0.5（弱绑定） | 引力——长程但弱的作用 |
| S1（电磁） | 1.0（标准） | 电磁——中程、标准强度 |
| S2（强力） | 3.0（强绑定） | 强力——短程但极强 |
| k | 3，N0=60，静态分区 |
| 耦合强度 | 0.3（跨子空间弱耦合） | 
| 种子 | 16 |
| 步数 | 5000 |

**预期**：
- S2（强绑定）：L1 形成更快、绑定更紧密、封口更早
- S0（弱绑定）：L1 形成更慢或不形成、绑定松散
- 跨子空间：强绑定子空间可能"牵引"弱绑定子空间的结构形成（如果耦合足够强）

**成功标准**：✅ 不同参数化导致可区分的子空间涌现特征，且特征与物理类比一致

**exp_153**：方向偏好分化

| 子空间 | 方向偏好 | 预期物理类比 |
|--------|---------|------------|
| S0 | 强单向（0.9） | 时间之箭——强方向性 |
| S1 | 弱偏好（0.6） | 可逆相互作用 |
| S2 | 零偏好（0.5）| 随机波动——无时间方向 |
| k | 3，N0=60，静态分区 |

### 3.4 P5: 涌现度量与理论回写

**目的**：将子空间分解的实验发现回写到差异论理论框架。

**核心问题**：
- 不同参数化的子空间会产生"不同种类的差异"吗？
- 跨子空间耦合等价于差异论中的"二阶差异"吗？
- 子空间特异性是否自然对应物理力的四种类型？

**产出**：
- `theory_synthesis_v2.md`：在 v1 基础上增加子空间分解章节
- `docs/decomposition_to_physics.md`：子空间实验结果 → 物理类比的理论映射
