## 一、问题陈述

exp_114（Phase 5 Track B1）发现：
- L1（INSTITUTIONAL）与 L2（CIVILIZATION）的 NSI 相关系数 = 0.976 ± 0.003
- L0（MINI）与 L1/L2 的相关系数仅为 0.53–0.57（中等耦合）
- 实际传导路径：L0/L2 同时激活（step 9）→ L1 延迟激活（~12步）

这与理论预期不符：如果"层级"机制正常工作，L1 和 L2 应该有独立的叙事轨迹，而非近乎完美的同步。

## 二、理论分析：九机制中的"层级"

### 2.1 理论定义

> "层级是差异聚簇进入上下、内外、整体与局部之间的组织分层。层级不是价值高低，也不等于权力等级，而是结构生成中的组织分层。"（§2.2）

关键要点：
1. **层级发生在聚簇之后**——只有差异已经聚簇，层级才能对这些聚簇进行组织分层
2. **层级不是预先存在的等级**——它是生成过程中的组织结果
3. **层级产生位置差**——上下、内外、整体与局部之间的差异

### 2.2 当前模拟机的架构问题

当前的 CSC（CrossScaleCoupling）机制同时连接 L0→L1 和 L0→L2：
- CSC 的 `update_mini_to_institutional()` 将 MINI 层聚簇提升到 INSTITUTIONAL 层
- CSC 的 `update_mini_to_civilization()` 将 MINI 层聚簇直接提升到 CIVILIZATION 层
- **两个提升路径使用相同的源（L0 聚簇）和相同的触发逻辑（ODI > threshold）**

这导致 L1 和 L2 的叙事轨迹完全同步——它们本质上是对同一批 L0 聚簇的两次"提升"，而非独立的层级组织。

### 2.3 理论解释：为什么 L1-L2 耦合不是 bug

从差异论角度看，这个结果有理论意义：

> "聚簇使差异形成局部，层级使局部进入组织"（§2.2）

在当前模拟中：
- **L0 是真正的"聚簇"层**——MINI 聚簇是差异的局部高密度区域
- **L1 和 L2 都试图对 L0 聚簇进行"组织分层"**——但它们使用相同的组织原则（CSC）
- **结果：L1 和 L2 没有形成真正的"层级"，它们只是同一聚簇的两种不同粒度的映射**

理论结论：**L1 和 L2 的耦合不是架构缺陷，而是"层级"机制尚未真正分化的表现**。在差异论中，真正的层级应该是：
- L1 组织 L0 聚簇 → 形成制度性边界和约束
- L2 组织 L1 制度 → 形成文明级叙事自我
- **L2 不应该直接读取 L0 聚簇**——它应该读取 L1 的制度输出

## 三、对 Track B2 的理论指导

### 3.1 修正 CSC 的传导路径

当前：`L0 → L1` 和 `L0 → L2`（并行）

理论正确的：`L0 → L1 → L2`（串行）

这意味着：
1. CSC 的 `update_mini_to_civilization()` 应该改为读取 **L1 的制度状态**，而非直接读取 L0 聚簇
2. L2 的叙事自我应该是对 L1 制度的"再组织"，而非对 L0 聚簇的"再提升"

### 3.2 Track B2 的实验设计

**假设 H30**：修正 CSC 传导路径（L0→L1→L2）后，L1↔L2 相关系数下降至 < 0.7

**假设 H31**：L0→L1 延迟 + L1→L2 延迟 > L0→L2 延迟（真正的层级传导）

**实验方案**：
1. 修改 `engine/cross_scale_coupling.py`：
   - 保留 `update_mini_to_institutional()`（L0→L1）
   - 修改 `update_mini_to_civilization()` 为 `update_institutional_to_civilization()`（L1→L2）
   - L2 的 NSI 计算应基于 L1 的制度稳定性，而非 L0 的 ODI
2. 运行 exp_115：8 seeds × 2000 steps，测量 L1↔L2 相关系数
3. 如果 H30 通过（r < 0.7），则理论预测得到验证

### 3.3 深层理论意义

如果 Track B2 成功解耦 L1 和 L2，则证明：

> "层级不是权力意义上的高低先行，而是结构生成中的组织分层。"（§2.2）

真正的层级应该是**不可逆的生成链条**：L0（聚簇）→ L1（制度组织）→ L2（文明叙事）。每一层都对前一层进行"再组织"，而非平行地读取同一源。

## 四、与"对象是差异关系的凝聚态"的连接

> "对象是差异关系在一定条件下形成的凝聚态。它之所以看起来稳定，是因为差异已经经过边界、重复、循环和锁定获得了相对持存。"（§1.2）

L1 和 L2 当前是同一"凝聚态"的两种表现形式（因为它们都直接从 L0 聚簇提升）。

修正后：
- L1 是 L0 差异关系的凝聚态（制度封装）
- L2 是 L1 差异关系的凝聚态（文明叙事）
- **每一层都是对前一层差异关系的再凝聚**，而非对原始差异的直接凝聚

这解释了为什么 L2 的重新封装（Track B2 的解封实验）会产生"新的"叙事自我——它不是复制原来的凝聚态，而是**在新的条件下重新组织 L1 的差异关系**。
