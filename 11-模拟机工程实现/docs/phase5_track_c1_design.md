## 一、总目标

Track B 验证了系统在最优规模（N0=48-72）下的多层级动力学。Track C 转向另一个方向：**当系统被资源约束时，核心功能能否维持？**

Track C1 聚焦于**空间资源约束**——缩小差异空间的比特数量 N0，找到系统维持叙事涌现的最小可行规模。

### 1.1 理论问题

差异论 §1.1-§1.3 论证了"差异先行"——任何结构化叙事都需要一个足够丰富的差异空间。如果差异空间太小：

1. **聚类无法形成**：N0 过小导致差异向量缺乏区分度，同质聚类（homogeneous clustering）无法产生
2. **跨尺度耦合失效**：CSC 需要至少一定数量的差异才能建立 Top-Down 约束
3. **叙事自我无法涌现**：NSE 需要足够的历史转折点构成叙事时间线
4. **CIV 事件消失或淹没**：文明事件需要可分辨的差异空间

### 1.2 工程问题

HierarchicalEvolver 的 N0 参数控制了 L0 底象层的比特数。从 Phase 4 的实验已知：

| N0 | Phase 4 结果 |
|----|-------------|
| 96 | H1-H8 全部通过，但过度聚类 |
| 72 | 最优规模，全部通过 |
| 48 | 全部通过，CIV 略低 |
| 32 | 未测试 — Track C1 探索点 |
| 24 | 未测试 — Track C1 探索点 |
| 16 | 未测试 — Track C1 探索点 |
| 8 | 可能不可行 — 工程极限 |

## 三、失败模式分析

### 3.1 H5 (CIV 范围) 失效在 N0=16

**原因**: CIV 事件需要跨尺度差异驱动。当 N0=16 时，可能的差异组合数量急剧下降。

量化: 假设每个比特独立变化，CIV 率与 N0 的相关性为 O(N0²)（两两组合）。N0 从 48 降到 16 意味着 CIV 率下降 (16/48)² ≈ 11%。

### 3.2 H8 (TopDown 激活) 失效在 N0=16

**原因**: TopDown 需要从 CIV 级别的叙事层向下传导约束。CIV 事件太少（H5 失效）意味着 TopDown 没有足够的触发事件。

### 3.3 H7 (CSCI 稳定) 失效在 N0=16

**原因**: CSCI 测量跨尺度相干的稳定性。极小的差异空间可能导致跨尺度相干退化为随机噪声。

## 四、与 Track B 的关系

Track C1 使用**真实 HierarchicalEvolver**（与 Phase 4 P2 Track B 一致），不是 Track B 的 SimulatedLayer 方法。这是必要的，因为：

1. 资源约束测试需要真实的 CSC+NSE 组件行为
2. CSC 的跨尺度耦合在模拟层中无法准确重现
3. NSE 的叙事自我涌现依赖真实的 NSE 组件
