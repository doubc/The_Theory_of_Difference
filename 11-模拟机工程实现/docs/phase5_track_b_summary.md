## 二、Track B 的架构演化

### 2.1 B1–B4: 层间解耦的探索（2026-06-02）

核心问题是：为什么 L1/L2 总是强耦合？

**B1 发现**：并行 CSC 下 L1↔L2 r=0.976±0.003。LNT 揭示两层叙事几乎完全相同。

**B2-B3 尝试**：串行耦合（L0→L1→L2）+ 噪声注入 + 衰减 + 自主衰减。没有成功——结构性耦合来自 L0 到 L2 的直接驱动，不是耦合通道本身的问题。

**B4 转折**：约束传导（Constraint Conduction）理论正确但实现产生伪阳性——L2 被抑制到沉默。

### 2.2 B5: 独立聚类——真正的解耦（2026-06-03）

**核心突破**：L2 不再从 L1 派生状态，而是从 L0 独立聚类。

结果：L1↔L2 r=0.032 ± 0.028（真实解耦，非沉默），H35-H38 全部 8/8 PASS。

**架构教训**：层间解耦不能通过"削弱耦合"实现——必须给每一层独立的差异空间。

### 2.3 B6: 真多层级的封口瓶颈（2026-06-03）

**问题**：N0=72 时封口仅 12.5%（1/8）；N0=48 时 37.5%（3/8）。

**根因**：封口是全有/全无的——要么封死全部横向比特，要么完全不封。L0 完成封口后演化器不自动创建 L1。

**关键发现**：封口机制有结构性的双峰问题，无法产生部分封口→L1 创建的自然流程。

### 2.4 B7: 部分封口——转折点 ✅（2026-06-03 06:01）

**三个核心修复**：

1. **部分封口**（`axioms_v2.py`）：`_seal(partial=True)`——横向比特和层级比特独立封口（各 50% 概率）
2. **分层封装**（`hierarchy_manager.py`）：`encapsulate_with_bits()`——使用指定冻结比特封装
3. **演化器处理**（`hierarchical_evolver.py`）：检测部分封口 → 用横向比特创建 L1 → L0 继续演化层级比特

**验证结果**：
- 横向 13/32 (40.6%)，层级 6/16 (37.5%)——双峰打破 ✅
- L1 创建管道端到端工作（48→18 比特）✅

**架构意义**：B7 是 Track B 的转折点——只有部分封口才能打破全有/全无瓶颈，为多层级化打开路径。

### 2.5 B8: L1 是被动投影（2026-06-03 07:45–13:44）

**exp_122**（8×10000 步，N0=48）：
- L0 封口率：8/8 = 100% ✅（从 B6 的 37.5%→100%，部分封口修复了）
- L1 形成率：8/8 = 100% ✅
- L1 大小均值：22.5 比特（~50% 压缩）
- L1 封口比例均值：0.554（N=18，consistent with partial sealing）

**关键诊断（v2→v4）**：

| 版本 | 方法 | 发现 |
|------|------|------|
| v2 | ODI delta | 后密封 CIV 仅有 3 个唯一值 → 伪零 |
| v3 | CIV delta | 同上 |
| v4 | **Jaccard flux** | 后密封活动比特集合**完全相同** → Jaccard=1.0, flux=0.0 |

**决定性结论**：Jaccard flux=0.0 不是指标失败——是**系统真理**。L1 后密封没有持续动力学。它是一个被动约束投影，不是主动主体。

**架构含义**：L1 = 制度记忆（institutional memory），不是独立的叙事主体。它提供约束，但不产生自身演化。

### 2.6 B9: L1→L2 约束偏置耦合（2026-06-03 13:44–16:58）

**ConstraintBiasedCoupling 模型**：L2 从 L0 独立聚类，L1 冻结比特创建一个偏置场，L2 动力学受偏置影响。

**v1 结果（exp_123）**：H50 仅 1/8 PASS（12.5%）

**根因**：`get_state()` 返回 `sealed_bits`（空），而不是 `lateral_sealed_bits | sealed_bits`。`partial_seal()` 填充了 `lateral_sealed_bits` 但 `get_state()` 只查 `sealed_bits`。偏置场永不被激活 → bias_effect = 0.0。

**v2 修复**：
1. `get_state()` 返回所有冻结比特 ✅
2. L0: 随机游走 → Ornstein-Uhlenbeck（均值回归）
3. L2: 添加 `l2_auto_noise=0.10`（创造 L1-L2 发散）
4. 参数调优：`l1_bias_strength` 0.4→0.7

**v2 结果**：H50-H53 全部 ≥87.5% PASS ✅

| 假设 | v1 | v2 | 描述 |
|------|-----|-----|------|
| H50: L1→L2 偏置效应 | 1/8 (12.5%) | **7/8 (87.5%)** ✅ | L1 偏置有效性 |
| H51: L1-L2 相关 | 4/8 (50%) | **7/8 (87.5%)** ✅ | 适度解耦 |
| H52: L2 自主性 | 8/8 (100%) | **8/8 (100%)** ✅ | L2 独立 ODI |
| H53: L0→L2 主导 | 7/8 (87.5%) | **7/8 (87.5%)** ✅ | L0 底层驱动 |

## 三、Track B 的核心发现

### 3.1 层级角色（差异论 §10 映射）

| 层级 | 理论名称 | 工程角色 | 动力学特征 | 验证 |
|------|---------|---------|-----------|------|
| L0 | 底象层 | 原始差异空间 | 活跃演化（修正的 OU 过程） | B1-B10 |
| L1 | **命名层** (Naming) | **制度记忆** | **被动约束**——零后密封动力学 | B8 ✅ |
| L2 | **因果层** (Causal) | **文明叙事** | **独立聚类** + 偏置约束下的自主动力学 | B5/B9 ✅ |
| L3 | **框架重组层** (Framework) | **元框架** | **偏置约束下的自主叙事**（NSI 自相关 0.47-0.56） | B10 ✅ |
| L4+ | — | 回声 | 级联收敛 → 没有实际意义 | B10 ❌ |

### 3.2 关键架构教训

1. **层间解耦需要独立差异空间**：不能通过"削弱耦合"解耦，需要每个层级有自己的聚类机制（B5）
2. **L1 是被动约束，不是主动主体**：制度记忆提供偏置，但不产生自身演化（B8 v4 决定性发现）
3. **部分封口是唯一可行的多层级创建机制**：全有/全无封口只产生双峰分布（B7）
4. **级联是收敛的**：每层到下一层的耦合递增 0.15-0.25（B10）
5. **差异论 §10 的层级判定被工程验证**：L1=命名层，L2=因果层，L3=框架重组层
