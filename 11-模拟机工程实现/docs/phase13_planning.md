## 0. Phase 12 回顾：已完成的基础

Phase 12 通过两个实验（exp_153, exp_154）建立了两个决定性结论：

### 核心发现 1：密封是一次性相变（一阶相变）
- 48/49 次密封运行中，密封发生在**单次 cascade 事件**中，一次性冻结所有 bits
- 平均 cascade 事件数 = 1.0（标准差 = 0）
- 密封不是"逐个 bit 逐渐冻结"或"反复冻融循环"，而是系统在临界密度的瞬间发生一次性分化

### 核心发现 2：普适标度律 cascade_size = 0.40 × N
- N 值扫描（36-96）确认该比例在全部范围内稳定（0.38-0.41）
- 密封比例不随系统规模变化——这是差异论世界的**第一个普适量纲关系**
- 相变边界修正：N0* ≈ 34（从 Phase 9 的 30.5 上调）

### 理论意义
- **共同反差的操作性定义**：当系统激活密度达到临界值（~73%），一次性分化决策将 N 维空间划分为"保持集"（~40%）和"冻结集"（~60%）
- **一阶相变是普适性质**，不是特定 N 或特定配置的产物

## 1. Phase 13 的核心问题

Phase 12 确认了"密封如何发生"（一阶相变、单次级联、普适标度律）。

Phase 13 要问的是：**密封之后发生了什么？**

核心问题链：
1. 密封后，被冻结的差异结构是否维持稳定？
2. 被压缩的差异（sealed bits）是否会重新激活？
3. L1 结构能否作为"基础层"支撑 L2 涌现？
4. 在多大差异密度下，多层结构（L1→L2→...）可以自发形成？

这个转向对应差异论 V1.6 的一个核心判断（来自阅读笔记 2026-06-07）：
> "被压缩的差异不会消失，只是从活跃态变成了背景态。"

Phase 12 研究了"压缩/密封"的瞬间；Phase 13 要研究密封后的**长效动力学**。

## 3. P1: 密封后持久性动力学（高优先级）

### 3.1 动机

当前模拟机在密封发生后即停止演化（或进入"完成"状态）。但从差异论的角度，密封不是终点，而是**结构稳定化的起点**。

关键问题：
- 如果继续注入差异（A1 持续 +1），密封后的系统会如何？
- sealed bits 是永久冻结，还是在新的差异压力下可以解封？
- 被冻结构是否因为"被压缩的差异"而产生内部张力？

### 3.2 理论基础

差异论 V1.6 Ch3（"事件从最撑不住的地方先发生"）和阅读笔记（2026-06-07, Ch3 解读）：

> "最近稳态"往往不是最优的。系统在 partial_sealing 之后会继续演化，还是永久卡在第一个局部最优？

Phase 12 确认了密封是**单次事件**。问题在于：这个单次事件是否产生了**锁定后的稳态**？

### 3.3 实验设计（exp_155）

**假设 H155-1**：密封后继续运行 L0（保持 A1 持续 +1），sealed bits 不会重新激活——密封是单向不可逆的。

**假设 H155-2**：密封后，若增加差异密度（N0 大幅提升，远超相变边界），sealed bits 可能解封并形成新的 L1 结构——差异被"重新激活"。

**参数**：
- N = 48（Phase 12 标准配置）
- 密封后继续运行 2000 步（密封前 + 密封后继续）
- 条件 A：密封后差异注入强度不变
- 条件 B：密封后差异注入强度提高 2x/5x/10x
- 条件 C：密封后进行 N 值扩张（从 48 → 96，增加额外活跃 bits）

**判定标准**：
- sealed bits 数量在密封后是否变化（增加/减少/不变）
- L1 形成率在密封后是否维持
- 是否有"二次相变"（新的密封事件）

### 3.4 理论预测

- H155-1 很可能成立：exp_154 的 100% 单次 cascade 说明密封决策是决定性的
- H155-2 可能成立也可能不成立：如果密封是"锁住差异"而非"消灭差异"，则额外压力可能激活
- 这对差异论的意义：如果密封确认为不可逆，则"差异压缩"是一次性事件；如果可逆，则存在"差异循环"

## 4. P2: 密封后 L2 涌现条件（高优先级）

### 4.1 动机

Phase 1-9 已经验证了 L0→L1→L2 的层级涌现（从离散 bit 到结构层到更高层）。
Phase 12 提供了 L0→L1 相变的精确图景。

P2 的核心问题：**L1 稳定后，什么条件下 L2 会涌现？**

### 4.2 理论基础

Phase 8 发现：L1 是"被动约束投影"（Jaccard flux=0.0），不是自主主体。
这意味着 L1 本身不能"驱动"L2 涌现——L2 可能需要额外的差异密度或外部条件。

Phase 5 B10 发现级联耦合递增：L0→L1 r~0.50, L1→L2 r~0.75。
这个模式暗示：越高层的涌现需要的差异关联性越强。

### 4.3 实验设计（exp_156）

**假设 H156-1**：L1 稳定密封后，仅靠继续运行 L0（相同参数），L2 不会自然涌现——L2 需要额外的差异密度或外部触发。

**假设 H156-2**：在密封后增加差异密度（扩大 N0），可以在已有 L1 结构基础上驱动 L2 涌现。

**假设 H156-3**：L2 涌现的临界差异密度高于 L0→L1 的临界密度（N0*_L2 > N0*_L1）。

**参数**：
- N = 48（L1 基础）
- 条件 A：密封后继续运行，不做任何调整
- 条件 B：密封后增加 N（新 bits 加入），梯度 [48→72, 48→96, 48→144]
- 条件 C：密封后增加绑定强度 binding_multiplier 梯度 [1.0, 1.5, 2.0, 5.0]

**判定标准**：
- L2 形成率（已有 L1 密封后，L2 是否出现）
- L2 形成时间与密封时间差
- L2 的 Jaccard flux vs L1 的 Jaccard flux（L2 是否也是"被动约束"？）

## 5. P3: 多空间耦合聚簇（中优先级）

### 5.1 动机

Phase 11 实现了子空间分解与耦合，Phase 12 研究了单空间聚簇。
P3 结合两者：**多子空间各自密封后，它们如何相互作用？**

### 5.2 理论基础

差异论 Ch2 关于"多重聚簇"（阅读笔记 2026-06-07）：
> "两个聚簇相遇时会发生什么？合并？排斥？寄生？"

Phase 11 的耦合机制（P4 修复后：coupling_bias 而非 binding_strength）提供了实验工具。

### 5.3 实验设计（exp_157）

**假设 H157-1**：两个已密封的子空间（均形成 L1）通过耦合会产生合并效应——子空间边界模糊。

**假设 H157-2**：一个已密封的大子空间通过耦合可以"主导"一个小型未密封子空间，驱动后者密封。

**假设 H157-3**：两个未密封但充分大的子空间（N ≥ 36）通过强耦合可以"协同"密封——耦合增强整体的聚簇概率。

**参数**：
- k=2 子空间，均 N=32（接近但低于 Phase 12 的 N0*~34）
- 耦合强度梯度 [0.0, 0.5, 1.0, 2.0, 5.0]
- 比较：隔离 vs 弱耦合 vs 强耦合

**判定标准**：
- 密封率 vs 耦合强度的函数关系
- 子空间密封同步性（同时密封，或一先一后）
- 密封后子空间间差异（合并还是保持独立）
