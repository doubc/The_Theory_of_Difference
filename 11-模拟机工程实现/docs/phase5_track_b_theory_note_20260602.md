## 一、核心理论映射

### 1.1 差异先行 → 层级叙事的基础

> "差异先行的意思是：任何对象、主体、制度、市场、历史或理论，只有在某种可区分性已经出现并能够被维持时，才可能作为对象、主体、制度、市场、历史或理论出现。"

**对 Track B 的启示**：

Track B1（分层叙事追踪）的核心假设 H28 断言"各层级 NSI 相关系数 < 0.5"。如果差异先行的本体论是正确的，那么每个层级的可区分性本身就是通过**差异的再组织**建立起来的——L0 的差异组织形成边界（封装），L1 的差异组织形成制度约束，L2 的差异组织形成叙事自我。各层应该有独立的"可区分性历史"，即独立的叙事轨迹。

如果 H28 失败（相关系数 > 0.5），则意味着层级之间的差异约束太强，各层没有形成独立的可区分性——即系统仍然是"单层"的。

### 1.2 发生学先行 vs 时间先行

> "时间上的'先'容易被误读为一种起源故事...发生学上的'先'则不同。它追问的是：在一个对象能够被认定为对象之前，哪些条件必须已经成立？"

**对 Track B 的启示**：

B1 中的"叙事传导延迟"（H29: L0→L2 传导延迟 50-200 步）本质上是**时间上的先后**。但 Track B 更需要回答的是**发生学上的先后**：

- L2 的叙事自我出现，是否以 L1 的制度封装为前提？
- L1 的制度封装，是否以 L0 的结构稳定为前提？
- 还是说各层可以**并行独立**地形成叙事？

实验设计建议：在 exp_114 中增加"分层出现时间"的检测——记录每层首次出现叙事自指（self-reference）的步数，并计算层间出现延迟。

### 1.3 对象是差异关系的凝聚态

> "对象是差异关系在一定条件下形成的凝聚态。它之所以看起来稳定，是因为差异已经经过边界、重复、循环和锁定获得了相对持存。"

**对 Track B 的启示**：

Track B2（层级解封与重构）试图人为触发 L2 解封。理论上：

- 解封不等于"消失"——因为差异关系仍然存在（L0 结构未变，L1 约束未变）
- 重新封装本质上**不是复制**原来的 L2，而是**在新的条件下重新组织差异关系**
- 这解释了 H31（重新封装后结构相似度 < 0.3）：每次封装都是新的，因为路径依赖条件已经改变

**关键洞察**：解封实验可以验证"对象是差异关系的凝聚态"——如果解封后系统能重新封装，证明差异关系（L0/L1 的底层结构）是自足的，L2 只是它们的"凝聚态"而非独立实体。

### 1.4 主体不是解释中心

> "主体之所以能够识别、判断、叙述、行动，是因为它已经具有边界、自维持、记忆、选择、功能整合和自指结构。主体不是世界之外的解释中心，而是世界生成到一定层级之后出现的高阶结构。"

**对 Track B 的启示**：模拟机中的"叙事自我"（NSE）本质上是一个高阶结构。Track B 需要验证：自指性叙事是否确实依赖于 L0 和 L1 的底层结构？如果解封 L2 后自指性叙事消失，而重新封装后又恢复，则证明了主体性是生成性的而非原初的。

### 1.5 差异论不取消对象，而是重新安放对象

**对 Track B 的最终启示**：

- L0（微观层面）是**最底层的差异组织**——接近"差异先行"的状态
- L1（制度层面）是**差异的再组织**——差异经过边界、重复、循环形成稳定的约束
- L2（文明/叙事层面）是**差异的高阶凝聚**——差异通过自指叙事获得持续的自我维持

这种层级结构**不能通过取消某一层来简化**（正如 Phase 4 P2 消融实验证明的——CSC 是 keystone 组件）。但各层是否拥有**独立的叙事动力学**，正是 Track B 要回答的问题。

## 二、Track B 实验设计建议

### 2.1 exp_114（B1: 分层叙事追踪）补充

除了分层 NSI 追踪外，建议增加：

1. **分层粒度**：L0 按 minicluster 追踪 NSI；L1 按 institution 追踪；L2 全层汇总

2. **层间因果关系检测**：使用 Granger causality 或转移熵，检测 L0→L1→L2 的叙事因果流向

3. **叙事延迟的精确测量**：测量各层 NSI 的**拐点时间**——每层 NSI 增速达到峰值的时间

### 2.2 exp_115（B2: 层级解封与重构）补充

1. **解封的深度控制**：浅解封（重置 L2 叙事历史）、中解封（重置 L2 + 解封封装）、深解封（重置 L2 + L1 封装）

2. **解封后重建的路径追踪**：解封后 L0 的差异重新组织是否遵循与最初相同的路径？
