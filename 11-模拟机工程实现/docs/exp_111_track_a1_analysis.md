> **日期**: 2026-06-02

> **架构**: CSC+NSE (simplified, no AMC/ILP)
> **总运行数**: 16 (4 seeds × 4 perturbation types)
> **假设**: H21, H22, H23

---

## 一、实验设计

### 1.1 扰动类型

| 类型 | 名称 | 描述 | 预期 |
|------|------|------|------|
| 0 | Control | 无扰动（基线） | 系统正常运行 |
| 1 | Mild | 随机翻转 3 个 L0 比特 | 立即恢复 |
| 2 | Moderate | 中等扰动 | 快速恢复 |
| 3 | Severe | 完全清空 NSE 历史积累 | 缓慢/无法恢复 |

### 1.2 测量指标

- `pre_nsi_mean` / `pre_nsi_max`: 扰动前的 NSI 水平
- `post_nsi_mean` / `post_nsi_max`: 扰动后的 NSI 水平
- `recovery_steps`: NSI 回到 > 0.5 所需的步数
- `recovered`: 是否在二进制意义上恢复（NSI > 0.5）

### 1.3 种子

seeds = [42, 142, 242, 342]

## 三、假设验证

### H21 (温和扰动恢复): ✅ PASS

> 类型 1 扰动后，NSI 在 200 步内恢复

**结果**: 所有 4 个种子在 0 步内恢复（NSI 从未低于 0.5）。实际上，温和扰动后的 NSI 立即超过了扰动前水平。

**解释**: CSC+NSE 架构对微小结构变化具有极强的吸收能力。3 比特翻转在 N0=72 的系统中只是 ~4% 的结构变化，CSC 的 Top-Down 约束和 Bottom-Up 反馈机制迅速将其整合进现有叙事。

### H22 (中度扰动恢复): ✅ PASS

> 类型 2 扰动后，NSI 在 500 步内恢复

**结果**: 所有 4 个种子在 0 步内恢复。

**解释**: 中度扰动同样被系统快速吸收。这表明 CSC+NSE 架构的鲁棒性远超预期——不仅温和扰动，即使是中度扰动也不足以破坏叙事连续性。

### H23 (剧烈扰动不可逆): ⚠️ PARTIAL PASS

> 类型 3 扰动后，NSI 无法恢复到扰动前水平（叙事自我被摧毁）

**结果**: 
- 所有种子在 100 步内恢复（NSI > 0.5），**未达到"不可逆"标准**
- 但 3/4 种子的 NSI **未回到扰动前水平**（差距 4-13%）
- 恢复需要 ~72 步（vs 0 步 for mild/moderate）

**解释**: H23 的预测部分正确。叙事自我没有被"摧毁"——系统确实从 NSE 历史清空中恢复。但恢复是不完全的：NSI 基线永久性下降。这是**部分可逆**而非**不可逆**。

叙事历史的积累是路径依赖的。清空历史等于抹去系统积累的"经验"，虽然 CSC 的结构耦合允许叙事自我重新涌现，但新的叙事自我缺少历史根基，NSI 无法达到原来的高度。

## 四、关键发现

### 4.1 系统具有极强的鲁棒性

CSC+NSE 架构对温和和中度扰动的抵抗能力超出预期。所有 8 个非严重扰动运行都在 0 步内恢复。

### 4.2 叙事历史是不可压缩的资源

Type 3 扰动（NSE 历史清空）是唯一能造成持久损害的扰动类型。这说明：
- 叙事自我不是纯粹的结构现象，而是**历史积累**的产物
- CSC 可以快速重建结构耦合，但 NSE 的历史深度需要时间重新积累
- 这与差异论的"路径依赖"预测一致

### 4.3 恢复时间尺度分离

| 扰动级别 | 恢复时间 | 恢复程度 |
|---------|---------|---------|
| Mild    | 0 步    | 100%+   |
| Moderate| 0 步    | 100%+   |
| Severe  | ~72 步  | 87-105% |

三个数量级的恢复时间差异表明系统对不同级别的扰动有明确的"响应分级"。

### 4.4 种子 342 的异常行为

Seed 342 在 Type 3 扰动后 NSI 不降反升（+5.2%）。这可能是因为：
- 该种子的叙事历史包含了"低效"的叙事模式
- 清空历史反而让系统从更干净的初始状态重建叙事
- 这是一种"创造性破坏"的雏形——但需要更多种子验证

## 五、与差异论的理论对照

### 5.1 "底图事件"的模拟

Type 3 扰动最接近差异论中的"底图事件"——旧资源无法在新空间中找到承接位置。系统确实产生了新的叙事组织方式（NSI 恢复），但新组织方式与旧的有结构性差异（NSI 基线下降）。

### 5.2 "螺旋"的韧性

差异论预测 P→E→M→S→R→P' 的螺旋运动。Type 3 扰动相当于在螺旋的某个点强制回到早期的 P' 状态。系统确实能重新启动螺旋，但新螺旋的"半径"（NSI 水平）可能不同。

### 5.3 层级保护机制

温和/中度扰动只影响 L0 层级的局部状态，CSC 的 Top-Down 约束和 ILP（虽已移除但 CSC 本身有保护功能）确保扰动不会向上传播。只有直接攻击 NSE 历史（元层级操作）才能造成持久影响。

## 六、限制与注意事项

1. **扰动后步数较短**: 只有 100 步，对于 Severe 扰动可能不够观察完全恢复
2. **样本量**: 每种扰动类型只有 4 个种子，统计显著性有限
3. **扰动类型的代表性**: 当前扰动类型是人为设计的，不一定代表"自然"扰动
4. **NSI 恢复阈值**: 使用 NSI > 0.5 作为恢复标准，可能过于宽松

---

## 七、下一步建议

1. **扩展 Type 3 实验**: 增加扰动后步数到 500-1000，观察 NSI 是否能完全回到扰动前水平
2. **Track A2 (CSC 耦合扫描)**: 测试不同耦合强度下系统的扰动恢复能力
3. **Track B1 (分层叙事追踪)**: 观察扰动在各层级的传播模式
4. **种子扩展**: 在 Track A3 中用更多种子验证种子 342 的异常行为

---

## 八、结论

Phase 5 Track A1 实验表明，CSC+NSE 架构具有**极强的鲁棒性**。温和和中等扰动被系统完全吸收，不会对叙事自我造成持久影响。即使是严重的 NSE 历史清空，系统也能在 ~72 步内恢复基本的叙事功能，但 NSI 基线会有小幅永久性下降。

核心发现：**叙事历史是不可压缩的资源**。结构可以快速重建，但历史积累需要时间。这为 Phase 5 Track B（多层级动力学）和 Track D（长期演化）提供了重要的基线数据。
