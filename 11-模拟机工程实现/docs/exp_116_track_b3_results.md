## 根因分析

### H30 失败根因

**核心问题：L1 和 L2 由同一 L0 聚簇动力学驱动，结构性相关无法通过噪声打破。**

1. **L2 稳定性来自 CSC 串行派生**：L2 稳定性 = f(L1 稳定性, 噪声)，虽然注入了噪声，但基础信号仍来自 L1
2. **L2 叙事活动来自 L2 稳定性**：L2 叙事 count = f(L2 稳定性, 噪声)，仍间接依赖 L1
3. **LNT 的 NSI 计算**：NSI = α·continuity + β·stability + γ·history_depth
   - stability 来自 CSC（L1 派生 + 噪声）
   - continuity 和 history_depth 来自叙事活动（L2 稳定性派生）
   - 所有三个分量都间接依赖 L1

4. **噪声注入的局限性**：
   - 噪声增加方差但不降低相关性
   - 当两个时间序列共享相同的低频驱动信号（L0 聚簇动态）时，噪声只能增加高频成分，不能打破低频相关性
   - L1 和 L2 的 NSI 都在响应相同的 L0 聚簇变化，只是幅度和相位不同

### H31 失败根因

**L0→L1 延迟无法检测，因为 L1 不由 L0 驱动。**

1. L1 的制度稳定性主要来自自身制度自稳（ILP 已移除，但制度自稳机制仍在）
2. L0 聚簇变化对 L1 的影响被制度自稳压制
3. 即使增加了 `serial_l0_to_l1_signal_weight=0.4`，L1 仍主要由自身动力学驱动

### H32 失败根因

**L2 叙事活动太低，无法产生非 silent 叙事。**

1. L2 稳定性均值约 0.1-0.2，叙事 count 约 2-4
2. 叙事递归算子需要更高的 count 才能产生 CIVILIZATION 级叙事
3. L1 和 L2 叙事都是 'silent'，自主性指数 = 0

## 理论洞察

### 串行耦合的结构性局限

串行耦合（L0→L1→L2）在理论上正确，但在当前实现中：

1. **L1 不由 L0 驱动**：L1 的制度自稳机制使其独立于 L0 聚簇变化
2. **L2 完全派生自 L1**：L2 没有独立的动力学，只是 L1 的衰减+噪声版本
3. **层级间缺乏真正的差异**：差异论要求每个层级有独立的"差异场"，但当前 L2 的差异场完全来自 L1

### 真正的层级需要独立的差异场

根据差异论 §2.2，层级是"结构生成中的组织分层"，每个层级应该有：
- 独立的差异场（差异产生的源头）
- 独立的动力学（不依赖于其他层级）
- 层间耦合（而非层间派生）

当前的串行耦合是"层间派生"而非"层间耦合"。L2 不是独立层级，而是 L1 的投影。

## 建议的下一步方向

### 方向 1: 为 L2 添加独立的差异场

- L2 应有自己的聚簇/结构形成机制，不依赖 L1
- L1→L2 的耦合应是约束传导，而非状态派生
- 这需要对 CSC 架构进行根本性重构

### 方向 2: 重新定义 H30

- 当前 H30（L1↔L2 r < 0.7）可能过于严格
- 在串行耦合架构下，L1-L2 相关性 r ≈ 0.85-0.95 可能是合理的
- 可以考虑将 H30 目标调整为 r < 0.95，或改用其他度量（如互信息、格兰杰因果）

### 方向 3: 转向 Track B4 — 层间约束传导

- 不再尝试"解耦"L1-L2，而是研究"约束传导"的机制
- H30 改为：L2 对 L1 约束的响应延迟 > 0
- H31 改为：L1 对 L0 约束的响应延迟 > 0
- 关注点从"相关性"转向"因果响应"
