**日期**: 2026-06-08

## 0. 战略定位

Phase 14 确立了关键理论发现：**死秩序是差异论单层世界的拓扑不变性**。
时间扰动、空间扰动、约束扰动三种正交路径均无法打破死秩序。
152 次运行、0 次突破、0/6 假设通过。

这意味着：**在同一层框架内，无论怎么改变参数，死秩序无法被打破。**
Phase 15 必须超越单层拓扑。

三个方案探索不同的超越路径：

| 方案 | 路径 | 核心问题 | 风险评估 |
|------|------|----------|----------|
| **A** | 非线性绑定 | 反馈机制的数学形式改变能否创造新行为？ | 低（α. 有先例证据表明指数衰减是关键限制因素） |
| **B** | 信息注入 | 结构性模式能否替代随机差异？ | 中（β. 打破 A1 随机选择假设，可能违反复现性基线） |
| **C** | 跨层演化器 | 密封结构作为上层约束能否创造层级涌现？ | 高（γ. 全新架构，存在多个失败路径） |

推荐执行顺序: **A → B → C** (α < β < γ 风险递进)

## 1. 方案 A：非线性绑定实验

### 1.1 理论基础

当前反馈机制（exp_162）使用指数衰减:
```python
bias[i] = fs * exp(-dist / decay_length)
```

Phase 14 数据表明这种平滑衰减被密封结构「吸收」或「中和」。
一个可能的解释是：**指数衰减的连续特性使得反馈信号无法在密封边界处产生足够的非线性跳变**，
从而无法触发相变。

### 1.2 实验设计

#### exp_163 — Sigmoid 绑定

| 参数 | 值 | 说明 |
|------|----|------|
| 反馈函数 | `bias[i] = fs / (1 + exp((dist - threshold) / steepness))` | Sigmoid 非线性 |
| 关键参数 | threshold ∈ {2, 4, 8, 16}, steepness ∈ {0.1, 0.5, 1.0, 2.0} | 4×4 = 16 种组合 |
| 基线 | fs ∈ {1.0, 3.0} × decay_length=8 | 32 runs |
| N | 48 | 保持一致性 |
| 总步数 | 8000 (pre-seal + post-seal) | |
| 输出指标 | secondary_sealing_rate, flip_enrichment boundary/non-boundary, dir_balance |

**理论预期**：
- 如果死秩序的突破需要相变式跳变，Sigmoid 应在 threshold ≈ decay_length 处产生效果
- 陡峭 Sigmoid (steepness=0.1) 应最接近 step 函数，效果最强

#### exp_164 — Step 绑定

| 参数 | 值 | 说明 |
|------|----|------|
| 反馈函数 | `bias[i] = fs if dist <= threshold else 0` | 硬阈值 |
| 关键参数 | threshold ∈ {2, 4, 8, 12, 16} | 5 种 |
| 基线 | fs ∈ {1.0, 2.0, 3.0, 5.0} | 20 runs |
| N | 48 | |
| 总步数 | 8000 | |
| 输出指标 | 同上 + boundary_active_lifetime |

**理论预期**：
- Step 函数最接近相变边界
- 如果死秩序真的是拓扑不变的，step 也应无效
- 但如果 Phase 14 的失败是因为信号不够「锐利」，step 可能在 fs≥3.0, threshold≤8 时产生效果

#### exp_165 — 协同反馈绑定

| 参数 | 值 | 说明 |
|------|----|------|
| 反馈函数 | `bias[i] = fs * max(0, 1 - dist/decay_length)²` | 二次衰减 |
| 关键参数 | decay_length ∈ {4, 8, 16, 32} | 4 种 |
| 基线 | fs ∈ {1.0, 3.0} | 8 runs |
| N | 48 | |
| 总步数 | 8000 | |
| 输出指标 | 同上 + spatial_selectivity (边界区域与非边界区域的 flip 差异) |

**理论预期**：
- 二次衰减比指数更快在近距离衰减，但在远距离更慢接近 0
- 可能产生更明确的「近程作用带」

### 1.3 成功标准

| 标准 | 测量方式 | 阈值 |
|------|----------|------|
| 强成功 | 出现至少 1 次二次密封事件 | secondary_sealing > 0 在 ≥1 运行中 |
| 中等成功 | flip_rate boundary vs non-boundary 差异 > 2σ | t-test p < 0.05 |
| 弱成功 | 方向场梯度出现非零模式 | dir_balance 偏离 0.5 ≥ 0.1 |

所有条件均不满足 → **方案 A 被否定，推进到方案 B**

## 2. 方案 B：信息注入实验

### 2.1 理论基础

差异论 §5 区分「差异」和「信息」：
- **差异**：比特翻转（量变）— Phase 14 全部使用
- **信息**：差异之间的关系模式（质变）— Phase 15 启动

核心假设：死秩序抵抗随机差异，但**结构性模式可能在密封结构内部创造自指涉回路**。

### 2.2 实验设计

#### exp_166 — 反转向差异注入

当前所有实验注入 0→1 差异。反转向 1→0 注入是否改变局面？

| 参数 | 值 | 说明 |
|------|----|------|
| 注入方向 | 1→0 (吸收→注入、注入→吸收) | 反转整个方向 |
| 基线 | n_inject ∈ {1, 2, 4} | 3 runs |
| N | 48 | |
| 总步数 | 5000 | |

**理论预期**：
- 如果系统只是镜面对称，结果应完全对称
- 但如果 A1/A8 机制有方向偏好，反转可能改变密封动力学
- **可能性低**：差异论的公理系统是对称的

#### exp_167 — 相关性注入

注入联动翻转的结构模式：

| 参数 | 值 | 说明 |
|------|----|------|
| 注入模式 | (a) 成对联动: (i,j) 同时注入，(b) 链式: i→j→k | 2 种模式 |
| 配对选择 | (a) 随机配对，(b) 空间邻近配对，(c) Hanning 距离配对 | 3 种策略 |
| 联动强度 | p_simultaneous ∈ {0.25, 0.5, 0.75, 1.0} | |
| 基线 | n_pairs ∈ {1, 2, 4} | |
| N | 48 | |
| 总步数 | 8000 | |
| 总运行数 | 2 × 3 × 4 × 3 = 72 runs | |

**理论预期**：
- 相关性注入可能在密封结构内部创造「相关性岛屿」
- 如果 A1' 横向演化能捕获这种相关性，可能产生局部结构

#### exp_168 — 时间模式注入 (振荡)

| 参数 | 值 | 说明 |
|------|----|------|
| 注入模式 | 振荡: sin(ωt) 调幅, on/off: 周期性开关 | 2 种 |
| 频率 | ω ∈ {0.01, 0.05, 0.1, 0.5} (每步弧度) | 4 种 |
| 振幅 | A ∈ {0.25, 0.5, 1.0} (相对于基线强度) | 3 种 |
| N | 48 | |
| 总步数 | 10000 (确保振荡周期覆盖) | |
| 总运行数 | 2 × 4 × 3 = 24 runs | |

**理论预期**：
- 共振频率可能意外地打破死秩序
- 低频率 (ω=0.01) 对应于 628 步一个周期，与密封时间尺度可能匹配

### 2.3 成功标准

与方案 A 相同。所有条件不满足 → **方案 B 被否定，推进到方案 C**

## 3. 方案 C：跨层演化器实验

### 3.1 理论基础

这是最根本的架构革新。核心思想：
- 密封结构不是「终点」，而是「新起点」
- 密封结构应成为**上层演化器的约束条件**
- L1 层在 L0 密封结构定义的约束空间中进行独立演化

类比：量子力学的能级结构 — 基态不变化，但激发态在基态之上独立存在。

### 3.2 架构设计

#### 3.2.1 跨层演化器架构

```
Layer 0 (L0): 当前 SpatialLongRangeEvolver
  ├── 运行直到密封
  ├── 密封结构 = {sealed_bits, binding_matrix, clusters}
  └── 密封后冻结（或继续以极小速率演化）

Layer 1 (L1): 新的 Layer1Evolver
  ├── 独立的比特空间 N1 (默认 = N0 = 48)
  ├── L0 密封结构 → L1 约束条件:
  │   ├── cluster_ids → L1 的分组约束
  │   ├── binding_strength → L1 的初始绑定强度
  │   └── direction_field → L1 的方向偏好
  └── 在 L0 约束之上进行独立演化

Layer 2 (L2): (如果 L1 密封) 递归
  └── ... 理论上可无限递归
```

        2. cluster 大小 → hierarchy 组中的 hierarchy bits 数量
        3. cluster 间 binding_strength → hierarchy 组间权重
        4. 未密封的 L0 bits → L1 的 lateral bits (模版)
        """
        ...

        L0 密封后方向场趋向 0 (方差→0)。
        但 L1 可以继承方向场的**差异**而非**值**：
        - 高方差的区域 → L1 的高分歧区域
        - 低方差区域 → L1 的稳定区域

### 3.4 成功标准

| 标准 | 定义 |
|------|------|
| 架构可行 | 跨层演化器能运行完整生命周期（L0 密封 → L1 启动 → L1 演化 → L1 结果） |
| L1 产生新结构 | L1 的密封模式反映 L0 约束的结构特征（非随机密封） |
| L2 涌现 | L1 密封后，L2 出现之前未观察到的结构模式（相关性、聚类、方向场） |
