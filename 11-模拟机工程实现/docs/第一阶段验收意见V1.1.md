### **一、最严重问题：`LocalConvModel` 仍然是 sigmoid residual，不是 delta 更新**

我看到的 `models/local_conv_model.py` 里，核心 forward 仍然是：

```python

out = state + dx

if self.use_reaction:
    reaction = self.reaction(state)
    out = out + self.reaction_scale * reaction

return torch.sigmoid(out)
```

这说明最关键的 P0 修复没有真正落地。这个版本仍然破坏恒等映射语义：当 `dx = 0` 且无 reaction 时，输出是：

```python
torch.sigmoid(state)
```

而不是：

```python
state
```

也就是说，对于输入 $state \in [0,1]$，即便模型什么都不想改变，输出也会被 sigmoid 推向 $0.5\sim0.73$ 区间。这会直接污染 A4
最小变易，使模型结构本身持续制造变化。

更严重的是，`tests/test_local_conv_model.py` 里有一个 `test_step_scale_learnable`，它断言：

```python

assert model.step_scale.requires_grad
```

但我看到的 `LocalConvModel.__init__()` 签名是：

```python

```

里面没有 `step_scale`。所以这组测试按当前源码应该会失败。也就是说：测试已经写成了“期待新版模型”，但模型实现仍是旧版。

这里建议立即改成下面这个版本：

```python
"""models/local_conv_model.py — 局部卷积模型

3×3 卷积 + delta 更新 + 反应分支。
模型预测局部变化量 delta，而不是直接预测下一状态。
"""

import torch
import torch.nn as nn


class LocalConvModel(nn.Module):
    """局部卷积模型，用于差异反应堆动力学。"""

    def __init__(
            self,
            channels: int = 32,
            use_reaction: bool = True,
            step_scale: float = 0.2,
            learnable_step_scale: bool = True,
    ):
        super().__init__()
        self.use_reaction = use_reaction

        if learnable_step_scale:
            self.step_scale = nn.Parameter(torch.tensor(float(step_scale)))
        else:
            self.register_buffer("step_scale", torch.tensor(float(step_scale)))

        self.backbone = nn.Sequential(
            nn.Conv2d(1, channels, kernel_size=3, padding=1),
            nn.GELU(),
            nn.Conv2d(channels, channels, kernel_size=3, padding=1),
            nn.GELU(),
            nn.Conv2d(channels, 1, kernel_size=3, padding=1),
        )

        if use_reaction:
            self.reaction = nn.Sequential(
                nn.Conv2d(1, channels, kernel_size=1),
                nn.Tanh(),
                nn.Conv2d(channels, 1, kernel_size=1),
                nn.Tanh(),
            )
            self.reaction_scale = nn.Parameter(torch.tensor(0.05))

    def forward(self, state: torch.Tensor) -> torch.Tensor:
        delta = torch.tanh(self.backbone(state))

        if self.use_reaction:
            reaction = self.reaction(state)
            delta = delta + self.reaction_scale * reaction

        scale = torch.clamp(self.step_scale, 0.0, 1.0)
        next_state = state + scale * delta
        return next_state.clamp(0.0, 1.0)
```

### **二、`L0BinaryLattice.measure_difference()` 仍然会在 1D 下返回空张量**

当前实现仍是：

```python
dx = (state[:, :, :, 1:] - state[:, :, :, :-1]).abs()
dy = (state[:, :, 1:, :] - state[:, :, :-1, :]).abs()

diff = (dx[:, :, :-1, :] + dy[:, :, :, :-1]) / 2.0
return diff
```

对于 `shape=(1, 50)`，`dx` 的 shape 是：

```text
(batch, channel, 1, 49)
```

但 `dx[:, :, :-1, :]` 会变成：

```text
(batch, channel, 0, 49)
```

`dy` 本身就是空高度：

```text
(batch, channel, 0, 50)
```

所以最后返回的仍然是空 tensor。这和 `tests/test_l0_binary_lattice.py` 里的预期冲突。测试里写的是：

```python
assert diff.shape == (1, 1, 1, 9)
```

但按当前实现，1D 情况下实际会是：

```text
(1, 1, 0, 9)
```

这说明 1D NaN/空张量修复没有真正完成。

建议替换为下面这个版本：

```python
def measure_difference(self, state: torch.Tensor) -> torch.Tensor:
    """返回空间差异场，兼容 1D/2D，绝不返回空 tensor。"""
    parts = []

    if state.shape[-1] > 1:
        dx = (state[:, :, :, 1:] - state[:, :, :, :-1]).abs()
        dx = F.pad(dx, (0, 1, 0, 0))
        parts.append(dx)

    if state.shape[-2] > 1:
        dy = (state[:, :, 1:, :] - state[:, :, :-1, :]).abs()
        dy = F.pad(dy, (0, 0, 0, 1))
        parts.append(dy)

    if not parts:
        return torch.zeros_like(state)

    return sum(parts) / len(parts)
```

这样：

```text
2D: 返回 shape 与 state 一致的差异场；
1D: 只使用 dx，并 pad 回原 shape；
单点世界: 返回 zeros_like(state)。
```

如果你更希望 `measure_difference()` 返回 batch 级标量，那就另起一个 `measure_difference_scalar()`。但当前
`WorldEngine._quick_pattern_persistence()` 明显把它当场来用，所以这里返回差异场更合适。

def apply_boundary_flow(
        self,
        state: torch.Tensor,
        source_strength: float = 1.0,
        sink_strength: float = 1.0,
):
    """同时应用源端注入和汇端吸收，并返回通量统计。"""
    before = state

    after_source = self.inject_difference(
        before,
        source_strength=source_strength,
    )
    injected = self.measure_invariant(after_source) - self.measure_invariant(before)

    after_sink = self.absorb_difference(
