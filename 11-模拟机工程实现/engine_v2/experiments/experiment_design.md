# 模拟机实验设计文档

> 日期: 2026-06-24
> 目标: 用模拟机独立检验 WorldBase 理论预测

## 一、实验哲学

### 模拟机的定位

模拟机不是"公理验证器"，而是**计算实验物理学家**。

```
理论物理学家: "从这十条公理，我推导出 D=3、Φ=-1/r、su(3)色禁闭..."
实验物理学家: "真的吗？让我跑一下看看。"
```

模拟机的角色是后者。它独立运行一个离散世界，观察其中发生的现象，然后与理论预测对比。

### 什么是"独立检验"

**独立**意味着：
1. 模拟机不知道理论公式，只执行九机制循环
2. 检测器从演化数据中直接测量，不调用理论公式
3. 测量值与预测值的对比在模拟完成后进行

**不是独立**的反例：
- 用公式 `Φ = -1/d_H` 计算 Φ，再验证它等于 `-1/d_H` → 同义反复
- 用公式 `m_W = ln(1+2/N)·m₀` 计算质量，再验证公式正确 → 同义反复

### 可证伪性

每个实验对应一个**可证伪的预测**：
- 预测失败 → 理论推导可能有误
- 预测成功 → 理论获得实验支持（不是"证明"）

## 二、理论预测清单

WorldBase 对模拟宇宙做出以下预测。我们逐一检验。

### P1: 自指闭环 vs 被动投影

**预测**: 补回 A9 自指后，涌现深度增加，L1 从死秩序变活秩序。

**来源**: engine_v2 README, Phase 17 修正。

**可证伪**: 如果自指和被动投影深度一样，或 L1 flux 都是 0，理论就错了。

**测量方法**:
- 跑 `self_encapsulate=True` 和 `False` 各 N 种子
- 比较涌现深度和 L1 Jaccard flux

### P2: 引力效应 — 重量趋向稳定值

**预测**: A5 守恒 + A1 差异源 → 系统被拉向某个重量层，重量有确定趋势。

**来源**: 定理 G (D_eff + A5 → Φ ∝ -1/r)。

**可证伪**: 如果重量做无偏随机游走（趋势=0），引力效应不存在。

**测量方法**:
- 跑模拟，记录每步汉明重量
- 计算重量时间序列的线性趋势
- 趋势显著 ≠ 0 → 引力效应存在

### P3: 强力效应 — 组织间出现非零绑定

**预测**: A1' 横向绑定 → 组织间绑定增强 → 色禁闭前兆。

**来源**: 定理 S (A4+A8+A9 → su(3))，定理 CONF-2 (线性势)。

**可证伪**: 如果组织间绑定始终为 0，强力效应不存在。

**测量方法**:
- 跑模拟，检测组织形成
- 测量组织间的绑定密度
- 非零绑定占比 > 0 → 强力效应存在

### P4: 弱力效应 — 中截面偏好

**预测**: A8 对称偏好 → 系统被拉向 w=N/2，远离中截面时有回复力。

**来源**: A8 (对称偏好)，WLEM (约束度函数)。

**可证伪**: 如果从 w≠N/2 出发，系统不做趋向 N/2 的运动，A8 效应不存在。

**测量方法**:
- 从远离 N/2 的初始状态出发
- 记录与中截面的距离变化
- 距离减小 → A8 效应存在

### P5: 电磁效应 — 电荷守恒

**预测**: A5 差异守恒 → 汉明重量围绕均值波动，方差有限。

**来源**: A5 (差异守恒)，定理 CD (电荷守恒)。

**可证伪**: 如果重量做无界随机游走（方差∝t），守恒律不成立。

**测量方法**:
- 跑模拟，记录每步汉明重量
- 计算重量方差
- 方差有限（不随时间增长）→ 守恒成立

### P6: 聚簇效应 — 绑定增长

**预测**: A1' 横向涌现 → 共活跃位的绑定随时间增强。

**来源**: A1' (横向涌现)，m1 聚簇机制。

**可证伪**: 如果绑定密度不增长（趋势=0），聚簇效应不存在。

**测量方法**:
- 跑模拟，记录每步活跃位间的平均绑定
- 计算绑定时间序列的趋势
- 趋势 > 0 → 聚簇效应存在

### P7: 层级涌现 — 组织形成

**预测**: m2 层级 → 从绑定图中提取组织，组织数从 0 增长。

**来源**: m2 层级机制。

**可证伪**: 如果始终无组织形成（最大组织数=0），层级涌现失败。

**测量方法**:
- 跑模拟，记录每步组织数量
- 最大组织数 > 0 → 层级涌现成功

### P8: 封口涌现 — 密封事件

**预测**: m8 锁定 → 组织位冻结 → 系统密封。

**来源**: m8 锁定机制，A9 自指触发条件。

**可证伪**: 如果永远不密封（step=max_steps 仍未 sealed），封口机制失败。

**测量方法**:
- 跑模拟，记录是否密封、密封步数
- 密封率 > 0% → 封口机制工作

## 三、实验设计

### 实验 A: 自指闭环基础检验 (P1)

```
配置:
  N0 = 36, seeds = 20, max_steps = 300
  对照组: self_encapsulate = False (被动投影)
  实验组: self_encapsulate = True (自指封装)

测量:
  - 涌现深度 (层数)
  - L1 Jaccard flux
  - L0 封口步数

判定:
  depth_self > depth_passive → P1a ✓
  flux_self_L1 > 0.05 → P1b ✓
  flux_passive_L1 < 0.05 → P1c ✓
```

### 实验 B: 力效应涌现检验 (P2-P7)

```
配置:
  N0 = 36, seeds = 10, max_steps = 200
  self_encapsulate = True
  
检测器 (每步调用):
  - gravity: 记录汉明重量
  - strong: 记录组织间绑定密度
  - weak: 记录与中截面距离
  - em: 记录汉明重量 (用于守恒检验)
  - binding: 记录活跃位间平均绑定
  - org_count: 记录组织数量

判定:
  P2: 重量趋势 ≠ 0 → ✓
  P3: 组织间绑定非零步 > 0 → ✓
  P4: 中截面偏离趋势 < 0 → ✓
  P5: 重量方差有限 → ✓
  P6: 绑定趋势 > 0 → ✓
  P7: 最大组织数 > 0 → ✓
```

### 实验 C: 封口机制检验 (P8)

```
配置:
  N0 = 36, seeds = 20, max_steps = 300
  self_encapsulate = True

测量:
  - 是否密封
  - 密封步数
  - 密封时的活跃位数

判定:
  密封率 > 50% → ✓
```

### 实验 D: 规模效应 (N 依赖)

```
配置:
  N0 = [16, 24, 36, 48], seeds = 10, max_steps = 300
  
测量:
  - 各 N 下的涌现深度
  - 各 N 下的 L1 flux
  - 各 N 下的封口率

目的:
  观察涌现质量如何随系统规模变化
  理论预测: N 越大，涌现越深
```

## 五、预期结果与风险

### 预期

| 预测 | 预期结果 | 信心 |
|------|---------|------|
| P1 自指深度 > 被动 | ✓ (已验证 20/20) | 高 |
| P2 引力效应 | 重量趋势 ≠ 0 | 高 |
| P3 强力效应 | 组织间绑定 > 0 | 中 |
| P4 弱力效应 | 趋向中截面 | 中 |
| P5 电磁效应 | 重量守恒 | 高 |
| P6 聚簇效应 | 绑定增长 | 高 |
| P7 层级涌现 | 组织出现 | 高 |
| P8 封口涌现 | 密封事件 | 高 |

### 风险

- P3 (强力): 组织间绑定可能很弱，需要灵敏检测器
- P4 (弱力): 如果初始状态已在中截面附近，效应不明显
- P8 (封口): max_steps 可能不够，需要调整参数
