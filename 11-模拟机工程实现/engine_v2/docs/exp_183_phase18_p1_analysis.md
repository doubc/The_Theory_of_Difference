### 2. 结构传递效应: 真实但温和

- **corr(k0, k1) = 0.658** — L0 组织数确实传递到 L1，但非一对一
- **corr(k0, N1) = 0.932** — m9 编码忠实: N1 = 2×k0 + n_res 几乎完全关联
- **corr(N1, k1) = 0.644** — 较大 N1 产生较多 L1 组织
- **L1 密封率 = 99.3%** — L1 几乎总是密封

### 3. 关键洞见: m9 增强是**线性叠加**而非**指数放大**

m9 提供了预绑定和颜色继承，但 L1 的 k1 仅仅是从 N1 中提取出 ~1-2 个组织。这与 P0 中随机初始条件在相同 N1 下的表现差异不大（1.0-1.7x）。

**为什么不是 3-4x？** 因为 N1 = 2×k0 + n_res ≈ N0/3 至 N0/6。m9 输出的 L1 场本身很小。即使结构化初始条件提高了 k/N 比例，由于 N1 太小，k1 的上限受 n_meta_colors=4 和 min_org_size=3 的硬限制。

### 4. Phase 17 涌现深度 4.62 的解释修正

exp_182 分析曾推测 m9 结构化 init 提供 3-4x 增强因子来解释深度 4.62。exp_183 证伪了该推测。

**深度链的真正逻辑**:
- L0(N0=48) → k0≈5 → N1≈15 → L1 → k1≈1.4 → N2≈6 → L2 → k2≈? → N3≈5 → …
- 深度 = 4.62 的关键: **k≥1 即可继续**, 不是 k 被大幅放大
- 密封机制在小 N 下也工作（P0 证实 N=12 密封率 100%）
- min_org_size=3, n_meta_colors=4 在 N≈5-15 范围仍能产生 1-2 个组织

**结论**: 涌现深度主要由**链的多步衰减**决定, 而非单步放大。k(N) 的衰减速度（每层 ~3x 减小）决定了深度 ~log_{3}(N0/3) ≈ 3-5 步。
