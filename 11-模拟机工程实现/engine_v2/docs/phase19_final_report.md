# Phase 19 综合报告: 开放系统（环境交互）

> **实验范围**: exp_185 (P0 强度扫描) + exp_186 (P1 复杂度扫描) + exp_187 (P2 时序扫描)
> **总计运行**: 80 + 64 + 64 = 208 runs
> **编写日期**: 2026-06-10 — Phase 19 最后实验完成后

---

## 1. 背景与问题

Phase 18 证明自指闭环的涌现深度存在渐近极限（iso_score ≈ 0.72）。**核心问题**: 外部环境能否突破这个极限？

Phase 13-16 的旧引擎无法回答这个问题——L1 flux=0（死秩序），环境交互要么直接写状态（破坏系统），要么被吸收（无效果）。

engine_v2 修复了自指缺失（A9），使 L1 flux≈0.22（活秩序）。在活的递归系统中，环境交互才有可能产生真正的"开放系统"行为。

**设计假设（phase19_design_open_system.md）**:
- H19-P0: iso_score > 0.65 → 环境不能重启自指（系统太稳定）
- H19-P1: iso_score < 0.50 → 环境可以重启自指（且时序敏感）
- H19-P2: 高复杂度环境 → 系统吸收环境结构
- H19-P3: 低复杂度环境 → 系统忽略环境

## 3. 结果汇总

### 3.1 涌现深度

| 条件 | Mean Depth | L3+% | vs 基线(无环境) |
|------|-----------|------|----------------|
| **基线(无环境)** | 4.56-4.62 | 93.8% | — |
| 强度=0.50 | 5.00 | **100%** | +0.38 / +6.2pp |
| 复杂度=0(噪声) | 4.88 | 100% | +0.26 / +6.2pp |
| 复杂度=1(弱聚簇) | 4.50 | 100% | -0.06 / +6.2pp |
| 复杂度=2(强聚簇) | 4.69 | 100% | +0.13 / +6.2pp |
| 时序=early(0) | 4.75 | 100% | +0.13 / +6.2pp |
| 时序=mid(5) | 4.56 | 100% | -0.06 / +6.2pp |
| 时序=late(密封后) | 4.88 | 100% | +0.25 / +6.2pp |

**关键模式**: 深度变化 ≤ 0.38（小于一层），但 L3+ 涌现率从 93.8% → 100% 在所有有环境的条件下。

### 3.2 L1/L2 Flux 抑制

| 条件 | L1 Flux | L2 Flux | Flux抑制比 |
|------|---------|---------|-----------|
| 基线(无环境) | 0.2065 | 0.5743 | — |
| 强度=0.50 | 0.1894 | 0.4942 | ↓8.3%/↓13.9% |
| 噪声(熵=0) | 0.1802 | 0.4165 | ↓12.7%/↓27.5% |
| 时序=early | 0.1839 | 0.4693 | ↓10.9%/↓18.3% |

**关键模式**: 所有环境配置都抑制 L1/L2 flux，与环境复杂度无关。

### 3.3 其他指标

| 指标 | 关键发现 |
|------|---------|
| L0 密封步数 | 环境**不改变** (~10.2-10.5) — L0 密封是内在的 |
| L0 组织数(k) | 环境**不改变** (~5.56-5.62) — 基线结构不变 |
| 结构吸收率 | ~0.39（目标>0.5❌）— 环境复杂度无差异 |
| 定殖率 | 100% — 环境比特无处不在 |
| 种子 8(边缘) | 基线 depth=2, 有环境 depth≥4 |
| env_flux | 与熵负相关, 但不影响系统响应 |

## 5. 核心发现

### 5.1 环境的三种假设全部落空

设计的假设体系建立在环境与系统的**二分法**上：
- 要么环境传递结构（H19-P2）
- 要么环境是噪声被忽略（H19-P3）
- 要么环境影响系统轨迹（H19-P1）

**实际结果**: 环境的效果既不来自"结构传递"也不来自"轨迹改变"，而来自**约束场效应**。

### 5.2 约束场效应的三个特征

1. **L2 flux 单调抑制**: 在所有实验、所有条件下，环境降低 L1/L2 flux（8-27%）。这对应差异论中"环境提供稳定性"的预测。
2. **边缘种子拯救**: 种子 8（基线 depth=2, 死秩序）在所有含环境的配置中 depth≥4。环境提供额外约束，防止系统过早锁定。
3. **L3+ 涌现率封顶**: 所有含环境的配置 L3+ 达到 100%（基线 93.8%）。6.2pp 的提升虽小但稳定。

### 5.3 结构传导的缺失

系统对三种环境复杂度（白噪声/弱聚簇/强聚簇）的结构吸收率几乎相同：
- 熵=0: 0.398
- 熵=1: 0.383
- 熵=2: 0.389

吸收主要来自**共享比特空间的必然重叠**（定殖率 100%），而非真正的结构传导。

### 5.4 时序无关性

环境的引入时机对系统的影响可以忽略：
- early(全程耦合): 影响可测但小
- mid(5步后): 几乎无影响
- late(密封后): 几乎无影响

**原因**: L0 密封在 ~10 步内完成。密封前的耦合窗口（<10步）不足以改变任何轨迹。

## 6. 理论意义

### 6.1 自指闭环的强鲁棒性

Phase 19 的**最核心发现**: engine_v2 的自指闭环(A9)一旦建立，外部环境的任何参数（强度、复杂度、时序）都无法改变其涌现深度。

这不是死秩序的"锁死"（Phase 13-16 的 L1 flux=0），而是活秩序的**鲁棒性**（L1 flux≈0.2, 但不受环境干扰）。

### 6.2 环境不是差异源

差异论的理论框架给出明确预测：
- **差异驱动演化**: 新差异来自闭包内部的解封/重组
- **环境提供约束**: 环境设定边界条件，但不产生新差异

Phase 19 的实验结果完全支持这一预测：环境提供约束（抑制 flux、抢救边缘种子），但不产生新差异源（不增加涌现深度）。

### 6.3 对开放系统的重新理解

"开放系统"不是环境向系统"注入信息"或"提供能量"的过程，而是：
1. **环境是约束场**: 提供稳定外部边界，改变内部动力学的参数
2. **环境的效果是定量而非定性的**: 不改变涌现深度，只改变涌现的稳定性（提高 L3+ 率几个百分点）
3. **环境的价值是防止死秩序**: 帮助边缘种子达到正常涌现

## 7. 对后续阶段的启示

### Phase 20: 并行子空间（推荐）

单个环境场只能提供约束，无法成为新的差异源。**并行子空间**让多个自指链互相作为"环境"——互相成为新的差异源。这是差异论中"差异的差异"的工程实现。

**工程可行性**: engine_v2 的自指闭环是计算轻量的（208 runs 在 ~20 分钟完成），多世界并行在计算上可行。

**理论支撑**: 差异论 §2.6 "差异的差异"——闭合递归间的交互产生新层。多个自指链之间互相作为对方的"环境"，但不是普通环境场（约束），而是有独立自指闭环的"活环境"（差异源）。

### 备选: 极端耦合测试

在 Phase 19 基础上做一个快速补充：耦合强度 1.0-2.0，看是否达到"结构传导"阈值。但这个方向大概率也失败——如果时序无关性成立，强度也很可能只是定量增加（更多 flux 抑制）而非定性改变。
