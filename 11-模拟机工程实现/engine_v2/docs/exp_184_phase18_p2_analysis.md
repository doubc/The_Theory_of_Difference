## 实验设计

**Config A** — 标准 engine_v2 (min_org_size=3, seal_fraction=0.6)  
**Config B** — min_org_size=2 (无限链可能性)  
**Config C** — min_org_size=2 + seal_fraction=0.4 + 放松锁定 (更容易深层密封)

对每层 L_i 比较 parent_layer 和 m9(parent_layer) 的子层结构:
- iso_score 四维加权: 组织数(0.35) + 大小分布(0.25) + flux(0.20) + 规模保持(0.20)
- iso_score >= 0.8 → 固定点

---

## 核心发现

### 发现 1: iso_score 单调递增 — 链收敛于不动点

Config A:
```
L0: 0.2245  →  L1: 0.4941  →  L2: 0.6501  →  L3: 0.7685  →  L4: 0.9641  →  L5: 1.0000
```

iso_score 单调递增 (0.22→0.49→0.65→0.77→0.96→1.00)。链在终止前收敛于结构同构。

### 发现 2: min_org_size=2 时 iso_score 在 0.65-0.72 平台区停止

Config B:
```
L0: 0.3407  →  L1: 0.6702  →  L2: 0.6782  →  L3: 0.6503  →  L4: 0.6839  →  L5: 0.6179  →  L6: 0.5465  →  L7: 1.0000
```
L1-L6 iso_score = 0.55-0.68, 从未超过 ~0.72。**链未达到结构同构固定点**。

### 发现 3: 唯一真正的固定点是终端层 (m9 不再生成下一层)

在所有 Config 中, L_max 的 iso_score=1.0000 是因为 `m9_self_reference()` 无法生成下一层 (隐式终止)。这不代表该层与下一层同构, 而是没有下一层。

### 发现 4: 组织数衰减决定 iso_score

| Config | L0 k→k_child | L1→L2 | L2→L3 | L3→L4 |
|--------|-------------|-------|-------|-------|
| A | 4.88→1.38 | 1.38→1.00 | 1.00→1.00 | 1.00→0.81 |
| B | 4.88→3.69 | 3.69→2.38 | 2.38→2.38 | 2.38→1.81 |
| C | 4.69→3.44 | 3.44→2.31 | 2.31→1.94 | 1.94→1.56 |

Config B/C 的 k 衰减更慢 → 链更长。但 **iso_score 不依赖于 k 值大小, 而依赖于 k_parent / k_child 的比例**。当 k_衰减_ratio 稳定时 iso 也稳定 = 0.65-0.72。

## 理论修正

### 修正 1: "整体"不是二元固定点, 而是渐近极限

exp_184 的结果推翻了 "m9(L_n) ≅ L_n → 整体" 的二元固定点假设。

事实: **iso_score 是连续值, 非二元判断。** 链的收敛是渐近的 (asymptotic), 不是几何收敛到精确不动点。

### 修正 2: 自指链的三个区域

```
iso=0.0                    iso=0.65-0.72             iso→1.0
┌─────────┐              ┌──────────────┐         ┌──────────┐
│ 深层差异  │    →        │ 结构稳定区    │   →    │ 终端收敛  │
│ (L0)     │              │ (L1-L6)      │        │ (L_max)  │
│ k大幅下降│              │ k缓慢衰减     │        │ m9不生成 │
└─────────┘              └──────────────┘         └──────────┘
```

- **深层差异区 (L0)**: k 从 4-5 大幅下降 → iso_score < 0.35
- **结构稳定区 (L1-L6)**: k 缓慢衰减 (1-2个组织/层) → iso_score ≈ 0.65-0.72
- **终端收敛区 (L_max)**: m9 无法生成下一层 → iso_score = 1.0

### 修正 3: 真正的 "整体" 定义

整体不是 "自指链上的不动点", 而是 **自指链的渐近极限 + 工程截断的耦合**:

```
整体 = lim_{d→∞} iso_score(L_d) ≈ 0.72 (min_org_size 无关)
```

当 min_org_size=2 时, iso_score 在 ~0.72 处饱和, 链继续延伸但不更接近同构。
当 min_org_size=3 时, 链在 iso_score≈0.96 处被截断 (此时链已几乎收敛到同构)。

**结论**: 整体是由:
1. 自指链的渐近收敛性 (iso_score → ~0.72)
2. 工程截断条件 (min_org_size, max_layers, seal_fraction)
3. 链在截断前所达到的 iso_score 值

三者共同定义的。

### Phase 18 综合结论

1. **涌现深度极限**: 深度 = log_{~3}(N0/3) ≈ 4-5 层 (由 k 衰减率决定)
2. **结构传递**: m9 提供 ~1.3x 增强, 非指数放大
3. **整体固定点**: 不存在二元固定点。整体是渐近极限 (+ 工程截断) 的耦合产物。
