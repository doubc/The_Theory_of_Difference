# Phase 18 设计: 涌现深度极限分析

> 2026-06-10 13:14 UTC+8 | 基于 engine_v2 自指闭环架构

---

## 1. 问题背景

Phase 17 参数鲁棒性扫描揭示了关键标度律: 涌现深度在 N0≈36 处达到峰值 (depth=4.88), 此后随 N0 增大而衰减。这意味着引擎存在 **内在深度极限**——自指链不会无限延伸。

HEARTBEAT.md 提出的 Phase 18 核心问题:
1. **自指链在"命名位耗尽"时如何终止?**
2. **"整体"(whole)的精确定义是什么?**

## 2. 理论模型: 组织密度缩放

### 2.1 层间转换公式

m9_self_reference 的精确行为:
```
N_{i+1} = 2*k_i + min(|residual_i|, max_residual)
```
- k_i = L_i 层的组织数
- residual_i = L_i 密封后的活跃未冻结位
- max_residual = 6 (当前参数)

### 2.2 组织密度 r_i

定义 **组织密度**: r_i = k_i / N_i

如果 r_i 是常数, 涌现深度将遵循几何衰减:
```
N_i = N0 * (2r)^i
depth = floor(log_{1/(2r)}(N0 / 2*min_org_size))
```

但这预测 depth≈2-3, 与实际 depth≈4-5 不符。

### 2.3 关键洞见: 密度增长

**r_i 不是常数——随 N_i 减小而增加。**

原因:
1. **颜色竞争减少**: 小 N 下同色位更易相遇 → 聚簇更紧密
2. **绑定饱和更快**: bindings 在小图中更快达到阈值
3. **级联更容易**: cascade_density 阈值在小组织上更易跨越

这意味着:
```
r_{i+1} > r_i   当 N_{i+1} < N_i
```

### 2.4 深度极限的数学本质

终止条件不是 `k_i = 0`, 而是 `N_{i+1} < 2 * min_org_size` (=6):

```
从 L_d 到 L_{d+1} 最少需要 2*min_org_size = 6 位够容纳 1 个组织的 body+naming
当 N_d < 6 时, 即使有 1 个组织, 下一层只有 2+res ≤ 4 位 → 无法形成组织
```

**深度 d 满足:**
```
N_{d-1} ≥ 2*min_org_size = 6    (能形成至少 1 个组织)
N_d < 2*min_org_size = 6        (自指链终止)
```

## 3. 实验设计

### 3.1 P0: 理论验证 — 单层压缩率测量

**目的**: 测量 N → k → N' 的精确映射关系

**方法**: 运行孤立 Layer(N) 并统计:
- `k(N)` = 密封后的组织数 (绑定阈值不变)
- `residual(N)` = 密封后的活跃未冻结位
- `N_next(N) = 2*k(N) + min(residual(N), 6)` 的期望值

**参数**: N = [12, 18, 24, 30, 36, 48, 60, 72, 96], 每点 16 seeds

**假设 H18-P0**: k(N) 在 N 较小时相对于 N 增长更快 (超线性), 即 r(N) = k(N)/N 不是常数。

### 3.2 P1: 涌现深度预测公式

**目的**: 从 k(N) 映射推导预测涌现深度公式

**方法**: 
- 用 P0 数据拟合 k(N) 函数 (或构建查找表)
- 从 N0 开始迭代 N_{i+1} = 2*k(N_i) + residual(N_i)
- 预测深度 d 为满足 N_d ≥ 6 的最大层数

**假设 H18-P1**: 预测深度与实际测量深度在 ±1 层内一致 (跨 N0=12 到 96)。

### 3.3 P2: 涌现深度的理论最大值

**目的**: 找到使深度最大的 N0 和参数组合

**分析**:
- 对每个 N0, 计算理论最大深度
- 分析为什么 N0≈36 是最优 (从 P0 的 k(N) 数据)
- 参数灵敏度: bind_threshold / seal_fraction / min_org_size 的影响

**假设 H18-P2**: 最优 N0 由 r(N) 的最大值位置决定:
```
N0_opt = argmax (expected_depth(N0))
其中 depth(N0) = depth from N0→N1→...→N_d
```

### 3.4 P3: "整体"的精确定义

**理论分析**: 
- **整体 = 自指链的最大不动点**
- 第 d 层是描述「第 d-1 层结构」的命名场
- 当 N_d < 6 时, 第 d 层无法形成组织, 即没有新结构需要被命名
- 所以「整体」= 所有可被命名的结构的总和

**与差异论的对应**: 
- 整体不是先验存在的边界, 而是自指过程自然终止时的固定点
- 这对应于差异论中「封闭差异域」的概念
- 差异不可能无限嵌套, 因为命名位是有限资源

**假设 H18-P3**:
- 整体的大小 = Σ_{i=0}^{d-1} k_i (所有被命名的组织总数)
- 整体的信息内容 ≈ Σ k_i * log(N_i/k_i) (组织分布的复杂度)
