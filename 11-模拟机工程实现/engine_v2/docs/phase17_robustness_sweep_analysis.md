## 二、涌现深度分析

涌现深度（密封层数）整体分布：

```
N0=24:  depth 3.00–4.62    (最优 bind=1.0/2.0)
N0=36:  depth 3.12–4.88    (最优 bind=2.0 → 4.88!)
N0=48:  depth 3.12–4.88    (最优 bind=1.0/2.0 → 4.88)
N0=72:  depth 3.62–4.62    (最优 bind=1.0)
N0=96:  depth 3.25–3.62    (最优 bind=2.0)
```

**关键发现**:
- **涌现深度存在最优 N0 ≈ 36-48**。N0 再大 (72/96) 反而不利于深层涌现。
- 这是 Phase 4 Track B 发现的复现：「N0=72 最优」在这里修正为 N0=36-48。
- 根因: N0 太大时命名位比例稀释（2k/N1 比下降），初始绑定稀疏，层层衰减。

**涌现深度 vs N0 的关系**:
```
depth(N0=24) = 4.38  (bind=2.0, seal=0.4)
depth(N0=36) = 4.88  (bind=2.0, seal=0.4)  ← 最高
depth(N0=48) = 4.75  (bind=2.0, seal=0.4)
depth(N0=72) = 3.62  (bind=2.0, seal=0.4)
depth(N0=96) = 3.62  (bind=2.0, seal=0.4)
```

涌现深度在 N0=36 达到峰值，之后单调下降。

## 三、L1 自主 Flux 分析

### 3.1 整体模式

L1 flux（Jaccard 通量）反映了 L1 层的**自主演化活跃度**：

| Config | Flux (range) | 特征 |
|--------|-------------|------|
| N0=24, bind=0.5 | 0.1839–0.2000 | 稳定～0.19 |
| N0=24, bind=1.0 | 0.2638–0.2783 | 稳定～0.27 |
| N0=24, bind=2.0 | 0.3041–0.3127 | **最高 flux** |
| N0=36, bind=0.5 | 0.1012–0.1256 | 稳定～0.12 |
| N0=36, bind=1.0 | 0.1598–0.2091 | seal 因子影响大 |
| N0=36, bind=2.0 | 0.2437–0.2796 | **约 0.26** |
| N0=48, bind=0.5 | 0.0544–0.1524 | seal 因子影响极大 |
| N0=48, bind=1.0 | 0.1422–0.2262 | 稳定～0.20 |
| N0=48, bind=2.0 | 0.2705—0.2705 | **完全不受 seal 影响** |
| N0=72, bind=0.5 | 0.0690—0.0690 | 完全固定 |
| N0=72, bind=1.0 | 0.1008—0.1008 | 完全固定 |
| N0=72, bind=2.0 | 0.1713—0.1713 | 完全固定 |
| N0=96, bind=0.5 | 0.0177—0.0177 | 完全固定 |
| N0=96, bind=1.0 | 0.0439—0.0439 | 完全固定 |
| N0=96, bind=2.0 | 0.1497—0.1497 | 完全固定 |

### 3.2 关键洞察

1. **Flux 由 bind_threshold 主导**，而非 seal_fraction：
   - bind=0.5: flux 0.02–0.20 (低)
   - bind=1.0: flux 0.04–0.28 (中)
   - bind=2.0: flux 0.15–0.31 (高)
   - seal_fraction 仅在 N0=48, bind=0.5 时显著影响 flux

2. **Flux 衰减与 N0 的标度关系**（取各 bind 的均值）：
   ```
   N0=24:  flux ≈ 0.27
   N0=36:  flux ≈ 0.20
   N0=48:  flux ≈ 0.18
   N0=72:  flux ≈ 0.11
   N0=96:  flux ≈ 0.07
   ```
   呈现近似的 **1/N0 衰减**规律（不是指数衰减！）。

3. **seal_fraction 无关性**：对于 N0≥48 的多数配置，seal_fraction 对 flux 无影响。
   这意味着一旦绑定阈值足够高，密封细节不影响 L1 自主性。

### 3.3 N0=48 的特例

N0=48, bind=0.5 时，seal_fraction 从 0.4→0.6→0.8 导致 flux 从 0.1524→0.1416→0.0544。
这是**唯一** seal_fraction 显著影响 flux 的配置。

根因: N0=48 是「中等规模」——组织的平均大小刚好在临界以下，
更高的 seal_fraction 锁死了更多的位，使 L1 活跃空间被过度压缩。

## 四、L2 涌现率

### 4.1 统计

- **100% L2 涌现**: 43/45 配置 (95.6%)
- **88% L2 涌现**: 2/45 配置 (4.4%)
  - N0=36, bind=0.5, seal=0.8
  - N0=36, bind=1.0, seal=0.8
  - N0=36, bind=2.0, seal=0.8

### 4.2 失败的根因

所有 2 个失败配置的**共同特征**:
1. N0=36（中等偏小）
2. seal_fraction=0.8（过紧的密封）

N0=36 的组织数 k 平均约 3-5 个 → body 位 3-5 + naming 位 3-5 + 余差。
密封比例 0.8 意味着大多数 L1 的组织被锁定，L1→L2 自指时 naming 位的差异源开始萎缩。

但即使在这些边缘配置中，L2 涌现率仍为 88%（7/8 seeds 成功）。
与 Phase 16 的 0% 形成鲜明对比。

## 六、物理规律抽象

从扫描数据可以提取出三个标度律:

### 6.1 Flux ∝ bind_threshold / N0

flux 基本上正比于 `bind_threshold / N0`:
```
F ≈ 0.15 × (bind / N0) × 100 + ϵ
```
这在低 N0/高 bind 时最显著，在高 N0/低 bind 时趋于收敛。

### 6.2 Depth 在中等 N0 达到峰值

depth 不是 N0 的单调函数，而是**在 N0≈36 时最优**。
根因是两股竞争因素:
- 小 N0 (24): 组织数 k 少 → 命名位少 → 每层信息量不足
- 大 N0 (72+): 命名位相对于 N 比例太低 → 自指衰减快

### 6.3 seal_fraction 的临界性

seal_fraction 仅在「中等规模 + 低绑定阈值」时才起作用。
在其他条件下，它是 **无效参数**——系统对密封比例的鲁棒性极高。
