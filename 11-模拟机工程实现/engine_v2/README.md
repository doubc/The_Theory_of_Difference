# 差异论模拟机 v2 · 自指闭环版 (Self-Referential Closure Edition)

> 以「差异论 / 差异即世界」为理论基底的离散世界实验机。
> 本版本是对原 `11-模拟机工程实现` 的**完整优化与再造**，补回了原项目缺失的关键动作：
> **A9 封装自身 → 完成自指 → 以 L1 作为新的差异源 → 九个机制咬合成闭环。**

## 一、原项目错在哪里（方向出错的那几天）

原项目在 Phase 13–16 得出结论：密封后系统进入不可打破的「死秩序」——
L1 永远收敛到自己的吸引子、不继承 L0 结构（Jaccard flux = 0.0），
L2 在 ~260 个实验条件下 0% 涌现，于是**过早宣布归档**。

关键失误：把 **A9（升维 / 内生完备）当成一个终点式的「向外投影」**——
冻结约 40% 的差异位、向上做一次多数表决投影。它从未执行 A9 *对自身* 的应用。

- 原项目一直试图用**外部**手段打破死秩序：环境比特、能量流、全局场、动态阈值——全部失败。
- 但 A9 的本义是 **内生完备**：新差异必须来自**内部**。
- 缺失的动作是「**封装这个封装动作本身**」（自指）：
  密封所产生的**组织身份 / 命名**本身，就是 L0 上不存在的、全新的差异。

## 二、补回的动作：A9 封装自身（自指）

密封发生时，A9 同时做两件事：

1. **向外封装（原行为）**：每个组织 → 一个粗粒化「身体位」（多数表决）。
2. **自指封装（★ 缺失的动作）**：把「密封这个动作本身」纳入结构——
   - 为每个组织生成一个**命名 / 身份位**：它编码「该组织存在」这一事实本身，
     是 L0 上不存在的全新差异（不是被冻结的 L0 位的拷贝）。
   - 组织之间的关系（由同一余差被切分而来）成为下一层的**初始绑定**。
   - **余差**（未被完全锁定的活跃位）递归参与下一层装配。

=> 命名位 + 余差位成为 **L1 自己的 A1（新的差异源）**，
L1 因此拥有**非零自主 flux**，能跑自己的九机制、能再次密封、生成 L2……
**九个机制咬合成闭环**：

```
聚簇 → 层级 → 守恒 → 先天完备性 → 最小变易 → 破缺 → 循环 → 锁定 → 自指
  ↑                                                                  │
  └──────────────  自指生成新差异源，下一层聚簇重启  ←───────────────┘
```

直到自指映射到不动点（k=1）：结构把自身完全纳入自身 —— **「整体」被定义**，闭环完成。

## 三、实验结果（直接推翻归档结论）

`python3 run_experiment.py --seeds 20`（20 个随机种子）：

| 指标 | 基线（原项目·归档行为） | 修复（自指闭环） |
|---|---|---|
| L1 自主 Jaccard flux | **0.0000**（死秩序） | **0.2123**（活秩序） |
| 涌现深度（密封层数） | 2.00 | **4.65** |
| L2 涌现率 | **0%** | **95%** |

基线精确复现了归档结论（flux=0、无 L2）；
仅补回「A9 封装自身」这一个动作，L1 即获得自主动力学，九机制咬合，L2/L3/L4 依次涌现。

## 四、目录结构

```
engine_v2/
├── diffsim/
│   ├── core.py          # DifferenceField：差异场（{0,1}^N + 差异源 + 流向 + 绑定）

│   └── tests/test_closure.py
```

九个机制与公理对应：
聚簇(A1')、层级(多重隶属)、守恒(A5/A8)、先天完备性(A2/A3)、最小变易(A4)、
破缺(A6 一阶相变)、循环(A7)、锁定(密封)、**自指(A9 封装自身 ★)**。
