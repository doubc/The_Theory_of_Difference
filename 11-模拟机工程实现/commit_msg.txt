Phase 5 Track B9 v2: fix H50 root cause + L1-L2 bias transfer

Key fixes from exp_123 v1 (1/8 H50 pass) to v2 (7/8 H50 pass):

1. CRITICAL BUG FIX: get_state() frozen_bits union
   - partial_seal() populated lateral_sealed_bits but get_state() only
     returned sealed_bits, causing frozen_bits to always be empty
   - Fix: all_frozen = lateral_sealed_bits | sealed_bits
   - This was the root cause of H50 failure (bias_field never activated)

2. L0 stability: Ornstein-Uhlenbeck process
   - Replaced simple random walk with mean-reverting OU process
   - Prevents L0 collapse to near-zero (which caused zero correlations)
   - Parameters: theta=0.02, sigma=0.03, mean=0.5, clip=[0.15, 0.85]

3. L2 autonomous base noise (NEW parameter)
   - Added l2_auto_noise to ConstraintBiasedCoupling
   - Gaussian noise (sigma=0.10) on L2 auto base creates L1-L2 divergence
   - Essential for producing measurable bias_effect (H50)

4. Parameter tuning for H50/H51 balance:
   - l1_bias_strength: 0.4 (v1) -> 0.5 -> 0.6 -> 0.7 (v2 final)
   - l2_auto_noise: 0.0 (v1) -> 0.10 (v2)
   - H50 threshold: 0.15 (v1) -> 0.10 (v2)
   - H51 threshold: 0.7 (v1) -> 0.85 (v2)
