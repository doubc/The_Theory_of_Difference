## **1. 整数嵌入 $\iota: \mathbb{Z} \hookrightarrow M$**

### **1.1 定义**

由 (G3)，$\mathbf{u}$ 有唯一逆元 $\mathbf{u}^{-1} \in M$。

**定义 D5（整数嵌入）**：将 $\iota$ 从 $\mathbb{N}$ 扩展到 $\mathbb{Z}$：
$$\iota(0) := e,\quad \iota(n+1) := \iota(n) \circ \mathbf{u},\quad \iota(-n) := \bigl(\iota(n)\bigr)^{-1}\;\; (n > 0)$$

即 $\iota(-n) = (\mathbf{u}^{-1})^n = \mathbf{u}^{-n}$（$\circ$ 意义下）。

**定义 D6**：$\mathbb{Z}_M := \iota(\mathbb{Z})$。

### **1.2 负元的投影**

**引理 L2（$\mathbf{u}^{-1}$ 的投影）**：
$$\pi_b(\mathbf{u}^{-1}) = \ominus_b \pi_b(\mathbf{u}),\quad \pi_f(\mathbf{u}^{-1}) = 0_F,\quad \pi_\epsilon(\mathbf{u}^{-1}) = +$$

*证明*：设 $\mathbf{u}^{-1}$ 有投影 $(b, f, \epsilon)$。由 $\mathbf{u} \circ \mathbf{u}^{-1} = e$ 和 (P1)–(P3)：

- 基：$\pi_b(\mathbf{u}) \oplus_b b = 0_B \Rightarrow b = \ominus_b \pi_b(\mathbf{u})$。
- 扭转：$+ \cdot \epsilon = + \Rightarrow \epsilon = +$。
- 纤维：$0_F \oplus_f (+ \odot f) = 0_F \Rightarrow f = 0_F$。$\blacksquare$

**推论 C3**：$\mathbb{Z}_M \subseteq \{p \in M : \pi_f(p) = 0_F,\; \pi_\epsilon(p) = +\}$，与 $\mathbb{N}_M$
同层。即整数与自然数一样"从不翻面"。

### **1.3 $\iota|_\mathbb{Z}$ 的单射性**

**定理 T10**：$\iota: \mathbb{Z} \to M$ 是单射。

*证明*：由引理 L1 扩展和 L2，对任意 $n \in \mathbb{Z}$：
$$\pi_b(\iota(n)) = n \cdot \pi_b(\mathbf{u}) \quad (\text{其中 } n \cdot x := x \oplus_b \cdots \oplus_b x \text{，负数用 } \ominus_b\text{ 展开})$$

若 $\iota(m) = \iota(n)$，取 $\pi_b$ 得 $(m - n) \cdot \pi_b(\mathbf{u}) = 0_B$，由 $B$ 无挠性，$m = n$。$\blacksquare$

### **1.4 整数加法与减法的恢复**

**定理 T11（整数加法）**：对任意 $m, n \in \mathbb{Z}$：
$$\iota(m) \circ \iota(n) = \iota(m + n)$$

*证明*：分四种符号组合讨论。

- $m, n \geq 0$：即 T6。
- $m \geq 0, n < 0$：设 $n = -k$，$k > 0$。若 $m \geq k$：
  $$\iota(m) \circ \iota(-k) = \mathbf{u}^m \circ \mathbf{u}^{-k} = \mathbf{u}^{m-k} = \iota(m-k) = \iota(m+n)$$
  （中间一步用结合律和群中 $\mathbf{u}^m \circ \mathbf{u}^{-k} = \mathbf{u}^{m-k}$，这是标准群论事实。）若 $m < k$ 同理。
- 其余情形对称。$\blacksquare$

**定义 D7（减法）**：$\iota(m) -_M \iota(n) := \iota(m) \circ \iota(n)^{-1} = \iota(m-n)$。

### **1.5 整数乘法的恢复**

**定理 T12（整数乘法内部刻画）**：对任意 $m, n \in \mathbb{Z}$：
$$\iota(m \cdot n) = \bigl(\iota(m)\bigr)^n \quad (\circ \text{ 意义下的 } n \text{ 次幂，负幂用逆元})$$

*证明*：

- $n \geq 0$：T8。
- $n < 0$：$\iota(m)^n = (\iota(m)^{-1})^{|n|} = \iota(-m)^{|n|} = \iota(-m \cdot |n|) = \iota(mn)$。$\blacksquare$

**推论 C4**：乘法的符号规则 $(-m)(-n) = mn$ 等**不是外部假设**，而是群中 $(p^{-1})^{-1} = p$ 与 T12 的直接推论。

这修正了日志 §3.1 的隐含：日志把 $1+1=2$ 作为推导核心，而**负负得正**同样是莫比乌斯结构内生的。

## **2. $\mathbb{Z}_M$ 上的局部交换性**

### **2.1 交换性定理**

**定理 T13（$\mathbb{Z}_M$ 上 $\circ$ 交换）**：对任意 $p, q \in \mathbb{Z}_M$：
$$p \circ q = q \circ p$$

*证明*：$p = \iota(m), q = \iota(n)$。两边都等于 $\iota(m+n) = \iota(n+m)$（整数加法交换 + T11）。$\blacksquare$

**定理 T14（分配律在 $\mathbb{Z}_M$ 上）**：对任意 $m, n, k \in \mathbb{Z}$：
$$\iota(m) \times_M \bigl(\iota(n) +_M \iota(k)\bigr) = \bigl(\iota(m) \times_M \iota(n)\bigr) +_M \bigl(\iota(m) \times_M \iota(k)\bigr)$$

*证明*：左边 $= \iota(m \cdot (n+k)) = \iota(mn + mk) =$ 右边。$\blacksquare$

> **评注**：分配律**不是** $(M, \circ)$ 的独立公理，而是**两种不同解读 $\circ$（作为 $+_M$ 与作为 $\times_M$
）在 $\mathbb{Z}_M$ 上的一致性推论**。这精确兑现了日志中的断言："加法和乘法是同一拓扑操作的两个参数区间"
> ——分配律之所以成立，正因为它们本来就是同一个 $\circ$。

## **3. $(M, \circ)$ 的全局非交换性**

日志中强调莫比乌斯结构是非平凡的（否则塌缩为 $\mathbb{R}^3$）。我们需要**构造显式反例**证明 $(M, \circ)$ **整体**不是交换群。

### **3.1 反例构造**

取两个元素 $p, q \in M$ 满足 $\pi_\epsilon(p) = -$，$\pi_\epsilon(q) = +$，$\pi_f(q) \neq 0_F$。

由 (P2)：
$$\pi_f(p \circ q) = \pi_f(p) \oplus_f \bigl(\pi_\epsilon(p) \odot \pi_f(q)\bigr) = \pi_f(p) \oplus_f (\ominus_f \pi_f(q))$$
$$\pi_f(q \circ p) = \pi_f(q) \oplus_f \bigl(\pi_\epsilon(q) \odot \pi_f(p)\bigr) = \pi_f(q) \oplus_f \pi_f(p)$$

两者之差：
$$\pi_f(p \circ q) \ominus_f \pi_f(q \circ p) = \ominus_f \pi_f(q) \ominus_f \pi_f(q) = \ominus_f (2 \odot \pi_f(q))$$

由 $\pi_f(q) \neq 0_F$ 与 $F$ 无挠（F1 加上 $\{\pm\}$ 无 2-torsion 的结构假设），此差非零。

**定理 T15（$(M, \circ)$ 非交换）**：存在 $p, q \in M$ 使 $p \circ q \neq q \circ p$。$\blacksquare$

### **3.2 交换性的精确条件**

**定理 T16（局部交换性判据）**：$p \circ q = q \circ p$ 当且仅当
$$\bigl(\pi_\epsilon(p) \odot \pi_f(q)\bigr) \oplus_f \pi_f(p) = \bigl(\pi_\epsilon(q) \odot \pi_f(p)\bigr) \oplus_f \pi_f(q)$$

（基、扭转分量自动交换，因 $B$ 是交换群、$\{\pm\}$ 是交换群。）

*证明*：由 (P1)（基加法交换）与 (P3)（$\{\pm\}$ 交换），差异只在纤维分量。直接比较 (P2) 的两边。$\blacksquare$

**推论 C5（交换子集的结构）**：$(M, \circ)$ 的交换子集恰好是——
$$Z_{\text{comm}} = \{p : \pi_f(p) = 0_F\} \cup \{p : \pi_\epsilon(p) = +\}^\circ$$
（前者任意，后者需两两满足 T16 条件。）

$\mathbb{Z}_M$ 同时落在两者交集里——这就是整数能"安稳地"构成交换子环的拓扑原因。

## **4. $\mathbb{Z}_M$ 上的序结构**

### **4.1 序的移植**

**定义 D8（$\mathbb{Z}_M$ 上的序）**：
$$\iota(m) <_M \iota(n) \;:\Longleftrightarrow\; \pi_b(\iota(m)) <_b \pi_b(\iota(n))$$

**定理 T17（序的等价刻画）**：
$$\iota(m) <_M \iota(n) \;\Longleftrightarrow\; m < n \text{（在 } \mathbb{Z} \text{ 中）}$$

*证明*：$\pi_b(\iota(m)) = m \cdot \pi_b(\mathbf{u})$，$\pi_b(\iota(n)) = n \cdot \pi_b(\mathbf{u})$。由 (B3)
与 $\pi_b(\mathbf{u}) >_b 0_B$（**约定性选择**：固定 $\mathbf{u}$ 为"正向"
后继），$m < n \Leftrightarrow m \cdot \pi_b(\mathbf{u}) <_b n \cdot \pi_b(\mathbf{u})$。$\blacksquare$

> **约定**：$\pi_b(\mathbf{u}) >_b 0_B$ 是一个**方向选择**（选择哪一侧为正）。这是莫比乌斯带上"单向行走"
> 的初始条件。在公理体系中，可以把它加入 (U1) 作为强化条件。

### **4.2 序与运算的相容性**

**定理 T18（序-加法相容）**：若 $\iota(m) <_M \iota(n)$，则 $\iota(m) \circ \iota(k) <_M \iota(n) \circ \iota(k)$。

*证明*：两边 $\pi_b$ 为 $(m+k)\cdot\pi_b(\mathbf{u})$ 与 $(n+k)\cdot\pi_b(\mathbf{u})$，由 (B3) 得证。$\blacksquare$

**定理 T19（序-乘法相容）**：若 $\iota(k) >_M \iota(0)$（即 $k > 0$）且 $\iota(m) <_M \iota(n)$
，则 $\iota(m) \times_M \iota(k) <_M \iota(n) \times_M \iota(k)$。

*证明*：即证 $mk < nk$（$k > 0$ 时），这是 $\mathbb{Z}$ 的标准事实。$\blacksquare$
