# 莫比乌斯代数（Möb）：整数嵌入、局部交换性、分配律与全局非交换性

---

## 一、前置定理

**元定理 M1（Möb 是 PA 的保守扩张）**：
$$\mathbf{Möb} \vdash \varphi \quad\Longleftrightarrow\quad \mathbf{PA} \vdash \varphi^\iota$$

其中 $\varphi^\iota$ 是把 $\varphi$ 中量化限制到 $\mathbb{N}_M$ 的翻译。

$$\mathbf{Möb} = \mathbf{PA} + \text{莫比乌斯拓扑结构（$B$, $F$, $\{\pm\}$ 三层）}$$

体系价值不在于公理更少，而在于：加法与乘法从同一运算 $\circ$ 涌现；$\pi, e, i$ 是拓扑不变量；解析延拓有统一拓扑图景。

### 记号约定

$$\boxed{M = (B \times F \times \{\pm\}) / \sim,\quad (b, f, \epsilon) \sim (b \oplus_b \omega,\; \ominus_f f,\; \bar{\epsilon})}$$

$\omega := \pi_b(\pi_M)$ 为粘合步长。

## 二、整数嵌入

由 (G3)，$\mathbf{u}$ 有唯一逆元 $\mathbf{u}^{-1} \in M$。

**定义 D5**：
$$\iota(0) := e, \qquad \iota(n+1) := \iota(n) \circ \mathbf{u}, \qquad \iota(-n) := (\iota(n))^{-1} \;\;(n > 0)$$

**定义 D6**：$\mathbb{Z}_M := \iota(\mathbb{Z})$。

### 负元的投影

**引理 L2**：
$$\pi_b(\mathbf{u}^{-1}) = \ominus_b \pi_b(\mathbf{u}), \qquad \pi_f(\mathbf{u}^{-1}) = 0_F, \qquad \pi_\epsilon(\mathbf{u}^{-1}) = +$$

*证明*：由 $\mathbf{u} \circ \mathbf{u}^{-1} = e$ 和 (P1)–(P3)：基 $\Rightarrow b = \ominus_b \pi_b(\mathbf{u})$；扭转 $\Rightarrow \epsilon = +$；纤维 $\Rightarrow f = 0_F$。✓ $\blacksquare$

**推论 C3**：$\mathbb{Z}_M \subseteq \{p : \pi_f(p) = 0_F,\; \pi_\epsilon(p) = +\}$，与 $\mathbb{N}_M$ 同层。

### 单射性

**定理 T10**：$\iota: \mathbb{Z} \to M$ 是单射。

*证明*：$\pi_b(\iota(n)) = n \cdot \pi_b(\mathbf{u})$。相等推 $(m-n) \cdot \pi_b(\mathbf{u}) = 0_B$，$B$ 无挠 $\Rightarrow m = n$。✓ $\blacksquare$

### 整数加法

**定理 T11**：$\iota(m) \circ \iota(n) = \iota(m+n)$。

*证明*：四种符号组合。$m, n \geq 0$：即 T6。$m \geq 0, n = -k < 0$：$\iota(m) \circ \iota(-k) = \mathbf{u}^m \circ \mathbf{u}^{-k} = \mathbf{u}^{m-k} = \iota(m+n)$。其余同理。✓ $\blacksquare$

**定义 D7**：$\iota(m) -_M \iota(n) := \iota(m) \circ \iota(n)^{-1} = \iota(m-n)$。

### 整数乘法

**定理 T12**：$\iota(m \cdot n) = (\iota(m))^n$（$\circ$ 意义下，负幂用逆元）。

*证明*：$n \geq 0$ 为 T8。$n < 0$：$\iota(m)^n = (\iota(m)^{-1})^{|n|} = \iota(-m)^{|n|} = \iota(mn)$。✓ $\blacksquare$

**推论 C4**：$(-m)(-n) = mn$ 不是外部假设，而是 $(p^{-1})^{-1} = p$ 与 T12 的推论。

## 三、$\mathbb{Z}_M$ 上的局部交换性

**定理 T13（交换律）**：$\iota(m) \circ \iota(n) = \iota(n) \circ \iota(m)$。

*证明*：两边都等于 $\iota(m+n) = \iota(n+m)$。✓ $\blacksquare$

**定理 T14（分配律）**：
$$\iota(m) \times_M (\iota(n) +_M \iota(k)) = (\iota(m) \times_M \iota(n)) +_M (\iota(m) \times_M \iota(k))$$

*证明*：左边 $= \iota(m(n+k)) = \iota(mn+mk) =$ 右边。✓ $\blacksquare$

分配律不是独立公理——它是两种不同解读 $\circ$（作为 $+_M$ 和作为 $\times_M$）在 $\mathbb{Z}_M$ 上的一致性推论。加法和乘法本来就是同一个 $\circ$。

## 四、$(M, \circ)$ 的全局非交换性

### 反例

取 $\pi_\epsilon(p) = -$，$\pi_\epsilon(q) = +$，$\pi_f(q) \neq 0_F$。由 (P2)：

$$\pi_f(p \circ q) = \pi_f(p) \oplus_f (\ominus_f \pi_f(q)), \qquad \pi_f(q \circ p) = \pi_f(q) \oplus_f \pi_f(p)$$

差值 $\ominus_f (2 \odot \pi_f(q)) \neq 0_F$（$F$ 无挠）。

**数值示例**（$B = F = \mathbb{R}$ 模型）：
$$p = (0, 1, -), \quad q = (0, 1, +)$$
$$p \circ q = (0, 0, -), \quad q \circ p = (0, 2, -)$$

先反面后正面，第二步振幅被负扭转抵消；先正面后反面，振幅原样累加。

**定理 T15**：$\exists p, q \in M$ 使 $p \circ q \neq q \circ p$。$\blacksquare$

### 交换性的精确条件

**定理 T16**：$p \circ q = q \circ p$ 当且仅当
$$(\pi_\epsilon(p) \odot \pi_f(q)) \oplus_f \pi_f(p) = (\pi_\epsilon(q) \odot \pi_f(p)) \oplus_f \pi_f(q)$$

**推论 C5**：交换子集 $Z_{\text{comm}} = \{p : \pi_f(p) = 0_F\} \cup \{p : \pi_\epsilon(p) = +\}^{\circ}$。$\mathbb{Z}_M$ 落在两者交集——整数安稳构成交换子环的拓扑原因。

## 五、$\mathbb{Z}_M$ 上的序结构

**定义 D8**：$\iota(m) <_M \iota(n) :\Longleftrightarrow \pi_b(\iota(m)) <_b \pi_b(\iota(n))$

**定理 T17**：$\iota(m) <_M \iota(n) \Longleftrightarrow m < n$（在 $\mathbb{Z}$ 中）。

*证明*：$\pi_b(\iota(m)) = m \cdot \pi_b(\mathbf{u})$，由 (B3) 和方向选择 $\pi_b(\mathbf{u}) >_b 0_B$。✓ $\blacksquare$

**定理 T18（序-加法相容）**：$\iota(m) <_M \iota(n) \Rightarrow \iota(m) \circ \iota(k) <_M \iota(n) \circ \iota(k)$。

**定理 T19（序-乘法相容）**：$\iota(k) >_M \iota(0)$ 且 $\iota(m) <_M \iota(n) \Rightarrow \iota(m) \times_M \iota(k) <_M \iota(n) \times_M \iota(k)$。
