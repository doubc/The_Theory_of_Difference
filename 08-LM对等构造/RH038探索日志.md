## 二、代码诊断：同义反复的三层包装

### 2.1 `_predict_by_phase` 的实际逻辑

完整解析 `素数贡献推测.py` 后，确认 `phase_accumulation` 方法本质上是**有界窗口内的素数搜索**，不是相位场预测。

完整逻辑链：

```
搜索窗口内枚举奇数
        ↓
筛选：无平方因子 + ω奇数 + ω≤3
        ↓
        ├── ω=1 的候选：素数
        └── ω=3 的候选：三素因子合数
        ↓
排序：优先 ω=1
        ↓
取第一个 → 搜索窗口内第一个奇素数
```

等价于直接写：
```python
for n in range(current_p + 2, search_end, 2):
    if is_prime(n):
        return n
```

**相位条件 $\phi(n) = \pi$ 的问题**：$\phi(n) = \pi \cdot \Omega(n) \bmod 2\pi$，所以 $\phi(n) = \pi$ 等价于 $\Omega(n)$ 为奇数——这是一个多对一映射，把 $\Omega = 1, 3, 5, \ldots$ 全部折叠到同一个值。mod 操作主动丢弃了区分它们所需的信息，相位条件携带的信息量仅等于 $\Omega(n) \bmod 2$。

**median_error = 0.0 的来源**：搜索窗口 $3\ln(p_k)$ 在测试范围（$p_{1000} \sim p_{2000}$）内远大于实际间隙中位数，真实的 $p_{k+1}$ 几乎必然落在窗口内，命中是必然的，与相位无关。

### 2.2 两个函数的本质区别

| | `gap_statistics` | `phase_accumulation` |
|---|---|---|
| 输出类型 | 连续实数估计 | 从候选集选整数 |
| 误差来源 | 间隙估计偏差 | 窗口是否覆盖真实值 |
| 相位的作用 | 无 | 实际上也无（被排序键覆盖）|
| 本质 | 统计回归 | 有界窗口内的素数搜索 |

median_error 的差距来自"连续估计 vs 离散搜索"的结构差异，和相位场无关。

### 2.3 代码的真实价值

`analyze_composite_contributions` 产生的数据是真实的——`mean_contribution_ratio ≈ 1/3`、`median_omega = 3.0` 是有效的统计规律。搜索框架本身的结构也是对的，问题在于筛选条件和排序键合在一起等价于素数判定，需要替换为不依赖素数判定的独立条件。

## 三、加法与乘法的结构性重新定义

### 3.1 传统定义的问题

传统定义把乘法描述为"重复加法"，把加法描述为"平移"。这两个定义是操作性的，没有揭示两种运算在结构层级上的本质差异。

### 3.2 新定义：相位语言

**加法 = 相位线性叠加**：

$$\phi(a + b) = \phi(a) + \phi(b)$$

差异沿同一方向累积，方向不变，可逆，零元透明（$a + 0 = a$，零不吸收信息）。对应莫比乌斯带的平行堆叠：$M \oplus M$，特征类 $w_1 + w_1 = 0 \pmod{2}$，扭转抵消，方向性恢复。

**乘法 = 相位扭转复合**：

$$\phi(a \times b) = \phi(a) \cdot \phi(b) \pmod{2\pi}$$

差异在复合时发生方向翻转，不可完全逆转，零元吸收（$a \times 0 = 0$，信息在零处销毁）。对应莫比乌斯带的扭转融合：$M \otimes M$，特征类 $w_1^2 \in H^2$，需要升维才能容纳。

### 3.3 两种运算的结构对比

| 性质 | 加法 | 乘法 |
|---|---|---|
| 相位操作 | 线性叠加 | 扭转复合 |
| 逆元 | 存在（$-a$） | 不存在（$1/a \notin \mathbb{Z}$）|
| 零元行为 | 透明（$a+0=a$）| 吸收（$a\times 0=0$）|
| 结构 | 群 | 幺半群 |
| 拓扑对应 | 圆柱（$M\oplus M$）| 克莱因瓶特征类（$M\otimes M$）|
| 维度效应 | 保持维度 | 升维 |
| 时间对应 | 单向流动 | 时间箭头折叠 |

### 3.4 定义域的自然限制

乘法的吸收元问题（$a \times 0 = 0$）在相位语言里的处理：$\phi(0)$ 未定义，定义域天然排除 0，规避了吸收元破坏群结构的问题。这不是人为回避，而是相位场本身的结构要求——相位是差异的编码，无差异态（$\mathbf{0}$）没有相位。

### 3.5 指数的位置

指数 = 扭转的扭转：

$$a^b = \underbrace{a \times a \times \cdots \times a}_{b \text{ 次}}$$

是对乘法操作本身的重复，即元操作。对应克莱因瓶在更高维度的推广——方向结构的完全升维。A3 的三阶迭代（加法→乘法→指数）在相位语言里是扭转层级的三个层次。

## 四、非面积点与 A9 筛选机制

### 4.1 非面积点的定义

**非面积点**：素数出现的位置，在这个位置上系统必须发生转折，无法继续待在已有元素张成的"面"里。

形式化：设前 $k$ 个素数 $p_1, \ldots, p_k$ 在乘法下生成半群

$$\langle p_1, \ldots, p_k \rangle_\times = \{p_1^{a_1} \cdots p_k^{a_k} : a_i \geq 0\}$$

这是已有结构张成的"面"。

**合数**：$n \in \langle p_1, \ldots, p_k \rangle_\times$，落在面内，可由已有元素组合生成，引入它是冗余的（违反 A9）。

**素数** $p_{k+1}$：$p_{k+1} \notin \langle p_1, \ldots, p_k \rangle_\times$，是面外第一个点，无法由已有元素生成，引入它是必要的（符合 A9）。

### 4.2 A9 筛选的精确表述

$$p_{k+1} = \min\{n > p_k : n \notin \langle p_1, \ldots, p_k \rangle_\times\}$$

这和 Eratosthenes 筛精确等价，但语言是"生成半群的补集"而非"划掉倍数"。两者的计算路径不同，语义不同——前者是结构性的（什么是必要节点），后者是操作性的（哪些数被排除）。

### 4.3 非面积点与莫比乌斯扭转的对应

- **乘法视角**：合数在已有素数张成的乘积面内，素数在面外，是新维度的起点
- **莫比乌斯扭转视角**：合数是已有扭转的复合（$\Omega > 1$），素数是第一次单次扭转（$\Omega = 1$），是扭转层级的原子单元

两者说的是同一件事：**素数是不可约的扭转单元，合数是可约的扭转复合。**

## 五、A5 守恒律的数论表达

### 5.1 A5 的两个层面

**物理层面**（`02-axioms.md`）：存在守恒量 $Q(x)$，在所有允许演化下保持不变。

**本体论层面**（差异论 V1.6）：差异不能被无代价清零——不是数值相加为零，而是任何消除都有其对应的代价。

### 5.2 候选命题：莫比乌斯反演是 A5 的数论表达

$$\sum_{d|n} \mu(d) = [n=1]$$

这个等式的结构与 A5 的两个层面精确对应：

- **守恒量存在**：对每个 $n > 1$，所有因子的莫比乌斯值之和恰好为零——差异在分解过程中完整守恒，没有净剩余
- **不能无代价清零**：每一项 $\mu(d)$ 是一次有代价的抵消，$+1$ 项和 $-1$ 项精确配对，没有任何差异被免费消除

### 5.3 A9 与 A5 的协同

两个公理在素数分布里形成完整的互补：

| 公理 | 作用 | 数论表达 |
|---|---|---|
| A9 | 确定哪些节点是必要的 | 素数 = 生成半群补集的最小元 |
| A5 | 确定分解过程如何守恒 | 莫比乌斯反演 $\sum_{d\|n}\mu(d)=[n=1]$ |

**叙事**：素数是 A9 筛选的必然涌现（必要节点），合数是 A5 守恒分配的结果（差异被完整分解到素因子），两者是同一个机制的两个侧面。

### 5.4 当前状态声明

这是**结构上自洽的候选解释**，不是证明。

缺口：从"莫比乌斯反演是 A5 的数论表达"到严格推导，需要：
1. 守恒量 $Q(x)$ 在素数分布语境下的具体形式
2. 证明 $Q$ 守恒必然导出 $\sum_{d|n}\mu(d) = [n=1]$
3. 这个守恒量与 $\zeta(s)$ 的连接（$\operatorname{Re}(s) > 1$ 的收敛条件）
