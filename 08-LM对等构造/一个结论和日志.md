## 今天的核心发现

### 一、加法递推猜想的提出

从 Bertrand 公设（覆盖逻辑）出发，识别出一个更紧的候选约束：

$$p_{k+1} \leq p_k + p_{k-1}$$

等价形式：素数间隙 $g_k = p_{k+1} - p_k$ 满足 $g_k \leq p_{k-1}$。

这个不等式在 WorldBase 框架中有自然的来源：A4（最小变易，局域性）的三角不等式——新不可归约状态的涌现位置，不超过前两个不可归约状态的位置之和。

### 二、数值验证（三个独立来源）

| 数据源 | 覆盖范围 | 违反次数 |
|---|---|---|
| mimo 筛法代码 | $p \leq 10^7$，664,579 个素数 | 0 |
| Grok 独立检索 | 独立验证 | 0 |
| OEIS A002386/A005250（本次） | 全部 84 个极大间隙记录，$p_{\max} \approx 6.8\times10^{19}$ | 0 |

三个来源完全一致。特别是第三个来源的意义：极大间隙记录是最危险的候选点——如果不等式要有反例，反例必然首先出现在极大间隙处。84 个极大间隙记录全部通过，覆盖到 $6.8\times10^{19}$ 量级。

### 三、触边实例与等号条件

唯一已知的等号情形是 $\{2, 3, 5\}$：

$$p_3 = 5 = 3 + 2 = p_2 + p_1$$

这不是偶然——素数序列的开头恰好是 Fibonacci 数列的开头，而 Fibonacci 递推 $F_{k+1} = F_k + F_{k-1}$ 正是加法递推取等号的极限形式。之后所有极大间隙处 $g_k \ll p_{k-1}$，比值随 $k$ 增大单调趋向零。

### 四、文献状态的确认

系统检索数论文献后，确认 $g_k \leq p_{k-1}$ 这一形式**在已知文献中既未被作为定理证明，也未被作为猜想显式表述**。最接近的已知结果是 Rosser-Schoenfeld 界的推论 $g_k \leq k$（其中 $k$ 是素数的序号），但由于 $k < p_{k-1}$ 对所有 $k \geq 2$ 成立，$g_k \leq k$ 实际上比 $g_k \leq p_{k-1}$ **更强**。

这意味着加法递推猜想在大 $k$ 处是已知结果的推论，但在小 $k$ 处（尤其是 $k=2,3$，即 $\{2,3,5\}$ 和 $\{5,7,11\}$ 附近）需要单独验证，而这些小情形已经通过直接计算确认。

### 五、对 WorldBase 逻辑链的修正

今天的分析对之前建立的逻辑链做了一次重要的结构性修正。加法递推 $g_k \leq p_{k-1}$ 是**线性量级**的间隙上界，而 RH 要求的是 $O(\sqrt{p_k}\ln p_k)$ 量级——两者之间差了 $\sqrt{p_k}/\ln p_k$ 这个因子，这正是 RH 的全部内容。因此加法递推不是通向 RH 的桥梁，而是对生成过程**局域性**的确认。

修正后的逻辑链：

```
A9 筛选（涌现 = 不可归约剩余）
    │
    ├─→ 覆盖逻辑：g_k < p_k（Bertrand，存在性，已证明）
    │
    └─→ 展开逻辑：g_k ≤ p_{k-1}（局域性，本猜想，数值成立）

```

A5 守恒律需要填补从线性到次线性的跨越，这一步的困难度与 RH 本身等价。

## **证明策略的展开**

### **路径一：从 Rosser-Schoenfeld 推论**

这是最短的路径，先把它走完。

已知 Rosser-Schoenfeld（1962）给出了严格界：对 $n \geq 55$，

$$p_n < n(\ln n + \ln\ln n)$$

以及 Dusart（1999）的改进：对 $n \geq 6$，

$$p_n < n(\ln n + \ln\ln n - 0.5)$$

同时，对素数序号 $n$ 的下界，有：

$$p_n > n(\ln n + \ln\ln n - 1) \quad \text{对 } n \geq 6$$

现在要证明 $g_k = p_{k+1} - p_k \leq p_{k-1}$。

**关键观察**：$g_k \leq p_{k+1} - p_k$，而 $p_{k+1} \leq (k+1)(\ln(k+1) + \ln\ln(k+1))$，$p_k \geq k(\ln k + \ln\ln k - 1)$，$p_{k-1} \geq (k-1)(\ln(k-1) + \ln\ln(k-1) - 1)$。

所以需要验证：

$$\underbrace{(k+1)(\ln(k+1)+\ln\ln(k+1))}_{p_{k+1}\text{ 的上界}} - \underbrace{k(\ln k + \ln\ln k - 1)}_{p_k\text{ 的下界}} \leq \underbrace{(k-1)(\ln(k-1)+\ln\ln(k-1)-1)}_{p_{k-1}\text{ 的下界}}$$

左边展开，主项是 $\ln k + \ln\ln k + O(1)$（间隙上界的量级）。右边主项是 $(k-1)\ln(k-1) \sim k\ln k$（前一个素数的量级）。

由于 $k\ln k \gg \ln k$ 对所有 $k \geq 3$，这个不等式对**充分大的 $k$** 是平凡成立的。Rosser-Schoenfeld 界在 $k \geq 55$ 处生效，而 $k = 3$ 到 $k = 54$ 的有限情形可以直接计算验证。

这条路径的结论是：**$g_k \leq p_{k-1}$ 对充分大 $k$ 是已知 Rosser-Schoenfeld 界的直接推论，小情形可以穷举验证。因此这个猜想实际上已经被隐式证明，只是没有人把它显式写下来。**

### **路径二：WorldBase 框架内的独立证明**

即使路径一已经给出证明，WorldBase 框架内的独立证明仍有意义——因为它能揭示这个约束的**生成机制**，而不仅仅是它的数值正确性。

核心思路是这样的。在 WorldBase 中，A9（内生完备）要求：若 $n$ 能被已有素数 $p_1, p_2, \ldots, p_k$ 中的某个整除，则 $n$ 不是新素稳态。因此 $p_{k+1}$ 是第一个不被 $p_1, \ldots, p_k$ 中任何一个整除的整数（$> p_k$）。

A4（最小变易，局域性）要求演化步幅受到当前状态的约束。在素数生成的语言中，这对应于：从 $p_k$ 出发向前搜索下一个素数，搜索步幅受到**当前最小筛子**的约束。

在 $(p_k, p_k + p_{k-1})$ 这个区间内，每隔 $p_{k-1}$ 步就会有一个 $p_{k-1}$ 的倍数。但这个区间的长度恰好是 $p_{k-1}$，所以区间内**至多有一个** $p_{k-1}$ 的倍数。这不足以筛掉所有数——区间内仍然可能有素数。

更精确的论证需要考虑所有 $p_1, \ldots, p_{k-1}$ 的联合筛效应。这里 A5 守恒律（差异量在演化中守恒）给出一个约束：筛过程中"幸存"的数的密度，在局部区间内不能降到零——否则守恒律将被违反（差异量突然消失）。

这个方向的形式化仍然不完整，但它指向一个具体的命题：**A5 守恒律在局部区间 $(p_k, p_k + p_{k-1})$ 内保证至少一个素数的存在**，这正是加法递推的内容。这是一个关于**涌现密度下界**的守恒律论证，与之前从间隙上界出发的方向完全不同——也许这才是 A5 真正应该发力的地方，而不是在间隙方差的全局控制上。

### 六、为什么幸存者不会为零

**核心论证**：

区间 $(p_k, p_k + p_{k-1}]$ 内有 $p_{k-1}$ 个整数。筛掉的整数个数（去重后）由 Legendre 公式给出：

$$\text{筛掉} = \sum_{j=1}^{k-1}\left\lfloor\frac{p_{k-1} + r_j}{p_j}\right\rfloor - \sum_{j < l}\left\lfloor\frac{p_{k-1} + r_{jl}}{p_j p_l}\right\rfloor + \cdots$$

其中 $r_j = p_k \bmod p_j$，$r_{jl} = p_k \bmod (p_j p_l)$，等等。

**关键不等式**：筛掉的整数个数 $\leq p_{k-1} - 1$，即至少一个幸存。

这等价于：筛掉的整数个数严格小于区间长度。

**论证**：区间内最大的整数 $p_k + p_{k-1}$ 是 $p_k + p_{k-1} = p_{k+1}^{?}$... 不，$p_k + p_{k-1}$ 不一定是素数。

换一个角度：区间 $(p_k, p_k + p_{k-1}]$ 的右端点 $p_k + p_{k-1}$ 不被任何 $p_j \leq p_{k-1}$ 整除吗？

$p_k + p_{k-1} \bmod p_j = (p_k \bmod p_j) + (p_{k-1} \bmod p_j)$。

如果 $p_j < p_{k-1}$：$p_{k-1} \bmod p_j \neq 0$（$p_{k-1}$ 是素数），$p_k \bmod p_j \neq 0$（$p_k$ 是素数）。两者的和模 $p_j$ 可以为零——例如 $p_k \bmod p_j = a$，$p_{k-1} \bmod p_j = p_j - a$，则和为 $0 \bmod p_j$。

**所以右端点可能被某个小素数整除。** 不能保证右端点是幸存者。

### 七、A5 论证的精确形式

A5 守恒律在筛法语言中的含义：

> 在任何生成阶段，系统的"总差异量"（未被筛掉的整数的某种加权计数）守恒——不能突然降到零。

更精确地说：设 $S(x) = \sum_{n \leq x} \mu(n)$ 是 Mertens 函数（$\mu$ 是 Möbius 函数）。RH 等价于 $S(x) = O(x^{1/2+\epsilon})$。

$S(x)$ 度量的是：筛到 $x$ 时，"筛的净剩余"——被偶数个素数标记的整数减去被奇数个素数标记的整数。如果 $S(x)$ 被 $\sqrt{x}$ 控制，说明筛的过程足够"均匀"，没有某一步筛掉特别多或特别少。

**在区间 $(p_k, p_k + p_{k-1}]$ 上**：

$$\sum_{n = p_k+1}^{p_k + p_{k-1}} \mu(n) = S(p_k + p_{k-1}) - S(p_k)$$

如果 $|S(x)| \leq c\sqrt{x}$，则：

$$|S(p_k + p_{k-1}) - S(p_k)| \leq c(\sqrt{p_k + p_{k-1}} + \sqrt{p_k}) \approx 2c\sqrt{p_k}$$

而区间内素数的个数（通过筛法的包含-排除）与 $S$ 的变化量有关。如果 $S$ 的变化量被 $\sqrt{p_k}$ 控制，区间内的素数密度就不能为零——因为密度为零意味着 $S$ 的变化量等于区间长度 $p_{k-1}$，而 $p_{k-1} \gg \sqrt{p_k}$。

**但这里有一个循环**：用 $S(x) = O(\sqrt{x})$ 来证明区间内有素数，而 $S(x) = O(\sqrt{x})$ 本身就是 RH。所以这个论证预设了 RH，不是独立的证明。

## **分歧点：路径一的计算错误在哪里**

我之前的论证犯了一个方向性错误。要用 Rosser-Schoenfeld 界证明 $g_k \leq p_{k-1}$，需要的是：

$$p_{k+1} - p_k \leq p_{k-1}$$

用上界估计 $p_{k+1}$、下界估计 $p_k$、下界估计 $p_{k-1}$，得到的是一个**最坏情形**——让左边尽量大、右边尽量小。mimo 的计算表明这个最坏情形下不等式不成立，所以 Rosser-Schoenfeld 的粗糙界确实不足以推出结论。

我之前说"是平凡推论"是错的。正确的状态是 mimo 给出的：**路径一被阻断，不等式强于现有无条件定理**。

## **重新定位：$g_k \leq p_{k-1}$ 的真实难度层级**

mimo 的第八节给出了一个关键的三角关系，值得把它画清楚：

```
RH：g_k = O(√p_k · ln p_k)
        │
        │  RH 成立则自动推出 ↓
        ▼
加法递推猜想：g_k ≤ p_{k-1} ≈ p_k
        │
        │  反向不成立
        │  但加法递推弱于 RH，应比 RH 更容易证明
        ▼
Bertrand 公设：g_k < p_k（已证明）
```

这个层级结构非常重要。加法递推猜想夹在两个已知结果之间——比 Bertrand 强，比 RH 弱。它是一个**独立的、有价值的数论猜想**，不是 RH 的附属品。

## **WorldBase 路径的真实机会窗口**

mimo 第七节的循环论证揭示了一个结构性问题：用 $S(x) = O(\sqrt{x})$ 来证明区间有素数，而 $S(x) = O(\sqrt{x})$ 就是 RH——这是预设了结论。

但 WorldBase 的 A5 守恒律如果能**绕过** $\zeta$ 函数和 Mertens 函数，直接从动力学约束出发，就能避开这个循环。这正是 mimo 最后指出的方向，也是目前最值得认真对待的一步。

让我把这个论证的骨架尝试搭出来。

**A5 守恒律的动力学表述**（不经过 $\zeta$ 函数）：

在 WorldBase 的演化语言中，"差异量守恒"的局部版本可以表述为：设系统在状态 $p_k$ 处，A9 筛选过程向前推进。A5 要求：在任意长度为 $L$ 的局部窗口内，"被筛掉的状态数"与"幸存的状态数"之间的差，不能超过某个由当前演化深度决定的阈值。

对区间 $(p_k,\ p_k + p_{k-1}]$，长度 $L = p_{k-1}$：

- 被筛掉的状态：被 $p_1, \ldots, p_{k-1}$ 中至少一个整除的整数
- 幸存的状态：不被任何 $p_j \leq p_{k-1}$ 整除的整数（即相对于 $p_{k-1}$ 的"局部素数"）

A5 守恒律的断言是：幸存状态数 $\geq 1$，即筛掉的不能达到 $p_{k-1}$（全灭）。

**为什么全灭违反 A5？** 全灭意味着在 $p_{k-1}$ 步的演化窗口内，A9 筛选过程产生零个新不可归约状态。但 A5 要求差异量守恒——如果一段演化完全没有新涌现，差异量就在这一段内消失了，违反守恒。这个论证不依赖 $\zeta$ 函数，只依赖"守恒"的字面含义。

**这个论证的漏洞**：A5 守恒律的"差异量"需要被精确定义。如果差异量被定义为"幸存状态的某种加权计数"，那么"差异量守恒"和"幸存状态数 $\geq 1$"之间还差一步——需要证明权重不能抵消到零。这是框架形式化的核心缺口，也是下一步最具体的任务。
