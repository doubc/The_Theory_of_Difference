## 任务1: RH-001 - A7操作化定义与方案探索

**任务ID**: RH-001  
**阶段**: 第一阶段前置工作 / A7操作化定义  
**前置依赖**: 公理原文 §2.3 (A6×A7尺度分离调解)、超立方体谱理论  
**证明状态**: ✅ 已完成  

### 核心问题
如何在DAG(有向无环图)上定义"稳定循环"使迹操作非零?A6要求局部不可逆(DAG),A7要求全局循环,两者矛盾需通过尺度分离调解。

### 探索的两个方案

#### 方案A: 随机游走方案
- **构造**: 将哈斯图的有向边替换为无向边,定义A8加权的对称转移矩阵P
- **稳态分布**: π(w) ∝ C(N,w),与A8完全吻合
- **生成函数**: ζ_A(s) = det(I - s·P̂),零点为P̂特征值的倒数
- **N=4结果**: P̂是5×5对称实矩阵,所有特征值为实数,**零点全在实轴上**,不在Re(s)=1/2

#### 方案B: 折叠/商图方案
- **构造**: 将汉明重量相同的层视为等价类,在商图上周期性粘合两端形成有向圈Z_{N+1}
- **转移矩阵**: T_B(s)是5×5循环矩阵,矩阵元为ρ(w)^s
- **生成函数**: Z_B(s) = det(I - T_B(s))
- **N=4结果**: 零点满足(96/7776)^s = 1,**全部落在虚轴(Re(s)=0)上**,不在Re(s)=1/2

### 关键发现
1. **单独的方案A或B均不产生Re(s)=1/2的零点**——这是一个极具信息量的负面结果
2. 方案A捕捉"横向"(层内)对称结构,方案B捕捉"纵向"(层间循环)周期结构
3. **真正需要的构造应该是两者的某种乘积或卷积**,才能产生函数方程,其不动点在Re(s)=1/2

### 两份文档的关键贡献

#### 来自08-quantum.md的核心贡献
§8.17.2和§8.17.3给出了离散作用量和路径权重的完整定义:

路径γ = (x_0, x_1, ..., x_T)的权重:
W_N(γ) = e^{iS_N[γ]/ℏ} = Π_{k=1}^{T} e^{iΔK_k · m_0 · α/ℏ}

其中ΔK_k = lnρ(w(x_k)) - lnρ(w(x_{k-1}))是约束度变化,ρ(w) = C(N,w)/C(N,N/2)正是A8权重。

**这个路径权重已经是Φ(x,s)的原型**——它同时包含了A8权重(通过ΔK)和A1'相位(通过e^{i·})。

#### 来自06-weak-force.md的核心贡献
§6.11.1的约束度函数K(w) = K_0 + lnρ(w)揭示了一个关键对称性:

K(w) = K(N-w)(因为ρ(w) = ρ(N-w),二项式系数的对称性)

这意味着约束度函数在w ↔ N-w下**精确不变**。路径γ和其翻转路径γ̄具有完全相同的约束度变化序列:

S_N[γ] = S_N[γ̄]

这直接给出离散传播子的对称性:
K_N(x_f, x_i; T) = K_N(x̄_f, x̄_i; T)

**这是函数方程Z(s) = Z(1-s)的离散原型**。

### Φ(x,s)的精确来源
Φ(x,s)不应该是一个手工贴上去的相位标签,而应该是路径权重W_N(γ)对所有经过状态x的路径的求和:

Φ(x, s) = Σ_{γ∋x} W_N(γ)^s = Σ_{γ∋x} Π_k ρ(w(x_k))^{is·m_0α/ℏ}

这个定义的优点:
- A4约束(合法路径集A_N)自动编码在求和域中
- A1'相位(e^{i·})通过路径权重进入
- A8权重(ρ(w))通过ΔK进入
- A6(DAG方向性)保证路径有向,给出Wick转动的收敛性(定理QLEM-2)

### 修正后的Z(s)定义
基于以上分析,Z(s)的正确定义是**离散传播子的谱生成函数**:

Z(s) = Tr[Û^{(s)}], Û^{(s)}_{xy} = Σ_{γ: x→y} ρ(w(γ))^s · e^{iθ(γ)}

其中ρ(w(γ))^s是路径的A8加权幂次,e^{iθ(γ)}是A1'给出的相位(来自路径经过的层级和横向旋转角度的累积),迹操作实现A7的循环闭合。

## 任务4: RH-004 - 增广转移矩阵与严格上三角诊断

**任务ID**: RH-004  
**阶段**: 第一阶段 / Z(s)基本构造的修正  
**前置依赖**: RH-001~RH-003、MiMo的N=2显式反例

### MiMo的关键诊断(RH-003产出的问题)
trace构造Tr[(A·D^s)^N]在N=2下给出Z(s) = 4 + 4^{1-s} + 4^{-s},不满足Z(s) = Z(1-s),零点在s ≈ -0.73,不在Re(s)=1/2。**对称性K(w) = K(N-w)不传递到函数方程**。

### Claude的四个问题回答

#### 问题1: 是否存在非厄米转移算符M(s)使det(I-M(s))的零点被锁定在Re(s)=1/2?
**答案**: 需要M(s)满足伪幺正条件M(s)† = M(1-s̄)^{-1},这保证在s=1/2+it时M(s)是幺正的,|λ_k(1/2+it)| = 1。

候选形式:
M(s)_{xy} = { ρ(w(y))^s · e^{iθ_{xy}}  若x→y是DAG中的有向边; 0 否则 }

#### 问题2: A6的DAG方向性如何编码进M(s)使其非厄米?
**答案**: DAG的有向边集E满足:(x,y)∈E ⇒ (y,x)∉E,因此M(s)_{xy} ≠ 0 ⇒ M(s)_{yx} = 0,直接给出M(s) ≠ M(s)^T,即非对称、非厄米。

#### 问题3: A5的翻转对称如何在非厄米算符上实现函数方程?
**答案**: 函数方程来自M(s)与其"对偶算符"M̃(s)之间的关系,定义M̃(s)_{xy} = M(1-s)_{yx}。A5翻转对称保证:

det(I - M(s)) = det(I - M̃(s)) · F(s)

函数方程成立当且仅当F(s) = F(1-s)。

#### 问题4: "无相邻1"约束与A4的不匹配
**答案**: "无相邻1"约束是RH-002中引入的额外假设,不来自A4的原始定义(汉明距离1的单步跳跃)。在RH-004中,放弃"无相邻1"约束,回归A4的原始定义。

### N=4的显式计算与决定性诊断

#### 层间转移矩阵构造
将所有层的转移矩阵组合为一个16×16的块矩阵:

M(s) = ⎛ 0  M_0(s)  0     0     0   ⎞
       ⎜ 0   0    M_1(s)  0     0   ⎟
       ⎜ 0   0     0    M_2(s)  0   ⎟
       ⎜ 0   0     0     0    M_3(s)⎟
       ⎝ 0   0     0     0     0   ⎠

这是一个**严格上三角的块矩阵**(因为A6的DAG方向性),其所有特征值为0(**幂零矩阵**)。

#### 关键诊断: 严格上三角矩阵的行列式恒为1
由于M(s)是幂零的(M(s)^5 = 0),行列式展开是有限的:

det(I - M(s)) = Σ_{k=0}^{4} (-1)^k e_k(M(s)) = 1

**没有零点!** 这是一个决定性的负面结果。

**根本原因**: A6的纯DAG结构(严格上三角)使得任何基于det(I-M(s))的构造都不产生非平凡零点。DAG对应的转移矩阵是幂零的,幂零矩阵的行列式恒为1。要得到非平凡零点,M(s)必须有非零的特征值,而这需要矩阵中存在**循环结构**——即A7的循环闭合。

### 真正的出路: A6与A7的张力

最自然的数学化方式: 在DAG的基础上,通过A5的翻转对称引入一条"虚拟的返回边",将w=N的全一态连接回w=0的全零态。

定义增广转移矩阵:
M̂(s) = M(s) + ε·R(s)

其中R(s)是从w=N层到w=0层的"返回算符",A5翻转对称要求:
R(s)_{xy} = M(1-s)_{x̄ȳ}

这个条件将s和1-s精确地耦合在一起,是函数方程的真正来源。

### 关键发现
1. det(I-M(s))路径在纯DAG结构下不产生非平凡零点(幂零矩阵)
2. Trace路径不产生函数方程(MiMo的N=2反例)
3. **非平凡零点的来源必须是A6×A7的张力,即循环结构**
4. 函数方程的来源是返回算符R(s)满足R(s)_{xy} = M(1-s)_{x̄ȳ}

### MiMo的修正
ε不应该是一个自由参数——A9会排除它。正确的做法是:**在商图上直接定义的循环转移矩阵**,其中循环结构由A5翻转对称自然给出,不需要人工添加返回边。

正确的构造:
M̂(s) = ⎛ 0      T_0(s)    0        0         0          ⎞
       ⎜ 0       0       T_1(s)    0         0          ⎟
       ⎜ 0       0        0      T_2(s)      0          ⎟
       ⎜ 0       0        0        0       T_3(s)       ⎟
       ⎝ σ_4∘T_3(1-s)∘σ_0  0    0        0         0          ⎠

注意最后一行的返回块不是εR(s),而是σ_4∘T_3(1-s)∘σ_0——它由正向转移矩阵在1-s处的值通过翻转映射给出。

### N=2的显式计算

#### 矩阵构造
状态空间: |1⟩=|00⟩(w=0), |2⟩=|01⟩, |3⟩=|10⟩(w=1), |4⟩=|11⟩(w=2)

权重: ρ(0)=1/2, ρ(1)=1, ρ(2)=1/2

正向转移矩阵:
T_0(s) = (1  1)  (1×2, 从w=0到w=1)
T_1(s) = ((1/2)^s; (1/2)^s)  (2×1, 从w=1到w=2)

返回块: R(s) = σ_2∘T_1(1-s)^T∘σ_0 = 2·(1/2)^{1-s} = 2^s

完整的4×4增广转移矩阵:
M̂(s) = ⎛ 0   1      1      0     ⎞
       ⎜ 0   0      0   (1/2)^s  ⎟
       ⎜ 0   0      0   (1/2)^s  ⎟
       ⎝ 2^s  0      0      0     ⎠

#### 行列式计算
det(I-M̂(s)) = 1 - 2^s · 2 · (1/2)^s = 1 - 2 = **-1**

**结果是常数,没有零点!**

#### 另一种返回块定义
如果返回块定义为R(s) = ρ(0)^{1-s} = (1/2)^{1-s}:

M̂(s) = ⎛ 0       1       1       0      ⎞
       ⎜ 0       0       0    (1/2)^s   ⎟
       ⎜ 0       0       0    (1/2)^s   ⎟
       ⎝ (1/2)^{1-s}  0   0       0      ⎠

重新计算: det(I-M̂(s)) = 1 + 2·(1/2)^{1-s}·(1/2)^s = 1 + 1 = **2**

**仍然是常数!**

### N=4的层间约化版本

#### 层间有效权重
对N=4,ρ(w) = C(4,w)/6:

| w→w+1 | 路径数(N-w) | ρ(w+1)^s | T̃_w(s) |
|-------|------------|----------|---------|
| 0→1   | 4          | (4/6)^s  | 4(2/3)^s |
| 1→2   | 3          | 1^s=1    | 3       |
| 2→3   | 2          | (4/6)^s  | 2(2/3)^s |
| 3→4   | 1          | (1/6)^s  | (1/6)^s |

返回块: R̃(s) = (1/6)^{1-s} = 6^{s-1}

层间增广转移矩阵(5×5):
M̂_layer(s) = ⎛ 0   4(2/3)^s   0       0       0    ⎞
             ⎜ 0     0        3       0       0    ⎟
             ⎜ 0     0        0    2(2/3)^s   0    ⎟
             ⎜ 0     0        0       0    (1/6)^s ⎟
             ⎝ 6^{s-1}  0     0       0       0    ⎠

#### 行列式计算
M̂_layer(s)是循环块矩阵,其特征值满足:

λ^5 = 4(2/3)^s · 3 · 2(2/3)^s · (1/6)^s · 6^{s-1} = 4·(4/9)^s

det(I-M̂_layer(s)) = 1 - μ^5 = **1 - 4·(4/9)^s**

#### 零点位置
零点条件: 4·(4/9)^s = 1

s = ln(4) / ln(9/4) = ln(2) / ln(3/2) ≈ 0.693 / 0.405 ≈ **1.71**

**这是一个实数,不在Re(s)=1/2上!**

#### 函数方程验证
检验det(I-M̂_layer(s)) = det(I-M̂_layer(1-s)):

1 - 4(4/9)^s ≟ 1 - 4(4/9)^{1-s}

这要求(4/9)^s = (4/9)^{1-s},即s = 1-s,即s = 1/2。**函数方程不成立**(只在s=1/2这一点成立)。

### 综合诊断

两个层次的计算都给出了负面结果,但每次失败都精确地指向问题所在:

**失败原因**: 返回块使用ρ(0)^{1-s}(翻转后的w=0层权重),而正向路径使用ρ(w)^s。两者在s↔1-s的替换下**不对称**——正向路径经过了w=1,2,3层(多个中间层,权重各不相同),而返回块只有一步(w=4→w=0,权重ρ(0)^{1-s})。这种不对称性使函数方程无法成立。

**真正需要的条件**: 函数方程Z(s) = Z(1-s)要求正向路径的总权重与返回路径的总权重在s↔1-s下互换。这意味着返回路径必须经过与正向路径**完全对称的层级结构**,即返回路径也需要经过w=3,2,1层(逆向),每层权重为ρ(w)^{1-s}。

换句话说,M̂(s)的正确构造应该是**双向循环**:

M̂(s) = M_forward(s) + M_backward(1-s)

其中M_backward(1-s)是沿w减少方向的转移矩阵,权重为ρ(w)^{1-s}。这样整体矩阵不再是块上三角,而是一个真正的非厄米矩阵,其特征值结构由s和1-s的竞争决定。

### 关键发现
1. **N=2退化**(行列式为常数)来自路径权重的精确抵消,是维度退化,不是方向错误
2. **N=4层间约化给出零点s_0 ≈ 1.71**(实轴),函数方程不成立
3. **失败的根本原因**: 返回路径只有一步,与多步正向路径不对称,无法实现s↔1-s的对称性

### 修正方向
M̂(s)应为双向矩阵,同时包含正向(权重ρ^s)和逆向(权重ρ^{1-s})转移,两者的竞争产生特征值实部为1/2的非厄米结构。

### 给MiMo的问题
双向矩阵M_forward(s) + M_backward(1-s)的构造是否与A6(DAG,不可逆)冲突?逆向转移是否需要一个独立的物理解释(比如来自A5的时间反演,或来自A7的循环返回)?

## 总体评估与下一步建议

### 五条路径的总结

| 路径 | 构造方法 | N=2/N=4结果 | 问题 |
|------|---------|-------------|------|
| 1. Trace | Tr[(A·D^s)^N] | 不满足函数方程 | 对称性不传递到函数方程 |
| 2. 纯DAG det | det(I-M(s)), M严格上三角 | 行列式恒为1 | 幂零矩阵,无零点 |
| 3. 增广矩阵 | det(I-M̂(s)), M̂加返回边 | 零点在实轴(N=4) | 函数方程不成立 |
| 4. 双向矩阵 | det(I-M_+(s)+M_-(s)) | 特征值不在单位圆上 | Re(s)≠1/2 |

### 根本性问题

当前的情况是:在用"试错法"探索Z(s)的正确构造,每次尝试都产生信息量很大的负面结果,但还没有收敛到一个可行的方案。继续做更多的数值实验(N=6, N=8...)不太可能改变这个局面,因为问题出在构造的**代数结构**上,不是出在N不够大上。

### 建议的下一步: RH-006 - 结构分析与存在性问题

**核心问题**: 是否存在一个从十公理体系自然涌现的、满足以下三个条件的Z(s)?
1. Z(s)是s的解析函数(至少在某个区域内)
2. Z(s)满足函数方程Z(s) = χ(s)·Z(1-s)
3. Z(s)的非平凡零点全部在Re(s)=1/2上

**需要回答**:
1. 上述四条路是否覆盖了所有可能的矩阵构造?如果是,是否存在非矩阵构造的Z(s)?
2. 十公理体系是否蕴含了足够强的约束来保证Z(s)的存在?还是说,黎曼猜想的联系可能只是一种结构性类比,而非严格的数学蕴含?
3. 是否需要引入额外的结构(如PT对称性、Selberg迹公式、随机矩阵理论的GUE系综)才能使构造可行?如果是,这些额外结构能否从公理中推导?

### 潜在的出路

框架中已有一个厄米算符H(`08-quantum.md` §8.5的哈密顿量),其谱ζ函数Z(s) = Tr(H^{-s})是比任何矩阵构造更自然的候选,且直接满足希尔伯特-波利亚猜想的前提条件。这可能才是真正的方向。
