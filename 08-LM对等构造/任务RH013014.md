# RH-014 任务一：定理 RH-A 的诚实审计

从最高优先级开始。这个审计必须是彻底的，不允许跳步，不允许事后修补。

---

## **第一步：重新建立 $Z_\gamma(s)$ 和 $Z_{\bar{\gamma}}(s)$ 的精确定义**

### **路径ζ函数的定义**

对排列 $\gamma: \{1, \ldots, N\} \to \{1, \ldots, N\}$（一个比特翻转顺序），路径ζ函数定义为：

$$Z_\gamma(s) = e^{i\pi(N+1)} \sum_{k=0}^{N} \rho(k)^s \cdot \omega^{-\Sigma_k(\gamma)}$$

其中：
- $\rho(k) = \frac{k(N-k)}{N^2}$（第 $k$ 步的差异量，即汉明重量 $k$ 处的归一化值）
- $\omega = e^{2\pi i/N}$（$N$ 次单位根）
- $\Sigma_k(\gamma) = \sum_{l=1}^{k} \gamma(l)$（路径前 $k$ 步翻转的比特位置之和），规定 $\Sigma_0 = 0$
- 逆序路径 $\bar{\gamma}$ 定义为 $\bar{\gamma}(l) = \gamma(N+1-l)$，即将翻转顺序反转

### **关键恒等式（RH-013 的核心）**

RH-013 声称：

$$Z_{\bar{\gamma}}(s) = e^{i\pi(N+1)} \overline{Z_\gamma(s)} \quad \text{（对所有复数 } s \text{ 精确成立）}$$

这是定理 RH-A 的基础。如果这个恒等式成立，则：

$$\mathcal{Z}_\gamma(s) = Z_\gamma(s) + Z_{\bar{\gamma}}(s) = Z_\gamma(s) + e^{i\pi(N+1)}\overline{Z_\gamma(s)}$$

---

## **第二步：对 $N=4$, $\gamma=(1,2,3,4)$ 进行完全显式计算**

### **参数确定**

- $N=4$，$\omega = e^{2\pi i/4} = i$
- $\gamma = (1,2,3,4)$，即按顺序翻转第 1、2、3、4 位比特

**$\rho(k)$ 的值**：

$$\rho(0) = 0, \quad \rho(1) = \frac{1 \cdot 3}{16} = \frac{3}{16}, \quad \rho(2) = \frac{2 \cdot 2}{16} = \frac{4}{16} = \frac{1}{4}, \quad \rho(3) = \frac{3 \cdot 1}{16} = \frac{3}{16}, \quad \rho(4) = 0$$

注意 $\rho(0) = \rho(4) = 0$，所以 $k=0$ 和 $k=4$ 项对任何 $s$ 都贡献为零（$0^s = 0$ 对 $\text{Re}(s) > 0$）。

**$\Sigma_k(\gamma)$ 的值**（$\gamma=(1,2,3,4)$）：

$$\Sigma_0 = 0, \quad \Sigma_1 = 1, \quad \Sigma_2 = 1+2=3, \quad \Sigma_3 = 1+2+3=6, \quad \Sigma_4 = 1+2+3+4=10$$

**$\omega^{-\Sigma_k}$ 的值**（$\omega = i$，即 $\omega^n = i^n$）：

$$\omega^{-\Sigma_0} = i^0 = 1$$
$$\omega^{-\Sigma_1} = i^{-1} = -i$$
$$\omega^{-\Sigma_2} = i^{-3} = i^{-3} = i^{4-3} = i^1 = i$$
$$\omega^{-\Sigma_3} = i^{-6} = i^{-6+8} = i^2 = -1$$
$$\omega^{-\Sigma_4} = i^{-10} = i^{-10+12} = i^2 = -1$$

**全局相位因子**：$e^{i\pi(N+1)} = e^{i\pi \cdot 5} = e^{i\pi} = -1$（因为 $N=4$，$N+1=5$，$e^{5\pi i} = e^{i\pi} = -1$）

### **$Z_\gamma(s)$ 的显式展开**

$$Z_\gamma(s) = (-1) \left[ \rho(0)^s \cdot 1 + \rho(1)^s \cdot (-i) + \rho(2)^s \cdot i + \rho(3)^s \cdot (-1) + \rho(4)^s \cdot (-1) \right]$$

由于 $\rho(0) = \rho(4) = 0$，这些项消失：

$$Z_\gamma(s) = (-1) \left[ \left(\frac{3}{16}\right)^s (-i) + \left(\frac{1}{4}\right)^s (i) + \left(\frac{3}{16}\right)^s (-1) \right]$$

$$= (-1) \left[ -i\left(\frac{3}{16}\right)^s + i\left(\frac{1}{4}\right)^s - \left(\frac{3}{16}\right)^s \right]$$

$$= (-1) \cdot \left(\frac{3}{16}\right)^s (-1 - i) + (-1) \cdot i\left(\frac{1}{4}\right)^s$$

$$\boxed{Z_\gamma(s) = (1+i)\left(\frac{3}{16}\right)^s - i\left(\frac{1}{4}\right)^s}$$

---

## **第三步：计算逆序路径 $\bar{\gamma}=(4,3,2,1)$**

对 $\bar{\gamma} = (4,3,2,1)$：

$$\bar{\Sigma}_0 = 0, \quad \bar{\Sigma}_1 = 4, \quad \bar{\Sigma}_2 = 4+3=7, \quad \bar{\Sigma}_3 = 4+3+2=9, \quad \bar{\Sigma}_4 = 10$$

**$\omega^{-\bar{\Sigma}_k}$ 的值**：

$$\omega^{-\bar{\Sigma}_0} = i^0 = 1$$
$$\omega^{-\bar{\Sigma}_1} = i^{-4} = 1$$
$$\omega^{-\bar{\Sigma}_2} = i^{-7} = i^{-7+8} = i^1 = i$$
$$\omega^{-\bar{\Sigma}_3} = i^{-9} = i^{-9+12} = i^3 = -i$$
$$\omega^{-\bar{\Sigma}_4} = i^{-10} = i^2 = -1$$

**$Z_{\bar{\gamma}}(s)$ 的显式展开**：

$$Z_{\bar{\gamma}}(s) = (-1)\left[\rho(1)^s \cdot 1 + \rho(2)^s \cdot i + \rho(3)^s \cdot (-i)\right]$$

（$\rho(0)=\rho(4)=0$ 项消失）

$$= (-1)\left[\left(\frac{3}{16}\right)^s + i\left(\frac{1}{4}\right)^s - i\left(\frac{3}{16}\right)^s\right]$$

$$= (-1)\cdot\left(\frac{3}{16}\right)^s(1 - i) + (-1)\cdot i\left(\frac{1}{4}\right)^s$$

$$\boxed{Z_{\bar{\gamma}}(s) = (-1+i)\left(\frac{3}{16}\right)^s + i\left(\frac{1}{4}\right)^s}$$

---

## **第四步：验证 RH-013 的关键恒等式 $Z_{\bar{\gamma}}(s) = e^{i\pi(N+1)}\overline{Z_\gamma(s)}$**

$e^{i\pi(N+1)} = -1$，所以需要验证 $Z_{\bar{\gamma}}(s) = -\overline{Z_\gamma(s)}$。

计算 $\overline{Z_\gamma(s)}$。设 $s = \sigma + it$，则：

$$\left(\frac{3}{16}\right)^s = \left(\frac{3}{16}\right)^\sigma e^{it\ln(3/16)}$$

$$\overline{\left(\frac{3}{16}\right)^s} = \left(\frac{3}{16}\right)^\sigma e^{-it\ln(3/16)} = \left(\frac{3}{16}\right)^{\bar{s}}$$

因此：

$$\overline{Z_\gamma(s)} = \overline{(1+i)\left(\frac{3}{16}\right)^s - i\left(\frac{1}{4}\right)^s}$$

$$= (1-i)\left(\frac{3}{16}\right)^{\bar{s}} + i\left(\frac{1}{4}\right)^{\bar{s}}$$

$$-\overline{Z_\gamma(s)} = (-1+i)\left(\frac{3}{16}\right)^{\bar{s}} - i\left(\frac{1}{4}\right)^{\bar{s}}$$

而 $Z_{\bar{\gamma}}(s) = (-1+i)\left(\frac{3}{16}\right)^s + i\left(\frac{1}{4}\right)^s$。

**比较**：$-\overline{Z_\gamma(s)}$ 含有 $\left(\frac{3}{16}\right)^{\bar{s}}$ 和 $-i\left(\frac{1}{4}\right)^{\bar{s}}$，而 $Z_{\bar{\gamma}}(s)$ 含有 $\left(\frac{3}{16}\right)^s$ 和 $+i\left(\frac{1}{4}\right)^s$。

**这两者只有在 $s = \bar{s}$（即 $s$ 为实数）时才相等，对一般复数 $s$ 不相等。**

这意味着 **RH-013 的关键恒等式 $Z_{\bar{\gamma}}(s) = e^{i\pi(N+1)}\overline{Z_\gamma(s)}$ 对一般复数 $s$ 不成立**。

---

## **第五步：直接计算 $\mathcal{Z}_\gamma(s) = Z_\gamma(s) + Z_{\bar{\gamma}}(s)$**

$$\mathcal{Z}_\gamma(s) = \left[(1+i) + (-1+i)\right]\left(\frac{3}{16}\right)^s + \left[-i + i\right]\left(\frac{1}{4}\right)^s$$

$$= 2i\left(\frac{3}{16}\right)^s + 0$$

$$\boxed{\mathcal{Z}_\gamma(s) = 2i\left(\frac{3}{16}\right)^s}$$

这与任务指令中给出的公式 $\mathcal{Z}_\gamma(s) = 2i[(2/3)^{1/2+it}-1]$ **不一致**——指令中的公式本身就是错误的。正确结果是 $\mathcal{Z}_\gamma(s) = 2i\left(\frac{3}{16}\right)^s$，**没有常数项 $-1$**。

---

## **第六步：寻找 $\mathcal{Z}_\gamma(s) = 0$ 的零点**

$$\mathcal{Z}_\gamma(s) = 2i\left(\frac{3}{16}\right)^s = 0$$

由于 $\left(\frac{3}{16}\right)^s = e^{s\ln(3/16)} \neq 0$ 对任何有限 $s$ 成立（指数函数无零点），这个方程**没有任何有限零点**。

$\mathcal{Z}_\gamma(s)$ 在整个复平面上恒非零。

---

## **审计结论**

### **定理 RH-A 不成立。**

失败原因逐层记录如下：

**层次一：关键恒等式的错误**。RH-013 声称 $Z_{\bar{\gamma}}(s) = e^{i\pi(N+1)}\overline{Z_\gamma(s)}$ 对所有复数 $s$ 成立。显式计算表明，这只在 $s$ 为实数时成立，对一般复数 $s$（包括 $s = 1/2 + it$，$t \neq 0$）不成立。

**层次二：$\mathcal{Z}_\gamma(s)$ 的实际形式**。对 $N=4$，$\gamma=(1,2,3,4)$，正确结果是 $\mathcal{Z}_\gamma(s) = 2i(3/16)^s$，这是一个**无零点函数**（指数函数在有限复平面上不消失）。

**层次三：$\mathcal{Z}_\gamma$ 的结构问题**。$Z_\gamma(s) + Z_{\bar{\gamma}}(s)$ 的求和在本例中导致 $(1/4)^s$ 项精确消去，$（3/16)^s$ 项的系数合并为 $2i$，结果是一个比原来更简单但也更退化的函数——它恰好是无零点的。这是一种新的退化形式：**不是恒等于零，而是恒不等于零**，同样无法用于研究零点分布。

**层次四：RH-013 中 N=4 零点计算的来源**。之前报告的零点 $t \approx 15.31, 17.90, \ldots$ 无法从这个计算中重现。这些数值很可能来自对 $Z_\gamma(s)$ 本身（而非 $\mathcal{Z}_\gamma(s)$）的零点计算，或来自某个不同的构造定义。需要重新追溯。

---

## **后续方向**

审计的结果实际上打开了一个更清晰的问题空间：

$Z_\gamma(s)$ 本身（单条路径，不对称化）是一个非退化的函数。对 $N=4$，$\gamma=(1,2,3,4)$：

$$Z_\gamma(s) = (1+i)\left(\frac{3}{16}\right)^s - i\left(\frac{1}{4}\right)^s$$

令 $Z_\gamma(s) = 0$，即 $(1+i)\left(\frac{3}{16}\right)^s = i\left(\frac{1}{4}\right)^s$，化简得：

$$\left(\frac{3/16}{1/4}\right)^s = \frac{i}{1+i} = \frac{1+i}{2}$$

$$\left(\frac{3}{4}\right)^s = \frac{1}{\sqrt{2}} e^{i\pi/4}$$

写 $s = \sigma + it$：

$$\left(\frac{3}{4}\right)^\sigma e^{it\ln(3/4)} = \frac{1}{\sqrt{2}} e^{i\pi/4}$$

取模：$\left(\frac{3}{4}\right)^\sigma = \frac{1}{\sqrt{2}}$，解得 $\sigma = \frac{\ln(1/\sqrt{2})}{\ln(3/4)} = \frac{-\ln 2/2}{\ln 3 - \ln 4} = \frac{\ln 2/2}{\ln 4 - \ln 3}$。

数值上：$\sigma = \frac{0.3466}{0.2877} \approx 1.205$。

**这个零点在 $\operatorname{Re}(s) \approx 1.205$，不在临界线上。** 这与 RH-012 报告的 $\sigma \approx 0.854$ 也不同，说明不同路径给出不同的偏离位置，且没有任何自然的临界线约束。

**$Z_\gamma(s)$ 单独使用时，零点不在临界线上，且不同路径给出不同的实部值。** 对称化 $\mathcal{Z}_\gamma$ 消去了零点（退化为无零点函数）。两条路都不通。

任务二的偏置算子分析，以及任务三的乘积结构 $\mathcal{W}_\gamma(s) = Z_\gamma(s) \cdot Z_{\bar{\gamma}}(s)$，是目前最值得探索的方向——乘积结构不会消去零点，而会**合并**两个函数各自的零点，可能产生更丰富的零点分布，且乘积的函数方程性质通常比和更容易分析。
# RH-014 任务二：偏置算子语言与路径ζ函数的结构对应

在任务一确认定理 RH-A 不成立之后，现在需要从更基础的结构层面重新理解 $Z_\gamma(s)$ 的构造。任务二的目标不是证明新定理，而是建立概念框架。

---

## **第一步：两个对象的精确定义并排**

### **路径ζ函数 $Z_\gamma(s)$**

$$Z_\gamma(s) = e^{i\pi(N+1)} \sum_{k=0}^{N} \rho(k)^s \cdot \omega^{-\Sigma_k(\gamma)}$$

这个对象有三个分量：
- **幅度部分** $\rho(k)^s$：由汉明重量层 $k$ 的差异密度决定，与路径 $\gamma$ 无关
- **相位部分** $\omega^{-\Sigma_k(\gamma)}$：由路径历史 $\Sigma_k(\gamma)$ 决定，携带路径的全部不对称信息
- **全局相位** $e^{i\pi(N+1)}$：常数因子，与 $s$ 和 $\gamma$ 均无关

### **偏置算子 $\mathcal{B}_\omega$（象界形式化文档）**

$$\mathcal{B}_\omega(x_n \to y) \propto w_0(x_n \to y) + \Delta w_\omega(x_n \to y)$$

其中：
- **基础权重** $w_0(x_n \to y)$：A8 的对称权重，只依赖汉明重量，对翻转对合 $x \mapsto \bar{x}$ 完全对称
- **路径修正** $\Delta w_\omega(x_n \to y)$：A1' 相位引入的不对称修正，打破 $w_0$ 的对称性

---

## **第二步：逐项对应分析**

### **$\rho(k)^s$ 对应 $w_0$**

$\rho(k) = k(N-k)/N^2$ 只依赖汉明重量 $k$，对 $k \leftrightarrow N-k$（即 $x \leftrightarrow \bar{x}$）满足 $\rho(k) = \rho(N-k)$，这是完全的翻转对称性。这正是 $w_0$ 的特征：**A8 基础权重，对称，不携带路径信息**。

$s$ 的作用是对 $\rho(k)$ 做幂次变形，相当于调节各汉明重量层的相对权重，但不改变对称性本身。在 $s=1$ 时恢复原始权重，在 $s \to \infty$ 时集中于 $\rho$ 最大的层（$k = N/2$），在 $s \to 0$ 时各层等权。**$s$ 是 $w_0$ 的变形参数，不是打破对称的因素。**

### **$\omega^{-\Sigma_k(\gamma)}$ 对应 $\Delta w_\omega$**

$\Sigma_k(\gamma) = \sum_{l=1}^k \gamma(l)$ 是路径 $\gamma$ 的历史累积，携带路径的全部不对称信息。$\omega^{-\Sigma_k}$ 是 A1' 相位在路径上的积累——每走一步翻转比特位 $\gamma(l)$，相位累积 $\omega^{-\gamma(l)}$。这正是 $\Delta w_\omega$ 的特征：**路径修正，不对称，打破 $w_0$ 的翻转对称性**。

注意一个精确的对应：在偏置算子中，$\Delta w_\omega$ 的作用是使某些转移更可能、某些更不可能；在路径ζ函数中，$\omega^{-\Sigma_k}$ 的作用是对不同汉明重量层赋予不同的复数相位权重。两者都是"在对称基础上叠加不对称修正"的结构。

### **全局相位 $e^{i\pi(N+1)}$ 的地位**

这个因子在两个对象中都没有直接对应。它来自 A1' 相位累积的全局归一化（$\theta(x) + \theta(\bar{x}) = \pi(N+1)$），是一个常数，不影响零点位置，也不影响函数方程的结构。在后续分析中可以暂时忽略。

---

## **第三步：对应关系的矩阵表示**

对 $N=4$，$\gamma=(1,2,3,4)$，将 $Z_\gamma(s)$ 写成两部分的乘积之和：

$$Z_\gamma(s) = e^{i\pi(N+1)} \sum_{k=1}^{3} \underbrace{\rho(k)^s}_{w_0 \text{ 部分}} \cdot \underbrace{\omega^{-\Sigma_k(\gamma)}}_{\Delta w_\omega \text{ 部分}}$$

（$k=0,4$ 项因 $\rho=0$ 消失）

| $k$ | $\rho(k)$ | $\rho(k)^s$（$w_0$ 部分） | $\Sigma_k$ | $\omega^{-\Sigma_k}$（$\Delta w_\omega$ 部分） |
|---|---|---|---|---|
| 1 | $3/16$ | $(3/16)^s$ | 1 | $-i$ |
| 2 | $1/4$ | $(1/4)^s$ | 3 | $i$ |
| 3 | $3/16$ | $(3/16)^s$ | 6 | $-1$ |

这个表格揭示了一个关键的结构问题：**$w_0$ 部分（$\rho(k)^s$）对 $k=1$ 和 $k=3$ 完全相等**（因为 $\rho(1) = \rho(3) = 3/16$），但 $\Delta w_\omega$ 部分（$\omega^{-\Sigma_k}$）对这两层不同（$-i$ vs $-1$）。这意味着路径 $\gamma$ 的不对称性完全由 $\Delta w_\omega$ 承载，$w_0$ 本身无法区分这两层。

---

## **第四步：$w_0$/$\Delta w_\omega$ 分离后的函数方程分析**

将 $Z_\gamma(s)$ 显式拆分：

$$Z_\gamma(s) = e^{i\pi(N+1)}\left[\sum_{k=1}^{N-1} \rho(k)^s \cdot 1 \cdot \omega^{-\Sigma_k(\gamma)}ight]$$

定义**对称部分**（仅保留 $w_0$，令所有相位为 1）：

$$Z_\gamma^{(0)}(s) = e^{i\pi(N+1)}\sum_{k=1}^{N-1} \rho(k)^s$$

对 $N=4$：$Z_\gamma^{(0)}(s) = -1 \cdot \left[2\left(\frac{3}{16}ight)^s + \left(\frac{1}{4}ight)^sright]$，这与 $\gamma$ 无关，且因为 $\rho(k) = \rho(N-k)$，满足 $Z^{(0)}(s) = Z^{(0)}(s)$（平凡的函数方程）。

定义**不对称修正**（$Z_\gamma(s) - Z_\gamma^{(0)}(s)$ 的结构）：

实际上更有用的分离是：

$$Z_\gamma(s) = Z_\gamma^{(0)}(s) \cdot \frac{Z_\gamma(s)}{Z_\gamma^{(0)}(s)}$$

即将 $Z_\gamma(s)$ 写成对称基础乘以一个比值因子。但这个比值因子没有简单形式，分析价值有限。

更有价值的分离是**按 $\rho(k)$ 的对称配对**来重组：

$$Z_\gamma(s) = e^{i\pi(N+1)}\sum_{k=1}^{N/2}\rho(k)^s\left[\omega^{-\Sigma_k(\gamma)} + \omega^{-\Sigma_{N-k}(\gamma)}ight]$$

（利用 $\rho(k) = \rho(N-k)$，将 $k$ 和 $N-k$ 配对）

对 $N=4$，$\gamma=(1,2,3,4)$，配对 $(k=1, k=3)$ 和 $(k=2)$（中间层单独）：

$$Z_\gamma(s) = (-1)\left[\left(\frac{3}{16}ight)^s\left(\omega^{-\Sigma_1} + \omega^{-\Sigma_3}ight) + \left(\frac{1}{4}ight)^s \omega^{-\Sigma_2}ight]$$

代入数值：

$$= (-1)\left[\left(\frac{3}{16}ight)^s(-i + (-1)) + \left(\frac{1}{4}ight)^s \cdot iight]$$

$$= (-1)\left[\left(\frac{3}{16}ight)^s(-1-i) + i\left(\frac{1}{4}ight)^sright]$$

$$= (1+i)\left(\frac{3}{16}ight)^s - i\left(\frac{1}{4}ight)^s$$

与之前的计算一致，验证正确。

**关键发现**：配对 $(k, N-k)$ 的相位之和 $\omega^{-\Sigma_k} + \omega^{-\Sigma_{N-k}}$ 决定了该配对的"有效权重"。若这个和为零，该配对对 $Z_\gamma(s)$ 无贡献（退化）；若不为零，该配对贡献一个 $\rho(k)^s$ 的幂次项。这正是 $\mathcal{Z}_\gamma(s) = Z_\gamma + Z_{\bar\gamma}$ 退化的根本原因——对称化操作使所有配对的有效权重都退化了。

---

## **第五步：对应关系中的不匹配之处**

诚实记录偏置算子与路径ζ函数之间**对应不成立**的地方：

**不匹配一：时间方向**。偏置算子 $\mathcal{B}_\omega$ 描述的是状态空间上的转移概率，是一个**动力学对象**，定义在转移 $x_n \to y$ 上。路径ζ函数 $Z_\gamma(s)$ 是一个**生成函数对象**，定义在整条路径的汇总统计量 $\Sigma_k(\gamma)$ 上。前者是局部的（单步转移），后者是全局的（整条路径的累积）。

**不匹配二：$s$ 的角色**。在偏置算子中，没有对应 $s$ 的参数——$\mathcal{B}_\omega$ 不对权重做幂次变形。$s$ 是路径ζ函数特有的复参数，它的引入使 $Z_\gamma(s)$ 成为一个解析函数而非单个数值。这个参数在偏置算子框架中没有自然的对应物。

**不匹配三：退化机制不同**。象界形式化文档中，偏置算子通过"多强连通分量的层级结构"解决退化问题。路径ζ函数的退化（全路径求和为零）是由初等对称多项式的代数消去引起的。这两种退化的数学机制不同，解决方案也不能直接互译。

**不匹配四：函数方程的来源**。偏置算子的对称性来自 A5（守恒律），而路径ζ函数的函数方程来自 A1'（相位累积的共轭关系）。两者公理来源不同，导致函数方程的结构也不同。

---

## **第六步：分离后的结构是否改进了函数方程？**

将 $Z_\gamma(s)$ 写成配对形式：

$$Z_\gamma(s) = e^{i\pi(N+1)}\sum_{k=1}^{N/2} \rho(k)^s \cdot A_k(\gamma)$$

其中 $A_k(\gamma) = \omega^{-\Sigma_k(\gamma)} + \omega^{-\Sigma_{N-k}(\gamma)}$（配对相位和）。

对 $Z_{\bar\gamma}(s)$，配对相位和为 $A_k(\bar\gamma) = \omega^{-\bar\Sigma_k} + \omega^{-\bar\Sigma_{N-k}}$。

若要使 $\mathcal{Z}_\gamma = Z_\gamma + Z_{\bar\gamma}$ 具有函数方程，需要 $A_k(\gamma) + A_k(\bar\gamma)$ 在 $k \leftrightarrow N/2$ 的某种对称性下不变。但任务一的计算已经表明，对 $N=4$，$A_1(\gamma) + A_1(\bar\gamma) = (-1-i) + (1-i) = -2i$，而 $A_2(\gamma) + A_2(\bar\gamma) = i + i = 2i$。这两个值的绝对值相等（$|-2i| = |2i| = 2$），但相位相差 $\pi$，不满足简单的函数方程条件。

**结论：$w_0/\Delta w_\omega$ 的分离没有改进函数方程的结构。** 分离揭示了问题的来源（对称配对的相位和），但没有提供解决方案。

---

## **任务二的整体结论**

**对应关系成立的部分**：$\rho(k)^s \leftrightarrow w_0$（对称基础），$\omega^{-\Sigma_k(\gamma)} \leftrightarrow \Delta w_\omega$（不对称修正）。这个对应在概念层面是清晰的，且与《象界》的生成链一致：$w_0$ 是"存在层"，$\Delta w_\omega$ 是"呈象层"。

**对应关系不成立的部分**：$s$ 参数、退化机制、函数方程来源。这三处不匹配说明，路径ζ函数和偏置算子是**同一公理系统的两个不同投影**，而不是同一对象的两种写法。

**对后续任务的启示**：$A_k(\gamma) = \omega^{-\Sigma_k} + \omega^{-\Sigma_{N-k}}$ 是路径ζ函数的最自然分解单元。任务三的乘积结构 $\mathcal{W}_\gamma(s) = Z_\gamma(s) \cdot Z_{\bar\gamma}(s)$ 可以用 $A_k$ 语言重写：

$$\mathcal{W}_\gamma(s) = e^{2i\pi(N+1)}\left(\sum_{k=1}^{N/2}\rho(k)^s A_k(\gamma)ight)\left(\sum_{j=1}^{N/2}\rho(j)^s A_j(\bar\gamma)ight)$$

这是一个 Dirichlet 级数的乘积，其零点结构比和结构更丰富，也更容易分析。这是任务三的出发点。

# RH-014 任务三：互补配对乘积结构 $\mathcal{W}_\gamma(s)$

## **第一步：$\mathcal{W}_\gamma(s)$ 的显式计算（$N=4$）**

$$\mathcal{W}_\gamma(s) = Z_\gamma(s) \cdot Z_{\bar\gamma}(s)$$

已知：

$$Z_\gamma(s) = (1+i)\left(\frac{3}{16}\right)^s - i\left(\frac{1}{4}\right)^s$$

$$Z_{\bar\gamma}(s) = (-1+i)\left(\frac{3}{16}\right)^s + i\left(\frac{1}{4}\right)^s$$

令 $a = \left(\frac{3}{16}\right)^s$，$b = \left(\frac{1}{4}\right)^s$，则：

$$Z_\gamma = (1+i)a - ib, \quad Z_{\bar\gamma} = (-1+i)a + ib$$

$$\mathcal{W}_\gamma = \left[(1+i)a - ib\right]\left[(-1+i)a + ib\right]$$

展开：

$$= (1+i)(-1+i)a^2 + (1+i)(i)ab + (-ib)(-1+i)a + (-ib)(ib)$$

逐项计算：

- $(1+i)(-1+i) = -1+i-i+i^2 = -1-1 = -2$
- $(1+i)(i) = i + i^2 = i - 1 = -1+i$
- $(-ib)(-1+i) \cdot a$：系数 $(-i)(-1+i) = i - i^2 = i+1 = 1+i$
- $(-ib)(ib) = -i^2 b^2 = b^2$

因此：

$$\mathcal{W}_\gamma(s) = -2a^2 + (-1+i)ab + (1+i)ab + b^2$$

$$= -2a^2 + \left[(-1+i)+(1+i)\right]ab + b^2$$

$$= -2a^2 + 2i \cdot ab + b^2$$

代回 $a = (3/16)^s$，$b = (1/4)^s$：

$$\boxed{\mathcal{W}_\gamma(s) = -2\left(\frac{3}{16}\right)^{2s} + 2i\left(\frac{3}{16}\right)^s\left(\frac{1}{4}\right)^s + \left(\frac{1}{4}\right)^{2s}}$$

注意 $(3/16)^s(1/4)^s = (3/64)^s$，$(3/16)^{2s} = (9/256)^s$，$(1/4)^{2s} = (1/16)^s$。

---

## **第二步：寻找 $\mathcal{W}_\gamma(s) = 0$ 的零点**

令 $u = \left(\frac{3/16}{1/4}\right)^s = \left(\frac{3}{4}\right)^s$，则 $a = (3/16)^s$，$b = (1/4)^s$，$a = u \cdot b$。

$$\mathcal{W}_\gamma(s) = b^2\left[-2u^2 + 2iu + 1\right] = 0$$

由于 $b^2 = (1/4)^{2s} \neq 0$，零点条件化为：

$$-2u^2 + 2iu + 1 = 0 \quad \Longleftrightarrow \quad 2u^2 - 2iu - 1 = 0$$

用求根公式：

$$u = \frac{2i \pm \sqrt{(2i)^2 + 8}}{4} = \frac{2i \pm \sqrt{-4+8}}{4} = \frac{2i \pm 2}{4} = \frac{i \pm 1}{2}$$

两个根：

$$u_+ = \frac{1+i}{2}, \quad u_- = \frac{-1+i}{2}$$

两者的模：$|u_\pm| = \frac{\sqrt{1^2+1^2}}{2} = \frac{\sqrt{2}}{2} = \frac{1}{\sqrt{2}}$

---

## **第三步：从 $u$ 的值求 $s$ 的零点位置**

$u = (3/4)^s$，写 $s = \sigma + it$：

$$u = \left(\frac{3}{4}\right)^\sigma e^{it\ln(3/4)}$$

**对根 $u_+ = \frac{1+i}{2} = \frac{1}{\sqrt{2}}e^{i\pi/4}$**：

取模：$\left(\frac{3}{4}\right)^\sigma = \frac{1}{\sqrt{2}}$

$$\sigma = \frac{\ln(1/\sqrt{2})}{\ln(3/4)} = \frac{-\frac{1}{2}\ln 2}{\ln 3 - \ln 4} = \frac{\frac{1}{2}\ln 2}{\ln 4 - \ln 3}$$

数值：$\sigma = \frac{0.3466}{0.2877} \approx 1.205$

取辐角：$t\ln(3/4) = \frac{\pi}{4} + 2\pi n$（$n \in \mathbb{Z}$）

$$t = \frac{\pi/4 + 2\pi n}{\ln(3/4)} = \frac{\pi(1/4 + 2n)}{\ln(3/4)}$$

由于 $\ln(3/4) < 0$，取 $n=0$：$t = \frac{\pi/4}{\ln(3/4)} = \frac{\pi/4}{-0.2877} \approx -2.73$（负值，取 $n=-1$：$t \approx \frac{\pi(-7/4)}{-0.2877} \approx 19.12$）

**对根 $u_- = \frac{-1+i}{2} = \frac{1}{\sqrt{2}}e^{i3\pi/4}$**：

取模相同：$\sigma \approx 1.205$

取辐角：$t\ln(3/4) = \frac{3\pi}{4} + 2\pi n$

$$t = \frac{3\pi/4 + 2\pi n}{\ln(3/4)} \approx \frac{3\pi/4}{-0.2877} \approx -8.18$$（取 $n=-1$：$t \approx \frac{3\pi/4 - 2\pi}{-0.2877} \approx 13.57$）

**结果**：$\mathcal{W}_\gamma(s)$ 的所有零点满足 $\operatorname{Re}(s) \approx 1.205$，**不在临界线 $\operatorname{Re}(s) = 1/2$ 上**。

---

## **第四步：比值结构 $\mathcal{Q}_\gamma(s) = Z_\gamma(s)/Z_{\bar\gamma}(s)$**

$$\mathcal{Q}_\gamma(s) = \frac{(1+i)a - ib}{(-1+i)a + ib}$$

$\mathcal{Q}_\gamma(s)$ 的零点就是 $Z_\gamma(s)$ 的零点（即 $\sigma \approx 1.205$，$u = (1+i)/2$），极点是 $Z_{\bar\gamma}(s)$ 的零点。

对 $Z_{\bar\gamma}(s) = 0$：$(-1+i)a + ib = 0$，即 $u = a/b = \frac{-ib}{(-1+i)b} \cdot \frac{1}{1} = \frac{i}{1-i} = \frac{i(1+i)}{2} = \frac{i-1}{2} = \frac{-1+i}{2}$。

这正好是 $u_-$，与 $\mathcal{W}_\gamma$ 的一个零点重合。所以 $\mathcal{Q}_\gamma(s)$ 的零点和极点都在 $\sigma \approx 1.205$ 上，比值结构没有改善零点位置。

---

## **任务三结论**

乘积结构 $\mathcal{W}_\gamma(s)$ 的零点在 $\operatorname{Re}(s) \approx 1.205$，不在临界线上。比值结构 $\mathcal{Q}_\gamma(s)$ 的零点和极点也在同一位置。

但这次计算有一个重要的正面发现：**$\mathcal{W}_\gamma(s)$ 的两个零点具有完全相同的实部**（$\sigma \approx 1.205$），尽管它们的虚部不同。这说明某种约束机制正在将零点的实部锁定到一个特定值——只是这个值不是 $1/2$，而是 $\approx 1.205$。

这个锁定来自 $|u_\pm| = 1/\sqrt{2}$ 的共同模，而 $|u| = |(3/4)^s| = (3/4)^\sigma$，模的固定直接锁定了 $\sigma$。**问题的核心因此变成：如何构造一个 $Z(s)$，使得其零点条件等价于 $|u| = 1$（而非 $1/\sqrt{2}$）？** 若 $|(3/4)^s| = 1$，则 $(3/4)^\sigma = 1$，即 $\sigma = 0$——这也不是 $1/2$。

若希望 $\sigma = 1/2$，需要零点条件化为 $|f(s)| = c$ 的形式，其中 $c$ 恰好对应 $\sigma = 1/2$。这要求函数的比值基底不是 $(3/4)^s$，而是某个在 $\sigma = 1/2$ 处模为 1 的函数——例如 $\rho(1)^s / \rho(N/2)^s$ 在 $\sigma = 1/2$ 时模为 $\sqrt{\rho(1)/\rho(N/2)}$，只有在 $\rho(1) = \rho(N/2)$（即各层等权）时才为 1。这又回到了退化问题。

**这个观察指向任务四的核心**：零点实部的锁定机制依赖于 $\rho(k)$ 的比值结构。若要将零点锁定在 $\sigma = 1/2$，需要在构造 $Z(s)$ 时引入一种使 $\rho(k)^s$ 在 $\sigma = 1/2$ 处满足某种归一化条件的机制——这正是 A8（权重集中于 $k = N/2$）与临界线之间关系的真正数学内容，也是后续工作的核心问题。
