## 核心猜想

### 陈述

对素数序列 $\{p_k\}_{k=1}^{\infty}$,成立以下不等式:

$$\boxed{p_{k+1} \leq p_k + p_{k-1}}$$

等价形式(素数间隙):

$$\boxed{g_k = p_{k+1} - p_k \leq p_{k-1}}$$

其中 $g_k$ 是第 $k$ 个素数间隙。

### 等号条件

唯一已知的等号情形是 **$\{2, 3, 5\}$**:

$$p_3 = 5 = 3 + 2 = p_2 + p_1$$

这是素数序列与 Fibonacci 数列的唯一交点——素数序列的开头恰好满足 Fibonacci 递推 $F_{k+1} = F_k + F_{k-1}$。对所有 $k \geq 3$,严格不等式 $g_k < p_{k-1}$ 成立,且比值 $g_k/p_{k-1}$ 随 $k$ 增大单调趋向零。

### 关键意义

第三个来源的意义特别重要:**极大间隙记录是最危险的候选点**。如果不等式要有反例,反例必然首先出现在极大间隙处。84 个极大间隙记录全部通过,覆盖到 $6.8\times10^{19}$ 量级,这提供了极强的数值证据。

## WorldBase 框架内的公理来源

### A4 (最小变易) → 局域性约束

加法递推猜想在 WorldBase 框架中有自然的物理解释:

**A4 (最小变易)** 要求演化步幅受到当前状态的约束。在素数生成的语言中,这对应于:从 $p_k$ 出发向前搜索下一个素数 $p_{k+1}$,搜索步幅受到**前两个不可归约状态** $(p_k, p_{k-1})$ 的联合约束。

三角不等式的离散版本:

$$p_{k+1} - p_k \leq p_{k-1}$$

这正是 A4 局域性的体现——新涌现的不可归约状态(素数)的位置,不超过前两个不可归约状态位置之和。

### A9 (内生完备) → 筛选机制

**A9 (内生完备)** 要求:若整数 $n$ 能被已有素数 $p_1, p_2, \ldots, p_k$ 中的某个整除,则 $n$ 不是新的素稳态。因此 $p_{k+1}$ 是第一个不被 $p_1, \ldots, p_k$ 中任何一个整除的整数($> p_k$)。

在区间 $(p_k, p_k + p_{k-1}]$ 内:
- 每隔 $p_{k-1}$ 步就有一个 $p_{k-1}$ 的倍数
- 但区间长度恰好是 $p_{k-1}$,所以区间内**至多有一个** $p_{k-1}$ 的倍数
- 结合其他小素数的筛效应,A9 保证区间内至少存在一个素数

## 与已知结果的关系

```
RH：g_k = O(√p_k · ln p_k)
        │
        │  RH 成立则自动推出 ↓
        ▼
加法递推猜想：g_k ≤ p_{k-1} ≈ p_k
        │
        │  反向不成立
        │  但加法递推弱于 RH，应比 RH 更容易证明
        ▼
Bertrand 公设：g_k < p_k（已证明）
```

这个层级结构非常重要。加法递推猜想夹在两个已知结果之间——比 Bertrand 强，比 RH 弱。它是一个**独立的、有价值的数论猜想**，不是 RH 的附属品。
