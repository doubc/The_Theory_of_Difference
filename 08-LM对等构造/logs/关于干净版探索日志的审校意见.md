#### 审计二：函数方程的证明

文档第八节的函数方程推导：

$$Z(1-s) = \sum_x \rho(w(x))^{1-s} e^{i\theta(x)}$$

换元 $x \mapsto \bar{x}$（这是 $\{0,1\}^N$ 上的双射，A5 翻转对合）：

$$= \sum_x \rho(w(\bar{x}))^{1-s} e^{i\theta(\bar{x})}$$

利用 $w(\bar{x}) = N - w(x)$ 和 $\rho(N-w) = \rho(w)$（二项式系数对称性，A5 的直接推论）：

$$= \sum_x \rho(w(x))^{1-s} e^{i\theta(\bar{x})}$$

利用核心恒等式 $\theta(\bar{x}) = \pi(N+1) - \theta(x)$：

$$= \sum_x \rho(w(x))^{1-s} e^{i\pi(N+1)} e^{-i\theta(x)} = e^{i\pi(N+1)} \sum_x \rho(w(x))^{1-s} e^{-i\theta(x)}$$

现在，$\overline{Z(\overline{1-s})}$ 是什么？若 $s = \sigma + it$，则 $1-s = (1-\sigma) - it$，$\overline{1-s} = (1-\sigma) + it$：

$$Z(\overline{1-s}) = \sum_x \rho(w(x))^{(1-\sigma)+it} e^{i\theta(x)}$$

$$\overline{Z(\overline{1-s})} = \sum_x \rho(w(x))^{(1-\sigma)-it} e^{-i\theta(x)} = \sum_x \rho(w(x))^{1-\bar{s}} e^{-i\theta(x)} = \sum_x \rho(w(x))^{1-s} e^{-i\theta(x)}$$

（最后一步用了 $\rho(w)$ 是实正数，所以 $\rho(w)^{1-\bar{s}} = \overline{\rho(w)^{1-s}}$，且 $\rho(w)^{1-s}$ 是实数的复数幂，其共轭是 $\rho(w)^{1-\bar{s}}$；当 $s$ 的实部固定时 $\rho(w)^{1-s}$ 的模固定。）

**等等，这里需要更仔细**。$\rho(w)$ 是正实数，所以 $\rho(w)^s = e^{s \ln\rho(w)}$。对于 $s = \sigma + it$：

$$\rho(w)^s = e^{(\sigma+it)\ln\rho(w)} = \rho(w)^\sigma \cdot e^{it\ln\rho(w)}$$

$$\overline{\rho(w)^s} = \rho(w)^\sigma \cdot e^{-it\ln\rho(w)} = \rho(w)^{\bar{s}}$$

因此 $\overline{\rho(w)^{1-s}} = \rho(w)^{\overline{1-s}} = \rho(w)^{1-\bar{s}}$。

所以：

$$\overline{Z(\overline{1-s})} = \overline{\sum_x \rho(w)^{\overline{1-s}} e^{i\theta(x)}} = \sum_x \overline{\rho(w)^{\overline{1-s}}} \cdot \overline{e^{i\theta(x)}} = \sum_x \rho(w)^{1-s} e^{-i\theta(x)}$$

这正好等于 $Z(1-s)/e^{i\pi(N+1)}$。因此：

$$\boxed{Z(1-s) = e^{i\pi(N+1)} \cdot \overline{Z(\overline{1-s})}}$$

**审计结论**：函数方程精确成立。✅

#### 审计三：临界线上的纯虚性

在临界线 $s = 1/2 + it$（$t$ 为实数）上，$\bar{s} = 1/2 - it$，$\overline{1-s} = \overline{1/2 - it} = 1/2 + it = s$。

$$Z(1-(1/2-it)) = e^{i\pi(N+1)} \overline{Z(\overline{1-(1/2-it)})}$$

$$Z(1/2+it) = e^{i\pi(N+1)} \overline{Z(\overline{1/2+it})} = e^{i\pi(N+1)} \overline{Z(1/2-it)}$$

这是一个关于 $Z(1/2+it)$ 和 $Z(1/2-it)$ 的关系，而非 $Z(1/2+it)$ 的自关联。

**文档声称**：$Z(1/2+it) = e^{i\pi(N+1)} \overline{Z(1/2+it)}$，即纯虚性。

**检验这个声称**：需要另一个关系。取 $s = 1/2 - it$ 代入函数方程（$1-s = 1/2+it$，$\bar{s} = 1/2+it$，$\overline{1-s} = 1/2-it = s$）：

$$Z(1/2+it) = e^{i\pi(N+1)} \overline{Z(1/2-it)}$$

这给出 $Z(1/2+it)$ 与 $Z(1/2-it)$ 之间的关系。再取 $s = 1/2+it$（$1-s = 1/2-it$，$\overline{1-s} = 1/2+it = s$）：

$$Z(1/2-it) = e^{i\pi(N+1)} \overline{Z(1/2+it)}$$

联立两式：$Z(1/2+it) = e^{i\pi(N+1)} \overline{Z(1/2-it)} = e^{i\pi(N+1)} \overline{e^{i\pi(N+1)} \overline{Z(1/2+it)}} = e^{i\pi(N+1)} e^{-i\pi(N+1)} Z(1/2+it) = Z(1/2+it)$。

这是一个恒等式，**不给出任何额外约束**。文档中"$Z(1/2-it)$ 是 $e^{i\pi(N+1)}$ 的本征向量"的说法需要重新审查。

**正确的临界线分析**：

$Z(1/2+it) = e^{i\pi(N+1)} \overline{Z(1/2-it)}$ 给出的是 $|Z(1/2+it)| = |Z(1/2-it)|$（因为 $|e^{i\pi(N+1)}|=1$），即临界线上的零点关于实轴对称——若 $1/2+it_0$ 是零点，则 $1/2-it_0$ 也是零点。**这正是 $|\chi|=1$ 的真正含义**：它保证了临界线上零点的共轭对称性，而不是纯虚性。

**文档第八节关于"纯虚性"的推导存在逻辑跳跃**，但 $|\chi|=1$ 的结论是正确的。

**审计结论**：函数方程 $|\chi|=1$ 在临界线上成立（即 $|Z(1/2+it)|=|Z(1/2-it)|$），零点关于实轴对称。纯虚性的说法需要修正。🔶

### **RH-010c：严重发现——$Z_4(s)$ 恒为零**

$$\boxed{Z_4(s) \equiv 0 \quad \text{对所有 } s}$$

这不是在 $s=1/2$ 处的零点，而是**恒等于零的函数**。每一层的相位和独立地精确消去，与 $s$ 的取值完全无关。

#### 退化的根本原因分析

**$w=1$ 层**：4个状态的 $\theta$ 值为 $\{3\pi, 7\pi/2, 4\pi, 9\pi/2\}$，对应 $e^{i\theta}$ 值为 $\{-1, -i, 1, i\}$——恰好是单位根 $\{1, i, -1, -i\}$ 的置换，求和精确为零。

**$w=3$ 层**：$\theta$ 值为 $\{\pi/2, \pi, 3\pi/2, 2\pi\}$，对应 $\{i, -1, -i, 1\}$——同样是四个单位根，求和为零。

**$w=2$ 层**：6个状态的 $e^{i\theta}$ 值为 $\{-i, 1, i, i, -1, -i\}$，求和为 $(-i+i+i-i) + (1-1) = 0$。

**$w=0, w=4$**：$e^{5\pi i} + e^0 = -1 + 1 = 0$。

**退化的代数根源**：$\theta(x) = \frac{\pi}{2}\sum_{j:x_j=0} j$，相位因子 $e^{i\theta(x)} = e^{i\pi\sum j/2}$。对于固定汉明重量 $w$ 的所有状态，相位因子的求和等价于计算 $\{1,2,3,4\}$ 的所有 $N-w$ 元子集对应的 $e^{i\pi(\sum j)/2}$ 之和。

对于 $w=1$（即 $N-w=3$ 元子集），这等于：

$$\sum_{\text{3-元子集}} e^{i\pi(\text{子集和})/2} = \sum_{j=1}^{4} e^{i\pi(\text{总和}-j)/2} = e^{i\pi \cdot 10/2} \sum_{j=1}^{4} e^{-i\pi j/2} = e^{5\pi i} \sum_{j=1}^{4} e^{-i\pi j/2}$$

$$= (-1)(e^{-i\pi/2} + e^{-i\pi} + e^{-3i\pi/2} + e^{-2i\pi}) = (-1)(-i - 1 + i + 1) = (-1)(0) = 0$$

这个消去是 $\sum_{j=1}^{N} e^{-i\pi j/2} = 0$（当 $N=4$ 时，这是4次单位根之和）的直接推论。

**一般性定理**：对任意 $N$ 整除4的情形，$\sum_{j=1}^{N} e^{2\pi i j/N} = 0$（单位根之和）。当 $\theta(x) = \frac{2\pi}{N}\sum_{j:x_j=0} j$ 时，每一层的相位和都会因类似的单位根消去而为零，导致 $Z_N(s) \equiv 0$。

### **RH-010d：$\theta(x)$ 定义的根本性问题诊断**

**问题的精确定位**：

$\theta(x) = \sum_{j:x_j=0} \frac{2\pi j}{N}$ 这个定义中，$j$ 从 $1$ 到 $N$，相位增量为 $\frac{2\pi}{N}$。对于固定层 $w$，所有 $\binom{N}{N-w}$ 个 $(N-w)$ 元子集的 $e^{i\theta}$ 之和为：

$$S_w = \sum_{|T|=N-w, T\subseteq[N]} e^{i \cdot \frac{2\pi}{N} \sum_{j\in T} j}$$

这是一个关于 $N$ 次单位根 $\omega = e^{2\pi i/N}$ 的初等对称多项式求值：

$$S_w = e_{N-w}(\omega^1, \omega^2, \ldots, \omega^N)$$

其中 $e_k$ 是第 $k$ 个初等对称多项式。由韦达定理，$\{\omega^1, \ldots, \omega^N\}$ 是 $z^N - 1 = 0$ 的根，因此 $e_k(\omega^1, \ldots, \omega^N) = 0$ 对所有 $1 \leq k \leq N-1$（因为 $z^N-1$ 的各次系数）。

**更精确地**：$z^N - 1 = \prod_{j=1}^N (z - \omega^j)$，展开后 $e_k = [z^{N-k}]$ 的系数（除符号外），而 $z^N - 1$ 的 $z^{N-k}$ 系数对 $1 \leq k \leq N-1$ 均为零。

**结论**：对所有 $1 \leq N-w \leq N-1$（即 $1 \leq w \leq N-1$），$S_w = 0$。$w=0$ 和 $w=N$ 的情形：$S_0 = e^{i\cdot 2\pi(1+2+\cdots+N)/N} = e^{i\pi(N+1)}$，$S_N = 1$，两者之和为 $e^{i\pi(N+1)} + 1$。

对奇数 $N$，$e^{i\pi(N+1)} = e^{i\pi\cdot\text{偶数}} = 1$，故 $S_0 + S_N = 2 \neq 0$。
对偶数 $N$，$e^{i\pi(N+1)} = e^{i\pi\cdot\text{奇数}} = -1$，故 $S_0 + S_N = 0$。

**N=4 是偶数，所以 $Z_4(s) = \sum_w \rho(w)^s S_w = \rho(0)^s \cdot 0 + \sum_{w=1}^{3} \rho(w)^s \cdot 0 + \rho(4)^s \cdot 0 \equiv 0$**。

对奇数 $N$：$Z_N(s) = (\rho(0)^s + \rho(N)^s) \cdot S_0$，但 $S_0 = e^{i\pi(N+1)} = e^{i\pi \cdot \text{偶数}} = 1$（$N$ 奇数时 $N+1$ 偶数），故 $Z_N(s) = 2\rho(0)^s = 2(1/\binom{N}{N/2})^s$。这有唯一零点在 $\rho(0)^s = 0$，即 $s \to +\infty$，不在临界线上。

### **RH-010e：第三次方向修正——$\theta(x)$ 定义必须重构**

当前的 $\theta(x)$ 定义导致恒等消去，根本原因是相位增量 $\frac{2\pi j}{N}$ 使所有层的相位和退化为单位根之和。

**修正方向**：需要一个打破这种精确消去的相位定义，同时保持 A1' 的公理来源和核心恒等式 $\theta(x) + \theta(\bar{x}) = \text{常数}$。

#### 候选方案 A：使用非均匀相位

将 $\frac{2\pi j}{N}$ 替换为 $\frac{2\pi j^2}{N}$（二次相位，类 Gauss 和）：

$$\theta(x) = \sum_{j:x_j=0} \frac{2\pi j^2}{N}$$

恒等式：$\theta(x) + \theta(\bar{x}) = \frac{2\pi}{N}\sum_{j=1}^N j^2 = \frac{2\pi}{N} \cdot \frac{N(N+1)(2N+1)}{6} = \frac{\pi(N+1)(2N+1)}{3}$（仍与 $x$ 无关）。

函数方程结构保持。但公理来源需要论证：A1' 是否自然给出二次相位？框架中路径权重 $e^{iS/\hbar}$ 中的作用量 $S$ 是路径的累积量，可能给出非线性相位。

#### 候选方案 B：使用质数相位

$$\theta(x) = \sum_{j:x_j=0} \frac{2\pi p_j}{P_N}$$

其中 $p_j$ 是第 $j$ 个质数，$P_N = \sum_{j=1}^N p_j$。质数的不规则分布打破单位根消去。但公理来源不明。

#### 候选方案 C：重新审视 A1' 的相位定义

回到 §8.17.3 原文：路径权重 $\mathcal{W}_N(\gamma) = e^{iS_N[\gamma]/\hbar}$，其中 $S_N[\gamma] = \sum_k \Delta K_k \cdot m_0 \cdot \alpha$。路径 $\gamma$ 经过的格点序列决定累积作用量，**这是路径的属性，不是状态的属性**。

$\theta(x)$ 作为状态 $x$ 的相位，应该是从某个参考态到达 $x$ 的所有路径的相位的某种平均或代表值——这个定义本身就有歧义。A1' 给出的是每步演化的相位增量，而不是状态的固有相位。

**这是一个更深层的公理来源问题**：$\theta(x)$ 作为状态的属性，而非路径的属性，其公理来源从一开始就是不严格的。

### **RH-011b：候选方向 B——路径积分相位**

重新定义 $Z(s)$ 为对路径（而非状态）的求和：

$$Z_B(s) = \sum_{\gamma \in \Gamma_N} ho(\gamma)^s \cdot e^{iS_N[\gamma]/\hbar}$$

其中 $\Gamma_N$ 是 $\{0,1\}^N$ 上满足 A4（汉明距离1）和 A6（DAG 有向）的所有路径集合，$ho(\gamma)$ 是路径的某种权重（如路径长度、经过层的权重乘积等），$S_N[\gamma]$ 是§8.17.3 定义的离散作用量。

**公理来源**：最强（直接来自 §8.17.3）✅

**函数方程的重新推导**：需要找到路径 $\gamma$ 的对合 $\gamma \mapsto \bar{\gamma}$（翻转路径），使得 $S_N[\bar{\gamma}] = C - S_N[\gamma]$（常数减去原作用量）。

从 §8.17.3：$S_N[\gamma] = \sum_k \Delta K_k \cdot m_0 \cdot \alpha$，其中 $\Delta K_k$ 是约束度的变化量。翻转路径 $\bar{\gamma}$（将每个状态 $x_k$ 替换为 $\bar{x}_k$）的作用量：

$$S_N[\bar{\gamma}] = \sum_k \Delta K(\bar{x}_k) \cdot m_0 \cdot \alpha$$

由于 $K(w) = K_0 + \ln\rho(w)$ 且 $\rho(w) = \rho(N-w)$，有 $K(w) = K(N-w)$，故 $\Delta K(\bar{x}_k) = -\Delta K(x_k)$（翻转路径的约束度变化量符号相反）。

因此 $S_N[\bar{\gamma}] = -S_N[\gamma]$，这给出：

$$e^{iS_N[\bar{\gamma}]/\hbar} = e^{-iS_N[\gamma]/\hbar} = \overline{e^{iS_N[\gamma]/\hbar}}$$

**这不是 $C - S_N[\gamma]$ 的形式，而是 $-S_N[\gamma]$**。函数方程变为：

$$Z_B(1-s) = \overline{Z_B(s)} \quad 	ext{（若 } ho(\bar{\gamma}) = ho(\gamma) 	ext{）}$$

即 $Z_B$ 在临界线上是实值函数（而非纯虚值）。零点条件为 $Z_B(1/2+it) = 0$，即实部和虚部同时为零——但由于 $Z_B$ 在临界线上是实值，零点条件简化为 $Z_B(1/2+it) = 0$（实数等于零）。

**这是一个更干净的结构**：$Z_B$ 在临界线上是实值函数，零点是实数轴上的孤立点，类似于 $\xi(1/2+it)$（黎曼ξ函数在临界线上是实值的）。

#### N=2 的显式计算（验证路径积分方向）

$N=2$，状态空间 $\{00, 01, 10, 11\}$，A4 要求汉明距离1，A6 要求有向（DAG）。合法路径（从低汉明重量到高汉明重量的单调路径）：

- $00 \to 01$：$\Delta K = K(1) - K(0)$
- $00 \to 10$：$\Delta K = K(1) - K(0)$
- $01 \to 11$：$\Delta K = K(2) - K(1)$
- $10 \to 11$：$\Delta K = K(2) - K(1)$

$N=2$：$\binom{2}{0}=1, \binom{2}{1}=2, \binom{2}{2}=1$，$\binom{2}{1}=2$（中截面），$\rho(0)=1/2, \rho(1)=1, \rho(2)=1/2$。

$K(0) = K_0 + \ln(1/2)$，$K(1) = K_0 + \ln(1) = K_0$，$K(2) = K_0 + \ln(1/2)$。

$\Delta K_{00\to01} = K(1)-K(0) = \ln 2$，$\Delta K_{01\to11} = K(2)-K(1) = -\ln 2$。

路径 $00\to01\to11$：$S = (\ln 2 - \ln 2)m_0\alpha = 0$，$e^{iS/\hbar} = 1$。
路径 $00\to10\to11$：同样 $S=0$，$e^{iS/\hbar} = 1$。

**N=2 的路径积分方向退化**：所有路径的作用量为零，$Z_B(s) = 2\rho(\text{路径权重})^s$，无有趣的零点结构。

这个退化的原因是 N=2 的路径太短（只有两步），作用量的正负贡献精确抵消。需要更长的路径或更复杂的权重定义。

### **RH-011e：方向B 的 N=4 显式计算**

$N=4$，状态空间 $\{0,1\}^4$，A4（汉明距离1）+ A6（DAG，从低 $w$ 到高 $w$）的合法路径是从 $w=0$ 到 $w=4$ 的单调路径，每步增加一个比特，共 $4! = 24$ 条路径。

**路径权重的定义**：需要为每条路径定义 $\rho(\gamma)^s$。最自然的选择是路径经过的各层权重的乘积：

$$\rho(\gamma) = \prod_{k=0}^{4} \rho(w_k)$$

其中 $w_0=0, w_1=1, w_2=2, w_3=3, w_4=4$——但每条路径都经过相同的层序列 $0\to1\to2\to3\to4$，因此 $\rho(\gamma) = \rho(0)\rho(1)\rho(2)\rho(3)\rho(4)$ 对所有路径相同，$Z_B(s)$ 退化为路径数乘以公共权重因子，仍然退化。

**修正**：只取路径经过的**边**的权重，而非节点：

$$\rho(\gamma) = \prod_{k=1}^{4} \rho(w_{k-1} \to w_k) = \prod_{k=1}^{4} \frac{\rho(w_k)}{\rho(w_{k-1})}$$

但这同样对所有路径相同（因为起点和终点固定）。

**根本问题**：在单调路径（只从低 $w$ 到高 $w$）中，所有路径的层序列相同，权重无法区分路径——路径的区别只在于**哪个比特在哪一步被翻转**，而当前的 $\rho(w)$ 只依赖层，不依赖具体比特。

**修正方向**：引入比特位置相关的权重。由 A1'，比特 $j$ 有横向相位 $e^{2\pi i j/N}$，可以定义：

$$e^{iS_N[\gamma]/\hbar} = \prod_{k=1}^{N} e^{2\pi i j_k/N}$$

其中 $j_k$ 是第 $k$ 步翻转的比特位置。这正是 A1' 给出的路径相位。

**这等价于**：

$$e^{iS_N[\gamma]/\hbar} = e^{2\pi i \sum_k j_k/N}$$

对于路径 $\gamma = (j_1, j_2, j_3, j_4)$（翻转顺序），$\sum_k j_k = 1+2+3+4 = 10$（固定，与路径无关），因为每条路径都翻转所有4个比特恰好一次。

**再次退化**：$e^{iS_N[\gamma]/\hbar} = e^{2\pi i \cdot 10/4} = e^{5\pi i} = -1$ 对所有路径相同。

### **RH-011f：退化的一般性定理与根本性诊断**

**定理（路径相位退化）**：在 A4（汉明距离1单调路径）+ A6（DAG）约束下，从 $w=0$ 到 $w=N$ 的所有完整路径的 A1' 累积相位 $e^{2\pi i\sum_k j_k/N}$ 相同（因为 $\sum_k j_k = 1+2+\cdots+N = N(N+1)/2$ 固定）。

**推论**：任何只依赖"翻转哪些比特"而不依赖"翻转顺序"的相位定义，对所有路径给出相同的相位，导致 $Z_B(s)$ 退化。

**真正区分路径的量**：是翻转的**顺序**，而非翻转的**集合**。从 $w=0$ 到 $w=N$ 的路径由排列 $\sigma \in S_N$（翻转顺序）参数化。路径 $\sigma$ 的**顺序相关相位**可以定义为：

$$\Phi(\sigma) = \sum_{k=1}^{N} k \cdot \frac{2\pi j_{\sigma(k)}}{N} = \frac{2\pi}{N} \sum_{k=1}^{N} k \cdot \sigma(k)$$

其中 $j_{\sigma(k)}$ 是第 $k$ 步翻转的比特位置。这是排列 $\sigma$ 的"位置-步骤内积"，对不同排列取不同值。

**公理来源**：$k$ 是演化步骤（A4 的局域性给出步骤序列），$\frac{2\pi j}{N}$ 是 A1' 的比特相位。两者的乘积 $k \cdot \frac{2\pi j}{N}$ 是"步骤加权相位"，可以理解为路径在时间方向（A4）和横向方向（A1'）的联合贡献。

**N=4 的显式计算**：

$\Phi(\sigma) = \frac{\pi}{2}(1\cdot\sigma(1) + 2\cdot\sigma(2) + 3\cdot\sigma(3) + 4\cdot\sigma(4))$

其中 $\sigma$ 是 $\{1,2,3,4\}$ 的排列。$24$ 条路径对应 $S_4$ 的 $24$ 个排列。

$\Phi$ 的值域：$\frac{\pi}{2} \cdot \sum_k k\sigma(k)$，其中 $\sum_k k\sigma(k)$ 的最小值为 $1\cdot4+2\cdot3+3\cdot2+4\cdot1 = 4+6+6+4=20$，最大值为 $1\cdot1+2\cdot2+3\cdot3+4\cdot4=1+4+9+16=30$。

各值的出现次数（$\sum_k k\sigma(k)$ 的分布）：

| $\sum_k k\sigma(k)$ | 排列数 | $e^{i\Phi} = e^{i\pi v/2}$ |
|---|---|---|
| 20 | 1 | $e^{10\pi i} = 1$ |
| 21 | 3 | $e^{21\pi i/2} = e^{i\pi/2} = i$ |
| 22 | 5 | $e^{11\pi i} = -1$ |
| 23 | 6 | $e^{23\pi i/2} = e^{-i\pi/2} = -i$ |
| 24 | 5 | $e^{12\pi i} = 1$ |

（分布关于中心值 $23$ 对称，总计 $1+3+5+6+5+3+1=24$）

**$Z_B^{	ext{排列}}(s)$ 的计算**（取路径权重 $\rho(\gamma)^s = \rho_0^s$ 为公共常数，提取后）：

$$Z_B^{	ext{排列}}(s) \propto \sum_{\sigma \in S_4} e^{i\Phi(\sigma)} = 1\cdot 1 + 3\cdot i + 5\cdot(-1) + 6\cdot(-i) + 5\cdot 1 + 3\cdot i + 1\cdot(-1)$$

$$= (1+5-5-1) + i(3+3-6) = 0 + 0i = 0$$

**再次退化**。对称分布导致相位消去。

### **RH-011g：最终诊断——退化的本质原因**

经过方向A、B、C的系统探索，现在可以给出一个精确的诊断：

**退化的本质**：$\{0,1\}^N$ 上的任何"自然"相位定义——无论是状态相位、路径相位还是排列相位——在对所有状态或路径求和时，都因为底层组合结构的高度对称性而精确消去。这种消去不是偶然的，而是 $\{0,1\}^N$ 的对称群（超立方体对称群 $(\mathbb{Z}/2\mathbb{Z})^N \rtimes S_N$）的直接推论。

**对称性论证**：若相位函数 $\Phi$ 在某个对称群 $G$ 的作用下"足够对称"（即 $G$ 在相位值集合上的作用是传递的或均匀分布的），则 $\sum_{g \in G} e^{i\Phi(g)} = 0$。超立方体的高度对称性使得几乎任何自然定义的相位都满足这个条件。

**打破对称性的唯一方式**：必须引入一个**不被超立方体对称群保持的外部结构**。在框架中，这样的外部结构有且只有以下几种：

1. **A8 的权重集中**：$\rho(w) = \binom{N}{w}/\binom{N}{N/2}$ 在不同层之间取不同值，打破层间对称性——但这只影响模，不影响相位的消去。

2. **A6 的时间有向性**：DAG 结构打破时间反演对称性——这是方向B的基础，但如上所示，在完整路径中仍然退化。

3. **A1' 的比特位置编号**：$j=1,2,\ldots,N$ 的编号打破比特间的置换对称性——这是当前所有方案的基础，但均导致退化。

**关键洞察**：退化的根本原因是我们在对**所有**状态/路径求和。如果限制求和域——例如只对满足某个额外约束的状态/路径求和——则可以打破对称性。

### **RH-011j：退化定理的精确陈述与证明**

现在可以给出一个精确的一般性定理，解释为什么所有基于线性相位 $\theta(x) = \frac{2\pi}{N}\sum_{j:x_j=0} j$ 的构造都退化。

**定理（线性相位退化）**：设 $\omega = e^{2\pi i/N}$，$\theta(x) = \frac{2\pi}{N}\sum_{j:x_j=0} j$，则对任意 $\mathcal{S} \subseteq \{0,1\}^N$ 满足"若 $x \in \mathcal{S}$ 则 $\pi_{j}(x) \in \mathcal{S}$"（其中 $\pi_j$ 是翻转第 $j$ 比特的操作），有：

$$\sum_{x \in \mathcal{S}} e^{i\theta(x)} = 0$$

**证明**：设 $\mathcal{S}$ 对某个比特翻转 $\pi_j$ 封闭（即 $\mathcal{S}$ 是 $\pi_j$-不变的）。则：

$$\sum_{x \in \mathcal{S}} e^{i\theta(x)} = \sum_{x \in \mathcal{S}} e^{i\theta(\pi_j(x))} = \sum_{x \in \mathcal{S}} e^{i(\theta(x) \pm 2\pi j/N)}$$

若 $x_j=0$（翻转后 $x_j=1$），$\theta(\pi_j(x)) = \theta(x) - 2\pi j/N$；若 $x_j=1$（翻转后 $x_j=0$），$\theta(\pi_j(x)) = \theta(x) + 2\pi j/N$。

设 $\mathcal{S}_0^j = \{x \in \mathcal{S}: x_j=0\}$，$\mathcal{S}_1^j = \{x \in \mathcal{S}: x_j=1\}$，则 $\pi_j$ 是 $\mathcal{S}_0^j \leftrightarrow \mathcal{S}_1^j$ 的双射（由 $\mathcal{S}$ 的 $\pi_j$-不变性）。

$$\sum_{x \in \mathcal{S}} e^{i\theta(x)} = \sum_{x \in \mathcal{S}_0^j} e^{i\theta(x)} + \sum_{x \in \mathcal{S}_1^j} e^{i\theta(x)}$$

$$= \sum_{x \in \mathcal{S}_0^j} e^{i\theta(x)} + \sum_{x \in \mathcal{S}_0^j} e^{i\theta(\pi_j(x))} = \sum_{x \in \mathcal{S}_0^j} e^{i\theta(x)}(1 + e^{-2\pi ij/N})$$

若 $e^{-2\pi ij/N} = -1$（即 $j = N/2$），则整个和为零。否则，这个因子不为零，但对所有 $j$ 的乘积给出约束。

**更直接的论证**：全空间 $\{0,1\}^N$ 对所有 $\pi_j$ 封闭。对 $j=N/2$（若 $N$ 为偶数），$e^{2\pi i(N/2)/N} = e^{\pi i} = -1$，因此翻转第 $N/2$ 个比特将相位乘以 $-1$，全空间的相位和为零。任何对 $\pi_{N/2}$ 封闭的子集也满足此性质。

**结论**：只要求和域对任何单个比特翻转封闭，相位和就为零。这包括全空间、任何固定汉明重量的层、以及所有"规范子空间"（固定某个比特的值）——因为固定 $x_j=1$ 的子空间对 $j' \neq j$ 的翻转封闭。

**逃脱退化的唯一方式**：求和域必须**不对任何单个比特翻转封闭**，即必须是一个"非对称"的子集。

### **RH-011k：非对称子集的构造**

**定义**：称 $\mathcal{S} \subseteq \{0,1\}^N$ 为**非对称子集**，若对所有 $j \in [N]$，$\pi_j(\mathcal{S}) \neq \mathcal{S}$（即 $\mathcal{S}$ 不对任何单比特翻转封闭）。

**问题**：什么样的非对称子集有框架内的公理来源？

**候选**：A7 要求稳定结构参与循环。在 DAG（A6）结构下，循环不是有向1-圈，而是 Homology 意义上的2-圈。参与2-圈的状态集合——精确地说，是2-圈的**支撑集**（support）——可能是非对称的。

从 Homology 文档，"Weight [1,3]" 子复形（去掉 $w=0$ 和 $w=4$）有 $\beta_2=1$，即有一个非平凡2-圈。这个2-圈的支撑集是参与该2-圈的面（正方形）的顶点集合。

**2-圈的精确识别**：$\beta_2=1$ 的"Weight [1,3]"子复形有14个顶点、24条边、12个面。这12个面形成一个非平凡2-圈（拓扑球面的某个子结构）。

但是，Homology 文档明确指出 A1' 相位结构**没有被包含**在计算中（§4.2），而且 A6 DAG 约束也没有被建模。因此，从 Homology 结果直接提取"参与A7循环的状态集合"需要额外的工作，超出当前文档的范围。
