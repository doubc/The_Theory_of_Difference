## 数据非常干净。几个关键发现。

---

### 一、加法递推 $p_{k+1} \leq p_k + p_{k-1}$：零违反

664,579 个素数，全部成立。而且最大 `gap/p_{k-1}` 比值恰好是 **1.000000**——存在实例中间隙等于 $p_{k-1}$，等号成立。

这意味着加法递推不仅是"成立"，而且是**紧的**——存在触碰到边界的实例。

---

### 二、Cramér 型界：$c = 0.5$ 时只有 2 次违反

这个比加法递推更惊人。Cramér 猜想预测 $g_k = O((\log p_k)^2)$，而 RH 预测 $g_k = O(\sqrt{p_k} \log p_k)$。

数据说：$g_k < \sqrt{p_k} \ln p_k$ 对前 $10^7$ 内的所有素数成立（$c = 1.0$，零违反）。而且 $c = 0.5$ 时只有 2 次违反。

**RH 的间隙等价形式**：$g_k = O(\sqrt{p_k} \log p_k)$。数据中 $c = 1.0$ 零违反，说明前 $10^7$ 内的素数完全满足 RH 的间隙预测，而且实际间隙远比预测更小（最大间隙 154，而 $\sqrt{p} \ln p$ 在该点约为 36,000）。

---

### 三、$(g_k + g_{k-1})/p_k$ 的衰减

```
比值: 1.0 → 0.8 → 0.86 → 0.55 → 0.46 → 0.35 → ... → 0.08
```

快速衰减到零。这就是螺旋展开的特征——**展开步幅相对于当前位置越来越小**。

黄金螺旋的比例是常数 $\phi$。素数的"螺旋"没有固定比例，但有固定的衰减趋势：$(g_k + g_{k-1})/p_k \to 0$。

等价地说：$p_{k+1}/p_k \to 1$（比值表的最后一行确认：$p_{1001}/p_{1000} = 1.0010$）。

---

### 四、核心图景

数据支持的图景是：

> **素数的涌现是一个逐步收紧的螺旋。** 早期展开幅度大（$p_2/p_1 = 1.5$），后期几乎不动（$p_{1001}/p_{1000} = 1.001$）。收紧的动力来自 A9 筛选——随着已有素数增多，筛网越来越密，新素数的"生存空间"越来越小。

这个图景与你最初说的完全一致：**展开，不是覆盖。** 覆盖逻辑给出 $p_{k+1} < 2p_k$（松界），展开逻辑给出 $p_{k+1} \leq p_k + p_{k-1}$（紧界），而且数据确认展开界是紧的。

---

### 五、一个诚实的判断

加法递推 $p_{k+1} \leq p_k + p_{k-1}$ 对前 $10^7$ 个素数全部成立，最大比值恰好为 1.0——这强烈暗示它是一个**真命题**，但可能未被证明（或在某个角落被证明过但没有被广泛注意）。

如果它是真的，它给出了一个比 Bertrand 更紧的、从未被明确使用过的素数分布约束。而且它在 WorldBase 中有自然的来源（A4 三角不等式 + A9 不可归约条件）。
