## **提升精度的三个方向（2026-04-17 新增）**

基于参数不敏感性的发现，进一步思考：**从拓扑构造而言，是否还有提升空间？比如把环做大，会不会更准确？**

### **理论分析**

#### **1. 增大R和w的效果**

当 R 和 w 同时增大时（保持比值不变）：
```
λ_mn = (m/R)² + (nπ/w)²  ↓ 本征值变小
γ_k = 0.5·√(4λ - 1)      ↓ 谱零点更密集
```

**物理意义**：更大的环 → 更高的谱分辨率 → 能捕捉素数分布中更精细的结构。

**类比傅里叶变换**：增加采样点数，频谱分辨率提高。

#### **2. 但是存在权衡（Trade-off）**

从参数优化结果看：
- 所有 (R, w) 组合的误差都在 0.336% 附近
- 变化不超过 0.001%
- **系统可能已经达到渐近极限**

为什么？因为预测误差由三部分组成：
```
总误差 = 主项截断误差 + 振荡项截断误差 + 数值误差
       ↓                ↓                  ↓
    ~O(1/log⁵n)      ~O(1/K_max)        ~机器精度
```

当前配置已经使用了 200×200=40,000 个本征模式，振荡项截断误差可能已经不是主要瓶颈。

#### **3. 真正的瓶颈在哪里？**

**最可能的瓶颈：振荡项振幅衰减律的选择**

当前代码使用：
```python
amplitude = 2.0 / ((k + 1) ** alpha)  # alpha=1.0（经验选择）
```

但根据黎曼ζ函数的显式公式（Explicit Formula），正确的振幅应该是：
```python
amplitude ~ 1/γ_k  # 与零点虚部成反比
```

**这是最有潜力的改进方向！**
