# 差异论V1.7升级提纲：从结构解释到生成式世界

---

## 导言

差异论自V1.0至V1.6，经历了从"差异作为第一范畴"的本体论宣言，到九条生成链的系统建构，再到应用层（期货市场解读）的初步检验。V1.6确立了四卷本结构——《WorldBase》奠基、《差异即世界》展开九机制、《象界》补足中段、《差异论》总成——并配备了可能性空间、事件、最小变易、最近稳态、叙事递归五套方法工具。

然而，V1.6的五方法论是分章展开的独立工具，各自自足，缺乏回环。最近的系统性对话揭示了若干对核心理论有实质推进的新发现：五方法论之间存在内在的螺旋循环结构；叙事递归的地位需要从方法论末位上调至世界生成的核心机制；需要区分结构事件与底图事件两类本质不同的现象；需要回答差异论与关系论、过程论、系统论等既有元理论的确切关系；需要对理论自身的命题类型和成立条件做出更精确的界定；需要证明九机制与《象界》八环节的发生学咬合；以及需要一套更严格的形式化表达系统。

本提纲将上述新发现整理为七个升级方向，作为V1.7正式修订的工作框架。

## 一、五方法论构成非重复螺旋循环

### 1.1 V1.6的问题

V1.6将可能性空间（P）、事件（E）、最小变易（M）、最近稳态（S）、叙事递归（R）作为五套独立的方法论工具分章展开。每一章各自完整，但彼此之间的关系是并列的——它们是"五把钥匙"，研究者根据案例需要选用。

这种安排的缺陷在于：它隐含地假设了世界可以被切割为独立的分析片段，而忽略了这五个环节之间存在内在的发生学顺序——它们不是五个可选工具，而是世界自身运转的一个完整回路。

### 1.2 螺旋循环的发现

对话中逐步确认：这五方法论不是并列的五把钥匙，而是世界生成的一个完整回路的五个环节：

**可能性空间 → 事件压缩可能性 → 最小变易 → 最近稳态 → 叙事递归 → 新可能性空间**

更关键的是，这不是一个封闭的圆环。每一轮回到可能性空间时，那个空间已经被前一轮的全部过程改写了。P_{t+1} ≠ P_t。叙事递归不只是对既有安排的解释，它会改变主体对自身、对他人、对制度、对未来的理解，从而重排可能性空间的结构。

形式化表达：

$$P_{t+1} = N(S(M(E(P_t))))$$

其中 N 表示叙事递归操作，S 表示最近稳态落定，M 表示最小变易选择，E 表示事件压缩。括号的嵌套顺序不是任意的，它反映了发生学的依赖关系：事件压缩以既有可能性空间为前提，最小变易以压缩后的空间为条件，最近稳态以变易方向为根据，叙事递归以稳态安排为对象，新可能性空间以叙事框架为边界。

### 1.3 非重复性的含义

"非重复"意味着：即使外部条件完全相同，由于叙事递归已经改写了可能性空间，世界不会回到原点。这不是黑格尔式的辩证循环（正题-反题-合题回到更高起点），也不是尼采式的永恒回归，而是：世界每经历一轮螺旋，其可能性空间的拓扑结构本身已经发生了不可逆的改变。

这意味着差异论的时间观不是周期性的，也不是线性的，而是螺旋递归的——每一轮都是新的，但每一轮的新又以前一轮为条件。

### 1.4 对V1.6的修订要求

- 在方法论部分增加一章"五方法论的螺旋循环"，放在五章之后作为总成
- 明确标注五环节之间的发生学依赖关系
- 形式化表达 P_{t+1} = N(S(M(E(P_t))))
- 以具体案例（如K12教培行业）展示一轮完整螺旋的运转

## 二、叙事递归的位置上调

### 2.1 V1.6的安排

V1.6将叙事递归放在第四卷（《差异论》）的方法论部分，位置偏后。它被理解为一种组织工具——主体通过叙事来组织行动、协调预期、赋予意义。这是一种"功能主义"的定位：叙事是有用的，但不是根本性的。

### 2.2 新发现：叙事递归是世界生成的核心机制

对话中反复出现一个模式：在每一个关键案例中，决定事件走向的不只是结构力量和资源条件，更是叙事递归的运作。叙事不是事后解释，而是事中生成。具体表现为：

**（一）叙事递归至少三层**

- **小叙事递归**：组织一次行情、一次危机中的共同行动。例如，期货市场中多头叙事与空头叙事的对抗如何改变持仓结构；一次企业危机中管理层的"自救叙事"如何动员资源。

- **制度正当化叙事**：稳定某个最近稳态。例如，"市场经济是最优资源配置方式"这一叙事如何稳定了市场经济的制度安排；"民主是最不坏的制度"如何稳定了代议制民主。

- **文明级生成叙事**：改写理解世界的基本坐标。例如：
  - 《资治通鉴》的"治乱兴衰"叙事如何塑造了中国传统政治的基本理解框架
  - 亚当·斯密"看不见的手"如何重构了经济行动的合法性基础
  - 凯恩斯"政府调控"如何创造了全新的政策可能性空间
  - 卡尼曼对"理性人假说"的系统修正如何动摇了整个新古典经济学的根基

**（二）高阶叙事递归的定义**

理论化解释进入世界后，改变主体理解自身、理解他者、理解制度、理解未来的方式，并由此重排可能性空间的过程。

这个定义的关键在于"进入世界后"——叙事递归不是在书斋里完成的，它必须进入流通、进入争论、进入实践，才能成为改变可能性空间的力量。凯恩斯的理论在1936年出版时只是文本，当它进入各国政策制定者的理解框架、进入大学教科书、进入央行决策逻辑后，它才成为改变世界经济可能性空间的叙事递归。

**（三）叙事的五个中介动作**

叙事是差异从分散状态进入共同行动状态的中介机制，它完成五个动作：

1. **筛选差异**：从无限的差异中选择哪些是重要的、相关的、需要回应的
2. **命名差异**：给被选中的差异赋予名称、范畴和归属
3. **连接差异**：将分散的差异编织成因果链或意义网络
4. **行动化差异**：将理解转化为可执行的行动方案
5. **递归验证差异**：行动结果反过来验证、修正或推翻原有叙事

这五个动作不是一次性的，而是循环运转的——每一次行动的结果都成为新一轮筛选的素材。

### 2.3 对V1.6的修订要求

- 将叙事递归从第四卷方法论末位上调，重新定位为世界生成的核心机制之一
- 在九机制链中明确叙事递归的位置——它至少在"自指"之后，属于高语义递归层
- 展开三层叙事递归的系统论述
- 给出高阶叙事递归的精确定义
- 展开叙事五中介动作的分析
- 在每个重要案例中展示叙事递归的运作

## 三、底图事件与底图承接

### 3.1 两类事件的区分

新东方K12案例证明了一类不同于普通结构事件的现象。2021年"双减"政策出台，K12学科培训的整个行业基础被政策性关闭。这不是某个企业在竞争中失利（结构事件），而是整个行业的可能性空间被改写（底图事件）。

需要区分两类事件：

**结构事件（Structural Event）**：旧结构还在，但路径变窄。例如LME镍事件——伦敦金属交易所的镍期货市场出现极端波动，多家机构被逼仓，但镍市场的基本结构（交易所、合约、参与者、定价机制）仍然存在。事件改变了短期路径，但没有改写底图。

**底图事件（Base-Map Event）**：旧结构赖以运转的可能性空间被改写。例如：
- K12教培行业：政策直接关闭了学科培训的商业空间
- 第一次工业革命：机器生产改写了手工劳动的整个可能性空间
- 鸦片战争：朝贡体系的可能性空间被强行改写
- 信息革命：信息生产和分发的整个底图被重绘

### 3.2 底图承接

"底图承接"描述的是：当底图事件发生后，旧资源、旧主体、旧叙事如何在新空间中寻找承接位置的过程。

底图承接不是简单的"转型"——转型暗示存在一个预设的目标路径，主体只需切换过去。底图承接更准确的描述是：在新可能性空间中，旧积累（技能、资本、品牌、关系网络、制度记忆）的价值需要被重新评估，某些旧积累可能获得新位置，某些可能彻底贬值，某些可能以扭曲的方式被重新利用。

**历史案例支撑：**

- **第一次工业革命**：城市手工工匠的技术积累在机器生产面前大幅贬值，但部分工匠转化为工厂技术管理人员，其"手艺人"身份转化为"技术监督者"身份
- **新世界发现**：旧大陆的贸易网络、航海技术、商业制度在新空间中找到了承接位置，但旧的香料贸易路线被绕过
- **1840年鸦片战争**：朝贡体系下的外交资源、贸易习惯、文化自信在新条约体系中被重新定价
- **信息革命**：传统媒体的采编能力、品牌公信力、分发渠道在数字空间中寻找承接
- **城市工匠转为工人阶级**：手艺技能转化为工厂纪律，行会制度转化为工会制度
- **马车夫被汽车替代**：驾驶技术部分可迁移，但马匹养护知识彻底贬值
- **羊吃人运动**：农业用地转化为牧场用地，农民的身份从"土地耕作者"变为"无地劳动者"
- **码农可能被AI替代**：编程能力的价值面临重新评估，但"理解问题、拆解问题"的能力可能获得新的承接位置

### 3.3 "底图承接"的概念定位

需要特别说明："底图承接"是一个描述性概念，不是一个新的核心名词。它不替代"最近稳态"，而是在底图事件这一特殊情境下，对最近稳态形成过程的一种具体化描述。在底图事件中，最近稳态的形成不是在既有结构内的路径选择，而是在整个底图改写后的资源重配。

### 3.4 对V1.6的修订要求

- 在事件理论部分增加"结构事件与底图事件"的区分
- 展开"底图承接"概念，明确其描述性定位
- 用多个历史案例展示底图承接的不同模式
- 不修改"最近稳态"等核心名词，但丰富其在底图事件情境下的内涵

## 四、差异论与其他元理论的归位

### 4.1 问题的提出

差异论面临一个常见的理论间题：关系论、过程论、系统论、生成论、约束论、主体论、叙事论等元理论，似乎各自也能解释差异论试图解释的现象。那么差异论与它们是什么关系？是竞争关系、互补关系，还是其他？

### 4.2 核心论证：它们是九机制不同环节的派生形态

这些元理论不是差异论的外部竞争者，而是九条生成链中不同环节的派生形态。它们共同的错误在于：把中途生成物误认为第一起点。

具体归位如下：

| 元理论 | 在九机制中的位置 | 误认的起点 |
|--------|------------------|------------|
| **关系论** | 聚簇、层级之后 | 把"关系"当作第一范畴，忽略了关系是差异聚簇和层级化的派生结果 |
| **过程论** | 最小变易、破缺、循环之间 | 把"过程"当作第一范畴，忽略了过程是差异变易和破缺的派生结果 |
| **系统论** | 层级、循环、锁定之后 | 把"系统"当作第一范畴，忽略了系统是差异层级化、循环和锁定的派生结果 |
| **生成论** | 整条九机制链 | 把"生成"当作第一范畴，但生成论本身无法解释生成的机制——这恰恰是九机制链所回答的 |
| **约束论** | 守恒、先天完备性、最小变易 | 把"约束"当作第一范畴，忽略了约束是差异守恒和完备性条件的派生结果 |
| **主体论** | 自指之后 | 把"主体"当作第一范畴，忽略了主体是差异自指的派生结果 |
| **叙事论** | 自指之后的高语义递归 | 把"叙事"当作第一范畴，忽略了叙事是高阶差异递归的派生结果 |

### 4.3 这一归位的理论意义

这一归位不是为了贬低其他理论，而是为了回答一个根本问题：差异论为什么不能被这些理论替代？

答案是：这些理论各自抓住了九机制链的某个环节，但无法解释该环节为什么发生、从何而来。关系论无法解释关系何以可能（因为关系预设了可区分的项），过程论无法解释过程何以启动（因为过程预设了差异），系统论无法解释系统何以涌现（因为涌现预设了层级化）。差异论通过从"可区分性之所以可能"出发，提供了一个统一的生成框架，使得这些理论各自的洞见可以在一个更大的框架中获得定位。

### 4.4 对V1.6的修订要求

- 增加一章"差异论与既有元理论的关系"
- 系统展开七种元理论的归位论证
- 说明归位不是贬低，而是定位
- 用具体案例说明每种元理论的解释力边界

## 五、差异论的不可绕开性与自我限缩

### 5.1 不可绕开性论证

差异论的第一性命题是："差异是可区分性之所以可能"。这个命题的特殊性在于：它不是一个经验命题，无法被普通的经验反例证伪。你不能通过举出一个"没有差异"的例子来反驳它，因为"举出一个例子"这个行为本身已经预设了可区分性。

这意味着差异论具有一种"不可绕开性"——任何试图反驳它的论证，都已经在使用它。

### 5.2 四类命题的区分

为了防止不可绕开性变成不可证伪性的万能盾牌，需要区分差异论中四类不同的命题：

1. **经验命题**：关于具体差异现象的陈述（如"K12行业经历了底图事件"）。这类命题可以被经验检验。

2. **方法命题**：关于分析方法的陈述（如"使用可能性空间框架分析市场"）。这类命题可以被实用检验——它是否有用、是否能产生新的洞见。

3. **生成命题**：关于生成机制的陈述（如"差异经聚簇生成结构"）。这类命题需要逻辑一致性和发生学论证。

4. **成立条件命题**：关于理论自身适用边界的陈述（如"差异论在主体尚未出现的物理世界中的解释力有限"）。这类命题是理论的自我限缩。

### 5.3 差异分层

为了更精确地使用"差异"概念，需要建立差异的分层体系：

- **D0：原初可区分性**——差异的最低条件，使得"这个不是那个"成为可能
- **D1：结构差异**——差异的聚簇和层级化形成的稳定结构
- **D2：分布差异**——差异在空间和时间上的不均匀分布
- **D3：语义差异**——差异被命名、范畴化、赋予意义
- **D4：事件差异**——差异的突然集中或释放，导致可能性空间的改写

每一层都有其自身的运作逻辑，不能简单还原为下一层。

### 5.4 限缩公约七条

为了防止差异论的滥用和过度延伸，需要遵守以下七条限缩公约：

1. **不替代具体学科研究**：差异论不替代历史学、经济学、社会学的具体研究，它提供的是分析框架而非答案。
2. **不预言具体事件**：差异论可以分析事件的结构，但不能预言下一个事件何时发生、以何种形式出现。
3. **不提供价值判断**：差异论是描述性和分析性的，不提供"好"或"坏"的判断。
4. **不替代主体判断**：差异论可以分析主体行动的结构性条件，但不替代主体自身的决策。
5. **承认适用边界**：差异论在物理世界、生物世界、社会世界中的适用程度不同，需要逐层论证。
6. **接受反证**：如果某个案例中差异论的核心机制无法成立，应当承认解释力有限而非强行套用。
7. **保持可修正性**：差异论的九机制链、形式化表达等都是可修正的，不构成教条。

### 5.5 四个内部检验标准

差异论的每一个核心主张都应当通过以下四个检验：

1. **必要性**：这个主张是否不可或缺？去掉它理论是否仍然成立？
2. **顺序性**：这个主张在生成链中的位置是否不可调换？
3. **完备性**：九机制链作为一个整体是否涵盖了从差异到世界的完整路径？
4. **派生力**：这个主张是否能派生出可检验的次级命题？

### 5.6 九机制的可能断点分析

每一环都需要证明其必要性和充分性。可能的断点包括：

- **聚簇→层级**：聚簇是否必然导致层级？是否存在非层级化的聚簇？
- **层级→守恒**：层级是否必然导致守恒？守恒的条件是什么？
- **守恒→先天完备性**：守恒是否必然导致完备性要求？完备性的边界在哪里？
- **先天完备性→最小变易**：完备性是否必然导致最小变易？最小变易的判据是什么？
- **最小变易→破缺**：最小变易是否必然导致破缺？破缺的触发条件是什么？
- **破缺→循环**：破缺是否必然导致循环？循环的稳定性条件是什么？
- **循环→锁定**：循环是否必然导致锁定？锁定的不可逆程度如何？
- **锁定→自指**：锁定是否必然导致自指？自指的涌现条件是什么？

每一个断点都需要独立的论证，不能仅凭"九机制链整体通顺"来掩盖个别环节的薄弱。

### 5.7 对V1.6的修订要求

- 在导论中增加"差异论的命题类型与适用边界"
- 建立差异分层体系（D0-D4）
- 列出限缩公约七条
- 展开四个内部检验标准
- 对九机制的每个环节进行断点分析
- 在结论中明确理论的自我限缩

## 六、九机制与《象界》八环节的咬合

### 6.1 《象界》的定位

《象界》在差异论四卷本中承担的角色是：补足从低语义结构到前主体态的中段。如果说《WorldBase》建立了差异的基底，《差异即世界》展开了九机制的生成链，那么《象界》回答的问题是：在差异经由九机制逐步生成世界的过程中，从"有结构"到"有主体"之间的那段路径是如何走过的？

### 6.2 八环节的发生学序列

《象界》的八环节不是任意排列的，它们有严格的发生学顺序：

1. **从存在到底象**：最原初的差异呈现为某种尚无边界的"底象"——差异在那里，但还没有被切割。
2. **边界界面化**：差异的聚集开始形成边界，边界成为界面——区分"内"与"外"的最初形式。
3. **循环自维持**：界面内部的差异运转开始形成自我维持的循环——一种不需要外部输入就能暂时保持的动态平衡。
4. **路径记忆化**：循环的重复使得某些路径被"记住"——不是有意识的记忆，而是结构上的路径依赖。
5. **重复复制化**：记忆化的路径开始自我复制——相似的循环在不同的位置出现。
6. **并存筛选化**：多种复制体并存，差异的不均匀分布导致某些复制体比另一些更稳定——一种无意识的"筛选"。
7. **耦合功能化**：筛选后的复制体之间形成耦合——一个的输出成为另一个的输入，功能开始出现。
8. **前主体态汇聚**：耦合和功能化达到某个阈值，前主体态出现——一种能够维持自身边界、响应环境变化、进行路径选择的准主体。

### 6.3 与九机制的咬合

八环节不是九机制的替代，而是九机制在"从结构到主体"这一特定区段的具体展开。两者的咬合关系如下：

| 《象界》环节 | 对应的九机制 | 咬合说明 |
|-------------|-------------|---------|
| 从存在到底象 | 聚簇 | 差异的最原初聚集 |
| 边界界面化 | 层级 | 聚集形成层级，层级形成边界 |
| 循环自维持 | 守恒、循环 | 循环使得差异在运转中保持守恒 |
| 路径记忆化 | 锁定 | 路径依赖是锁定的一种形式 |
| 重复复制化 | 守恒（跨实例） | 守恒使得结构能够复制 |
| 并存筛选化 | 最小变易、破缺 | 筛选是一种最小变易，差异的不均匀分布导致破缺 |
| 耦合功能化 | 层级（高阶）、循环 | 耦合是高阶的层级化和循环 |
| 前主体态汇聚 | 自指 | 前主体态是自指的前奏 |

### 6.4 理论意义

这一咬合证明了两件事：

第一，九机制不是任意排列的——它们的发生顺序有《象界》八环节的发生学论证作为支撑。

第二，主体不能从底层差异中直接生成——必须先有前主体态，而前主体态的形成需要八环节的完整过程。这意味着那些试图从简单的差异直接跳到主体理论的做法（如某些信息论进路、某些自创生理论）忽略了中段的全部复杂性。

### 6.5 对V1.6的修订要求

- 在《差异即世界》与《象界》之间增加衔接章
- 明确展示八环节与九机制的咬合对应关系
- 论证发生学顺序的不可调换性
- 用案例说明中段的不可跳跃性

## 七、形式化表达

### 7.1 结构分析函数

差异论的核心分析结构可以形式化为：

$$P_{t+1} = R(S(M(E(P_t, D_t))))$$

其中：
- $P_t$：t时刻的可能性空间
- $D_t$：t时刻的差异积累
- $E(P_t, D_t)$：事件函数——差异积累在可能性空间中压缩为事件
- $M(\cdot)$：最小变易函数——主体在压缩后的空间中选择最小阻力路径
- $S(\cdot)$：最近稳态函数——系统落在最近的临时可维持状态
- $R(\cdot)$：叙事递归函数——新安排被解释、命名、合法化或反噬
- $P_{t+1}$：被叙事递归改写后的新可能性空间

### 7.2 叙事递归函数

叙事递归的运作可以进一步展开为：

$$N_{t+1} = \Psi(N_t, P_t, A_t, I_t, R_t)$$

其中：
- $N_t$：t时刻的叙事状态（主导叙事、竞争叙事、反叙事的分布）
- $P_t$：t时刻的可能性空间
- $A_t$：t时刻的行动状态（谁在做什么、以什么方式）
- $I_t$：t时刻的制度状态（规则、规范、权力结构）
- $R_t$：t时刻的资源状态（物质资源、人力资源、符号资源）
- $\Psi$：叙事递归算子——综合上述输入生成新的叙事状态

叙事递归函数的意义在于：它不是从外部给定的，而是从可能性空间、行动、制度、资源的互动中生成的。叙事不是"人的主观建构"，而是差异运转到高语义层后的必然产物。

### 7.3 制度相变

当差异积累达到某个临界点时，制度发生相变——从一种稳定状态突然跳到另一种：

$$\Gamma_t = \begin{cases} 0, & \text{若 } K_t > K^* \\ 1, & \text{若 } K_t \leq K^* \end{cases}$$

其中：
- $K_t$：t时刻的差异密度（单位结构中差异的集中程度）
- $K^*$：临界差异密度
- $\Gamma_t$：制度相变指示变量（0表示旧制度仍在，1表示相变已发生）

制度相变的意义在于：它不是渐进的，而是突变的。差异积累是渐进的，但相变是突然的——这解释了为什么许多制度变革看起来"突然"发生，但实际上差异已经积累了很长时间。

### 7.4 扩展状态空间

完整的差异论状态空间可以表示为：

$$X_t = (P_t, V_t, OI_t, I_t, B_t, \sigma_t, C_t, D_t, K_t, A_t, N_t, \Gamma_t)$$

其中：
- $P_t$：可能性空间
- $V_t$：变易状态（正在发生的最小变易的分布）
- $OI_t$：旧身份状态（主体的传统身份和自我理解）
- $I_t$：制度状态
- $B_t$：信念状态（主体的预期、信念、叙事）
- $\sigma_t$：波动率状态（差异释放的速度和幅度）
- $C_t$：承接状态（旧资源在新空间中的承接位置）
- $D_t$：差异积累
- $K_t$：差异密度
- $A_t$：行动状态
- $N_t$：叙事状态
- $\Gamma_t$：制度相变状态

这个状态空间的意义在于：它提供了一个完整的"快照"，描述世界在任何时刻的全部相关状态。研究者的任务是：根据具体案例，确定哪些状态变量是关键的，哪些是次要的，哪些可以忽略。

### 7.5 套利条件改写

在应用层（如期货市场分析），差异论对传统套利条件的改写为：

$$P_1 - P_2 > c + \lambda$$

其中：
- $P_1 - P_2$：两个市场或两个时点之间的价格差异
- $c$：传统交易成本（佣金、滑点、资金成本等）
- $\lambda$：界面断裂成本——差异论特有的概念，指两个市场之间的结构性断裂所造成的额外成本

界面断裂成本 $\lambda$ 不是传统意义上的交易成本。它包括：信息不对称的结构性根源、制度差异造成的摩擦、叙事差异造成的预期分歧、身份差异造成的行为模式差异。

这个改写的意义在于：它解释了为什么某些明显的套利机会长期存在而不被消除——不是因为交易成本太高，而是因为界面断裂成本 $\lambda$ 太高，使得两个市场实际上处于不同的"世界"中。

## 结语：V1.7的核心推进

V1.7相对于V1.6的核心推进可以概括为一句话：**从结构解释到生成式世界。**

V1.6回答的问题是："世界是什么结构？"——差异经由九机制生成结构。

V1.7要回答的问题是："世界如何生成自身？"——差异经由九机制生成结构，结构经由叙事递归改写可能性空间，改写后的可能性空间成为下一轮生成的起点。世界不是一个静态的结构，而是一个不断生成自身的过程。

这个推进要求差异论在以下方面做出实质性修订：

1. **方法论的统一**：五方法论不再是独立工具，而是螺旋循环的五个环节。
2. **叙事递归的上调**：从方法论末位上升为世界生成的核心机制。
3. **事件理论的精细化**：区分结构事件与底图事件，引入底图承接概念。
4. **理论关系的厘清**：与其他元理论的关系从模糊的竞争变为清晰的归位。
5. **自我限缩的明确化**：建立命题类型、差异分层、限缩公约、内部检验标准。
6. **中段的补足**：九机制与《象界》八环节的咬合。
7. **形式化的深化**：从松散的概念到严格的函数表达。

这七项修订不改变差异论的基本立场——差异先于对象，世界由差异生成——但使得这一立场获得了更精确、更可检验、更自我约束的理论形式。

差异论V1.7的目标不是成为一个"无所不能"的理论，而是成为一个"诚实面对自身限度"的理论。一个知道自己的解释力边界在哪里的理论，比一个声称能解释一切的理论更有力量。
