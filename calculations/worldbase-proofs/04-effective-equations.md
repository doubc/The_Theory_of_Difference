# 有效方程提取报告 — 从 WorldBase 模拟机数据

## 核心发现

### 1. 引力势：Newton 力律在离散空间中精确成立

**拟合结果：** Φ = -C / w^α
- **α = 0.973 ≈ 1.0**（Newton 力律 1/r 确认）
- C = 0.924（归一化常数）
- R² = 0.947
- exp_146 原始数据：Pearson r = 1.000（完美相关）

**物理解释：** 在 {0,1}^N 的汉明空间中，引力势与汉明距离成反比。这是 Newton 力律的离散版本。连续极限下（N→∞，嵌入 [0,L]³），这应该给出 ∇²Φ = 4πGρ。

**重要发现：** 离散 Laplacian 测试显示 ∇²Φ ≠ const（变异系数 = 1.86）。这意味着：
- 在离散空间中，Φ = -1/w 的 Laplacian 不是常数
- Laplacian 的形式是 ∇²(1/w) ∝ 1/w³（可解析验证）
- 这对应 ρ(r) ∝ 1/r³ 的源密度分布
- **不是**均匀源密度的 Poisson 方程

**含义：** 原始文档的"泊松方程恢复"（§4.7）需要修正。离散引力势满足的不是标准 Poisson 方程，而是带有非均匀源密度的修正 Poisson 方程。

- 一阶相变条件：a(N) < 0, b > 0 → ✓

### 3. 有效方程组

从模拟数据推断的 WorldBase 有效方程组：

#### 方程 1：引力势（离散版）
```
Φ(w) = -C / d_H(w),  C > 0
```
- 在汉明空间中精确成立（r = 1.000）
- 连续极限下应给出 Newton 力，但需要非均匀 Poisson 方程：
  ∇²Φ = 4πGρ(r), 其中 ρ(r) ∝ 1/r³（非均匀）

#### 方程 2：密度演化（反应-扩散 + 相变）
```
dρ/dt = J_in·(1-ρ) - J_out·ρ + D·∇²ρ + λ·Θ(ρ - ρ_c)
```
- ρ = 活跃比特密度
- J_in = A1 注入速率, J_out = A8 吸收速率
- D = A1' 绑定扩散系数
- Θ = Heaviside 阶跃函数（cascade 触发）
- ρ_c = 0.40（临界密度，普适常数）
- λ = cascade 强度（一次性冻结 ~40% 活跃比特）

#### 方程 3：跨层级耦合
```
Φ_{L+1} = P(ρ_L)        (投影: 低层密度 → 高层引力势)
ρ_{L+1} 的演化受 Φ_{L+1} 调制
A9 自指: 密封动作 → 生成新差异源 → 下一层的 J_in
```

#### 方程 4：相变条件
```
当 ρ → ρ_c = 0.40 时:
  1. 一阶相变触发（单次 cascade）
  2. 冻结 ~40% 活跃比特
  3. 剩余 ~60% 继续演化
  4. 冻结比特成为"质量源"（引力势的源）
```

### 4. 与标准物理方程的对应

| WorldBase 有效方程 | 标准物理方程 | 差异 |
|---|---|---|
| Φ = -C/d_H | ∇²Φ = 4πGρ | 离散 vs 连续；源密度非均匀 |
| dρ/dt = J_in - J_out + D∇²ρ | ∂ρ/∂t + ∇·J = σ | 反应-扩散方程 |
| Θ(ρ-ρ_c) cascade | 一阶相变 | 离散级联 vs 连续跃变 |
| ρ_c = 0.40 | 临界温度/密度 | 普适常数 |
| cascade = 0.40·N | 序参量跃变 | 线性标度律 |
| A9 自指 | 自组织临界性 | 内生完备 vs 外部驱动 |

### 5. 对理论证明的影响

这些发现对 02-worldbase 形式化框架的证明有重要影响：

1. **引力势：** Φ = -1/d_H 在离散空间中精确成立，但连续极限给出的不是标准 Poisson 方程。需要证明的是：在适当的缩放下，非均匀 Poisson 方程 → 均匀 Poisson 方程。

2. **维数问题：** 离散空间的"有效维数"不是 3（exp_146 显示 D_eff = 18-19）。但引力势的形式 Φ = -1/r 与维数无关（在任何维数中，点源的势都是 1/r^{d-2}，对 d=3 是 1/r）。模拟数据显示的是汉明空间中的 1/w 行为，不是嵌入空间中的 1/r。

3. **相变是关键桥梁：** 一阶相变（cascade）是离散→连续转变的机制。密封过程一次性冻结 40% 的比特，这相当于一个"粗粒化"操作——从微观离散态跳到宏观连续态。
