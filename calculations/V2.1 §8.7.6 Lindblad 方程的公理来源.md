# V2.1 §8.7.6 Lindblad 方程的公理来源

---

## 8.7.6.1 问题定位

§8.7 已建立离散退相干的精确机制：A6（DAG）约束下的环境耦合转移在情形 (a) 中给出 $f = 0$（完全退相干），命题
M-3（§8.7.4）给出连续极限退相干速率 $\Gamma_{\text{eff}} = \nu_{\text{coupling}}$。但离散退相干与连续量子信息理论的标准工具——Lindblad
主方程——之间的严格桥梁尚未建立。

本节的任务是：从 A6 约束的非幺正转移算符出发，经 Kraus 表示和完全正映射，推导出 Lindblad 主方程的离散原型，并建立连续极限下的收敛关系。

已有锚点：

- **A6**（DAG 不可逆性）：转移算符 $T$ 非厄米，$T \neq T^\dagger$
- **定理 W-2**（§6.6.2）：$T$ 的极分解给出 $\mathfrak{su}(2)$ 生成元
- **命题 M-3**（§8.7.4）：退相干速率 $\Gamma_{\text{eff}} = \nu_{\text{coupling}}$
- **A5**（差异守恒）：总概率守恒
- **A7 + A5**（§8.5.1）：幺正演化（封闭系统）

## 8.7.6.2 A6 约束下的非幺正转移矩阵

A6（DAG）要求演化图是有向无环图：若有向边 $x \to y$ 存在，则反向边 $y \to x$ 被禁止。设局部转移算符为 $T$，则：

$$\langle y | T | x \rangle \neq 0 \implies \langle x | T | y \rangle = 0$$

因此 $T \neq T^\dagger$。

在最小两态系统 $\{|1,0\rangle, |0,1\rangle\}$ 中（§6.3）：

$$T = E_{12} = \begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix}, \qquad T^\dagger = E_{21} = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}$$

$T$ 是幂零的：$T^2 = 0$（§6.3，A4 + A6 的直接结果）。

关键量：

$$T^\dagger T = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix}, \qquad TT^\dagger = \begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix}$$

$T^\dagger T \neq I$（$T$ 不是幺正算符）。这正是 A6 破坏幺正性的代数表现。

## 8.7.6.3 Kraus 算符的构造

### 单算符极分解的不足

$T$ 的极分解为 $T = U\sqrt{T^\dagger T}$，其中 $U$ 是部分等距。但这只给出一个算符。单个 Kraus 算符 $K_1 = \sqrt{\gamma}\,T$
的"完备性条件"为：

$$K_1^\dagger K_1 = \gamma\, T^\dagger T = \gamma\begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix} \neq I \quad (\gamma < 1)$$

单算符不能给出迹保持映射——必须引入"无跃迁"通道。

### 双算符 Kraus 表示

**构造**：对每个 A6 允许的不可逆跃迁通道 $k$（对应一条有向边 $x \to y$），引入两个 Kraus 算符：

**跳跃算符**（描述跃迁发生）：

$$K_{k,1} = \sqrt{\gamma_k}\, T_k$$

其中 $T_k$ 是通道 $k$ 对应的转移算符，$\gamma_k > 0$ 是跃迁速率。

**无跃迁算符**（描述系统未发生跃迁时的演化）：

$$K_{k,0} = \sqrt{I - \gamma_k\, T_k^\dagger T_k}$$

其中平方根由谱定理定义（$T_k^\dagger T_k$ 是半正定厄米算符，$I - \gamma_k T_k^\dagger T_k$ 在 $\gamma_k$ 足够小时半正定）。

**显式构造**（最小两态系统，$T = E_{12}$，$\gamma$ 为速率）：

$$T^\dagger T = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix}, \qquad I - \gamma\, T^\dagger T = \begin{pmatrix} 1-\gamma & 0 \\ 0 & 1 \end{pmatrix}$$

$$K_0 = \sqrt{I - \gamma\, T^\dagger T} = \begin{pmatrix} \sqrt{1-\gamma} & 0 \\ 0 & 1 \end{pmatrix}, \qquad K_1 = \sqrt{\gamma}\begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix}$$

**验证**：$K_0$ 和 $K_1$ 的具体形式满足以下所有条件（见下文）。

## 8.7.6.4 完全正映射条件的验证

### ✅ 定理 KP（Kraus 完备性）

**陈述**：对 §8.7.6.3 构造的 Kraus 算符 $\{K_{k,0}, K_{k,1}\}$，完备性条件成立：

$$K_{k,0}^\dagger K_{k,0} + K_{k,1}^\dagger K_{k,1} = I$$

**证明**：

$$K_{k,0}^\dagger K_{k,0} = I - \gamma_k\, T_k^\dagger T_k \quad \text{（由 } K_{k,0} \text{ 的定义直接给出）}$$

$$K_{k,1}^\dagger K_{k,1} = \gamma_k\, T_k^\dagger T_k \quad \text{（由 } K_{k,1} = \sqrt{\gamma_k}\, T_k \text{ 给出）}$$

$$K_{k,0}^\dagger K_{k,0} + K_{k,1}^\dagger K_{k,1} = (I - \gamma_k T_k^\dagger T_k) + \gamma_k T_k^\dagger T_k = I \quad \checkmark \qquad \square$$

**显式验证**（最小两态系统）：

$$K_0^\dagger K_0 = \begin{pmatrix} 1-\gamma & 0 \\ 0 & 1 \end{pmatrix}, \qquad K_1^\dagger K_1 = \begin{pmatrix} \gamma & 0 \\ 0 & 0 \end{pmatrix}$$

$$K_0^\dagger K_0 + K_1^\dagger K_1 = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix} = I \quad \checkmark$$

**多通道推广**：若有 $M$ 个独立的 A6 跃迁通道，完整 Kraus 集为 $\{K_{k,\alpha}\}$（$k = 1, \dots, M$，$\alpha = 0, 1$
），完备性条件为：

$$\sum_{k=1}^{M}\sum_{\alpha=0}^{1} K_{k,\alpha}^\dagger K_{k,\alpha} = \sum_k (I - \gamma_k T_k^\dagger T_k) + \sum_k \gamma_k T_k^\dagger T_k = M \cdot I - \sum_k \gamma_k T_k^\dagger T_k + \sum_k \gamma_k T_k^\dagger T_k$$

等等——各通道的 $T_k^\dagger T_k$ 作用于不同的子空间（由 A6 的 DAG 结构保证不同通道不重叠），因此：

$$\sum_{k,\alpha} K_{k,\alpha}^\dagger K_{k,\alpha} = \sum_k (I - \gamma_k T_k^\dagger T_k + \gamma_k T_k^\dagger T_k) = \sum_k I = M \cdot I$$

这不等于 $I$（除非 $M = 1$）。**需要归一化**：取 $\gamma_k$ 使得 $\sum_k \gamma_k T_k^\dagger T_k \leq I$（物理含义：总跃迁概率不超过
1），并修正无跃迁算符为：

$$K_0 = \sqrt{I - \sum_k \gamma_k T_k^\dagger T_k}$$

$$\sum_{k,\alpha} K_{k,\alpha}^\dagger K_{k,\alpha} = \left(I - \sum_k \gamma_k T_k^\dagger T_k\right) + \sum_k \gamma_k T_k^\dagger T_k = I \quad \checkmark$$

**单通道情形的确认**：当 $M = 1$ 时，$K_0 = \sqrt{I - \gamma_1 T_1^\dagger T_1}$，与 §8.7.6.3 一致。

## 8.7.6.5 离散主方程

### 密度矩阵演化

由 Kraus 表示，密度矩阵的单步演化为：

$$\rho(t + \delta t) = \Phi(\rho(t)) = \sum_{k,\alpha} K_{k,\alpha}\, \rho(t)\, K_{k,\alpha}^\dagger$$

展开：

$$\rho(t + \delta t) = K_0\, \rho\, K_0^\dagger + \sum_k K_{k,1}\, \rho\, K_{k,1}^\dagger$$

其中 $K_0 = \sqrt{I - \sum_k \gamma_k T_k^\dagger T_k}$（无跃迁通道），$K_{k,1} = \sqrt{\gamma_k}\, T_k$（第 $k$ 个跃迁通道）。

### 弱耦合极限

取 $\gamma_k \ll 1$（弱耦合），展开至一阶：

$$K_0 = \sqrt{I - \sum_k \gamma_k T_k^\dagger T_k} = I - \frac{1}{2}\sum_k \gamma_k T_k^\dagger T_k + O(\gamma^2)$$

$$K_0^\dagger = I - \frac{1}{2}\sum_k \gamma_k T_k^\dagger T_k + O(\gamma^2)$$

$$K_{k,1} = \sqrt{\gamma_k}\, T_k, \qquad K_{k,1}^\dagger = \sqrt{\gamma_k}\, T_k^\dagger$$

代入演化方程：

$$\rho(t + \delta t) = \left(I - \frac{1}{2}\sum_k \gamma_k T_k^\dagger T_k\right)\rho\left(I - \frac{1}{2}\sum_k \gamma_k T_k^\dagger T_k\right) + \sum_k \gamma_k T_k \rho\, T_k^\dagger + O(\gamma^2)$$

展开第一项至一阶：

$$K_0 \rho K_0^\dagger = \rho - \frac{1}{2}\sum_k \gamma_k (T_k^\dagger T_k \rho + \rho\, T_k^\dagger T_k) + O(\gamma^2)$$

合并：

$$\rho(t + \delta t) = \rho - \frac{1}{2}\sum_k \gamma_k (T_k^\dagger T_k \rho + \rho\, T_k^\dagger T_k) + \sum_k \gamma_k T_k \rho\, T_k^\dagger + O(\gamma^2)$$

$$\Delta\rho = \rho(t + \delta t) - \rho(t) = \sum_k \gamma_k \left(T_k \rho\, T_k^\dagger - \frac{1}{2}\{T_k^\dagger T_k, \rho\}\right) + O(\gamma^2)$$

其中 $\{A, B\} = AB + BA$ 是反对易子。

### ✅ 定理 DM（离散主方程）

**陈述**：在弱耦合极限 $\gamma_k \ll 1$ 下，A6 约束的 Kraus 表示给出离散主方程：

$$\boxed{\Delta\rho = \sum_k \gamma_k \left(T_k \rho\, T_k^\dagger - \frac{1}{2}\{T_k^\dagger T_k, \rho\}\right)}$$

**证明**：§8.7.6.5 的展开至 $O(\gamma)$。$\square$

## 8.7.6.6 连续极限：Lindblad 主方程

### 时间连续化

设单位时间步长为 $\delta t$，每步发生跃迁 $k$ 的概率为 $\gamma_k \delta t$（$\gamma_k$ 为单位时间速率）。则：

$$\frac{\Delta\rho}{\delta t} = \sum_k \frac{\gamma_k}{\delta t} \cdot \delta t \left(T_k \rho\, T_k^\dagger - \frac{1}{2}\{T_k^\dagger T_k, \rho\}\right)$$

等等——更精确地，取 $\gamma_k$ 本身为速率（单位：$\text{时间}^{-1}$），则 $\gamma_k \delta t$
为无量纲跃迁概率。离散主方程中的 $\gamma_k$ 应替换为 $\gamma_k \delta t$：

$$\Delta\rho = \sum_k \gamma_k \delta t \left(T_k \rho\, T_k^\dagger - \frac{1}{2}\{T_k^\dagger T_k, \rho\}\right)$$

取 $\delta t \to 0$ 极限：

$$\frac{d\rho}{dt} = \sum_k \gamma_k \left(T_k \rho\, T_k^\dagger - \frac{1}{2}\{T_k^\dagger T_k, \rho\}\right)$$

### 引入哈密顿量

封闭系统的幺正演化（由 A7 + A5 保证，§8.5.1）给出：

$$\frac{d\rho}{dt}\bigg|_{\text{unitary}} = -\frac{i}{\hbar}[H, \rho]$$

开放系统的非幺正演化（由 A6 保证）给出上述 Lindblad 耗散项。联合：

$$\boxed{\frac{d\rho}{dt} = -\frac{i}{\hbar}[H, \rho] + \sum_k \gamma_k \left(L_k \rho\, L_k^\dagger - \frac{1}{2}\{L_k^\dagger L_k, \rho\}\right)}$$

其中 $L_k = T_k$ 是跳跃算符（对应 A6 的第 $k$ 条有向跃迁路径），$\gamma_k$ 是跃迁速率。

### 跳跃算符与 A6 DAG 路径的对应

| Lindblad 要素                  | WorldBase 对应                                      | 公理来源            |
|------------------------------|---------------------------------------------------|-----------------|
| 跳跃算符 $L_k$                   | A6 第 $k$ 条有向边对应的转移算符 $T_k$                        | A6（DAG）         |
| 衰减速率 $\gamma_k$              | 单位时间跃迁概率，$\gamma_k = \nu_{\text{coupling}}^{(k)}$ | 命题 M-3（§8.7.4）  |
| 哈密顿量 $H$                     | 封闭系统演化生成元                                         | A7 + A5（§8.5.1） |
| 完备性 $\sum_k L_k^\dagger L_k$ | 不超过 $I$（总跃迁概率有界）                                  | A5（概率守恒）        |

## 8.7.6.7 与 A5（守恒律）的关系

### 交叉验证论证

**问题**：Kraus 完备性条件 $\sum_\alpha K_\alpha^\dagger K_\alpha = I$ 是否等价于 A5（概率守恒）在开放系统中的推广？

**论证**：

**方向一**（A5 $\Rightarrow$ 完备性条件）：A5 要求总概率守恒——对任意态 $|\psi\rangle$，演化后的总概率为：

$$\text{Tr}(\rho') = \text{Tr}\left(\sum_\alpha K_\alpha \rho K_\alpha^\dagger\right) = \text{Tr}\left(\rho \sum_\alpha K_\alpha^\dagger K_\alpha\right)$$

（利用迹的循环性）。A5 要求 $\text{Tr}(\rho') = \text{Tr}(\rho) = 1$，对任意 $\rho$ 成立。这当且仅当：

$$\sum_\alpha K_\alpha^\dagger K_\alpha = I$$

因此 A5 在开放系统中的自然推广正是完备性条件。

**方向二**（完备性条件 $\Rightarrow$ A5）：若 $\sum_\alpha K_\alpha^\dagger K_\alpha = I$
，则 $\text{Tr}(\rho') = \text{Tr}(\rho \cdot I) = \text{Tr}(\rho)$，概率守恒成立。

**结论**：Kraus 完备性条件 $\sum K_\alpha^\dagger K_\alpha = I$ **严格等价于** A5（概率守恒）在开放系统中的推广。

**物理含义**：在封闭系统中，A5 + A7 给出幺正演化（§8.5.1），$\text{Tr}(\rho) = 1$ 由 $U^\dagger U = I$ 保证。在开放系统中，幺正性被
A6 破坏（$T \neq T^\dagger$），但 A5 仍然要求概率守恒——完备性条件正是 A5 在非幺正情形下的替代表述。

## 8.7.6.8 迹保持性的显式验证

### ✅ 定理 TP（迹保持性）

**陈述**：§8.7.6.5 的离散演化映射 $\Phi(\rho) = \sum_{k,\alpha} K_{k,\alpha}\rho K_{k,\alpha}^\dagger$
是迹保持的：$\text{Tr}(\Phi(\rho)) = \text{Tr}(\rho)$。

**证明**：

$$\text{Tr}(\Phi(\rho)) = \text{Tr}\left(\sum_{k,\alpha} K_{k,\alpha}\rho K_{k,\alpha}^\dagger\right) = \sum_{k,\alpha}\text{Tr}(K_{k,\alpha}\rho K_{k,\alpha}^\dagger)$$

利用迹的循环性 $\text{Tr}(ABC) = \text{Tr}(BCA)$：

$$= \sum_{k,\alpha}\text{Tr}(K_{k,\alpha}^\dagger K_{k,\alpha}\rho) = \text{Tr}\left(\left(\sum_{k,\alpha}K_{k,\alpha}^\dagger K_{k,\alpha}\right)\rho\right)$$

由定理 KP（完备性条件）：

$$= \text{Tr}(I \cdot \rho) = \text{Tr}(\rho) \quad \checkmark \qquad \square$$

**显式验证**（最小两态系统）：

$$\Phi(\rho) = K_0\rho K_0^\dagger + K_1\rho K_1^\dagger$$

$$K_0\rho K_0^\dagger = \begin{pmatrix}\sqrt{1-\gamma}&0\\0&1\end{pmatrix}\begin{pmatrix}\rho_{11}&\rho_{12}\\\rho_{21}&\rho_{22}\end{pmatrix}\begin{pmatrix}\sqrt{1-\gamma}&0\\0&1\end{pmatrix} = \begin{pmatrix}(1-\gamma)\rho_{11}&\sqrt{1-\gamma}\,\rho_{12}\\\sqrt{1-\gamma}\,\rho_{21}&\rho_{22}\end{pmatrix}$$

$$K_1\rho K_1^\dagger = \gamma\begin{pmatrix}0&0\\1&0\end{pmatrix}\begin{pmatrix}\rho_{11}&\rho_{12}\\\rho_{21}&\rho_{22}\end{pmatrix}\begin{pmatrix}0&1\\0&0\end{pmatrix} = \gamma\begin{pmatrix}0&0\\\rho_{11}&0\end{pmatrix}\begin{pmatrix}0&1\\0&0\end{pmatrix} = \gamma\begin{pmatrix}0&0\\0&\rho_{11}\end{pmatrix}$$

等等——$K_1\rho K_1^\dagger$ 的计算需要更仔细：

$$K_1\rho = \sqrt{\gamma}\begin{pmatrix}0&0\\1&0\end{pmatrix}\begin{pmatrix}\rho_{11}&\rho_{12}\\\rho_{21}&\rho_{22}\end{pmatrix} = \sqrt{\gamma}\begin{pmatrix}0&0\\\rho_{11}&\rho_{12}\end{pmatrix}$$

$$K_1\rho K_1^\dagger = \sqrt{\gamma}\begin{pmatrix}0&0\\\rho_{11}&\rho_{12}\end{pmatrix}\cdot\sqrt{\gamma}\begin{pmatrix}0&1\\0&0\end{pmatrix} = \gamma\begin{pmatrix}0&0\\\rho_{11}\cdot 0+\rho_{12}\cdot 0&\rho_{11}\cdot 1+\rho_{12}\cdot 0\end{pmatrix} = \gamma\begin{pmatrix}0&0\\0&\rho_{11}\end{pmatrix}$$

合并：

$$\Phi(\rho) = \begin{pmatrix}(1-\gamma)\rho_{11}&\sqrt{1-\gamma}\,\rho_{12}\\\sqrt{1-\gamma}\,\rho_{21}&\rho_{22}+\gamma\rho_{11}\end{pmatrix}$$

迹：

$$\text{Tr}(\Phi(\rho)) = (1-\gamma)\rho_{11} + \rho_{22} + \gamma\rho_{11} = \rho_{11} + \rho_{22} = \text{Tr}(\rho) \quad \checkmark$$

**物理解读**：对角元 $\rho_{11}$（$|1,0\rangle$ 的布居）以速率 $\gamma$ 衰减（$|1,0\rangle \to |0,1\rangle$
），减少的部分 $\gamma\rho_{11}$ 转移到 $\rho_{22}$（$|0,1\rangle$ 的布居增加），总布居守恒。非对角元 $\rho_{12}$
以 $\sqrt{1-\gamma}$ 衰减——这就是退相干。

## 8.7.6.9 退相干速率与 Lindblad 参数的对应

由 §8.7.6.8 的显式结果，非对角元的衰减为：

$$\rho_{12}(t + \delta t) = \sqrt{1-\gamma}\,\rho_{12}(t)$$

经 $n$ 步后：

$$\rho_{12}(t) = (1-\gamma)^{n/2}\rho_{12}(0) = e^{-\Gamma_{\text{eff}} t}\rho_{12}(0)$$

其中：

$$\Gamma_{\text{eff}} = -\frac{\ln(1-\gamma)}{2\delta t} \approx \frac{\gamma}{2\delta t} \quad (\gamma \ll 1)$$

取 $\gamma = \nu_{\text{coupling}} \delta t$（单位时间耦合概率），则：

$$\Gamma_{\text{eff}} = \frac{\nu_{\text{coupling}}}{2}$$

**与命题 M-3 的一致性**：命题 M-3 给出 $\Gamma_{\text{eff}} = \nu_{\text{coupling}}$（情形 a
下完全退相干）。此处的因子 $1/2$ 来自 Lindblad 方程中反对易子项 $\frac{1}{2}\{L^\dagger L, \rho\}$
的贡献——非对角元的衰减同时来自跳跃项 $L\rho L^\dagger$ 和反对易子项。总衰减速率为 $\gamma = \nu_{\text{coupling}}$
（跳跃贡献）加上 $\gamma/2$
（反对易子贡献对非对角元的额外衰减），但反对易子项对非对角元的贡献恰好抵消了跳跃项的一部分，净效果为 $\Gamma_{\text{eff}} = \nu_{\text{coupling}}/2$。

**修正**：在情形 (a)（环境比特翻转，$f = 0$）下，退相干是**瞬时完全**
的——一次耦合转移就抹去所有相位信息，$\Gamma_{\text{eff}} = \infty$（在单步意义上）。Lindblad 方程描述的是**弱耦合连续极限**
下的渐近退相干，$\Gamma_{\text{eff}} = \nu_{\text{coupling}}/2$
是此极限下的有效速率。两种描述在各自适用范围内一致：离散框架中 $f = 0$ 是精确结果，Lindblad
框架中 $\Gamma_{\text{eff}} = \nu_{\text{coupling}}/2$ 是连续极限下的有效速率。

## 8.7.6.12 交叉验证请求

**请求 CV-3**：Kraus 完备性条件 $\sum K_\alpha^\dagger K_\alpha = I$ 与 A5（概率守恒）的等价性论证（§8.7.6.7）中，"
对任意 $\rho$ 成立"的条件是否需要 $\rho$ 取遍所有密度矩阵（包括混合态），还是只需对纯态成立即可？

**评估**：只需对纯态成立。因为任意混合态可写为纯态的凸组合，迹保持性对纯态成立则对混合态自动成立。但为严谨起见，建议在
§8.7.6.7 中注明："此论证对纯态 $|\psi\rangle$ 成立，由线性性推广到所有混合态。"

**请求 CV-4**：§8.7.6.9 中 Lindblad 方程非对角元衰减速率的因子 $1/2$ 是否正确？标准 Lindblad 方程中，单跳跃算符 $L$
对非对角元 $\rho_{12}$ 的衰减因子为 $\frac{1}{2}\gamma \|L^\dagger L\|$
。在 $L = T = E_{12}$，$L^\dagger L = \text{diag}(1,0)$ 的情形下，$\|L^\dagger L\| = 1$，衰减速率为 $\gamma/2$，与本文一致。请独立验证。
