## 第二件事：多源配置的度规构造

### **设定**

取两个占据位：$S = \{u_1, u_2\}$，其中 $u_1 = (0,0,0)$，$u_2 = (2,0,0)$（沿 $x$ 轴相距 2）。

定义多源势场：

$$\Phi(u) = -Gm\left(\frac{1}{d_H(u, u_1)} + \frac{1}{d_H(u, u_2)}\right)$$

其中 $d_H(u, u_k) = |u^1 - u_k^1| + |u^2 - u_k^2| + |u^3 - u_k^3|$（曼哈顿距离，即比特配置下的汉明距离在宏观坐标上的体现）。

取 $G=m=1$ 简化计算。

---

### **计算格点 $(1,1,1)$ 处的势场值**

到 $u_1=(0,0,0)$ 的距离：$d_1 = 1+1+1 = 3$

到 $u_2=(2,0,0)$ 的距离：$d_2 = |1-2|+1+1 = 3$

$$\Phi(1,1,1) = -\frac{1}{3} - \frac{1}{3} = -\frac{2}{3}$$

逐一计算所需的邻近格点势场值：

| 格点 $u$ | $d_1 = d_H(u,u_1)$ | $d_2 = d_H(u,u_2)$ | $\Phi(u)$ |
|---|---|---|---|
| $(1,1,1)$ | $3$ | $3$ | $-2/3$ |
| $(2,1,1)$ | $4$ | $2$ | $-1/4-1/2=-3/4$ |
| $(3,1,1)$ | $5$ | $1$ | $-1/5-1=-6/5$ |
| $(0,1,1)$ | $2$ | $4$ | $-1/2-1/4=-3/4$ |
| $(1,2,1)$ | $4$ | $4$ | $-1/4-1/4=-1/2$ |
| $(1,3,1)$ | $5$ | $5$ | $-1/5-1/5=-2/5$ |

### **计算度规分量**

**对角分量 $g_{11}$（沿 $x$ 方向）**：

$$g_{11}(1,1,1) = -\left[\Phi(3,1,1) - 2\Phi(2,1,1) + \Phi(1,1,1)\right]$$

$$= -\left[-\frac{6}{5} - 2\left(-\frac{3}{4}\right) + \left(-\frac{2}{3}\right)\right]$$

$$= -\left[-\frac{6}{5} + \frac{3}{2} - \frac{2}{3}\right]$$

通分（公分母 30）：

$$= -\left[-\frac{36}{30} + \frac{45}{30} - \frac{20}{30}\right] = -\left[-\frac{11}{30}\right] = \frac{11}{30}$$

**对角分量 $g_{22}$（沿 $y$ 方向）**：

$$g_{22}(1,1,1) = -\left[\Phi(1,3,1) - 2\Phi(1,2,1) + \Phi(1,1,1)\right]$$

$$= -\left[-\frac{2}{5} - 2\left(-\frac{1}{2}\right) + \left(-\frac{2}{3}\right)\right]$$

$$= -\left[-\frac{2}{5} + 1 - \frac{2}{3}\right]$$

通分（公分母 15）：

$$= -\left[-\frac{6}{15} + \frac{15}{15} - \frac{10}{15}\right] = -\left[-\frac{1}{15}\right] = \frac{1}{15}$$

由 $y$-$z$ 对称性（两个源都在 $x$ 轴上，$y$ 和 $z$ 方向等价）：$g_{33} = g_{22} = \frac{1}{15}$。

注意 $g_{11} = \frac{11}{30} 
eq g_{22} = \frac{1}{15} = \frac{2}{30}$——**$x$ 方向（源连线方向）的度规分量与 $y,z$ 方向不同**，这正是两个源打破各向同性的体现。

**非对角分量 $g_{12}$**：

$$g_{12}(1,1,1) = -\frac{1}{2}\left[\Phi(2,2,1) - \Phi(2,1,1) - \Phi(1,2,1) + \Phi(1,1,1)\right]$$

$$= -\frac{1}{2}\left[-\frac{8}{15} - \left(-\frac{3}{4}\right) - \left(-\frac{1}{2}\right) + \left(-\frac{2}{3}\right)\right]$$

$$= -\frac{1}{2}\left[-\frac{8}{15} + \frac{3}{4} + \frac{1}{2} - \frac{2}{3}\right]$$

通分（公分母 60）：

$$= -\frac{1}{2}\left[-\frac{32}{60} + \frac{45}{60} + \frac{30}{60} - \frac{40}{60}\right] = -\frac{1}{2}\cdot\frac{3}{60} = -\frac{1}{40}$$

**非对角分量 $g_{13}$**（由 $y$-$z$ 对称性，$g_{13} = g_{12} = -\frac{1}{40}$）

**非对角分量 $g_{23}$**：

$$g_{23}(1,1,1) = -\frac{1}{2}\left[\Phi(1,2,2) - \Phi(1,2,1) - \Phi(1,1,2) + \Phi(1,1,1)\right]$$

$$= -\frac{1}{2}\left[-\frac{2}{5} - \left(-\frac{1}{2}\right) - \left(-\frac{1}{2}\right) + \left(-\frac{2}{3}\right)\right]$$

$$= -\frac{1}{2}\left[-\frac{2}{5} + 1 - \frac{2}{3}\right] = -\frac{1}{2}\cdot\left(-\frac{1}{15}\right) = \frac{1}{30}$$

**时间分量**：

$$g_{00}(1,1,1) = -\left(1 + 2\Phi\right) = -\left(1 - \frac{4}{3}\right) = \frac{1}{3}$$

等一下——$g_{00} = +1/3 > 0$，这里出了问题。原因是 $\Phi(1,1,1) = -2/3$，代入 $-(1+2\Phi) = -(1-4/3) = -(-1/3) = +1/3$。

**这是一个真实的物理效应，不是计算错误**：两个质量的叠加势场在中点处足够深（$\Phi = -2/3$，而 $2|\Phi| = 4/3 > 1$），超出了弱场近似的适用范围（弱场要求 $|\Phi|/c^2 \ll 1$，即 $Gm/r \ll c^2$）。在自然单位 $c=1$ 下，$|\Phi| = 2/3$ 已经不是弱场。

**修正**：恢复 $c$ 的量纲，取 $c=10$（模拟弱场），则：

$$g_{00}(1,1,1) = -\left(1 + \frac{2\Phi}{c^2}\right) = -\left(1 - \frac{4/3}{100}\right) = -\frac{296}{300} \approx -0.987 < 0 \quad \checkmark$$

或者等价地，取 $Gm = 0.1$（弱场参数），则 $\Phi(1,1,1) = -0.1\cdot(1/3+1/3) = -1/15$，

$$g_{00} = -\left(1 + 2\cdot(-1/15)\right) = -\frac{13}{15} < 0 \quad \checkmark$$

以下用 $Gm=0.1$ 继续。重新计算所有势场值乘以 $0.1$，度规的空间分量也相应缩小 10 倍，但**比值和结构不变**。

## 第三件事：连续极限的数值收敛分析

### **设定**

在 $N=6,12,24$ 三个格点尺度上，对单源配置重复度规计算，提取 $g_{11}$ 随 $N$ 的变化，观察收敛速率。

**关键参数**：格点间距 $\epsilon_N = L/\sqrt{N}$（$L$ 为系统尺度），$N$ 增大时 $\epsilon_N$ 减小。

**固定物理点**：保持宏观坐标 $u=(1,1,1)$（以物理单位计）不变，只改变格点精度。

---

### **$N=6$ 的结果（已完成）**

格点间距 $\epsilon_6 = L/\sqrt{6}$，在 $(1,1,1)$ 处：

$$g_{11}^{(6)} = \frac{1}{30}, \quad g_{12}^{(6)} = \frac{1}{60}$$

---

### **$N=12$ 的计算**

$N=12$，每维 4 比特，宏观坐标 $u^k \in \{0,1,2,3,4\}$，格点间距 $\epsilon_{12} = L/\sqrt{12} = \epsilon_6/\sqrt{2}$。

物理点 $(1,1,1)$ 对应格点坐标 $(2,2,2)$（每维翻倍）。

势场（$Gm=1$）：

$$\Phi(u) = -\frac{1}{u^1+u^2+u^3}$$

在 $(2,2,2)$ 处，步长为 2（对应物理单位 1）：

$$g_{11}^{(12)} = -\frac{1}{4}\left[\Phi(6,2,2) - 2\Phi(4,2,2) + \Phi(2,2,2)\right]$$

（系数 $1/4 = 1/(\Delta u)^2$，步长 $\Delta u = 2$）

$$= -\frac{1}{4}\left[-\frac{1}{10} - 2\left(-\frac{1}{8}\right) + \left(-\frac{1}{6}\right)\right]$$

$$= -\frac{1}{4}\left[-\frac{1}{10} + \frac{1}{4} - \frac{1}{6}\right]$$

通分（公分母 60）：

$$= -\frac{1}{4}\left[-\frac{6}{60} + \frac{15}{60} - \frac{10}{60}\right] = -\frac{1}{4}\cdot\frac{-1}{60} = \frac{1}{240}$$

等等——这与 $N=6$ 的 $g_{11}^{(6)} = 1/30$ 不同，差了 8 倍。原因是步长不同导致有限差分的物理单位不同，需要统一归一化。

**归一化**：有限差分的度规定义中，步长 $\Delta u$ 进入分母的平方。$N=6$ 时步长为 1，$N=12$ 时步长为 2，所以 $N=12$ 的结果需要乘以 $(\Delta u_{12})^2 / (\Delta u_6)^2 = 4$：

$$g_{11}^{(12),	ext{归一化}} = \frac{1}{240} \times 4 = \frac{1}{60}$$

还是不对，再检查一遍。$N=6$ 时步长 $\Delta u = 1$，$N=12$ 时物理单位不变但格点加密，步长 $\Delta u = 1$（物理单位），只是格点坐标变成了 2。

**重新设定**：统一用物理步长 $\Delta u = 1$ 计算，$N=12$ 时格点坐标 $(2,2,2)$，步长仍为 1 格点单位但物理长度为 $1/2$。

更清晰的方式是固定**物理步长**为 $\delta = 1$，改变**格点密度**：

$$g_{11}^{(N)}(u) = -\frac{1}{\delta^2}\left[\Phi\left(u+\frac{2\delta}{N/6}\hat{e}_1\right) - 2\Phi\left(u+\frac{\delta}{N/6}\hat{e}_1\right) + \Phi(u)\right]$$

这里 $N/6$ 是相对于基准 $N=6$ 的格点密度比。

---

### **统一框架下的三尺度计算**

为避免步长混淆，改用以下统一定义：

在物理坐标 $r$ 处，势场 $\Phi(r) = -Gm/r$（连续形式），离散版本在格点间距 $\epsilon$ 下为：

$$g_{11}^\epsilon(r) = -\frac{\Phi(r+2\epsilon) - 2\Phi(r+\epsilon) + \Phi(r)}{\epsilon^2}$$

取 $r=3$（对应格点 $(1,1,1)$ 的径向距离 $r = 1+1+1 = 3$），$Gm=1$：

**$\epsilon = 1$（对应 $N=6$）**：

$$g_{11}^{\epsilon=1} = -\frac{(-1/5) - 2(-1/4) + (-1/3)}{1} = -\left[-\frac{1}{5}+\frac{1}{2}-\frac{1}{3}\right] = \frac{1}{30}$$

**$\epsilon = 1/2$（对应 $N=12$）**：

$$g_{11}^{\epsilon=1/2} = -\frac{(-1/3.5) - 2(-1/3) + (-1/2.5) \cdot... }{}$$

等等，$r=3$，步长 $\epsilon=1/2$：

$$\Phi(3) = -\frac{1}{3}, \quad \Phi(3.5) = -\frac{1}{3.5} = -\frac{2}{7}, \quad \Phi(4) = -\frac{1}{4}$$

$$g_{11}^{\epsilon=1/2} = -\frac{\Phi(4) - 2\Phi(3.5) + \Phi(3)}{(1/2)^2} = -\frac{-\frac{1}{4} + \frac{4}{7} - \frac{1}{3}}{1/4}$$

通分（公分母 84）：

$$= -4\left[-\frac{21}{84} + \frac{48}{84} - \frac{28}{84}\right] = -4\cdot\left(-\frac{1}{84}\right) = \frac{4}{84} = \frac{1}{21}$$

**$\epsilon = 1/4$（对应 $N=24$）**：

$$\Phi(3) = -\frac{1}{3},\quad \Phi(3.25) = -\frac{4}{13},\quad \Phi(3.5) = -\frac{2}{7}$$

$$g_{11}^{\epsilon=1/4} = -\frac{\Phi(3.5) - 2\Phi(3.25) + \Phi(3)}{(1/4)^2}$$

$$= -16\left[-\frac{2}{7} + \frac{8}{13} - \frac{1}{3}\right]$$

通分（公分母 273）：

$$-\frac{2}{7} = -\frac{78}{273},\quad \frac{8}{13} = \frac{168}{273},\quad -\frac{1}{3} = -\frac{91}{273}$$

$$= -16\cdot\frac{-78+168-91}{273} = -16\cdot\frac{-1}{273} = \frac{16}{273}$$

### **连续极限的理论值**

对 $\Phi(r) = -Gm/r$，连续二阶导数：

$$\partial_r^2\Phi = -\frac{d^2}{dr^2}\left(\frac{1}{r}\right) = -\frac{2}{r^3}$$

在 $r=3$，$Gm=1$：

$$g_{11}^\infty = -\partial_r^2\Phi\big|_{r=3} = \frac{2}{27}$$

### **收敛分析**

| $N$ | $\epsilon$ | $g_{11}^\epsilon$ | 误差 $|g_{11}^\epsilon - g_{11}^\infty|$ |
|---|---|---|---|
| $6$ | $1$ | $1/30 \approx 0.03333$ | $|0.03333 - 0.07407| = 0.04074$ |
| $12$ | $1/2$ | $1/21 \approx 0.04762$ | $|0.04762 - 0.07407| = 0.02645$ |
| $24$ | $1/4$ | $16/273 \approx 0.05861$ | $|0.05861 - 0.07407| = 0.01546$ |
| $\infty$ | $0$ | $2/27 \approx 0.07407$ | $0$ |

误差比值：

$$\frac{e_6}{e_{12}} = \frac{0.04074}{0.02645} \approx 1.54, \quad \frac{e_{12}}{e_{24}} = \frac{0.02645}{0.01546} \approx 1.71$$

理论上，二阶有限差分的误差应为 $O(\epsilon^2)$，即步长减半误差缩小为 $1/4$。实际观测到的比值约为 $1.5\sim1.7$，不是 4。

**这个偏差的来源**：$\Phi(r) = -1/r$ 的三阶导数在 $r=3$ 处不小（$\Phi''' = -6/r^4 = -6/81$），导致 $O(\epsilon^2)$ 修正项的系数较大，在 $\epsilon=1$ 时高阶项已经不可忽略。随着 $\epsilon$ 减小，收敛速率逐渐接近理论值 $O(\epsilon^2)$。

更精确地，有限差分误差的展开为：

$$g_{11}^\epsilon - g_{11}^\infty = \frac{\epsilon^2}{12}\Phi^{(4)}(r) + O(\epsilon^4)$$

其中 $\Phi^{(4)}(r) = 24/r^5$，在 $r=3$ 处 $\Phi^{(4)} = 24/243 \approx 0.099$。

预测误差：$\frac{\epsilon^2}{12}\cdot 0.099$

- $\epsilon=1$：$0.099/12 \approx 0.0082$（实际 $0.041$，高阶项贡献大）
- $\epsilon=1/2$：$0.099/48 \approx 0.0021$（实际 $0.026$）
- $\epsilon=1/4$：$0.099/192 \approx 0.0005$（实际 $0.015$）

高阶项在 $r=3$ 处仍然显著，说明 $r=3$ 还不够"远场"。

**在 $r=10$ 处重复**（更远场，高阶项更小）：

$$g_{11}^\infty\big|_{r=10} = \frac{2}{1000} = 0.002$$

$$g_{11}^{\epsilon=1}\big|_{r=10} = -\left[-\frac{1}{12}+\frac{2}{11}-\frac{1}{10}\right] = -\left[\frac{-110+240-132}{1320}\right] = \frac{2}{1320} \approx 0.001515$$

$$g_{11}^{\epsilon=1/2}\big|_{r=10} = -4\left[-\frac{1}{11}+\frac{2}{10.5}-\frac{1}{10}\right] = -4\left[\frac{-210+400-231}{2310}\right] \approx -4\cdot\frac{-41}{2310}$$

等等，重新算：

$$-\frac{1}{11}+\frac{2}{10.5}-\frac{1}{10} = -0.09091+0.19048-0.10000 = -0.00043$$

$$g_{11}^{\epsilon=1/2}\big|_{r=10} = -4\cdot(-0.00043) \approx 0.001730$$

误差比：$|0.001515-0.002|/|0.001730-0.002| = 0.000485/0.000270 \approx 1.80$

在 $r=10$ 处误差比接近 2，理论值为 4（$O(\epsilon^2)$）——仍然有偏差，但趋势是正确的，随 $\epsilon$ 减小收敛速率在提高。
