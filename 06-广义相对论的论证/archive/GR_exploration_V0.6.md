## 二、$g_{00}$ 的 $\Phi$ 修正：从 A9 到 $-(1+2\Phi/c^2)$ 的严格推导

### 2.1 A9 的精确含义

A9（最小完备）的核心陈述：系统不引入多余自由度——每一个可观测量都由比特配置 $\{x_i\}$ 唯一确定，没有任何外部参数。

对时间步长 $\alpha$ 的约束：$\alpha$ 不是独立的外部参数，而是由比特配置的内在能量尺度确定的。

### 2.2 内在能量的定义

在比特配置 $\{x_i\}$ 中，占据位集合 $S$ 的内在能量由势场 $\Phi$ 决定。单个占据位在势场中的能量为：

$$E(s) = m_0 c^2 + m_0 \Phi(s)$$

其中 $m_0$ 是基本质量单位（由 A9 的最小完备性确定），$m_0 c^2$ 是静止能量，$m_0 \Phi(s)$ 是势能。

**公理来源**：A5（守恒律）给出能量守恒的结构，$d_H$ 定义的势场 $\Phi$ 给出势能的具体形式。

### 2.3 时间步长由内在能量唯一确定

A9 要求时间步长 $\alpha$ 由内在能量尺度唯一确定。在 A1'（相位结构）给出的框架中，比特配置的相位 $e^{i\theta}$ 的时间变化率是角频率 $\omega = d\theta/dt$。A4（最小变易）约束相位变化的最小单位，A9（最小完备）要求时间步长由相位变化率唯一确定：

$$\alpha = \frac{2\pi}{\omega}$$

结合 $E = \hbar\omega$（来自 A1' 的相位结构 + A4 的最小变易），得到：

$$\alpha = \frac{\hbar}{E(s)} = \frac{\hbar}{m_0 c^2 + m_0 \Phi(s)} = \frac{\hbar}{m_0 c^2} \cdot \frac{1}{1 + \Phi(s)/c^2}$$

**公理来源**：A1'（相位）+ A4（最小变易）+ A9（最小完备）联合。

### 2.4 时间步长红移公式

在弱场极限 $|\Phi| \ll c^2$ 下，展开到一阶：

$$\alpha_\Phi = \frac{\hbar}{m_0 c^2} \cdot \frac{1}{1 + \Phi/c^2} \approx \frac{\hbar}{m_0 c^2}\left(1 - \frac{\Phi}{c^2}\right) = \alpha_0\left(1 - \frac{\Phi}{c^2}\right)$$

其中 $\alpha_0 = \hbar/(m_0 c^2)$ 是零势场下的基准时间步长。

**物理含义**：$\Phi < 0$（引力势为负），所以 $\alpha_\Phi > \alpha_0$——在引力势中，时间步长变长（时钟变慢）。这正是引力时间膨胀的离散来源。

### 2.5 从时间步长红移到 $g_{00}$

时间度规分量 $g_{00}$ 的物理含义：固有时间 $d\tau$ 与坐标时间 $dt$ 的比值的平方，

$$g_{00} = -\left(\frac{d\tau}{dt}\right)^2$$

负号来自 A6（不可逆性）。

在离散框架中，固有时间是演化步数 $n$ 乘以局部时间步长 $\alpha_\Phi$，坐标时间是演化步数 $n$ 乘以基准时间步长 $\alpha_0$：

$$d\tau = \alpha_\Phi \cdot dn, \qquad dt = \alpha_0 \cdot dn$$

因此：

$$\frac{d\tau}{dt} = \frac{\alpha_\Phi}{\alpha_0} = 1 - \frac{\Phi}{c^2} + O\left(\frac{\Phi^2}{c^4}\right)$$

代入 $g_{00}$ 的定义：

$$g_{00} = -\left(\frac{d\tau}{dt}\right)^2 = -\left(1 - \frac{\Phi}{c^2}\right)^2 = -\left(1 - \frac{2\Phi}{c^2} + \frac{\Phi^2}{c^4}\right)$$

在弱场极限 $|\Phi| \ll c^2$ 下，忽略 $O(\Phi^2/c^4)$ 项：

$$\boxed{g_{00} = -\left(1 + \frac{2\Phi}{c^2}\right)}$$

注意 $\Phi < 0$，所以 $-2\Phi/c^2 > 0$，$|g_{00}| < 1$，符合引力势中时钟变慢的物理图像。

### 2.6 公理来源的完整追踪

| 步骤 | 内容 | 公理来源 |
|------|------|---------|
| 内在能量 $E(s) = m_0(c^2 + \Phi)$ | 占据位在势场中的总能量 | A5（守恒）+ $d_H$（势场） |
| 相位结构 $e^{i\theta}$ | 比特配置的相位自由度 | A1'（相位结构） |

## 三、内部一致性验证

### 3.1 与 $g_{00}$ 背景值的一致性

当 $\Phi = 0$（无引力场）时：

$$g_{00} = -(1 + 0) = -1$$

与 V0.5 中 A4 + A6 推导的背景值 $g_{00}^{(0)} = -1$ 完全一致。

### 3.2 与 Schwarzschild 度规的一致性

Schwarzschild 外部解在弱场极限下：

$$g_{00}^{\text{Sch}} = -\left(1 - \frac{2GM}{c^2 r}\right)$$

WorldBase 的势场 $\Phi = -GM/r$，代入得：

$$g_{00} = -\left(1 + \frac{2(-GM/r)}{c^2}\right) = -\left(1 - \frac{2GM}{c^2 r}\right) = g_{00}^{\text{Sch}}$$

与 Schwarzschild 度规精确一致（弱场极限）。

### 3.3 与 $g_{kk}$ 的关系

在弱场参数 $Gm = 0.1$ 下，格点 $(1,1,1)$ 处：

$$g_{00} = -\left(1 + 2 \cdot \left(-\frac{1}{30}\right)\right) = -\frac{14}{15}$$

$$g_{11} = \frac{1}{300}$$

$$g_{00} - g_{11} = -\frac{14}{15} - \frac{1}{300} = -\frac{281}{300} \approx -2 + O(\Phi)$$

在弱场极限下 $g_{00} - g_{kk} \to -2$，与 V0.5 的框架内生关系一致。

## 十四、核心洞察

1. **度规全部 10 个分量从公理推导**。不需要等效原理，不需要假设度规形式，不需要引入外部参数。

2. **等效原理成为推论**。引力场中时钟变慢是 A1' + A4 + A5 + A6 + A9 联合推导的必然结果，不是独立假设。

4. **离散度规单调收敛到连续度规**。远场收敛速率趋向 $O(\epsilon^2)$。

5. **$g_{0i} = 0$ 从 A5 + A8 推导**。静态条件不是假设，而是守恒律 + 对称偏好的联合推论。

6. **$g_{00}$ 的完整形式从 A1' + A4 + A5 + A6 + A9 推导**。符号来自 A6，背景值来自 A4 + A6，$\Phi$ 修正来自 A1' + A4 + A9（时间步长红移）。

7. **爱因斯坦方程是自洽性条件，不是动力学方程**。物质和几何来自同一比特配置，方程是两者一致性的表达。

8. **爱因斯坦的 7 个假设全部变成推论或被消解**。WorldBase 用 10 条更基本的公理给出了同一个物理，但不需要任何外部假设。
