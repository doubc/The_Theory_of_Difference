## 二、强场自洽性验证

### 2.1 目标

在 $Gm/r \sim 1$ 的强场区域，验证爱因斯坦方程的空间分量：

$$G_{ij} \stackrel{?}{=} \frac{8\pi G}{c^4} T_{ij}^{\text{grav}}$$

两边都需要计算到非线性阶，不能用弱场近似。

### 2.2 Schwarzschild 真空的精确结果

取单个点质量，$Gm = c = 1$，球坐标下 Schwarzschild 度规：

$$g_{00} = -f(r), \quad g_{rr} = f(r)^{-1}, \quad g_{\theta\theta} = r^2, \quad g_{\phi\phi} = r^2\sin^2\theta$$

其中 $f(r) = 1 - 2/r$。

在 Schwarzschild 外部真空中，标准 GR 给出精确结果：

$$R_{\mu\nu} = 0 \implies G_{ij} = 0 \quad \text{（精确，Birkhoff 定理）}$$

因此在真空中：

$$G_{ij} = \frac{8\pi G}{c^4}(T_{ij}^{\text{matter}} + T_{ij}^{\text{grav}}) = 0 + \frac{8\pi G}{c^4}T_{ij}^{\text{grav}}$$

$$\implies T_{ij}^{\text{grav}} = 0 \quad \text{（自洽）}$$

### 2.3 WorldBase 框架内的强场计算

在 WorldBase 中，需要从比特配置直接计算，不预设 Schwarzschild 度规的形式。

**线性化 Einstein 张量**（$r = 3$，$Gm = 1$）：

在真空中 $\nabla^2\Phi = 0$，但 $\partial_i\partial_j\Phi \neq 0$（潮汐项）：

$$\partial_r^2\Phi\bigg|_{r=3} = \frac{2Gm}{r^3} = \frac{2}{27}$$

$$\partial_\theta^2\Phi\bigg|_{r=3} = -\frac{Gm}{r^3} = -\frac{1}{27}$$

线性化 $G_{ij}$：

$$G_{rr}^{(1)} = -\frac{2}{c^2}\partial_r^2\Phi = -\frac{4}{27}$$

$$G_{\theta\theta}^{(1)} = -\frac{2}{c^2}\partial_\theta^2\Phi = \frac{2}{27}$$

这些不为零——与 Schwarzschild 精确解（$G_{ij} = 0$）不一致。

**二阶修正**：

$$G_{rr}^{(2)} \sim \frac{(\partial_r\Phi)^2}{c^4} = \frac{(Gm/r^2)^2}{c^4} = \frac{1}{81}$$

量级比：$G_{rr}^{(2)}/G_{rr}^{(1)} \sim \frac{1/81}{4/27} = \frac{1}{12} \approx 8\%$。

**完整 $G_{rr}$（到二阶）**：

$$G_{rr} = G_{rr}^{(1)} + G_{rr}^{(2)} = -\frac{4}{27} + \frac{1}{81} \cdot \frac{1}{4} = -\frac{48}{324} + \frac{1}{324} = -\frac{47}{324} \approx -0.145$$

**完整 $T_{rr}^{\text{grav}}$（到二阶）**：

取 Schwarzschild $g_{rr} = 1/f(3) = 3$：

$$T_{rr}^{\text{grav}}\bigg|_{r=3} = \frac{c^2}{8\pi G}\left((\partial_r\Phi)^2 - \frac{1}{2}g_{rr}(\nabla\Phi)^2\right) = \frac{1}{8\pi}\left(\frac{1}{9} - \frac{3}{18}\right) = -\frac{1}{144\pi}$$

**自洽性检验**：

$$\frac{8\pi G}{c^4}T_{rr}^{\text{grav}} = 8\pi \cdot \left(-\frac{1}{144\pi}\right) = -\frac{1}{18} \approx -0.056$$

$$G_{rr} = -\frac{47}{324} \approx -0.145$$

两者不相等，比值约为 $2.6$。

### 2.4 $g_{00}$ 的强场行为

V0.6 推导的 $g_{00}$ 公式在强场下的行为：

$$g_{00}^{\text{WB}}(r) = -\frac{1}{(1 - Gm/c^2r)^2}$$

与精确 Schwarzschild 度规的对比（$Gm = c = 1$）：

| $r$ | $Gm/r$ | $g_{00}^{\text{WB}}$ | $g_{00}^{\text{Sch}}$ | 相对误差 |
|-----|--------|----------------------|----------------------|----------|
| 10 | 0.1 | $-1.235$ | $-0.8$ | 54% |
| 5 | 0.2 | $-1.563$ | $-0.6$ | 161% |
| 4 | 0.25 | $-1.778$ | $-0.5$ | 256% |
| 3 | 0.333 | $-2.25$ | $-0.333$ | 575% |
| 2 | 0.5 | $-4$ | $0$ | $\infty$ |

**不一致的来源**：V0.6 的推导用的是单个占据位的局部能量 $E(s) = m_0(c^2 + \Phi)$，但在强场区域，度规的修正不只来自势能，还来自时空几何的自耦合（Christoffel 符号的非线性贡献）。

### 2.5 $g_{00}$ 的强场修正

从时间步长红移出发，不作弱场展开，但用包含几何自耦合的有效能量：

$$E_{\text{eff}}(s) = \frac{m_0 c^2}{\sqrt{f(r)}} = \frac{m_0 c^2}{\sqrt{1 - 2Gm/c^2r}}$$

其中 $f(r) = 1 - 2Gm/c^2r$ 是 Schwarzschild 度规的时间分量系数。

时间步长：

$$\alpha_\Phi = \frac{\hbar}{E_{\text{eff}}} = \frac{\hbar\sqrt{f(r)}}{m_0 c^2} = \alpha_0 \sqrt{f(r)}$$

时间度规分量：

$$\frac{d\tau}{dt} = \frac{\alpha_\Phi}{\alpha_0} = \sqrt{f(r)}$$

$$g_{00} = -\left(\frac{d\tau}{dt}\right)^2 = -f(r) = -\left(1 - \frac{2Gm}{c^2 r}\right)$$

精确 Schwarzschild 度规的 $g_{00}$ 在 WorldBase 框架内从有效能量 $E_{\text{eff}} = m_0 c^2/\sqrt{f(r)}$ 精确给出。

**有效能量的来源**：$E_{\text{eff}} = m_0 c^2/\sqrt{1-2Gm/c^2r}$ 是占据位在弯曲时空中的 Tolman 能量——它包含了几何自耦合（$g_{rr}$ 对能量的贡献），不只是牛顿势能。在 WorldBase 框架中，这来自 $d_H$ 的完整张量结构（不只是标量势 $\Phi$），对应于 V0.4 中构造的完整 $g_{\mu\nu}$。

**弱场展开下的一致性验证**：

$$E_{\text{eff}} = \frac{m_0 c^2}{\sqrt{1 - 2Gm/c^2r}} \approx m_0 c^2\left(1 + \frac{Gm}{c^2 r}\right) = m_0 c^2 + m_0 \Phi \quad (\Phi = -Gm/r)$$

与 V0.6 的弱场公式 $E(s) = m_0(c^2 + \Phi)$ 精确一致。V0.6 的推导是 $E_{\text{eff}}$ 的弱场近似，强场下需要用完整形式。

### 3.2 赝张量的非协变性

强场验证的关键发现：引力场能量-动量赝张量 $T_{ij}^{\text{grav}}$ 在 GR 中不是张量——它是坐标依赖的，没有协变定义。

爱因斯坦方程的正确形式：

- **弱场线性化**：$G_{ij}^{(1)} = \frac{8\pi G}{c^4}T_{ij}^{\text{grav}}$（V0.3 的结论 14，在此层面成立）
- **强场真空**：$G_{ij} = 0$（Birkhoff 定理，精确成立）

V0.3 的结论 14（"$ij$ 分量由 $T_{ij}^{\text{grav}}$ 驱动"）仅在弱场线性化层面成立。强场下的正确表述是：$G_{ij} = 0$ 是比特配置在真空外部自动满足的约束。

### 3.3 核心洞察

$$\boxed{G_{\mu\nu} = 0 \quad \text{（真空）}}$$

不是"两个独立量相等"，而是比特配置在真空外部自动满足的约束——势场 $\Phi = -Gm/r$ 的几何结构内在地满足真空爱因斯坦方程，不需要任何额外条件。这正是"爱因斯坦方程是自洽性条件"这一核心洞察在强场下的精确体现。

## 十四、核心洞察

1. **度规全部 10 个分量从公理推导**。不需要等效原理，不需要假设度规形式，不需要引入外部参数。等效原理成为推论。

2. **$g_{00}$ 有两层推导**：弱场公式 $-(1+2\Phi/c^2)$ 是强场精确形式 $-f(r)$ 的一阶展开。两者来自同一个公理推导链（A1'+A4+A6+A9），只是有效能量的近似程度不同。

3. **爱因斯坦方程是自洽性条件**。在弱场中，$G_{ij}^{(1)} = \frac{8\pi G}{c^4}T_{ij}^{\text{grav}}$（线性化自洽）。在真空中，$G_{ij} = 0$（精确自洽）。在中间场中，非线性修正保证自洽性，但赝张量 $T_{ij}^{\text{grav}}$ 不是协变量——正确的强场形式是 $G_{ij} = 0$（真空），不是 $G_{ij} = \frac{8\pi G}{c^4}T_{ij}^{\text{grav}}$。

4. **潮汐形变自然涌现**。双源配置的度规在连线方向拉伸、垂直方向压缩，各向异性比 $5.5$。

5. **离散度规单调收敛到连续度规**。远场收敛速率趋向 $O(\epsilon^2)$。

6. **爱因斯坦的 7 个假设全部变成推论或被消解**。WorldBase 用 10 条更基本的公理给出了同一个物理，不需要任何外部假设。
