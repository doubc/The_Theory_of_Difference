## 一、$g_{0i} = 0$ 的公理推导

### 1.1 离散有限差分定义

$g_{0k}$ 的离散有限差分定义：

$$g_{0k}(u) = -\frac{1}{2}\left[\Phi(u+\hat{e}_0+\hat{e}_k) - \Phi(u+\hat{e}_0) - \Phi(u+\hat{e}_k) + \Phi(u)\right]$$

### 1.2 静态条件的公理来源

**A5（守恒律）**约束：占据位总数守恒，$S$ 中的元素不消失。每步演化只翻转 $S$ 之外的比特（"真空涨落"），$S$ 本身保持不变。

**A8（对称偏好）**约束：占据位不沿任何空间方向净移动。净移动破坏势场的空间对称性。

**联合推论**：A5 + A8 联合约束下，$S$ 不随时间变化，占据位不沿空间方向净流动。因此势场不显含时间：

$$\Phi(u + \hat{e}_0) = \Phi(u)$$

### 1.3 代数推导

代入 $g_{0k}$ 的定义：

$$g_{0k}(u) = -\frac{1}{2}\left[\Phi(u+\hat{e}_k) - \Phi(u) - \Phi(u+\hat{e}_k) + \Phi(u)\right] = 0 \quad \square$$

**依赖关系**：

| 步骤 | 公理来源 |
|------|---------|
| 占据位总数守恒，$S$ 不消失 | A5 |
| 占据位不沿空间方向净移动 | A8 |
| $\Phi$ 不显含时间 | A5 + A8 联合 |
| $g_{0k} = 0$ | 代数恒等式 |

**状态**：从 A5 + A8 推导，不依赖等效原理。

## 二、$g_{00}$ 来源的精确化

$g_{00} = -(1+2\Phi/c^2)$ 拆分为三个独立层次。

### 2.1 符号：$g_{00} < 0$

来源：A6（DAG 不可逆性）。时间方向不可逆，空间方向可逆。度规中不可逆方向对应负号，可逆方向对应正号。

$$g_{00} < 0 \quad \text{（A6，已推导）}$$

### 2.2 背景值：$g_{00}^{(0)} = -1$

来源：A4 + A6 联合。A4 给出演化的离散步长 $\alpha$（每步 $\Delta w = 1$），A6 给出演化的方向性（不可逆）。时间维度的基准值由演化序列的结构确定：

- 空间基准值：$+1$（来自配置截面的可逆性，A1/A1'）
- 时间基准值：$-1$（来自演化序列的不可逆性，A6）

$$g_{00}^{(0)} = -1 \quad \text{（A4 + A6，已推导）}$$

### 2.3 强场关系：$g_{00} - g_{kk} = -2$

来源：时间维度和空间维度的不同涌现机制。时间来自演化序列（基准值 $-1$），空间来自配置截面（基准值 $+1$），两者的差 $-1 - 1 = -2$ 是框架内生的。

$$g_{00} - g_{kk} = -2 \quad \text{（框架内生，弱场下成立）}$$

### 2.4 $\Phi$ 修正项：$-2\Phi/c^2$（A9 候选论证）

这是度规构造中最后一个依赖外部假设的部分。在 WorldBase 框架内有一个候选来源。

**候选论证**：时间步长 $\alpha$（基本时间量子）在引力势 $\Phi$ 中会发生红移。具体地，在势场 $\Phi$ 中，演化的有效速率变为：

$$\alpha_\Phi = \alpha\left(1 + \frac{\Phi}{c^2}\right)$$

这来自 A9（最小完备）：系统的时间步长由其内在能量尺度确定，而在引力势中，内在能量尺度被 $\Phi$ 修正。

时间度规分量正比于时间步长的平方：

$$g_{00} \propto -\alpha_\Phi^2 = -\alpha^2\left(1 + \frac{\Phi}{c^2}\right)^2 \approx -\left(1 + \frac{2\Phi}{c^2}\right) \quad (|\Phi| \ll c^2)$$

这正好给出弱场形式 $-(1+2\Phi/c^2)$，来源是 A9（最小完备）对时间步长的约束，不依赖等效原理。

$$g_{00} = -\left(1 + \frac{2\Phi}{c^2}\right) \quad \text{（A9 候选论证，待严格化）}$$

## 三、多源配置的度规构造

### 3.1 设定

两个占据位：$S = \{u_1, u_2\}$，$u_1 = (0,0,0)$，$u_2 = (2,0,0)$（沿 $x$ 轴相距 2）。

多源势场：

### 3.3 度规分量计算

**对角分量 $g_{11}$（沿 $x$ 方向，源连线方向）**：

$$g_{11} = -\left[\Phi(3,1,1) - 2\Phi(2,1,1) + \Phi(1,1,1)\right]$$

$$= -\left[-\frac{3}{25} + \frac{3}{20} - \frac{1}{15}\right]$$

通分（公分母 300）：

$$= -\left[-\frac{36}{300} + \frac{45}{300} - \frac{20}{300}\right] = \frac{11}{300} \approx 0.0367$$

**对角分量 $g_{22}$（沿 $y$ 方向，横向）**：

$$g_{22} = -\left[\Phi(1,3,1) - 2\Phi(1,2,1) + \Phi(1,1,1)\right]$$

$$= -\left[-\frac{1}{25} + \frac{1}{10} - \frac{1}{15}\right]$$

通分（公分母 150）：

$$= -\left[-\frac{6}{150} + \frac{15}{150} - \frac{10}{150}\right] = \frac{1}{150} \approx 0.0067$$

由 $y$-$z$ 对称性：$g_{33} = g_{22} = 1/150$。

**非对角分量 $g_{12}$**：

$$g_{12} = -\frac{1}{2}\left[\Phi(2,2,1) - \Phi(2,1,1) - \Phi(1,2,1) + \Phi(1,1,1)\right]$$

$$= -\frac{1}{2}\left[-\frac{4}{75} + \frac{3}{40} + \frac{1}{20} - \frac{1}{15}\right]$$

通分（公分母 600）：

$$= -\frac{1}{2}\left[-\frac{32}{600} + \frac{45}{600} + \frac{30}{600} - \frac{40}{600}\right] = -\frac{1}{2} \cdot \frac{3}{600} = -\frac{1}{400}$$

由 $y$-$z$ 对称性：$g_{13} = g_{12} = -1/400$。

**非对角分量 $g_{23}$**：

$$g_{23} = -\frac{1}{2}\left[\Phi(1,2,2) - \Phi(1,2,1) - \Phi(1,1,2) + \Phi(1,1,1)\right]$$

$$= -\frac{1}{2}\left[-\frac{1}{25} + \frac{1}{20} + \frac{1}{20} - \frac{1}{15}\right]$$

通分（公分母 300）：

$$= -\frac{1}{2}\left[-\frac{12}{300} + \frac{15}{300} + \frac{15}{300} - \frac{20}{300}\right] = -\frac{1}{2} \cdot \left(-\frac{2}{300}\right) = \frac{1}{300}$$

**时间分量**（$Gm = 0.1$，弱场）：

$$g_{00} = -\left(1 + 2\Phi\right) = -\left(1 - \frac{2}{15}\right) = -\frac{13}{15} \approx -0.867 < 0$$

### 3.6 物理解读

**潮汐形变**：两个质量沿 $x$ 轴排列，引力场在连线方向拉伸空间（$g_{11}$ 增大 $10\%$），在垂直方向压缩空间（$g_{22}$、$g_{33}$ 减小 $80\%$）。这是潮汐力的几何体现——引力波探测器（LIGO）工作原理的弱场版本。

**非对角分量符号翻转**：单源时 $g_{12} = +1/60 > 0$，双源时 $g_{12} = -1/400 < 0$。两个源的叠加改变了空间的剪切结构。

**各向异性**：$g_{11}/g_{22} = (11/300)/(1/150) = 11/2 = 5.5$。源连线方向的度规分量约为横向的 $5.5$ 倍。共形平坦要求 $g_{11} = g_{22}$，而 $5.5 \neq 1$。

**全部自然涌现，无需额外假设。**

## 四、连续极限的数值收敛分析

### 4.1 统一框架

在物理坐标 $r$ 处，势场 $\Phi(r) = -Gm/r$（连续形式）。离散度规在格点间距 $\epsilon$ 下定义为：

$$g_{11}^{\epsilon}(r) = -\frac{\Phi(r+2\epsilon) - 2\Phi(r+\epsilon) + \Phi(r)}{\epsilon^2}$$

连续极限值：

$$g_{11}^{\infty}(r) = -\partial_r^2 \Phi \Big|_r = \frac{2Gm}{r^3}$$

### 4.2 三尺度计算（$r=3$，$Gm=1$）

连续极限值：

$$g_{11}^{\infty}\Big|_{r=3} = \frac{2}{27} \approx 0.07407$$

**$\epsilon = 1$（对应 $N=6$）**：

$$g_{11}^{\epsilon=1} = -\left[-\frac{1}{5} + \frac{1}{2} - \frac{1}{3}\right] = \frac{1}{30} \approx 0.03333$$

**$\epsilon = 1/2$（对应 $N=12$）**：

$$g_{11}^{\epsilon=1/2} = -4\left[-\frac{1}{4} + \frac{4}{7} - \frac{1}{3}\right]$$

通分（公分母 84）：

$$= -4\left[-\frac{21}{84} + \frac{48}{84} - \frac{28}{84}\right] = -4 \cdot \left(-\frac{1}{84}\right) = \frac{1}{21} \approx 0.04762$$

**$\epsilon = 1/4$（对应 $N=24$）**：

$$g_{11}^{\epsilon=1/4} = -16\left[-\frac{2}{7} + \frac{8}{13} - \frac{1}{3}\right]$$

通分（公分母 273）：

$$= -16\left[-\frac{78}{273} + \frac{168}{273} - \frac{91}{273}\right] = -16 \cdot \left(-\frac{1}{273}\right) = \frac{16}{273} \approx 0.05861$$

### 4.3 收敛数据（$r=3$）

| $N$ | $\epsilon$ | $g_{11}^{\epsilon}$ | 误差 | 收敛率 |
|-----|-----------|---------------------|------|--------|
| 6 | 1 | $1/30 \approx 0.0333$ | $0.0408$ | — |
| 12 | $1/2$ | $1/21 \approx 0.0476$ | $0.0265$ | $\sim\epsilon^{0.62}$ |
| 24 | $1/4$ | $16/273 \approx 0.0586$ | $0.0155$ | $\sim\epsilon^{0.77}$ |
| $\infty$ | 0 | $2/27 \approx 0.0741$ | 0 | — |

### 4.4 远场验证（$r=10$）

$$g_{11}^{\infty}\Big|_{r=10} = \frac{2}{1000} = 0.002$$

**$\epsilon = 1$**：

$$g_{11}^{\epsilon=1}\Big|_{r=10} = -\left[-\frac{1}{12} + \frac{2}{11} - \frac{1}{10}\right] = \frac{1}{660} \approx 0.001515$$

误差：$0.000485$。

**$\epsilon = 1/2$**：

$$g_{11}^{\epsilon=1/2}\Big|_{r=10} = -4\left[-\frac{2}{21} + 0.2 - \frac{2}{19}\right] \approx 0.002004$$

误差：$0.000004$。

远场误差比 $\approx 121$，远超 $O(\epsilon^2)$ 预期。原因是 $r=10$ 处 $\Phi^{(4)} = 0.00024$ 极小，$\epsilon=1/2$ 时误差已接近计算精度下限。

### 4.5 收敛分析

**偏差来源**：$\Phi(r) = -1/r$ 的高阶导数在近场较大：

$$\Phi^{(4)}(r) = \frac{24}{r^5}\bigg|_{r=3} = \frac{24}{243} \approx 0.099$$

有限差分误差展开：

$$g_{11}^{\epsilon} - g_{11}^{\infty} = \frac{\epsilon^2}{12}\Phi^{(4)}(r) + O(\epsilon^4)$$

高阶项在 $r=3$ 处显著，近场收敛慢于远场。

### 4.6 收敛结论

| 结论 | 内容 | 状态 |
|------|------|------|
| 单调收敛 | $g_{11}^{\epsilon}$ 随 $\epsilon \to 0$ 单调趋近 $g_{11}^{\infty}$ | 已验证 |
| 收敛方向 | 离散值低于连续值，从下方趋近 | 已验证 |
| 近场收敛速率 | $\alpha \in [0.6, 0.8]$，趋向 $O(\epsilon^2)$ | 待严格化 |
| 远场收敛速率 | 趋向 $O(\epsilon^2)$，高阶项可忽略 | 已验证 |
| 与定理 CL 的一致性 | $O(\epsilon_N^{\alpha})$，$\alpha \approx 1$ 近场有效值 | 待严格化 |
