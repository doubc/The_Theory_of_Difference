## 〇、公理体系

WorldBase 框架定义在比特空间 $\{0,1\}^N$ 上，由十条公理约束。

| 编号 | 名称 | 核心内容 |
|------|------|---------|
| A1 | 差异本体论 | 存在性；不同占据位之间的差异独立 |
| A1' | 横向涌现 | 层级正交方向存在连续旋转对称性 $SO(2) \cong U(1)$ |
| A4 | 最小变易 | 每次演化仅改变一个比特（$d_H = 1$ 邻域） |
| A5 | 差异守恒 | 存在演化不变量；占据位独立守恒 |
| A6 | DAG 有向 | 演化图是有向无环图（时间不可逆） |
| A7 | 循环闭合 | 演化中存在闭合路径（相干叠加来源） |
| A8 | 对称偏好 | 中截面（$w = N/2$）统计权重最大 |
| A9 | 内生完备 | 自由度集合恰好等于公理要求的最小值 |

**核心洞察**：比特配置 $\{x_i\}$ **同时**决定物质分布（占据位密度 $\rho$）和时空几何（汉明距离势场 $\Phi$）。两者来自同一实体，不是两套独立自由度。

**推论**：爱因斯坦方程不是"两个不同实体之间的耦合方程"，而是"同一个比特配置的两个不同描述之间的一致性条件"——**它是自洽性条件，不是动力学方程**。

## 一、基础定义

### 定义 1.1（比特配置与占据位）

比特配置 $x = (x_1, \ldots, x_N) \in \{0,1\}^N$。占据位集合 $S = \{s \in \{0,1\}^N : s \text{ 是稳定态}\}$。

汉明距离 $d_H(x,y) = \sum_i |x_i - y_i|$。汉明重量 $w(x) = \sum_i x_i$。

### 定义 1.2（分块嵌入与宏观坐标）

将 $N$ 个比特均分为三组 $G_1, G_2, G_3$（每组 $N/3$ 比特），宏观坐标：

$$u^k = \epsilon_N \sum_{i \in G_k} x_i, \quad k = 1,2,3$$

其中 $\epsilon_N = 3L/N$ 是格点间距（三分组嵌入方案），$L$ 是系统线度。

### 定义 1.3（离散势场）

对占据位集合 $S$，离散势场定义为线性叠加：

$$\Phi_S(u) = -Gm \sum_{s \in S} \frac{1}{d_H(u, s)}$$

唯一性由定理 MULTI 保证。

### 定义 1.4（有限差分度规）

空间分量：

$$g_{kk}(u) = -\left[\Phi(u+2\hat{e}_k) - 2\Phi(u+\hat{e}_k) + \Phi(u)\right]$$

$$g_{kl}(u) = -\frac{1}{2}\left[\Phi(u+\hat{e}_k+\hat{e}_l) - \Phi(u+\hat{e}_k) - \Phi(u+\hat{e}_l) + \Phi(u)\right] \quad (k \neq l)$$

时空交叉分量：$g_{0k}(u) = 0$（定理 G0I）。

时间分量：$g_{00}(u) = -(1 + 2\Phi/c^2)$（弱场）/ $-f(r) = -(1-2Gm/c^2r)$（强场精确）。

## 五、连续极限

### 定理 CL-T（度规张量连续极限）

**陈述**：在双尺度条件 $\epsilon_N \ll \delta \ll L$ 下，离散度规 $g_{kl}^{(N)}(u)$ 在任意紧致区域 $K \subset \mathbb{R}^3 \setminus S$ 上一致收敛到连续度规 $g_{kl}^\infty(u) = -\partial_k\partial_l\Phi(u)$：

$$\sup_{u \in K} |g_{kl}^{(N)}(u) - g_{kl}^\infty(u)| \leq C_1(K)\epsilon_N + C_2(K)\epsilon_N^2$$

其中 $C_1(K) = O(Gm/r_{\min}^4)$，$C_2(K) = O(Gm/r_{\min}^5)$。

**证明结构**（四步）：

1. 势场格点化误差：$|\Phi_N - \Phi| \leq M_1 \epsilon_N$
2. 有限差分算子误差：对角 $O(\epsilon_N^2)$，非对角 $O(\epsilon_N)$
3. 度规误差合并：线性误差被二阶差分消去效应抑制
4. $\epsilon$-$\delta$ 构造：$N_0 = \lceil L^2/\epsilon_0^2 \rceil$

**数值验证**：$N=6,12,24$ 三尺度数据，误差比值 $1.2\sim1.6$ 与常数估计一致。

**来源版本**：V0.8

**论证文件**：`张量CL定理.md`

### 定理 MULTI（多源势场唯一性）

**陈述**：多源势场的唯一正确定义是线性叠加 $\Phi_S(u) = -Gm\sum_s 1/d_H(u,s)$。

**唯一性证明**：由三个约束联合确定——

1. A5 独立性 → $\Phi_S = \sum_s f(d_H(u,s))$
2. 单源退化 → $f(d) = -Gm/d$
3. 泊松方程相容性 → 线性叠加满足 $\nabla^2\Phi_S = 4\pi G\rho_S$

候选 A（最近距离）因 Voronoi 不连续被排除。

**推论**：多源度规叠加原理 $g_{kl}^S = \sum_s g_{kl}^{\{s\}}$，CL-T 多源推广（常数 $\propto |S|$）。

**来源版本**：V0.9

## 六、Lorentz 变换

### 定理 LT（Lorentz 群涌现）

**陈述**：惯性系之间的坐标变换群同构于正时向固有 Lorentz 群 $SO^+(1,3)$。

**证明结构**：

**引理 LT-0**（$c$ 的观察者独立性）：A1' 各向同性 → $c$ 对所有方向唯一。A9 禁止额外自由度 → $c$ 对所有观察者相同。

**引理 LT-1**（不变量的必然形式）：A1' 排除时空混合项。光锥条件 $Q|_{\text{light}} = 0$ → $A = -Bc^2$。A9 归一化 → $ds^2 = -c^2dt^2 + d\mathbf{x}^2$。

**定理 LT**：
1. 保持 $ds^2$ 的线性变换群 = $O(1,3)$（定义）
2. A6 排除时间反演 + A1' 连通性 → $SO^+(1,3)$
3. A4 保证线性性 + A9 排除非线性 → 线性 Lorentz boost
4. 显式 $\gamma = 1/\sqrt{1-v^2/c^2}$ 从 $ds^2$ 不变性唯一确定

**公理来源**：A1' + A4 + A6 + A9

**来源版本**：V0.12, V0.12a（加固）

**论证文件**：`lorentz变换.md`，`定理LT的显式构造补充.md`

### 定理 FE（场方程闭合）

**陈述**：在双尺度条件下，离散 Einstein 张量均匀化后收敛到连续 Einstein 张量：

$$\|\overline{G^{(N)}_{\mu\nu}} - G_{\mu\nu}\|_{L^2_{\text{loc}}} = O(\epsilon_N) + O(\epsilon_N^2/\delta^2)$$

完整场方程闭合：

$$G_{\mu\nu} + \Lambda g_{\mu\nu} = \frac{8\pi G}{c^4}T_{\mu\nu}$$

**证明结构**（五步定理链）：

| 定理 | 内容 | 误差 |
|------|------|------|
| NP | 非线性乘积收敛 $\mathcal{H}_\delta(g \cdot g) \to g \cdot g$ | $O(\epsilon_N^2)$ |
| CC | Christoffel 符号收敛 | $O(\epsilon_N) + O(\epsilon_N^2/\delta^2)$ |
| RC | Ricci 张量收敛 | 同上 |
| EC | Einstein 张量收敛 | 同上 |
| FE | 场方程闭合 | 同上 |

**核心困难与解决**：均匀化算子不保持乘积 → 分解 $g = \bar{g} + \delta g$（宏观平均+微观涨落）→ 交叉项因零均值统计抵消 → 涨落二阶矩 $O(\epsilon_N^2)$。

**离散自耦合机制**：度规和物质源通过比特配置相互依赖——$d_H$ 决定势场（→ 度规），$S$ 决定物质（→ $T_{\mu\nu}$），两者来自同一 $\{x_i\}$。

**来源版本**：V0.14

## 八、宇宙学常数

### 定理 Λ（宇宙学常数公理推导）

**陈述**：

$$\Lambda = \frac{8\pi G}{c^4}\rho_{\text{vac}} > 0$$

其中 $\rho_{\text{vac}} = \frac{m_0 c_0}{L^3}$（$c_0 = 1/4$ 为 $N \to \infty$ 解析极限；$N=6$ 数值验证 $c_0 = 0.2405$，偏差 3.9%），**无自由参数**。

**推导链**：

1. A8 对称偏好 → 权重函数 $\rho(w) = \binom{N}{w}/\binom{N}{N/2}$ → 偏离中截面的能量代价 $\Delta E = -\ln\rho(w) \cdot m_0$
2. 真空能密度 $\rho_{\text{vac}}^{(N)} = \frac{m_0}{L^3 Z}\sum_w \binom{N}{w}\rho(w)[-\ln\rho(w)]$ → 连续极限 $\rho_{\text{vac}} = m_0 c_0/L^3$
3. Bianchi 恒等式 $\nabla_\mu T^{\text{vac}\,\mu\nu} = 0$ + 各向同性（A8）→ $T^{\text{vac}}_{\mu\nu} = -\rho_{\text{vac}} g_{\mu\nu}$（唯一形式）
4. A5 正定能量 → $\rho_{\text{vac}} > 0$；定理 LT → $g_{00} < 0$ → $T^{\text{vac}}_{00} > 0$ → $\Lambda > 0$

**与标准 QFT 的区别**：A8 给出的是偏离中截面的统计平均，不是零点能求和——为"宇宙学常数问题"提供新视角。

**来源版本**：V0.14
